# LEVEL 3: WHITE MATTER TRACTS - ULTRA-GRANULAR SPECIFICATION

**Purpose**: Complete white matter tract anatomy for neurosurgical knowledge base
**Scope**: All major cerebral white matter pathways with clinical correlations
**Format**: Hierarchical tree structure for deep extraction and RAG retrieval
**Standards**: Neurocore architecture, medical ontology accuracy

---

## INTRODUCTION TO WHITE MATTER ⭐⭐⭐

├─ Definition: Myelinated axonal projections connecting different brain regions
│
├─ Composition:
│   ├─ Axons: Long-range projections between neurons
│   ├─ Myelin: Lipid-rich insulation (oligodendrocytes in CNS)
│   ├─ Glial cells: Oligodendrocytes, astrocytes, microglia
│   └─ Blood vessels: Supporting vasculature
│
├─ Function: High-speed information transfer between brain regions
│   ├─ Myelin: Saltatory conduction (50-100 m/s vs 0.5-2 m/s unmyelinated)
│   └─ Network integration: Connects distributed neural systems
│
├─ Volume: ~50% of human brain volume (increases with evolution)
│
└─ Clinical significance:
    ├─ White matter diseases: MS, leukodystrophies, leukoencephalopathies
    ├─ Disconnection syndromes: Functional deficits from tract interruption
    ├─ Surgical planning: Avoid eloquent white matter tracts
    └─ DTI/tractography: Imaging modality for tract visualization


## CLASSIFICATION OF WHITE MATTER TRACTS ⭐⭐⭐

White matter tracts are classified into three major categories based on the regions they connect:

├─ PROJECTION FIBERS:
│   ├─ Definition: Connect cerebral cortex ↔ subcortical structures (thalamus, brainstem, spinal cord)
│   ├─ Direction: Bidirectional (ascending sensory + descending motor)
│   ├─ Examples: Corona radiata, internal capsule, cerebral peduncles
│   └─ Clinical: Motor/sensory deficits if damaged
│
├─ COMMISSURAL FIBERS:
│   ├─ Definition: Connect homologous regions between left and right hemispheres
│   ├─ Direction: Interhemispheric (left ↔ right)
│   ├─ Examples: Corpus callosum, anterior commissure, posterior commissure
│   └─ Clinical: Disconnection syndromes (split-brain, alien hand)
│
└─ ASSOCIATION FIBERS:
    ├─ Definition: Connect different regions within the same hemisphere
    ├─ Direction: Intrahemispheric (within one side)
    │
    ├─ Short association fibers (U-fibers):
    │   ├─ Connect adjacent gyri (arcuate course under sulcus)
    │   └─ Most numerous but less studied (diffuse distribution)
    │
    └─ Long association fibers:
        ├─ Connect distant lobes within hemisphere
        ├─ Examples: SLF, arcuate, ILF, IFOF, uncinate, cingulum
        └─ Clinical: Language, memory, visuospatial deficits

## I. PROJECTION FIBERS ⭐⭐⭐

### CORONA RADIATA:

├─ Definition: Fan-shaped projection fibers radiating from internal capsule to cortex
│
├─ Location: Deep cerebral white matter (centrum semiovale)
│   ├─ Superior to internal capsule
│   └─ Inferior to cortical gray matter
│
├─ Components (continuous with internal capsule):
│   ├─ Anterior corona radiata: From anterior limb of internal capsule
│   │   └─ Connects: Prefrontal cortex ↔ thalamus (anterior nucleus, mediodorsal)
│   │
│   ├─ Superior corona radiata: From genu and posterior limb
│   │   ├─ Corticospinal fibers: Motor cortex → internal capsule
│   │   └─ Thalamocortical fibers: Thalamus → sensory/motor cortex
│   │
│   └─ Posterior corona radiata: From retrolenticular internal capsule
│       └─ Connects: Parietal/occipital cortex ↔ thalamus (pulvinar, lateral geniculate)
│
├─ Somatotopy (preserved from cortex):
│   ├─ Medial: Leg representation (paracentral lobule)
│   ├─ Lateral: Arm/hand representation (lateral precentral gyrus)
│   └─ Most lateral: Face representation (inferior precentral gyrus)
│
└─ Clinical significance:
    ├─ Strokes: Corona radiata infarcts → pure motor hemiparesis
    │   └─ Lacunar infarcts: Small vessel disease (hypertension, diabetes)
    ├─ Tumors: Infiltrating gliomas commonly involve corona radiata
    ├─ Surgical: Transcortical approaches must preserve corona radiata
    └─ DTI: Tractography used to identify corticospinal tract location


### INTERNAL CAPSULE ⭐⭐⭐:

├─ Definition: Dense white matter structure containing all projection fibers between cortex and brainstem/spinal cord
│
├─ Location: Between thalamus (medial) and lentiform nucleus (lateral)
│   ├─ Caudate nucleus: Medial (in lateral ventricle)
│   ├─ Thalamus: Medial and posterior
│   ├─ Lentiform nucleus: Lateral (putamen + globus pallidus)
│   └─ "V-shaped" on axial MRI (with apex pointing medially)
│
├─ Anatomical divisions (five segments):
│   ├─ 1. Anterior limb ⭐:
│   │   ├─ Location: Between caudate nucleus (medial) and lentiform nucleus (lateral)
│   │   │
│   │   ├─ Contents:
│   │   │   ├─ Anterior thalamic radiations:
│   │   │   │   ├─ Prefrontal cortex ↔ mediodorsal thalamus (executive function)
│   │   │   │   └─ Anterior cingulate ↔ anterior thalamic nucleus (limbic)
│   │   │   │
│   │   │   ├─ Frontopontine fibers:
│   │   │   │   ├─ Frontal cortex → pontine nuclei → cerebellum
│   │   │   │   └─ Cognitive/motor planning loop
│   │   │   │
│   │   │   └─ Caudate connections: Frontal cortex ↔ caudate (working memory, attention)
│   │   │
│   │   └─ Clinical:
│   │       ├─ Pure lesions rare (overlaps with genu)
│   │       └─ Executive dysfunction, apathy, abulia if damaged
│   │
│   ├─ 2. Genu ⭐⭐:
│   │   ├─ Location: "Knee" or apex of internal capsule (most medial point)
│   │   │
│   │   ├─ Contents:
│   │   │   ├─ Corticobulbar tract ⭐:
│   │   │   │   ├─ Motor cortex (face area) → cranial nerve motor nuclei (CN V, VII, IX, X, XI, XII)
│   │   │   │   ├─ Bilateral innervation: Most cranial nerve nuclei (except lower CN VII, XII - contralateral)
│   │   │   │   └─ Clinical: Central facial palsy (forehead spared - bilateral upper face innervation)
│   │   │   │
│   │   │   └─ Corticospinal fibers (upper extremity - beginning):
│   │   │       └─ Transition from anterior limb to posterior limb
│   │   │
│   │   └─ Clinical:
│   │       ├─ Lacunar infarct: Pure motor hemiparesis (face + arm)
│   │       └─ Dysarthria-clumsy hand syndrome: Corticobulbar + corticospinal
│   │
│   ├─ 3. Posterior limb ⭐⭐⭐:
│   │   ├─ Location: Between thalamus (medial) and lentiform nucleus (lateral)
│   │   ├─ Most clinically important segment (largest, most fibers)
│   │   │
│   │   ├─ Contents (somatotopically organized):
│   │   │   ├─ Corticospinal tract ⭐⭐⭐:
│   │   │   │   ├─ Motor cortex → brainstem → spinal cord
│   │   │   │   ├─ Somatotopy (anterior to posterior):
│   │   │   │   │   ├─ Face: Anterior (near genu)
│   │   │   │   │   ├─ Upper extremity: Middle
│   │   │   │   │   └─ Lower extremity: Posterior (near retrolenticular)
│   │   │   │   │
│   │   │   │   └─ Clinical: Pure motor hemiparesis (most common lacunar syndrome)
│   │   │   │
│   │   │   ├─ Superior thalamic radiations ⭐:
│   │   │   │   ├─ VPL nucleus → primary sensory cortex (postcentral gyrus)
│   │   │   │   ├─ Sensory: Touch, pain, temperature, proprioception from body
│   │   │   │   ├─ Somatotopy: Face (anterior) → arm (middle) → leg (posterior)
│   │   │   │   └─ Clinical: Pure sensory stroke (VPL infarct or posterior limb)
│   │   │   │
│   │   │   └─ Corticopontine fibers:
│   │   │       └─ Sensorimotor cortex → pontine nuclei → cerebellum
│   │   │
│   │   └─ Clinical syndromes (posterior limb lesions) ⭐⭐:
│   │       ├─ Pure motor hemiparesis (most common):
│   │       │   ├─ Contralateral face + arm + leg weakness
│   │       │   ├─ No sensory loss, no visual field defect, no aphasia
│   │       │   └─ Lacunar infarct from lenticulostriate artery occlusion
│   │       │
│   │       ├─ Sensorimotor stroke:
│   │       │   ├─ Combined motor + sensory loss (corticospinal + thalamic radiations)
│   │       │   └─ Larger lesion than pure motor
│   │       │
│   │       └─ Lacunar syndromes:
│   │           ├─ Pure motor: Posterior limb, basis pontis, medullary pyramid
│   │           ├─ Pure sensory: VPL thalamus
│   │           ├─ Sensorimotor: Posterior limb + thalamus
│   │           ├─ Ataxic hemiparesis: Posterior limb + adjacent structures
│   │           └─ Dysarthria-clumsy hand: Genu/anterior posterior limb
│   │
│   ├─ 4. Retrolenticular segment:
│   │   ├─ Location: Posterior to lentiform nucleus
│   │   │
│   │   ├─ Contents:
│   │   │   ├─ Optic radiations (posterior portion) ⭐:
│   │   │   │   ├─ Lateral geniculate nucleus → primary visual cortex (V1)
│   │   │   │   ├─ Course: Sweeps posteriorly through temporal and parietal lobes
│   │   │   │   └─ See "Optic radiations" section below for details
│   │   │   │
│   │   │   ├─ Posterior thalamic radiations:
│   │   │   │   └─ Pulvinar → parietal/occipital cortex
│   │   │   │
│   │   │   └─ Parietopontine fibers:
│   │   │       └─ Parietal cortex → pontine nuclei → cerebellum
│   │   │
│   │   └─ Clinical:
│   │       └─ Homonymous hemianopia: Contralateral visual field loss
│   │
│   └─ 5. Sublenticular segment:
│       ├─ Location: Inferior to lentiform nucleus (below putamen and globus pallidus)
│       │
│       ├─ Contents:
│       │   ├─ Optic radiations (anterior portion - Meyer's loop) ⭐:
│       │   │   ├─ Lateral geniculate nucleus → temporal lobe (Meyer's loop) → occipital lobe
│       │   │   ├─ Meyer's loop: Anterior sweep through temporal lobe
│       │   │   └─ Represents superior visual field (inferior retina)
│       │   │
│       │   ├─ Auditory radiations:
│       │   │   ├─ Medial geniculate nucleus → primary auditory cortex (Heschl's gyrus)
│       │   │   └─ Bilateral representation (each ear → both hemispheres)
│       │   │
│       │   └─ Temporopontine fibers:
│       │       └─ Temporal cortex → pontine nuclei → cerebellum
│       │
│       └─ Clinical:
│           ├─ "Pie in the sky" visual field defect: Meyer's loop damage ⭐
│           │   ├─ Superior quadrantanopia (contralateral)
│           │   └─ Common in temporal lobe surgery (anterior temporal lobectomy for epilepsy)
│           │
│           └─ Auditory dysfunction: Rare (bilateral representation)
│
├─ Blood supply ⭐⭐:
│   ├─ Lenticulostriate arteries (from MCA M1 segment):
│   │   ├─ Supply: Posterior limb, genu (most of internal capsule)
│   │   ├─ Perforating branches: Small end arteries (no collaterals)
│   │   └─ Clinical: "Arteries of stroke" - common lacunar infarct site
│   │
│   ├─ Anterior choroidal artery (from ICA):
│   │   ├─ Supply: Posterior limb (posterior), retrolenticular, sublenticular
│   │   └─ Clinical: Triad if occluded (hemiparesis, hemianesthesia, hemianopia)
│   │
│   └─ Recurrent artery of Heubner (from ACA):
│       └─ Supply: Anterior limb (medial part), head of caudate
│
└─ Clinical summary:
    ├─ Most common stroke location: Posterior limb (lenticulostriate territory)
    ├─ Most common syndrome: Pure motor hemiparesis
    ├─ Surgical risk: Transcortical approaches must avoid corticospinal tract
    └─ Imaging: MRI T2/FLAIR hyperintensity (acute), DTI tractography (surgical planning)


### OPTIC RADIATIONS (GENICULOCALCARINE TRACT) ⭐⭐⭐:

├─ Definition: Visual pathway from lateral geniculate nucleus (LGN) to primary visual cortex (V1)
│
├─ Course (complex three-dimensional trajectory):
│   ├─ Origin: Lateral geniculate nucleus (thalamus)
│   │   ├─ 6 layers: Alternating ipsilateral (2, 3, 5) and contralateral (1, 4, 6) eye inputs
│   │   └─ Retinotopic organization: Maintained throughout pathway
│   │
│   ├─ Initial course: Lateral and posterior from LGN
│   │
│   ├─ Subdivisions based on trajectory:
│   │   ├─ Meyer's loop (inferior/anterior bundle) ⭐⭐:
│   │   │   ├─ Course: Sweeps anteriorly into temporal lobe (around temporal horn of lateral ventricle)
│   │   │   ├─ Anterior extent: Variable (20-40 mm anterior to LGN, mean 30 mm)
│   │   │   ├─ Then turns posteriorly: Travels through temporal lobe → occipital lobe
│   │   │   ├─ Visual field representation: Superior visual field (inferior retina) ⭐
│   │   │   │
│   │   │   └─ Clinical significance ⭐⭐:
│   │   │       ├─ Vulnerable in temporal lobe surgery: Anterior temporal lobectomy (epilepsy surgery)
│   │   │       ├─ Risk factors for injury:
│   │   │       │   ├─ Resection >3 cm posterior to temporal pole: 50-100% risk of visual field defect
│   │   │       │   ├─ Resection <3 cm: 10-20% risk
│   │   │       │   └─ Dominant hemisphere: Higher risk (left > right)
│   │   │       │
│   │   │       ├─ Visual field defect pattern: "Pie in the sky" defect
│   │   │       │   ├─ Contralateral superior quadrantanopia
│   │   │       │   ├─ Example: Left Meyer's loop injury → right superior quadrantanopia
│   │   │       │   └─ Shape: Wedge-shaped defect in superior visual field
│   │   │       │
│   │   │       └─ Intraoperative monitoring: Not standard (DTI preoperative planning preferred)
│   │   │
│   │   ├─ Central bundle (middle fibers):
│   │   │   ├─ Course: Directly posterior from LGN (most direct path)
│   │   │   ├─ Visual field: Horizontal meridian
│   │   │   └─ Travels through: Parietal and occipital white matter
│   │   │
│   │   └─ Superior bundle (dorsal/superior fibers):
│   │       ├─ Course: Superior and posterior from LGN
│   │       ├─ Travels through: Parietal lobe white matter
│   │       ├─ Visual field representation: Inferior visual field (superior retina)
│   │       └─ Clinical: Injury causes contralateral inferior quadrantanopia
│   │
│   └─ Termination: Primary visual cortex (V1, striate cortex, calcarine cortex)
│       ├─ Location: Banks of calcarine sulcus (medial occipital lobe)
│       ├─ Retinotopic map:
│       │   ├─ Macular (central) vision: Posterior occipital pole (large representation)
│       │   ├─ Peripheral vision: Anterior calcarine cortex
│       │   ├─ Superior visual field: Inferior calcarine (cuneus)
│       │   └─ Inferior visual field: Superior calcarine (lingual gyrus)
│       │
│       └─ Blood supply: PCA (dual blood supply at occipital pole: PCA + MCA)
│
├─ Retinotopic organization (preserved throughout pathway):
│   ├─ Contralateral visual field: Each optic radiation carries contralateral visual field
│   ├─ Superior/inferior organization: Superior radiation → inferior field, inferior (Meyer's) → superior field
│   └─ Macular sparing: Dual vascular supply (PCA + MCA) at occipital pole
│
├─ Lesion localization (visual field defects) ⭐⭐:
│   ├─ Monocular visual loss: Lesion anterior to optic chiasm (eye or optic nerve)
│   │
│   ├─ Bitemporal hemianopia: Optic chiasm (crossing nasal fibers) - pituitary tumor
│   │
│   ├─ Homonymous hemianopia: Lesion posterior to chiasm (contralateral)
│   │   ├─ Complete homonymous hemianopia: Entire optic radiation or V1
│   │   ├─ "Incongruous" (dissimilar in both eyes): Optic tract (uncommon lesion site)
│   │   └─ "Congruous" (identical in both eyes): Optic radiation or V1
│   │
│   ├─ Superior quadrantanopia ("pie in the sky"): Meyer's loop (temporal lobe)
│   │   └─ Contralateral superior quadrant loss
│   │
│   ├─ Inferior quadrantanopia ("pie on the floor"): Superior optic radiation (parietal lobe)
│   │   └─ Contralateral inferior quadrant loss
│   │
│   └─ Macular sparing: Occipital lobe lesion (dual blood supply at pole)
│       ├─ Central vision preserved despite homonymous hemianopia
│       └─ PCA infarct: Macular sparing (MCA collateral to occipital pole)
│
└─ Clinical applications:
    ├─ Temporal lobectomy: DTI tractography to visualize Meyer's loop (minimize visual field defects)
    ├─ Tumor resection: Parietal/occipital tumors (preserve optic radiations)
    ├─ Stroke: PCA territory (occipital lobe) → homonymous hemianopia with macular sparing
    └─ MCA territory: Deep (optic radiation) → homonymous hemianopia without macular sparing


### CEREBRAL PEDUNCLES ⭐⭐:

├─ Definition: White matter stalks connecting cerebral hemispheres to brainstem
│
├─ Location: Ventral midbrain (between diencephalon and pons)
│   └─ Anterior to substantia nigra
│
├─ Anatomical divisions:
│   ├─ Crus cerebri (basis pedunculi):
│   │   ├─ Location: Anterior (ventral) to substantia nigra
│   │   ├─ Contents: Descending motor fibers
│   │   │
│   │   ├─ Somatotopic organization (medial to lateral):
│   │   │   ├─ Medial 1/5: Frontopontine fibers (frontal cortex → pontine nuclei)
│   │   │   ├─ Middle 3/5: Corticospinal + corticobulbar tracts ⭐
│   │   │   │   ├─ Face fibers: Most medial
│   │   │   │   ├─ Arm fibers: Middle
│   │   │   │   └─ Leg fibers: Most lateral
│   │   │   └─ Lateral 1/5: Parietopontine, temporopontine, occipitopontine fibers
│   │   │
│   │   └─ Clinical: Infarct → contralateral hemiparesis (Weber syndrome if CN III involved)
│   │
│   └─ Tegmentum:
│       ├─ Location: Posterior (dorsal) to substantia nigra
│       ├─ Contents: Ascending sensory, cranial nerve nuclei, reticular formation
│       └─ Not typically considered part of cerebral peduncles proper
│
├─ Blood supply:
│   ├─ PCA (P1 segment): Perforating branches
│   └─ Basilar artery: Perforating branches
│
└─ Clinical syndromes:
    ├─ Weber syndrome (medial midbrain) ⭐:
    │   ├─ Lesion: Crus cerebri + CN III fascicles
    │   ├─ Ipsilateral: CN III palsy (ptosis, "down and out" eye, mydriasis)
    │   ├─ Contralateral: Hemiparesis (face, arm, leg)
    │   └─ Cause: PCA occlusion (perforating branches)
    │
    └─ Benedikt syndrome (tegmentum + red nucleus):
        ├─ Lesion: Tegmentum (medial lemniscus, red nucleus, CN III)
        ├─ Ipsilateral: CN III palsy
        ├─ Contralateral: Ataxia, tremor (red nucleus), hemianesthesia (medial lemniscus)
        └─ Cause: PCA occlusion (paramedian branches)

## II. COMMISSURAL FIBERS ⭐⭐⭐

### CORPUS CALLOSUM ⭐⭐⭐:

├─ Definition: Largest commissural tract connecting homologous cortical regions between hemispheres
│
├─ Size: ~200 million axons (largest white matter tract in human brain)
│
├─ Location: Deep interhemispheric structure (roof of lateral ventricles)
│
├─ Anatomical divisions (anterior to posterior):
│   ├─ 1. Rostrum:
│   │   ├─ Location: Anterior-inferior extension (below genu)
│   │   ├─ Connects: Orbital frontal cortex (bilateral)
│   │   └─ Continuous with lamina terminalis inferiorly
│   │
│   ├─ 2. Genu ⭐:
│   │   ├─ Location: Anterior "knee" (curves anteriorly)
│   │   ├─ Connects: Prefrontal cortex (bilateral)
│   │   ├─ Fibers: Forceps minor (anterior forceps)
│   │   │   └─ Curve anteriorly into frontal lobes
│   │   └─ Size: Thicker than rostrum
│   │
│   ├─ 3. Body (trunk):
│   │   ├─ Location: Central portion (longest segment)
│   │   ├─ Connects: Supplementary motor area, primary motor cortex, primary sensory cortex
│   │   ├─ Somatotopic organization:
│   │   │   ├─ Anterior body: Motor (face, upper limb)
│   │   │   ├─ Middle body: Motor (lower limb), sensory (body)
│   │   │   └─ Posterior body: Posterior parietal cortex
│   │   └─ Clinical: Alien hand syndrome if damaged (anterior body)
│   │
│   ├─ 4. Isthmus:
│   │   ├─ Location: Narrow segment (posterior to body, anterior to splenium)
│   │   └─ Connects: Posterior parietal, superior temporal regions
│   │
│   └─ 5. Splenium ⭐:
│       ├─ Location: Posterior "bulge" (thickest part)
│       ├─ Connects: Posterior parietal, occipital, medial temporal regions
│       ├─ Fibers: Forceps major (posterior forceps)
│       │   └─ Curve posteriorly into occipital lobes
│       │
│       └─ Clinical: Visual disconnection syndrome if damaged (see below)
│
├─ Topographic organization (somatotopic and functional):
│   ├─ Anterior-posterior: Frontal → parietal → occipital connections
│   ├─ Superior-inferior: Dorsal cortex → ventral cortex
│   └─ Homotopic: Primarily connects homologous regions (mirror areas)
│
├─ Function:
│   ├─ Interhemispheric communication: Transfer information between hemispheres
│   ├─ Motor coordination: Bimanual tasks (both hands working together)
│   ├─ Sensory integration: Combine sensory information from both sides
│   └─ Cognitive processes: Language, memory, attention
│
├─ Development:
│   ├─ Myelination: Continues through adolescence (posterior → anterior)
│   └─ Agenesis: Congenital absence (1/4000 births) - variable symptoms
│
├─ Blood supply:
│   ├─ Anterior: ACA (A2 segment) - pericallosal artery
│   ├─ Posterior: PCA (splenial branches)
│   └─ Watershed zone: Between ACA and PCA territories (vulnerable)
│
└─ Clinical syndromes - callosal lesions ⭐⭐:
    ├─ Split-brain syndrome (complete callosotomy):
    │   ├─ Historical: Surgical treatment for intractable epilepsy (prevents seizure spread)
    │   │
    │   ├─ Clinical manifestations (subtle in daily life):
    │   │   ├─ Visual disconnection:
    │   │   │   ├─ Object in left visual field: Cannot name (visual → right hemisphere, no language transfer to left)
    │   │   │   └─ Can identify with left hand: Right hemisphere controls left hand
    │   │   │
    │   │   ├─ Tactile disconnection:
    │   │   │   ├─ Object in left hand: Cannot name (sensory → right hemisphere)
    │   │   │   └─ Object in right hand: Can name (sensory → left hemisphere with language)
    │   │   │
    │   │   ├─ Apraxia of left hand:
    │   │   │   ├─ Cannot follow verbal commands with left hand
    │   │   │   └─ Language (left hemisphere) cannot communicate with right motor cortex (left hand)
    │   │   │
    │   │   └─ Intact functions:
    │   │       ├─ Simple motor tasks: Bilateral (subcortical pathways sufficient)
    │   │       └─ Midline sensory: Bilateral representation
    │   │
    │   └─ Modern outcome: Most patients adapt well (subcortical communication compensates)
    │
    ├─ Alien hand syndrome (corpus callosum lesion, anterior body) ⭐:
    │   ├─ Definition: Involuntary, purposeful movements of one hand (usually left)
    │   ├─ Etiology:
    │   │   ├─ Corpus callosum infarct: ACA territory (pericallosal artery)
    │   │   ├─ Tumor: Corpus callosum glioma (butterfly glioblastoma)
    │   │   ├─ Corpus callosum surgery: Anterior callosotomy
    │   │   └─ Anterior cerebral artery (ACA) infarct: With supplementary motor area (SMA)
    │   │
    │   ├─ Clinical presentation:
    │   │   ├─ Involuntary movements: Hand acts "on its own" (patient surprised by actions)
    │   │   ├─ Inter-manual conflict: One hand opposes the other (e.g., one buttons shirt, other unbuttons)
    │   │   ├─ Grasping behavior: Involuntary grasping of objects
    │   │   └─ Patient awareness: Recognizes hand as their own (vs somatoparaphrenia)
    │   │
    │   ├─ Mechanism:
    │   │   ├─ Disconnection: Left hemisphere (dominant) cannot inhibit right hemisphere motor programs
    │   │   └─ Contralateral control lost: Non-dominant hemisphere acts independently
    │   │
    │   └─ Treatment: No specific treatment (may improve over months with rehabilitation)
    │
    ├─ Alexia without agraphia (disconnection syndrome) ⭐:
    │   ├─ Etiology: Left PCA infarct (splenium + left occipital lobe)
    │   │
    │   ├─ Anatomy of lesion:
    │   │   ├─ Left occipital lobe: Visual input from right visual field lost
    │   │   ├─ Splenium of corpus callosum: Visual input from right occipital cannot reach left language areas
    │   │   └─ Result: No visual information reaches language areas (Wernicke's area)
    │   │
    │   ├─ Clinical presentation:
    │   │   ├─ Cannot read: Text presented visually not understood ⭐
    │   │   ├─ Can write: Language production intact (motor/frontal areas preserved)
    │   │   ├─ Can comprehend spoken language: Auditory input intact
    │   │   ├─ Right homonymous hemianopia: Left occipital lobe infarct
    │   │   └─ Can identify letters by tracing: Tactile input bypasses visual system
    │   │
    │   └─ Mechanism: Visual information cannot reach language areas (disconnection)
    │
    ├─ Marchiafava-Bignami disease:
    │   ├─ Etiology: Chronic alcoholism (rare)
    │   ├─ Pathology: Demyelination/necrosis of corpus callosum (especially body)
    │   ├─ Clinical: Dementia, confusion, seizures, interhemispheric disconnection
    │   └─ Imaging: MRI shows T2/FLAIR hyperintensity in corpus callosum
    │
    └─ Callosal tumors:
        ├─ Butterfly glioblastoma:
        │   ├─ High-grade glioma crossing corpus callosum (bilateral hemispheres)
        │   ├─ "Butterfly" shape on imaging (crosses midline)
        │   └─ Poor prognosis (bilateral involvement, high grade)
        │
        └─ Lymphoma: Primary CNS lymphoma (commonly involves corpus callosum)


### ANTERIOR COMMISSURE ⭐:

├─ Definition: Small commissural tract connecting temporal lobes and olfactory regions
│
├─ Size: Much smaller than corpus callosum (~1 million axons vs 200 million)
│
├─ Location: Anterior wall of third ventricle (rostral to fornix columns)
│   └─ Landmarks: Anterior to hypothalamus, below rostrum of corpus callosum
│
├─ Anatomical divisions:
│   ├─ Anterior limb:
│   │   ├─ Smaller portion
│   │   └─ Connects: Olfactory bulbs and anterior olfactory nuclei (bilateral)
│   │
│   └─ Posterior limb:
│       ├─ Larger portion
│       └─ Connects: Middle and inferior temporal gyri, amygdala, parahippocampal gyrus (bilateral)
│
├─ Function:
│   ├─ Olfactory integration: Transfer olfactory information between hemispheres
│   ├─ Temporal lobe communication: Memory, emotion (amygdala)
│   └─ Language: Minor role (some language transfer in split-brain patients)
│
└─ Clinical significance:
    ├─ Split-brain compensation: Some interhemispheric transfer via anterior commissure (if corpus callosum absent)
    ├─ Surgical landmark: Used in third ventricle approaches (endoscopic, open)
    └─ Agenesis: Can be absent with corpus callosum agenesis (Probst bundles instead)


### POSTERIOR COMMISSURE:

├─ Definition: Small commissure at posterior third ventricle (dorsal to cerebral aqueduct)
│
├─ Location: Junction of diencephalon and midbrain (epithalamus)
│   └─ Landmarks: Above cerebral aqueduct, below pineal gland
│
├─ Contents:
│   ├─ Pretectal nuclei connections: Pupillary light reflex pathway (bilateral)
│   └─ Superior colliculus connections: Vertical gaze control
│
├─ Function:
│   ├─ Pupillary light reflex: Consensual light reflex (afferent from one eye → efferent to both eyes)
│   └─ Vertical eye movements: Coordination between hemispheres
│
└─ Clinical significance:
    ├─ Parinaud syndrome (dorsal midbrain syndrome) ⭐:
    │   ├─ Etiology: Pineal tumor (compresses posterior commissure), hydrocephalus
    │   │
    │   ├─ Clinical triad:
    │   │   ├─ Upgaze palsy: Cannot look up (vertical gaze center compressed)
    │   │   ├─ Light-near dissociation: Pupils do not constrict to light, but constrict to near (accommodation)
    │   │   └─ Lid retraction ("Collier sign"): Upper eyelid retraction (surprised expression)
    │   │
    │   ├─ Additional signs:
    │   │   ├─ Convergence-retraction nystagmus: Eyes retract into orbit on attempted upgaze
    │   │   └─ Papilledema: If hydrocephalus present (aqueductal compression)
    │   │
    │   └─ Causes: Pineal germinoma, tectal glioma, hydrocephalus (aqueductal stenosis)
    │
    └─ Surgical: Landmark for pineal region surgery (infratentorial supracerebellar approach)

## III. ASSOCIATION FIBERS ⭐⭐⭐

### SUPERIOR LONGITUDINAL FASCICULUS (SLF) ⭐⭐⭐:

├─ Definition: Long association fiber connecting frontal, parietal, and temporal lobes within each hemisphere
│
├─ Historical note: Historically considered synonymous with arcuate fasciculus (now considered separate)
│
├─ Subdivisions (SLF I, II, III - distinct trajectories) ⭐:
│   ├─ SLF I (dorsal/superior) ⭐:
│   │   ├─ Course: Superior parietal lobule ↔ dorsal/medial frontal cortex (SMA, preSMA)
│   │   ├─ Trajectory: Arches over lateral ventricle (most superior pathway)
│   │   │
│   │   ├─ Function:
│   │   │   ├─ Visuomotor integration: Eye-hand coordination
│   │   │   ├─ Motor planning: Reaching movements (parietal → frontal motor)
│   │   │   └─ Spatial working memory
│   │   │
│   │   └─ Clinical: Lesion → apraxia (reaching movements), working memory deficits
│   │
│   ├─ SLF II (middle) ⭐⭐:
│   │   ├─ Course: Inferior parietal lobule (angular gyrus, supramarginal gyrus) ↔ dorsolateral prefrontal cortex
│   │   ├─ Trajectory: Arches around insula (lateral to SLF I)
│   │   │
│   │   ├─ Function:
│   │   │   ├─ Visuospatial attention: Dorsal attention network
│   │   │   ├─ Spatial working memory: Frontoparietal network
│   │   │   └─ Visual perception: Object manipulation
│   │   │
│   │   └─ Clinical: Lesion → spatial neglect (especially right hemisphere), visuospatial deficits
│   │
│   ├─ SLF III (ventral/inferior) ⭐⭐:
│   │   ├─ Course: Supramarginal gyrus (inferior parietal) ↔ ventrolateral prefrontal cortex, inferior frontal gyrus
│   │   ├─ Trajectory: Most ventral of SLF components (lateral to arcuate)
│   │   │
│   │   ├─ Function:
│   │   │   ├─ Articulatory network: Speech motor planning (Broca's ↔ parietal)
│   │   │   ├─ Phonological processing: Sound-based language
│   │   │   └─ Working memory: Verbal working memory (phonological loop)
│   │   │
│   │   └─ Clinical: Lesion → conduction aphasia (especially left hemisphere), phonological deficits
│   │
│   └─ Relationship to arcuate fasciculus:
│       ├─ Arcuate: Connects temporal → frontal (language pathway)
│       ├─ SLF III: Connects parietal → frontal (overlaps with arcuate in frontal termination)
│       └─ Modern view: Arcuate is distinct from SLF (but closely related)
│
├─ Lateralization:
│   ├─ Left hemisphere: Language (SLF III, arcuate), verbal working memory
│   └─ Right hemisphere: Visuospatial attention (SLF II), spatial working memory
│
└─ Clinical significance:
    ├─ Right SLF II lesion: Spatial neglect syndrome (most common)
    ├─ Left SLF III lesion: Conduction aphasia (with arcuate)
    ├─ Tumor infiltration: Gliomas commonly infiltrate SLF (preserve function during resection)
    └─ DTI tractography: Surgical planning (map SLF components intraoperatively)


### ARCUATE FASCICULUS ⭐⭐⭐:

├─ Definition: Long association fiber connecting Wernicke's area (posterior superior temporal) to Broca's area (inferior frontal)
│
├─ Historical significance: Core component of classic language network (Geschwind model)
│
├─ Course (left hemisphere - language dominant) ⭐:
│   ├─ Origin: Wernicke's area (posterior superior temporal gyrus, BA 22)
│   │   └─ Language comprehension region
│   │
│   ├─ Trajectory: Arches around Sylvian fissure (lateral to insula)
│   │   ├─ Sweeps laterally and superiorly: Through temporoparietal junction
│   │   ├─ Courses around posterior end of Sylvian fissure
│   │   └─ Then anteriorly and inferiorly: Into inferior frontal lobe
│   │
│   └─ Termination: Broca's area (inferior frontal gyrus, BA 44/45)
│       └─ Language production/articulation region
│
├─ Modern anatomy (revised model - direct and indirect pathways):
│   ├─ Direct pathway (long segment) ⭐:
│   │   ├─ Direct connection: Wernicke's (posterior STG) → Broca's (IFG)
│   │   ├─ Lateralized: Left > right (language dominance)
│   │   └─ Function: Phonological processing (sound-based language)
│   │
│   ├─ Indirect anterior segment:
│   │   ├─ Connection: Inferior parietal (supramarginal, angular) → Broca's (IFG)
│   │   ├─ Overlaps with: SLF III territory
│   │   └─ Function: Articulatory planning
│   │
│   └─ Indirect posterior segment:
│       ├─ Connection: Wernicke's (STG) → inferior parietal (supramarginal, angular)
│       └─ Function: Phonological working memory
│
├─ Function (language network) ⭐⭐:
│   ├─ Phonological processing: Transfer sound-based language information
│   │   ├─ Repetition: Hear word (Wernicke's) → repeat word (Broca's)
│   │   └─ Phonological working memory: Hold sound information temporarily
│   │
│   ├─ Speech production: Link comprehension to articulation
│   │   └─ Auditory-motor integration
│   │
│   └─ Reading: Connect visual word form → phonology → articulation
│
├─ Lateralization:
│   ├─ Left hemisphere (95% right-handed, 70% left-handed):
│   │   ├─ Larger, more robust arcuate fasciculus
│   │   └─ Critical for language function
│   │
│   └─ Right hemisphere:
│       ├─ Present but smaller
│       └─ Function: Prosody (emotional tone), non-verbal communication
│
├─ Development: Continues to mature through childhood (ages 5-15)
│   └─ Myelination: Correlates with language acquisition milestones
│
└─ Clinical syndromes - arcuate fasciculus lesions ⭐⭐⭐:
    ├─ Conduction aphasia (classic disconnection syndrome):
    │   ├─ Etiology: Left arcuate fasciculus lesion (ischemic stroke, tumor, trauma)
    │   │   └─ Most common: MCA territory stroke (supramarginal gyrus region)
    │   │
    │   ├─ Classic triad:
    │   │   ├─ Impaired repetition ⭐: Cannot repeat sentences/phrases
    │   │   │   ├─ Most severe deficit: Out of proportion to other language functions
    │   │   │   └─ Example: Can understand "close your eyes" and obey, but cannot repeat phrase
    │   │   │
    │   │   ├─ Preserved comprehension: Wernicke's area intact
    │   │   │   └─ Can understand spoken language
    │   │   │
    │   │   └─ Preserved fluent speech: Broca's area intact
    │   │       └─ Speech is fluent but with phonemic paraphasias (sound errors)
    │   │
    │   ├─ Additional features:
    │   │   ├─ Phonemic (literal) paraphasias: Sound substitutions ("tephelone" for "telephone")
    │   │   ├─ Awareness of errors: Attempts to self-correct (vs Wernicke's aphasia - unaware)
    │   │   ├─ Naming difficulties: Anomia present (but can circumlocute)
    │   │   └─ Written language: Similar pattern (impaired writing, preserved reading comprehension)
    │   │
    │   ├─ Mechanism: Disconnection between Wernicke's (intact comprehension) and Broca's (intact production)
    │   │   └─ Information cannot transfer from posterior language → anterior language areas
    │   │
    │   └─ Prognosis: Better than Broca's or Wernicke's aphasia (cortical areas intact)
    │       ├─ Spontaneous recovery: 50-70% show significant improvement
    │       └─ Repetition may remain impaired
    │
    ├─ Surgical considerations ⭐:
    │   ├─ Awake craniotomy: Direct cortical/subcortical stimulation
    │   │   ├─ Object naming task: Tests arcuate integrity (phonological retrieval)
    │   │   ├─ Repetition task: Direct test of arcuate function
    │   │   └─ Speech arrest: If stimulation disrupts arcuate
    │   │
    │   ├─ Preoperative DTI: Visualize arcuate fasciculus trajectory
    │   │   └─ Surgical planning: Maximal resection with preserved language
    │   │
    │   └─ Glioma infiltration: Low-grade gliomas can infiltrate without obvious deficit
    │       └─ Plasticity: Language network can reorganize if tumor grows slowly
    │
    └─ Right hemisphere lesion:
        └─ Prosody deficits: Impaired emotional tone of speech (aprosody)


### INFERIOR LONGITUDINAL FASCICULUS (ILF) ⭐⭐:

├─ Definition: Long association fiber connecting occipital and temporal lobes (ventral visual stream)
│
├─ Course:
│   ├─ Origin: Occipital lobe (extrastriate visual cortex, V2-V5)
│   ├─ Trajectory: Runs anteriorly along lateral wall of temporal horn
│   │   └─ Parallel to optic radiations (lateral and inferior to optic radiations)
│   └─ Termination: Anterior temporal lobe (temporal pole, fusiform gyrus, parahippocampal gyrus)
│
├─ Function (ventral "what" stream) ⭐:
│   ├─ Object recognition: Visual object processing (occipital → temporal)
│   │   └─ "What" pathway: Identify objects (vs "where" pathway - dorsal)
│   │
│   ├─ Face recognition: Fusiform face area (FFA) connections
│   │   ├─ Right hemisphere dominant: Face-specific processing
│   │   └─ Lesion → prosopagnosia (face blindness)
│   │
│   ├─ Visual memory: Link visual perception → medial temporal lobe memory systems
│   │   └─ Occipital visual areas → hippocampus (via parahippocampal gyrus)
│   │
│   └─ Reading: Visual word form area (VWFA) connections (left hemisphere)
│       └─ Occipital → fusiform gyrus (visual word recognition)
│
├─ Lateralization:
│   ├─ Left hemisphere: Reading (visual word form), object naming
│   └─ Right hemisphere: Face recognition (fusiform face area), object recognition
│
└─ Clinical significance:
    ├─ Prosopagnosia (face blindness) ⭐:
    │   ├─ Etiology: Bilateral fusiform/ILF lesion (or right dominant hemisphere)
    │   ├─ Clinical: Cannot recognize familiar faces (even family members)
    │   ├─ Intact: Can recognize by voice, gait, context
    │   └─ Mechanism: Disconnection of occipital visual input from fusiform face area
    │
    ├─ Pure alexia (alexia without agraphia):
    │   ├─ Etiology: Left occipital lobe + splenium lesion (PCA infarct)
    │   ├─ Mechanism: Visual information cannot reach left VWFA (disconnection)
    │   └─ See corpus callosum section for details
    │
    ├─ Temporal lobectomy: Anterior temporal lobectomy (epilepsy surgery)
    │   ├─ Risk: ILF damage → object recognition deficit, visual memory impairment
    │   └─ Right > left: Right anterior temporal lobectomy → face recognition decline
    │
    └─ Tumor infiltration: Temporo-occipital gliomas (preserve ILF for object recognition)


### INFERIOR FRONTO-OCCIPITAL FASCICULUS (IFOF) ⭐⭐:

├─ Definition: Long association fiber connecting frontal, parietal, occipital, and temporal lobes (longest association tract)
│
├─ Course (most ventral association pathway):
│   ├─ Origin: Occipital lobe (extrastriate cortex), parietal lobe
│   ├─ Trajectory: Runs through external capsule and extreme capsule
│   │   ├─ Passes through temporal stem (inferior to insula)
│   │   ├─ Lateral to uncinate fasciculus (IFOF more posterior)
│   │   └─ Courses along external capsule (between claustrum and putamen)
│   └─ Termination: Frontal lobe (inferior frontal gyrus, orbitofrontal cortex, dorsolateral prefrontal)
│
├─ Function (ventral attention and semantic network) ⭐:
│   ├─ Visual attention: Ventral attention network (reorienting to salient stimuli)
│   │   └─ Occipital visual → frontal attention control
│   │
│   ├─ Semantic processing: Meaning-based language (left hemisphere)
│   │   ├─ Visual object → semantic knowledge (frontal)
│   │   └─ Reading comprehension: Link visual word → meaning
│   │
│   ├─ Visual perception and recognition: Occipital → frontal feedback
│   │   └─ Top-down modulation of visual processing
│   │
│   └─ Emotional processing: Orbitofrontal ↔ visual (especially right hemisphere)
│       └─ Emotional significance of visual stimuli
│
├─ Anatomical controversy:
│   ├─ Existence debated: Some DTI studies question if true direct pathway
│   ├─ May represent: Multiple indirect pathways (not single fasciculus)
│   └─ Modern consensus: Exists but anatomically heterogeneous
│
├─ Lateralization:
│   ├─ Left hemisphere: Semantic processing, reading comprehension
│   └─ Right hemisphere: Ventral attention, emotional processing
│
└─ Clinical significance:
    ├─ Semantic deficits: Left IFOF lesion → semantic aphasia (difficulty with word meaning)
    ├─ Visual attention: Right IFOF lesion → attentional deficits
    ├─ Surgical: Insular/temporal glioma resection (preserve IFOF for semantic function)
    └─ DTI: Difficult to separate from UF and ILF (close proximity)


### UNCINATE FASCICULUS (UF) ⭐⭐⭐:

├─ Definition: Hook-shaped association fiber connecting anterior temporal lobe to frontal lobe (orbitofrontal, inferior frontal)
│
├─ Course ("hook" shape - "uncus" = hook in Latin):
│   ├─ Origin: Anterior temporal lobe (temporal pole, amygdala, anterior hippocampus)
│   ├─ Trajectory: Curves anteriorly and superiorly (hook shape)
│   │   ├─ Passes through limen insulae (anterior perforated substance)
│   │   └─ Sharp anterior turn around frontal stem
│   └─ Termination: Frontal lobe (orbitofrontal cortex, inferior frontal gyrus)
│
├─ Function (limbic-frontal connection) ⭐⭐:
│   ├─ Emotional regulation: Amygdala ↔ orbitofrontal cortex
│   │   ├─ Fear conditioning: Amygdala → frontal cortex (emotional learning)
│   │   └─ Emotion regulation: Frontal inhibition of amygdala (top-down control)
│   │
│   ├─ Memory retrieval: Hippocampus ↔ prefrontal cortex
│   │   ├─ Episodic memory: Retrieve personal memories
│   │   └─ Semantic memory: Access stored knowledge
│   │
│   ├─ Social cognition: Emotional and social judgment (theory of mind)
│   │   └─ Orbitofrontal cortex integrates emotional and social information from temporal lobe
│   │
│   └─ Language (left hemisphere): Semantic retrieval, proper name retrieval
│
├─ Lateralization:
│   ├─ Left hemisphere: Semantic retrieval, language (proper names)
│   └─ Right hemisphere: Emotional processing (stronger), social cognition
│
├─ Development: Continues to mature through adolescence (emotional regulation develops)
│
└─ Clinical significance ⭐⭐:
    ├─ Temporal lobe epilepsy: Anterior temporal lobectomy (epilepsy surgery)
    │   ├─ UF transected: Inevitable in anterior temporal lobectomy
    │   ├─ Post-surgical deficits:
    │   │   ├─ Proper name retrieval: Difficulty recalling people's names (especially left)
    │   │   ├─ Semantic memory: Object naming deficits
    │   │   └─ Emotional changes: Subtle emotional regulation deficits
    │   └─ Usually well-tolerated: Contralateral UF compensates
    │
    ├─ Psychiatric disorders (microstructural changes on DTI):
    │   ├─ Depression: Reduced UF integrity (emotional dysregulation)
    │   ├─ Anxiety disorders: Altered UF (amygdala-frontal dysconnectivity)
    │   ├─ Schizophrenia: Reduced UF integrity (especially left - semantic deficits)
    │   └─ Antisocial behavior: UF abnormalities (impaired emotional regulation)
    │
    ├─ Alzheimer's disease: Early degeneration of UF (memory, semantic deficits)
    │
    ├─ Frontotemporal dementia: UF degeneration (behavioral variant)
    │   └─ Emotional and social dysfunction
    │
    └─ Surgical: Preserve UF when possible during anterior temporal/frontal surgery


### CINGULUM ⭐⭐:

├─ Definition: C-shaped association fiber within cingulate gyrus connecting frontal, parietal, temporal, and limbic regions
│
├─ Course (follows cingulate gyrus) ⭐:
│   ├─ Origin: Orbitofrontal cortex, subcallosal area (ventral frontal)
│   ├─ Trajectory: Follows cingulate gyrus (medial surface of hemisphere)
│   │   ├─ Arches superiorly: Along corpus callosum (above corpus callosum)
│   │   ├─ Curves posteriorly: Into parietal lobe (retrosplenial cortex)
│   │   └─ Curves inferiorly: Into temporal lobe (parahippocampal gyrus)
│   └─ Termination: Medial temporal lobe (entorhinal cortex, parahippocampal gyrus)
│
├─ Subdivisions (functional):
│   ├─ Anterior cingulum (subgenual/perigenual):
│   │   ├─ Connects: Orbitofrontal, anterior cingulate cortex (ACC), medial prefrontal
│   │   └─ Function: Emotional regulation, error monitoring, conflict resolution
│   │
│   ├─ Dorsal cingulum:
│   │   ├─ Connects: Dorsal ACC, supplementary motor area (SMA), motor cortex
│   │   └─ Function: Motor control, pain processing, cognitive control
│   │
│   ├─ Posterior cingulum (retrosplenial):
│   │   ├─ Connects: Posterior cingulate cortex (PCC), retrosplenial cortex, precuneus
│   │   └─ Function: Default mode network, spatial orientation, autobiographical memory
│   │
│   └─ Temporal cingulum (parahippocampal):
│       ├─ Connects: Parahippocampal gyrus, entorhinal cortex, hippocampus
│       └─ Function: Episodic memory, spatial navigation (Papez circuit)
│
├─ Function (Papez circuit and limbic system) ⭐⭐:
│   ├─ Emotion: Limbic system integration (cingulate cortex hub for emotional processing)
│   ├─ Memory: Papez circuit (hippocampus → fornix → mammillary bodies → thalamus → cingulate → hippocampus)
│   ├─ Attention: Anterior cingulate (conflict monitoring, error detection)
│   ├─ Pain processing: Anterior cingulate (affective component of pain)
│   └─ Default mode network: Posterior cingulate (self-referential thought, mind-wandering)
│
├─ Lateralization: Bilateral and symmetric (less lateralized than other association tracts)
│
└─ Clinical significance:
    ├─ Alzheimer's disease ⭐:
    │   ├─ Early degeneration: Cingulum (especially posterior) degenerates early
    │   ├─ Correlation: Cingulum integrity predicts memory decline
    │   └─ Hippocampus disconnection: Disrupts memory consolidation
    │
    ├─ Depression:
    │   ├─ Anterior cingulum: Reduced integrity (emotional regulation)
    │   └─ Treatment: Deep brain stimulation (DBS) of subgenual cingulate (Brodmann area 25)
    │
    ├─ Chronic pain: Cingulotomy historically used (anterior cingulate lesion)
    │   ├─ Mechanism: Disrupts affective component of pain
    │   └─ Modern: Rarely performed (medication/neuromodulation preferred)
    │
    ├─ Schizophrenia: Abnormal cingulum (emotional, cognitive deficits)
    │
    └─ Surgical: Cingulate gyrus tumor resection (must consider functional connectivity)

## IV. CLINICAL IMAGING OF WHITE MATTER ⭐⭐⭐

### DIFFUSION TENSOR IMAGING (DTI) ⭐⭐⭐:

├─ Principle: MRI technique measuring water diffusion along white matter tracts
│   ├─ Anisotropic diffusion: Water diffuses preferentially along axons (parallel > perpendicular)
│   └─ Myelin barrier: Restricts perpendicular diffusion → enhances anisotropy
│
├─ Key metrics:
│   ├─ Fractional anisotropy (FA) ⭐:
│   │   ├─ Range: 0-1 (0 = isotropic/no directionality, 1 = perfectly anisotropic)
│   │   ├─ White matter: FA 0.4-0.8 (high anisotropy)
│   │   ├─ Gray matter: FA 0.1-0.3 (low anisotropy)
│   │   └─ Clinical: Reduced FA indicates white matter injury/disease
│   │
│   ├─ Mean diffusivity (MD):
│   │   ├─ Average magnitude of diffusion (all directions)
│   │   ├─ Increased MD: Edema, infarction, demyelination
│   │   └─ Decreased MD: Cytotoxic edema (acute stroke)
│   │
│   ├─ Axial diffusivity (AD):
│   │   ├─ Diffusion parallel to axons
│   │   └─ Decreased AD: Axonal injury
│   │
│   └─ Radial diffusivity (RD):
│       ├─ Diffusion perpendicular to axons
│       └─ Increased RD: Demyelination (loss of myelin barrier)
│
├─ Tractography ⭐⭐:
│   ├─ Definition: 3D reconstruction of white matter pathways from DTI data
│   │
│   ├─ Methods:
│   │   ├─ Deterministic: Follow principal diffusion direction at each voxel
│   │   │   ├─ Advantage: Fast, intuitive
│   │   │   └─ Limitation: Fails at crossing fibers
│   │   │
│   │   └─ Probabilistic: Model probability of connection
│   │       ├─ Advantage: Handles crossing fibers better
│   │       └─ Limitation: More computationally intensive
│   │
│   ├─ Limitations:
│   │   ├─ Crossing fibers: Cannot resolve fibers crossing at same voxel (>50% of white matter)
│   │   ├─ False positives: May show tracts that don't exist anatomically
│   │   └─ False negatives: May miss tracts (especially small or crossing)
│   │
│   └─ Advanced techniques:
│       ├─ High angular resolution diffusion imaging (HARDI): Better crossing fiber resolution
│       └─ Constrained spherical deconvolution (CSD): Model multiple fiber orientations per voxel
│
└─ Clinical applications ⭐:
    ├─ Surgical planning: Map eloquent white matter tracts (corticospinal, arcuate, optic radiations)
    │   ├─ Glioma resection: Maximal safe resection (preserve function)
    │   ├─ Epilepsy surgery: Map Meyer's loop (anterior temporal lobectomy)
    │   └─ Intraoperative navigation: Register DTI to surgical field
    │
    ├─ Traumatic brain injury: Quantify diffuse axonal injury (reduced FA)
    ├─ Stroke: Predict recovery (corticospinal tract integrity)
    ├─ Multiple sclerosis: Monitor disease progression (white matter lesion burden)
    └─ Neurodegenerative disease: Early detection (Alzheimer's, ALS)


### CONVENTIONAL MRI:

├─ T1-weighted imaging:
│   ├─ White matter: Bright (high signal intensity)
│   ├─ Gray matter: Darker (lower signal)
│   └─ Clinical: Anatomical detail, myelination assessment
│
├─ T2-weighted imaging:
│   ├─ White matter: Dark (low signal intensity)
│   ├─ Gray matter: Brighter (higher signal)
│   ├─ Pathology: Hyperintense (bright) - edema, infarction, demyelination
│   └─ Clinical: Best for white matter lesion detection
│
└─ FLAIR (Fluid-Attenuated Inversion Recovery):
    ├─ Suppresses CSF signal: Dark CSF (enhances lesion conspicuity)
    ├─ White matter lesions: Hyperintense (bright)
    └─ Clinical: MS lesions, periventricular white matter disease, small vessel ischemia

## V. WHITE MATTER DISEASES ⭐⭐⭐

### MULTIPLE SCLEROSIS (MS) ⭐⭐⭐:

├─ Definition: Autoimmune demyelinating disease of CNS white matter
│
├─ Pathophysiology:
│   ├─ Immune attack: T cells cross blood-brain barrier → attack myelin
│   ├─ Demyelination: Oligodendrocyte loss → myelin destruction
│   ├─ Plaques: Discrete white matter lesions (inflammation, demyelination, gliosis)
│   └─ Location: Periventricular, corpus callosum, optic nerves, brainstem, spinal cord
│
├─ Clinical patterns:
│   ├─ Relapsing-remitting (RRMS) - 85% at onset:
│   │   ├─ Episodes: Acute attacks (days to weeks) with recovery
│   │   └─ Between attacks: Stable (no progression)
│   │
│   ├─ Secondary progressive (SPMS):
│   │   ├─ Evolution: From RRMS (50% convert within 10 years)
│   │   └─ Progressive decline: Gradual worsening (with or without relapses)
│   │
│   ├─ Primary progressive (PPMS) - 15%:
│   │   └─ Progressive from onset: No distinct attacks
│   │
│   └─ Progressive-relapsing:
│       └─ Progressive with superimposed attacks
│
├─ Common presenting symptoms:
│   ├─ Optic neuritis (25%): Vision loss, pain with eye movement, color desaturation
│   ├─ Sensory symptoms: Numbness, tingling, Lhermitte's sign
│   ├─ Motor: Weakness, spasticity (corticospinal tract)
│   ├─ Brainstem: Diplopia, vertigo, facial numbness
│   └─ Cerebellar: Ataxia, intention tremor, dysarthria
│
├─ Imaging (MRI) ⭐:
│   ├─ McDonald criteria (diagnosis requires dissemination in space + time):
│   │   ├─ Dissemination in space: ≥2 of 5 CNS regions (periventricular, juxtacortical, infratentorial, spinal cord, optic nerve)
│   │   └─ Dissemination in time: ≥2 attacks OR new lesions on follow-up MRI
│   │
│   ├─ Lesion characteristics:
│   │   ├─ Periventricular: Ovoid lesions perpendicular to ventricles ("Dawson fingers")
│   │   ├─ Corpus callosum: Involves inferior surface (characteristic)
│   │   ├─ T2/FLAIR: Hyperintense (bright)
│   │   └─ T1 post-contrast: Enhancing if acute (blood-brain barrier disruption)
│   │
│   └─ "Black holes": T1 hypointense lesions (severe demyelination/axonal loss)
│
└─ Treatment:
    ├─ Acute relapses: High-dose IV methylprednisolone (1 g daily × 3-5 days)
    ├─ Disease-modifying therapy (DMT):
    │   ├─ Injectable: Interferon-beta, glatiramer acetate
    │   ├─ Oral: Fingolimod, dimethyl fumarate, teriflunomide
    │   └─ Infusion: Natalizumab, ocrelizumab, alemtuzumab
    └─ Symptomatic: Spasticity (baclofen), fatigue (amantadine), neuropathic pain (gabapentin)


### LEUKODYSTROPHIES:

├─ Definition: Inherited disorders of myelin formation or maintenance
│
├─ Adrenoleukodystrophy (X-linked):
│   ├─ Pathophysiology: Peroxisomal disorder → accumulation of very long-chain fatty acids (VLCFA)
│   ├─ Clinical: Childhood cerebral form (4-8 years) - progressive cognitive decline, behavioral changes
│   ├─ Imaging: Symmetric posterior white matter (parieto-occipital) hyperintensity, contrast enhancement at edge
│   └─ Treatment: Hematopoietic stem cell transplant (if early), Lorenzo's oil
│
├─ Metachromatic leukodystrophy (MLD):
│   ├─ Pathophysiology: Arylsulfatase A deficiency → sulfatide accumulation
│   ├─ Clinical: Infantile (most common) - motor regression, peripheral neuropathy, cognitive decline
│   ├─ Imaging: Diffuse white matter hyperintensity ("tigroid" pattern in thalamus)
│   └─ Treatment: Supportive, enzyme replacement (emerging)
│
└─ Krabbe disease (globoid cell leukodystrophy):
    ├─ Pathophysiology: Galactocerebrosidase deficiency → psychosine accumulation
    ├─ Clinical: Infantile - extreme irritability, seizures, optic atrophy
    └─ Treatment: Hematopoietic stem cell transplant (if before symptom onset)


### SMALL VESSEL DISEASE (CEREBRAL MICROANGIOPATHY) ⭐:

├─ Definition: Chronic white matter changes from small vessel ischemia
│
├─ Risk factors:
│   ├─ Age: Most important (almost universal >80 years)
│   ├─ Hypertension: Chronic (damages small vessels)
│   ├─ Diabetes: Microvascular disease
│   └─ Smoking: Endothelial damage
│
├─ Pathophysiology:
│   ├─ Chronic hypoperfusion: Small vessel stenosis/occlusion
│   ├─ Blood-brain barrier dysfunction: Plasma protein leakage
│   └─ White matter injury: Demyelination, axonal loss, gliosis
│
├─ Imaging ⭐:
│   ├─ White matter hyperintensities (WMH): T2/FLAIR bright lesions
│   │   ├─ Location: Periventricular, deep white matter
│   │   ├─ Grading: Fazekas scale (0-3)
│   │   └─ Progression: Worsens with age
│   │
│   └─ Lacunar infarcts: Small (<15 mm) subcortical infarcts
│       └─ Locations: Basal ganglia, thalamus, internal capsule, brainstem
│
├─ Clinical:
│   ├─ Cognitive decline: Executive dysfunction, processing speed
│   ├─ Gait impairment: Small-stepped gait, falls
│   └─ Urinary urgency/incontinence
│
└─ Treatment:
    ├─ Risk factor modification: Blood pressure control (most important), diabetes management, smoking cessation
    └─ Antiplatelet therapy: Aspirin or clopidogrel (prevent further ischemia)


### PROGRESSIVE MULTIFOCAL LEUKOENCEPHALOPATHY (PML):

├─ Etiology: JC virus (polyomavirus) reactivation in immunocompromised
│   └─ Risk factors: AIDS (most common), immunosuppression (natalizumab, rituximab, chemotherapy)
│
├─ Pathophysiology: JC virus infects oligodendrocytes → demyelination
│
├─ Clinical:
│   ├─ Progressive: Weeks to months (rapid deterioration)
│   ├─ Multifocal neurological deficits: Weakness, visual deficits, cognitive decline, ataxia
│   └─ No fever: Distinguishes from infection
│
├─ Imaging:
│   ├─ T2/FLAIR: Multifocal hyperintense lesions (subcortical white matter, U-fibers involved)
│   ├─ No enhancement: Typically (vs MS - enhancing plaques)
│   └─ Asymmetric: Irregular borders
│
└─ Diagnosis: MRI + CSF JC virus PCR (high specificity)
└─ Treatment: Immune reconstitution (stop immunosuppression, HAART for AIDS)

## VI. DISCONNECTION SYNDROMES - SUMMARY ⭐⭐

├─ Split-brain syndrome: Complete corpus callosum section → interhemispheric disconnection
├─ Alien hand syndrome: Anterior corpus callosum lesion → involuntary hand movements
├─ Alexia without agraphia: Left PCA infarct (splenium + occipital) → cannot read but can write
├─ Conduction aphasia: Arcuate fasciculus lesion → impaired repetition, preserved comprehension/production
├─ Apraxia: SLF I lesion → impaired motor planning (reaching)
├─ Spatial neglect: Right SLF II lesion → ignore left hemispace
├─ Prosopagnosia: Bilateral ILF/fusiform lesion → cannot recognize faces
└─ Pure alexia: Left occipital + splenium → cannot read (visual disconnection)

## VII. SURGICAL CONSIDERATIONS ⭐⭐⭐

### AWAKE CRANIOTOMY:

├─ Indications: Tumors near eloquent cortex or white matter
│
├─ Cortical mapping: Direct electrical stimulation
│   ├─ Motor cortex: Muscle twitching (positive motor response)
│   ├─ Language cortex: Speech arrest, anomia (Broca's, Wernicke's)
│   └─ Sensory cortex: Tingling, numbness
│
└─ Subcortical mapping ⭐:
    ├─ Corticospinal tract: Motor response at low current (<10 mA)
    ├─ Arcuate fasciculus: Speech arrest, anomia during language tasks
    ├─ Optic radiations: Visual phenomena (phosphenes)
    └─ Stopping criteria: Functional disruption (halt resection at functional boundary)


### PREOPERATIVE MAPPING:

├─ DTI tractography:
│   ├─ Visualization: 3D reconstruction of white matter tracts
│   ├─ Surgical planning: Identify eloquent tracts, plan approach
│   └─ Registration: Fuse with intraoperative navigation
│
├─ Functional MRI (fMRI):
│   ├─ Language lateralization: Determine dominant hemisphere
│   └─ Motor mapping: Localize motor cortex
│
└─ Magnetoencephalography (MEG):
    └─ Language mapping: Higher spatial resolution than fMRI


### GLIOMA RESECTION PRINCIPLES:

├─ Maximal safe resection: Balance oncological benefit vs functional preservation
├─ Eloquent white matter: Preserve corticospinal, arcuate, optic radiations
├─ Low-grade gliomas: Often infiltrate tracts without obvious deficit (plasticity)
└─ Postoperative deficits: Transient deficits common (resolve in weeks to months)
