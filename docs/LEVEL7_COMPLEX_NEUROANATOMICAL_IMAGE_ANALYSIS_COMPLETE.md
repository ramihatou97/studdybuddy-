# LEVEL 7: COMPLEX NEUROANATOMICAL IMAGE ANALYSIS ⭐⭐⭐
## ULTRA-GRANULAR SPECIFICATION FOR SURGICAL NEURORADIOLOGY

**PURPOSE**: Reference-grade image analysis for neurosurgical planning, anatomical localization, pathology identification, vascular mapping, and intraoperative guidance across all neuroimaging modalities.

**SCOPE**: CT, MRI (all sequences), angiography (DSA, CTA, MRA), perfusion imaging, diffusion imaging (DWI, DTI tractography), functional MRI, PET, SPECT, intraoperative imaging, image-guided surgery, pediatric neuroimaging, emergency protocols.

**CLINICAL INTEGRATION**: Surgical approach planning, lesion localization, eloquent cortex mapping, vascular anatomy, pathology characterization, treatment response assessment, complication detection.

## I. COMPUTED TOMOGRAPHY (CT) NEUROIMAGING ⭐⭐⭐

### NON-CONTRAST HEAD CT (NCCT) ⭐⭐⭐:

├─ Technical parameters:
│   ├─ Slice thickness: 5mm (routine), 2.5mm (posterior fossa), 0.625mm (bone windows)
│   ├─ Windowing ⭐⭐:
│   │   ├─ Brain window: Width 80 HU, Level 40 HU (optimal gray-white differentiation)
│   │   ├─ Subdural window: Width 200-250 HU, Level 75-100 HU (acute blood detection)
│   │   ├─ Bone window: Width 2000-4000 HU, Level 300-700 HU (fractures, skull base)
│   │   └─ Stroke window: Width 8 HU, Level 32 HU (early ischemia detection)
│   └─ Radiation dose: 2-4 mSv (standard head CT)
│
├─ Hounsfield unit (HU) values for anatomical structures ⭐⭐⭐:
│   ├─ CSF: 0-15 HU (similar to water)
│   ├─ White matter: 20-30 HU ⭐
│   ├─ Gray matter: 30-40 HU ⭐
│   │   └─ Gray-white differentiation: Normal 10 HU difference
│   ├─ Acute blood (hyperacute 0-6h): 50-90 HU ⭐⭐⭐
│   ├─ Calcification: >100 HU (physiologic: pineal, choroid plexus, basal ganglia)
│   ├─ Bone: >300 HU
│   └─ Air: -1000 HU (pneumocephalus detection)
│
├─ Anatomical identification on axial NCCT ⭐⭐⭐:
│   ├─ Skull base level (inferior → superior):
│   │   ├─ Foramen magnum level:
│   │   │   ├─ Medulla oblongata (midline, anterior to cerebellar tonsils)
│   │   │   ├─ Cerebellar tonsils (posterolateral to medulla)
│   │   │   └─ Vertebral arteries (anterolateral to medulla, enter foramen magnum)
│   │   ├─ Pons level:
│   │   │   ├─ Pons (central, anterior to fourth ventricle)
│   │   │   ├─ Fourth ventricle (diamond-shaped, posterior to pons) ⭐
│   │   │   ├─ Cerebellar hemispheres (lateral to fourth ventricle)
│   │   │   ├─ Middle cerebellar peduncles (connect pons to cerebellum)
│   │   │   └─ Basilar artery (anterior midline groove of pons)
│   │   └─ Midbrain level:
│   │       ├─ Midbrain (central, "Mickey Mouse" appearance on axial) ⭐
│   │       ├─ Cerebral aqueduct (midline, connects third and fourth ventricles)
│   │       ├─ Quadrigeminal cistern (posterior to midbrain)
│   │       ├─ Ambient cisterns (lateral to midbrain)
│   │       └─ Basilar artery tip (bifurcation into PCAs)
│   │
│   ├─ Suprasellar level:
│   │   ├─ Suprasellar cistern (pentagonal/star-shaped) ⭐⭐:
│   │   │   ├─ Circle of Willis (arterial ring, normal not well-seen on NCCT)
│   │   │   ├─ Optic chiasm (anterior suprasellar cistern)
│   │   │   ├─ Pituitary stalk (infundibulum, connects to pituitary)
│   │   │   └─ Interpeduncular cistern (posterior, between cerebral peduncles)
│   │   ├─ Temporal lobes (lateral, containing temporal horns)
│   │   ├─ Third ventricle (midline, between thalami)
│   │   └─ Sylvian fissures (lateral, separate frontal and temporal lobes)
│   │
│   ├─ Basal ganglia level ⭐⭐⭐:
│   │   ├─ Lateral ventricles (frontal horns anteriorly, bodies posteriorly)
│   │   ├─ Caudate nucleus head (indents lateral wall of frontal horn) ⭐
│   │   ├─ Putamen (lateral to internal capsule)
│   │   ├─ Globus pallidus (medial to putamen, slightly hyperdense)
│   │   ├─ Internal capsule ⭐⭐⭐:
│   │   │   ├─ Anterior limb (between caudate and lentiform nucleus)
│   │   │   ├─ Genu (knee, angulation point)
│   │   │   └─ Posterior limb (between thalamus and lentiform nucleus) ⭐
│   │   │       └─ Location of corticospinal tract (hand-knob area medially)
│   │   ├─ Thalamus (medial to internal capsule, borders third ventricle)
│   │   ├─ External capsule (lateral to putamen)
│   │   ├─ Extreme capsule (lateral to external capsule)
│   │   └─ Insular cortex (deep to Sylvian fissure, lateral to external capsule)
│   │
│   ├─ Centrum semiovale level:
│   │   ├─ Lateral ventricle bodies (parallel to midline)
│   │   ├─ Corona radiata (white matter fan projecting from internal capsule) ⭐
│   │   ├─ Gray-white junction (sharp demarcation in normal brain)
│   │   └─ Frontal, parietal lobes (convex outer surface)
│   │
│   └─ High convexity level:
│       ├─ Superior frontal gyri (anterior)
│       ├─ Precentral gyrus (motor cortex, anterior to central sulcus) ⭐
│       ├─ Central sulcus (separates frontal and parietal lobes)
│       ├─ Postcentral gyrus (sensory cortex, posterior to central sulcus) ⭐
│       ├─ Superior parietal lobule (posterior)
│       └─ Interhemispheric fissure (midline, contains falx cerebri)
│
└─ Acute pathology detection on NCCT ⭐⭐⭐:
    ├─ Acute intracranial hemorrhage ⭐⭐⭐:
    │   ├─ Epidural hematoma (EDH):
    │   │   ├─ Appearance: Biconvex (lentiform), hyperdense (60-80 HU) ⭐
    │   │   ├─ Location: Does NOT cross sutures (limited by dural attachments)
    │   │   ├─ Associated: Skull fracture (>90% of cases), middle meningeal artery injury
    │   │   └─ "Lucid interval": 20-50% of patients (initial unconsciousness → recovery → deterioration)
    │   ├─ Subdural hematoma (SDH):
    │   │   ├─ Appearance: Crescent-shaped, hyperdense (acute 50-90 HU) ⭐
    │   │   ├─ Location: CROSSES sutures (beneath dura, not limited by sutures) ⭐
    │   │   ├─ Does NOT cross: Midline (limited by falx), tentorium
    │   │   └─ Evolution: Acute (hyperdense) → subacute (isodense, 1-3 weeks) → chronic (hypodense, >3 weeks)
    │   ├─ Subarachnoid hemorrhage (SAH) ⭐⭐⭐:
    │   │   ├─ Appearance: Hyperdense blood in CSF spaces (cisterns, sulci, fissures)
    │   │   ├─ Distribution patterns:
    │   │   │   ├─ Aneurysmal SAH: Diffuse cisternal blood, sylvian fissures
    │   │   │   │   ├─ AComA aneurysm: Blood in interhemispheric fissure ⭐
    │   │   │   │   ├─ MCA aneurysm: Blood in ipsilateral sylvian fissure ⭐
    │   │   │   │   └─ PComA aneurysm: Blood in interpeduncular/ambient cisterns
    │   │   │   └─ Perimesencephalic SAH: Blood around midbrain only, benign (no aneurysm)
    │   │   ├─ Grading (modified Fisher scale) ⭐:
    │   │   │   ├─ Grade 0: No SAH
    │   │   │   ├─ Grade 1: Thin SAH, no IVH
    │   │   │   ├─ Grade 2: Thin SAH, with IVH
    │   │   │   ├─ Grade 3: Thick SAH (>1mm), no IVH ⭐ (highest vasospasm risk)
    │   │   │   └─ Grade 4: Thick SAH, with IVH
    │   │   └─ Sensitivity: 95-100% if <6 hours, drops to 50% at 1 week
    │   ├─ Intraparenchymal hemorrhage (IPH):
    │   │   ├─ Appearance: Hyperdense (50-90 HU) mass within brain tissue ⭐
    │   │   ├─ Hypertensive hemorrhage locations (in order of frequency):
    │   │   │   ├─ Basal ganglia/putamen: 35-50% (most common) ⭐
    │   │   │   ├─ Thalamus: 15-25%
    │   │   │   ├─ Pons: 5-10%
    │   │   │   ├─ Cerebellum: 5-10%
    │   │   │   └─ Subcortical white matter (lobar): 10-20%
    │   │   └─ "Spot sign" on CTA: Contrast extravasation → active bleeding, hematoma expansion risk
    │   └─ Intraventricular hemorrhage (IVH):
    │       ├─ Appearance: Blood filling ventricles, fluid-blood level if acute
    │       └─ Complications: Obstructive hydrocephalus (requires EVD placement)
    │
    ├─ Acute ischemic stroke ⭐⭐⭐:
    │   ├─ Early CT signs (within 6 hours) ⭐⭐:
    │   │   ├─ Hyperdense MCA sign ⭐:
    │   │   │   ├─ Appearance: Increased density of MCA (acute thrombus, 60-80 HU)
    │   │   │   ├─ Location: Sylvian fissure (M1 segment)
    │   │   │   └─ Significance: Acute large vessel occlusion → candidate for thrombectomy
    │   │   ├─ Loss of gray-white differentiation ⭐:
    │   │   │   ├─ Insular ribbon sign: Loss of definition of insular cortex
    │   │   │   ├─ Lentiform nucleus obscuration: Blurred putamen border
    │   │   │   └─ Significance: Cytotoxic edema, irreversible injury
    │   │   ├─ Sulcal effacement: Sulci compressed/disappear (early edema)
    │   │   └─ ASPECTS score (Alberta Stroke Program Early CT Score) ⭐⭐:
    │   │       ├─ Scoring: 10-point scale (10 = normal, 0 = diffuse infarction)
    │   │       ├─ Regions: MCA territory divided into 10 regions
    │   │       │   ├─ Basal ganglia level (4): Caudate, lentiform, internal capsule, insular ribbon
    │   │       │   ├─ Supraganglionic level (6): M1-M6 cortical territories
    │   │       │   └─ Scoring: Subtract 1 point for each abnormal region
    │   │       └─ Interpretation ⭐:
    │   │           ├─ ASPECTS ≥6: Good outcome with thrombectomy (used in trials)
    │   │           └─ ASPECTS ≤5: Poor outcome, hemorrhagic transformation risk
    │   └─ Established infarction (>6-24 hours):
    │       ├─ Hypodensity: Edematous tissue (20-25 HU, similar to CSF)
    │       ├─ Mass effect: Sulcal effacement, midline shift, herniation
    │       └─ Hemorrhagic transformation: Petechial or confluent blood within infarct
    │
    ├─ Traumatic brain injury (TBI) ⭐⭐⭐:
    │   ├─ Diffuse axonal injury (DAI):
    │   │   ├─ NCCT: Often normal or subtle hypodensities (MRI more sensitive)
    │   │   ├─ Typical locations (increasing severity):
    │   │   │   ├─ Grade 1: Gray-white junction (lobar, frontal/temporal)
    │   │   │   ├─ Grade 2: Corpus callosum (splenium > body) ⭐
    │   │   │   └─ Grade 3: Brainstem (dorsolateral midbrain, pons) ⭐⭐
    │   │   └─ Mechanism: Shear injury from rotational acceleration-deceleration
    │   ├─ Skull fractures:
    │   │   ├─ Linear fracture: Radiolucent line through bone (bone windows)
    │   │   ├─ Depressed fracture: Inward displacement >1 table thickness ⭐
    │   │   │   └─ Surgical indication: Depressed >1 table thickness, open, or cosmetic
    │   │   └─ Basilar skull fracture: Pneumocephalus, air-fluid level in sphenoid sinus
    │   └─ Cerebral contusion:
    │       ├─ Appearance: Heterogeneous hyperdense (hemorrhage) and hypodense (edema)
    │       ├─ Coup injury: At site of impact
    │       └─ Contrecoup injury: Opposite side (frontal/temporal poles against sphenoid wing)
    │
    └─ Hydrocephalus ⭐⭐⭐:
        ├─ Obstructive (non-communicating) hydrocephalus:
        │   ├─ Ventricular enlargement PROXIMAL to obstruction ⭐
        │   ├─ Common sites of obstruction:
        │   │   ├─ Foramen of Monro: Colloid cyst, intraventricular tumor
        │   │   ├─ Cerebral aqueduct: Aqueductal stenosis, tectal glioma, pineal tumor ⭐
        │   │   └─ Fourth ventricle outlet foramina: Posterior fossa tumor, Dandy-Walker
        │   └─ Appearance: Ballooned ventricles proximal, normal/small distal
        ├─ Communicating hydrocephalus:
        │   ├─ All ventricles enlarged (obstruction at arachnoid granulations or subarachnoid space)
        │   ├─ Causes: Post-hemorrhagic, post-infectious, idiopathic
        │   └─ Normal pressure hydrocephalus (NPH) triad ⭐:
        │       ├─ Gait disturbance: Magnetic gait (feet stuck to floor)
        │       ├─ Urinary incontinence
        │       └─ Dementia (reversible with shunt placement 50-80%)
        └─ CT findings:
            ├─ Periventricular hypodensity: Transependymal CSF flow ⭐
            ├─ Evans index: Frontal horn width / skull width (>0.3 = ventriculomegaly)
            └─ Temporal horn enlargement: Early sign of hydrocephalus

### CT ANGIOGRAPHY (CTA) HEAD AND NECK ⭐⭐⭐:

├─ Technical parameters:
│   ├─ Contrast: 80-100 mL iodinated contrast (Omnipaque 350, Visipaque 320)
│   ├─ Injection rate: 4-5 mL/sec (power injector)
│   ├─ Timing: Bolus tracking (trigger at 100 HU in aortic arch or carotid)
│   ├─ Delay: 15-20 seconds (arterial phase), 60-70 seconds (venous phase)
│   └─ Slice thickness: 0.5-0.625 mm (allows high-quality 3D reconstructions)
│
├─ Vascular anatomy identification ⭐⭐⭐:
│   ├─ Circle of Willis ⭐⭐⭐:
│   │   ├─ Anterior circulation:
│   │   │   ├─ Internal carotid artery (ICA) segments (Bouthillier classification):
│   │   │   │   ├─ C1 (cervical): Carotid bifurcation → skull base
│   │   │   │   ├─ C2 (petrous): Carotid canal in temporal bone
│   │   │   │   ├─ C3 (lacerum): Above foramen lacerum (not through it)
│   │   │   │   ├─ C4 (cavernous): Within cavernous sinus (horizontal then vertical)
│   │   │   │   ├─ C5 (clinoid): Between cavernous sinus and subarachnoid space
│   │   │   │   ├─ C6 (ophthalmic): Ophthalmic artery origin
│   │   │   │   └─ C7 (communicating): PComA and AChA origin → ICA terminus
│   │   │   ├─ Anterior cerebral artery (ACA):
│   │   │   │   ├─ A1 segment: ICA terminus → AComA ⭐
│   │   │   │   │   ├─ Medial lenticulostriate arteries: Supply anterior basal ganglia
│   │   │   │   │   └─ Hypoplastic A1: 10% (contralateral A1 supplies both ACAs via AComA)
│   │   │   │   ├─ A2 segment: AComA → rostrum of corpus callosum (vertical ascent)
│   │   │   │   └─ A3-A5: Distal branches (pericallosal, callosomarginal arteries)
│   │   │   └─ Anterior communicating artery (AComA):
│   │   │       ├─ Connects: Left and right A1 segments
│   │   │       └─ Aneurysm location: Most common site (30-35% of all aneurysms) ⭐⭐⭐
│   │   ├─ Middle cerebral artery (MCA) ⭐⭐⭐:
│   │   │   ├─ M1 segment (sphenoidal/horizontal) ⭐:
│   │   │   │   ├─ Course: ICA bifurcation → MCA bifurcation/trifurcation
│   │   │   │   └─ Lateral lenticulostriate arteries ⭐⭐:
│   │   │       │   ├─ Supply: Basal ganglia, internal capsule (posterior limb) ⭐
│   │   │       │   └─ Clinical: Hypertensive hemorrhage, lacunar infarcts
│   │   │   ├─ M2 segment (insular):
│   │   │   │   ├─ Course: Within Sylvian fissure, overlying insula
│   │   │   │   └─ Divisions: Superior and inferior divisions (or trifurcation)
│   │   │   ├─ M3 segment (opercular): Exits Sylvian fissure
│   │   │   └─ M4 segment (cortical): Cortical surface branches
│   │   └─ Posterior circulation:
│   │       ├─ Vertebral arteries:
│   │       │   ├─ V1 segment: Subclavian artery → C6 transverse foramen
│   │       │   ├─ V2 segment: Ascending through C6-C2 transverse foramina
│   │       │   ├─ V3 segment: C2 → foramen magnum (horizontal loop, prone to dissection)
│   │       │   ├─ V4 segment: Intradural, gives off PICA → joins contralateral VA
│   │       │   └─ PICA (posterior inferior cerebellar artery):
│   │       │       ├─ Origin: V4 segment (lateral medullary syndrome if occluded)
│   │       │       └─ Supply: Lateral medulla, inferior cerebellum, choroid plexus of 4th ventricle
│   │       ├─ Basilar artery ⭐⭐:
│   │       │   ├─ Formation: Confluence of vertebral arteries at pontomedullary junction
│   │       │   ├─ Course: Anterior to pons in prepontine cistern
│   │       │   ├─ Branches:
│   │       │   │   ├─ AICA (anterior inferior cerebellar artery): Pons-cerebellar angle
│   │       │   │   ├─ Pontine perforators: Supply pons (pontine infarcts)
│   │       │   │   └─ SCA (superior cerebellar artery): Just below PCA origin
│   │       │   └─ Terminus: Bifurcates into bilateral PCAs
│   │       ├─ Posterior cerebral artery (PCA):
│   │       │   ├─ P1 segment: Basilar terminus → PComA ⭐
│   │       │   │   └─ Thalamoperforating arteries: Supply thalamus (thalamic infarcts)
│   │       │   ├─ P2 segment: PComA → ambient cistern
│   │       │   ├─ P3 segment: Quadrigeminal cistern
│   │       │   └─ P4 segment: Cortical branches (occipital, temporal)
│   │       └─ Posterior communicating artery (PComA):
│   │           ├─ Connects: ICA (C7) to PCA (P1)
│   │           └─ Aneurysm: Third most common site (25% of aneurysms) ⭐⭐
│   │               └─ Presentation: CN III palsy (pupil-involving, "down and out")
│   │
│   └─ Anatomical variants ⭐⭐:
│       ├─ Incomplete circle of Willis: 50-60% of population
│       ├─ Fetal PCA: PCA supplied by ICA (via large PComA) instead of basilar (15-25%)
│       ├─ Azygos ACA: Single midline A2 (no AComA, 1-2%)
│       └─ Fenestrated vertebrobasilar system: Split vessel (2-3%)
│
├─ Aneurysm detection and characterization ⭐⭐⭐:
│   ├─ Saccular (berry) aneurysms:
│   │   ├─ Location frequency ⭐⭐:
│   │   │   ├─ AComA: 30-35% (most common) ⭐
│   │   │   ├─ PComA: 25% (CN III palsy if ruptured/enlarging)
│   │   │   ├─ MCA bifurcation: 20%
│   │   │   ├─ Basilar tip: 5-10%
│   │   │   ├─ PICA: 5%
│   │   │   └─ Others: Distal ACA, SCA, pericallosal
│   │   ├─ Size classification and rupture risk ⭐⭐⭐:
│   │   │   ├─ Small: <7 mm (annual rupture risk 0.1-0.5%)
│   │   │   ├─ Large: 7-24 mm (rupture risk 2-5%/year)
│   │   │   ├─ Giant: ≥25 mm (rupture risk 6-10%/year, mass effect)
│   │   │   └─ PHASES score: Population, Hypertension, Age, Size, Earlier SAH, Site
│   │   ├─ Morphological features ⭐:
│   │   │   ├─ Dome: Maximum diameter measurement
│   │   │   ├─ Neck: Width of aneurysm base (neck <4mm = narrow-necked, favorable for coiling)
│   │   │   ├─ Aspect ratio: Dome height / neck width (>1.6 = increased rupture risk)
│   │   │   ├─ Daughter sac/bleb: Irregular outpouching (rupture risk marker) ⭐
│   │   │   └─ Multilobulated: Multiple daughter sacs (increased rupture risk)
│   │   └─ CTA vs DSA sensitivity:
│   │       ├─ CTA sensitivity: 95-99% for aneurysms >3mm
│   │       └─ DSA (gold standard): 100% sensitivity, dynamic flow assessment
│   │
│   └─ Vascular occlusive disease ⭐⭐:
│       ├─ Atherosclerotic stenosis:
│       │   ├─ ICA stenosis (NASCET criteria) ⭐:
│       │   │   ├─ Percent stenosis = [1 - (narrowest diameter / normal ICA diameter)] × 100
│       │   │   ├─ Mild: <50% stenosis
│       │   │   ├─ Moderate: 50-69% stenosis
│       │   │   ├─ Severe: 70-99% stenosis (symptomatic → CEA indicated) ⭐
│       │   │   └─ Occlusion: 100% (no flow)
│       │   └─ Carotid endarterectomy (CEA) indications:
│       │       ├─ Symptomatic ≥50% stenosis (recent stroke/TIA)
│       │       └─ Asymptomatic ≥70% stenosis (in selected patients)
│       └─ Acute large vessel occlusion (LVO) ⭐⭐⭐:
│           ├─ Imaging findings:
│           │   ├─ Abrupt vessel cutoff (no contrast beyond occlusion site) ⭐
│           │   ├─ Hyperdense vessel sign on NCCT (clot)
│           │   └─ Collateral flow assessment (leptomeningeal collaterals)
│           └─ Thrombectomy eligibility (based on trials):
│               ├─ Time window: <6 hours (DEFUSE 3: up to 24h with perfusion imaging)
│               ├─ Location: ICA terminus, M1, basilar artery
│               └─ Imaging criteria: ASPECTS ≥6, small core/large penumbra on CTP
│
└─ Extracranial vascular pathology ⭐⭐:
    ├─ Carotid dissection:
    │   ├─ CTA findings:
    │   │   ├─ Intimal flap: Linear filling defect (separates true and false lumens)
    │   │   ├─ "String sign": Long segment narrowing (compressed true lumen)
    │   │   └─ Double lumen sign: True and false lumens visible
    │   └─ Location: C1-C2 (extracranial ICA), spontaneous or traumatic
    └─ Vertebral artery dissection:
        ├─ Location: V3 segment (C1-C2, rotational injury)
        └─ Presentation: Posterior circulation stroke, Wallenberg syndrome

### CT PERFUSION (CTP) ⭐⭐⭐:

├─ Technical principles:
│   ├─ Contrast bolus tracking: Dynamic imaging during first-pass contrast
│   ├─ Coverage: 8-16 cm (whole brain with multi-detector CT)
│   ├─ Temporal resolution: 1-2 second intervals over 60-90 seconds
│   └─ Post-processing: Generates parametric maps (CBF, CBV, MTT, Tmax)
│
├─ Perfusion parameters ⭐⭐⭐:
│   ├─ Cerebral blood flow (CBF):
│   │   ├─ Definition: Volume of blood per 100g tissue per minute (mL/100g/min)
│   │   ├─ Normal values: Gray matter 50-80 mL/100g/min, white matter 20-30 mL/100g/min
│   │   └─ Interpretation: ↓CBF = hypoperfusion (ischemia, infarct)
│   ├─ Cerebral blood volume (CBV):
│   │   ├─ Definition: Volume of blood per 100g tissue (mL/100g)
│   │   ├─ Normal values: Gray matter 4-5 mL/100g, white matter 2-3 mL/100g
│   │   └─ Interpretation:
│   │       ├─ ↑CBV with ↓CBF: Penumbra (autoregulatory vasodilation) ⭐⭐
│   │       └─ ↓CBV with ↓CBF: Infarct core (irreversible) ⭐⭐
│   ├─ Mean transit time (MTT):
│   │   ├─ Definition: Average time for blood to traverse cerebral vasculature (seconds)
│   │   ├─ Normal values: 3-6 seconds
│   │   └─ Interpretation: ↑MTT = delayed perfusion (upstream occlusion)
│   └─ Time to maximum (Tmax):
│       ├─ Definition: Time to peak contrast arrival (delay compared to normal)
│       ├─ Normal: <4 seconds
│       └─ Interpretation:
│           ├─ Tmax >6 seconds: Critically hypoperfused tissue (penumbra if CBV preserved) ⭐⭐
│           └─ Tmax >10 seconds: Severe hypoperfusion (likely to infarct)
│
└─ Clinical applications ⭐⭐⭐:
    ├─ Acute ischemic stroke imaging ⭐⭐⭐:
    │   ├─ Infarct core estimation:
    │   │   ├─ CBV <2.0 mL/100g (severe reduction) ⭐
    │   │   ├─ rCBF <30% of normal contralateral hemisphere
    │   │   └─ Corresponds to DWI lesion on MRI (irreversible injury)
    │   ├─ Penumbra (salvageable tissue) ⭐⭐⭐:
    │   │   ├─ Definition: Hypoperfused but viable tissue (target for reperfusion)
    │   │   ├─ Criteria: Tmax >6 seconds with preserved CBV (>2.0 mL/100g)
    │   │   └─ Mismatch ratio: Penumbra volume / core volume
    │   │       ├─ Mismatch >1.8 with core <70 mL: Thrombectomy benefit (DEFUSE 3) ⭐⭐
    │   │       └─ Large penumbra: Extended time window for intervention (up to 24h)
    │   └─ Collateral assessment:
    │       ├─ Good collaterals: Maintained CBV despite delayed MTT/Tmax
    │       └─ Poor collaterals: Matched CBF/CBV deficits (large core)
    │
    └─ Vasospasm detection (post-SAH) ⭐⭐:
        ├─ CTA: Vessel narrowing (>50% diameter reduction)
        └─ CTP: ↓CBF, ↑MTT in vascular territory (delayed ischemia)

## II. MAGNETIC RESONANCE IMAGING (MRI) ⭐⭐⭐

### MRI SEQUENCES - TECHNICAL PRINCIPLES ⭐⭐⭐:

├─ T1-weighted imaging:
│   ├─ Parameters: Short TR (400-700 ms), short TE (10-30 ms)
│   ├─ Signal characteristics ⭐⭐:
│   │   ├─ High signal (bright):
│   │   │   ├─ Fat (subcutaneous, marrow)
│   │   │   ├─ Subacute hemorrhage (methemoglobin, 3 days - 3 weeks) ⭐
│   │   │   ├─ Melanin (melanoma metastases)
│   │   │   ├─ Proteinaceous fluid (colloid cyst, craniopharyngioma)
│   │   │   └─ Gadolinium contrast enhancement
│   │   └─ Low signal (dark):
│   │       ├─ CSF (ventricles, cisterns, sulci)
│   │       ├─ Cortical bone (calvarium, skull base)
│   │       ├─ Air (paranasal sinuses, mastoid air cells)
│   │       └─ Flow voids (arteries, veins with fast flow)
│   └─ Clinical uses: Anatomical imaging, post-contrast enhancement, hemorrhage dating
│
├─ T2-weighted imaging:
│   ├─ Parameters: Long TR (2000-6000 ms), long TE (80-120 ms)
│   ├─ Signal characteristics ⭐⭐:
│   │   ├─ High signal (bright):
│   │   │   ├─ CSF (ventricles, cisterns) - "bright water" ⭐
│   │   │   ├─ Edema (vasogenic, cytotoxic)
│   │   │   ├─ Demyelination (MS plaques)
│   │   │   ├─ Infarction (chronic, >24-48 hours)
│   │   │   └─ Cystic lesions (arachnoid cyst, epidermoid)
│   │   └─ Low signal (dark):
│   │       ├─ Cortical bone
│   │       ├─ Hemosiderin (chronic hemorrhage, >months) ⭐
│   │       ├─ Calcification
│   │       └─ Flow voids
│   └─ Clinical uses: Edema detection, cysts, chronic hemorrhage, MS plaques
│
├─ FLAIR (Fluid-Attenuated Inversion Recovery) ⭐⭐⭐:
│   ├─ Parameters: Long TR, long TE, inversion pulse (TI 2000-2500 ms to null CSF)
│   ├─ Signal characteristics:
│   │   ├─ CSF: Suppressed (dark) ⭐
│   │   ├─ Periventricular white matter lesions: Bright (MS, small vessel disease) ⭐⭐
│   │   ├─ Sulcal abnormalities: SAH (hyperintense sulci), meningitis, leptomeningeal mets
│   │   └─ Hyperacute stroke: Hyperintense vessel sign (slow flow in occluded artery)
│   └─ Clinical uses ⭐⭐:
│       ├─ White matter disease (MS, chronic small vessel ischemia) - best sequence ⭐
│       ├─ Cortical/subcortical lesions (without CSF artifact)
│       └─ Subarachnoid hemorrhage detection (more sensitive than CT at >5 days)
│
├─ Gradient echo (GRE) / Susceptibility-weighted imaging (SWI) ⭐⭐⭐:
│   ├─ Principle: Sensitive to magnetic susceptibility (paramagnetic substances)
│   ├─ Signal characteristics:
│   │   ├─ "Blooming" dark signal ⭐⭐:
│   │   │   ├─ Deoxyhemoglobin (hyperacute hemorrhage, <24h)
│   │   │   ├─ Hemosiderin (chronic hemorrhage, months-years) ⭐
│   │   │   ├─ Calcification
│   │   │   └─ Microhemorrhages (hypertension, amyloid angiopathy, DAI)
│   │   └─ Clinical uses ⭐⭐:
│   │       ├─ Hemorrhage detection (most sensitive sequence for blood products) ⭐
│   │       ├─ Microbleeds (cerebral amyloid angiopathy, chronic hypertension)
│   │       ├─ Vascular malformations (cavernoma = "popcorn" appearance)
│   │       └─ Diffuse axonal injury (GRE > FLAIR > CT for detection)
│   │
│   └─ Cerebral amyloid angiopathy (CAA) ⭐:
│       ├─ Boston criteria: Multiple cortical/subcortical microbleeds on GRE/SWI
│       └─ Distribution: Lobar (vs deep/basal ganglia in hypertensive angiopathy)
│
└─ Diffusion-weighted imaging (DWI) ⭐⭐⭐:
    ├─ Principle: Detects restricted water diffusion (cytotoxic edema)
    ├─ Signal characteristics ⭐⭐⭐:
    │   ├─ Acute infarction ⭐⭐⭐:
    │   │   ├─ DWI: High signal (bright) - restricted diffusion ⭐
    │   │   ├─ ADC (apparent diffusion coefficient): Low signal (dark) - confirms restriction ⭐
    │   │   ├─ Timing: Positive within 30 minutes of symptom onset
    │   │   ├─ Sensitivity: 90-95% for acute stroke (within 6 hours)
    │   │   └─ Evolution: DWI remains bright for 7-10 days, then "pseudo-normalizes"
    │   ├─ T2 "shine-through":
    │   │   ├─ Caution: High T2 signal can appear bright on DWI without true restriction
    │   │   └─ Solution: Check ADC map (should be dark if truly restricted)
    │   └─ Other restricted diffusion patterns:
    │       ├─ Abscess (high cellularity) - DWI bright, ADC dark
    │       ├─ Epidermoid cyst - DWI bright, ADC dark (vs arachnoid cyst)
    │       ├─ Creutzfeldt-Jakob disease (CJD) - cortical ribboning, basal ganglia
    │       └─ Hypercellular tumors (lymphoma, medulloblastoma, glioblastoma)
    │
    └─ Clinical applications ⭐⭐⭐:
        ├─ Acute stroke diagnosis (gold standard, most sensitive) ⭐⭐⭐
        ├─ DWI-FLAIR mismatch: DWI+/FLAIR- = stroke <4.5 hours (wake-up stroke imaging)
        ├─ Abscess vs necrotic tumor (abscess restricts, tumor does not typically)
        └─ Posterior reversible encephalopathy syndrome (PRES): Vasogenic edema (↑ADC)

### CONTRAST-ENHANCED MRI ⭐⭐⭐:

├─ Gadolinium-based contrast agents:
│   ├─ Mechanism: Shortens T1 relaxation time → bright on T1-weighted images
│   ├─ Dosing: 0.1 mmol/kg (standard), 0.2 mmol/kg (double dose, rarely used)
│   ├─ Routes: IV (standard), intrathecal (myelography, cisternography)
│   └─ Contraindications:
│       ├─ Severe renal impairment (eGFR <30): Nephrogenic systemic fibrosis (NSF) risk
│       └─ Gadolinium deposition: Dentate nucleus, globus pallidus (clinical significance unclear)
│
├─ Enhancement patterns ⭐⭐⭐:
│   ├─ Blood-brain barrier (BBB) disruption ⭐⭐:
│   │   ├─ Normal brain: Intact BBB, no enhancement
│   │   ├─ BBB breakdown: Contrast leaks into extravascular space → enhancement
│   │   └─ Causes: Tumor, infection, inflammation, subacute infarction
│   │
│   ├─ Tumor enhancement patterns ⭐⭐⭐:
│   │   ├─ Ring enhancement:
│   │   │   ├─ Glioblastoma (GBM) ⭐:
│   │   │   │   ├─ Thick irregular ring, nodular components
│   │   │   │   ├─ Extensive vasogenic edema (white matter "fingers")
│   │   │   │   ├─ Central necrosis (T2 bright, restricted diffusion at periphery)
│   │   │   │   └─ Crosses corpus callosum ("butterfly glioma")
│   │   │   ├─ Metastasis:
│   │   │   │   ├─ Smooth thin ring enhancement
│   │   │   │   ├─ Multiple lesions at gray-white junction ⭐
│   │   │   │   ├─ Disproportionate edema (large edema for lesion size)
│   │   │   │   └─ Common primaries: Lung, breast, melanoma, renal, colon
│   │   │   ├─ Abscess:
│   │   │   │   ├─ Smooth thin rim enhancement
│   │   │   │   ├─ Restricted diffusion in center (pus, high cellularity) ⭐⭐
│   │   │   │   └─ Daughter abscesses (small satellite lesions)
│   │   │   └─ Subacute infarct (1-4 weeks):
│   │   │       ├─ Gyral/cortical enhancement (follows vascular territory)
│   │   │       └─ Fogging effect: Isointense on T1/T2 at 1-2 weeks
│   │   ├─ Homogeneous enhancement:
│   │   │   ├─ Meningioma ⭐⭐:
│   │   │   │   ├─ Intense homogeneous enhancement (very vascular) ⭐
│   │   │   │   ├─ Extra-axial (broad dural base, "dural tail" sign) ⭐
│   │   │   │   ├─ CSF cleft between lesion and brain (vs intra-axial)
│   │   │   │   └─ Hyperostosis of adjacent bone (bone window on CT)
│   │   │   ├─ Schwannoma (vestibular schwannoma/acoustic neuroma):
│   │   │   │   ├─ Intense enhancement, CPA cistern location
│   │   │   │   ├─ "Ice cream cone" appearance: mass in IAC extending into CPA
│   │   │   │   └─ Presentation: Sensorineural hearing loss, tinnitus, imbalance
│   │   │   └─ Pituitary adenoma:
│   │   │       ├─ Microadenoma (<1 cm): Hypointense on T1, delayed enhancement
│   │   │       └─ Macroadenoma (≥1 cm): Homogeneous enhancement, suprasellar extension
│   │   └─ No enhancement:
│   │       ├─ Low-grade glioma (grade 1-2) ⭐:
│   │       │   ├─ Infiltrative T2/FLAIR hyperintensity without enhancement
│   │       │   └─ Oligodendroglioma: "Cortical thickening," calcification (CT)
│   │       ├─ Arachnoid cyst: CSF signal (T1 dark, T2 bright), no enhancement
│   │       └─ Epidermoid cyst: DWI bright (vs arachnoid), T1/T2 variable, no enhancement
│   │
│   └─ Infectious/inflammatory enhancement ⭐⭐:
│       ├─ Meningitis: Leptomeningeal enhancement (pia-arachnoid)
│       ├─ Multiple sclerosis (MS) plaques:
│       │   ├─ Active plaques: Ring or nodular enhancement (BBB breakdown)
│       │   ├─ Chronic plaques: No enhancement (gliosis, T2/FLAIR bright)
│       │   └─ Dawson fingers: Periventricular perpendicular orientation ⭐
│       └─ Sarcoidosis: Leptomeningeal enhancement (basilar meninges), cranial neuropathies
│
└─ Post-operative imaging ⭐⭐:
    ├─ Expected post-op findings:
    │   ├─ Blood products: Hemosiderin (GRE/SWI dark)
    │   ├─ Pneumocephalus: Air in subdural or epidural space (resolves in days-weeks)
    │   └─ Pachymeningeal enhancement: Dural enhancement (normal post-craniotomy)
    └─ Residual/recurrent tumor:
        ├─ Nodular enhancement: Suggests residual tumor (vs linear dural enhancement)
        └─ Timing: Wait 3-6 months post-op for baseline (inflammation settles)

### DIFFUSION TENSOR IMAGING (DTI) AND TRACTOGRAPHY ⭐⭐⭐:

├─ Technical principles:
│   ├─ DTI: Measures directional diffusion of water along white matter tracts
│   ├─ Fractional anisotropy (FA) ⭐:
│   │   ├─ Definition: Degree of directional diffusion (0 = isotropic, 1 = perfectly anisotropic)
│   │   ├─ Normal values: White matter FA 0.4-0.8 (corpus callosum ~0.7-0.8)
│   │   └─ Interpretation:
│   │       ├─ High FA: Organized white matter tracts (corticospinal tract, corpus callosum)
│   │       └─ Low FA: Disrupted white matter (edema, tumor infiltration, degeneration)
│   ├─ Mean diffusivity (MD):
│   │   ├─ Average diffusion magnitude (regardless of direction)
│   │   └─ ↑MD in edema, ↓MD in cytotoxic edema
│   └─ Tractography:
│       ├─ Reconstructs 3D white matter pathways from DTI data
│       └─ Color-coding: Red = left-right, green = anterior-posterior, blue = superior-inferior
│
├─ Clinical applications ⭐⭐⭐:
│   ├─ Preoperative surgical planning ⭐⭐⭐:
│   │   ├─ Tumor resection:
│   │   │   ├─ Corticospinal tract mapping ⭐:
│   │   │   │   ├─ Identifies motor pathway relative to tumor
│   │   │   │   ├─ Guides maximal safe resection (avoid motor deficit)
│   │   │   │   └─ Tract displacement vs infiltration vs destruction
│   │   │   ├─ Arcuate fasciculus mapping:
│   │   │   │   ├─ Language pathway (Broca's to Wernicke's)
│   │   │   │   └─ Awake craniotomy planning (dominant hemisphere tumors)
│   │   │   └─ Optic radiations:
│   │   │       ├─ Meyer's loop anterior extent (temporal lobe surgery)
│   │   │       └─ Visual field deficit risk assessment
│   │   ├─ Epilepsy surgery:
│   │   │   └─ Temporal lobectomy: Memory pathway preservation (uncinate, ILF)
│   │   └─ DBS targeting:
│   │       └─ STN localization for Parkinson's disease
│   │
│   ├─ Tumor characterization ⭐⭐:
│   │   ├─ Glioma infiltration:
│   │   │   ├─ Low FA in peritumoral "edema" suggests tumor infiltration (vs pure edema)
│   │   │   └─ Tract disruption: Worse prognosis (invasion vs displacement)
│   │   └─ Meningioma:
│   │       └─ Tract displacement without infiltration (FA preserved in adjacent WM)
│   │
│   └─ Traumatic brain injury ⭐:
│       ├─ Diffuse axonal injury: Decreased FA in corpus callosum, internal capsule
│       └─ Mild TBI/concussion: Subtle FA changes (prognostic value)
│
└─ Limitations:
    ├─ Crossing fibers: Cannot resolve multiple tracts in single voxel (advanced models: HARDI, Q-ball)
    ├─ Edema/mass effect: Distorts tract anatomy, reduces FA
    └─ Motion artifact: Long acquisition time (10-20 minutes)

### FUNCTIONAL MRI (fMRI) ⭐⭐⭐:

├─ Technical principles:
│   ├─ BOLD (blood oxygen level dependent) contrast:
│   │   ├─ Active neurons: ↑ metabolic demand → ↑ regional cerebral blood flow (rCBF)
│   │   ├─ ↑ rCBF exceeds ↑ oxygen consumption → ↑ oxyhemoglobin, ↓ deoxyhemoglobin
│   │   ├─ Deoxyhemoglobin: Paramagnetic (↓ signal), oxyhemoglobin: Diamagnetic
│   │   └─ BOLD signal: ↑ oxyhemoglobin → ↑ T2* signal (bright) in active regions ⭐
│   ├─ Task-based fMRI: Subject performs task (motor, language) during scanning
│   └─ Resting-state fMRI: No task, identifies intrinsic connectivity networks (DMN, etc.)
│
├─ Preoperative mapping ⭐⭐⭐:
│   ├─ Motor mapping:
│   │   ├─ Task: Hand/foot/tongue movements (sequential finger tapping)
│   │   ├─ Activation: Contralateral primary motor cortex (M1) ⭐
│   │   └─ Somatotopy: "Motor homunculus" (foot medial, hand lateral, face inferior)
│   ├─ Language mapping (dominant hemisphere) ⭐⭐:
│   │   ├─ Tasks:
│   │   │   ├─ Verbal fluency: Word generation (activates Broca's area, BA 44/45)
│   │   │   ├─ Semantic decision: Category judgment (Wernicke's area, BA 22)
│   │   │   └─ Silent reading/comprehension
│   │   ├─ Lateralization ⭐:
│   │   │   ├─ Left hemisphere dominant: 95% right-handed, 70% left-handed
│   │   │   ├─ Wada test: Gold standard (intracarotid amobarbital, sedates one hemisphere)
│   │   │   └─ fMRI laterality index: (L-R)/(L+R) activation (>0.2 = left dominant)
│   │   └─ Concordance with awake mapping: 85-95% (fMRI underestimates eloquent cortex)
│   └─ Memory mapping (temporal lobe epilepsy surgery):
│       ├─ Scene encoding task: Hippocampal activation
│       └─ Predicts post-operative memory decline (dominant temporal resection)
│
└─ Limitations ⭐:
    ├─ Tumor-related neurovascular uncoupling: BOLD signal unreliable near tumor
    ├─ Patient compliance: Requires task performance (difficult in impaired patients)
    ├─ False-negative risk: Does not replace intraoperative mapping (awake craniotomy)
    └─ Spatial resolution: 3-4 mm (vs <1 cm with direct cortical stimulation)

### MAGNETIC RESONANCE ANGIOGRAPHY (MRA) ⭐⭐⭐:

├─ Time-of-flight (TOF) MRA ⭐⭐:
│   ├─ Principle: Inflowing blood (unsaturated spins) appears bright vs stationary tissue (saturated)
│   ├─ Technique: 3D acquisition (0.5-1 mm slices), no contrast needed
│   ├─ Advantages:
│   │   ├─ Non-invasive, no contrast, high spatial resolution
│   │   └─ Excellent for circle of Willis, intracranial aneurysms
│   ├─ Disadvantages:
│   │   ├─ Overestimates stenosis (turbulent flow appears as signal loss)
│   │   ├─ Slow flow not well-visualized (dural sinuses, small veins)
│   │   └─ Long acquisition time (10-15 minutes, motion sensitive)
│   └─ Clinical uses: Aneurysm screening, circle of Willis anatomy, stenosis detection
│
├─ Contrast-enhanced MRA (CE-MRA):
│   ├─ Principle: Gadolinium shortens T1 → bright blood regardless of flow
│   ├─ Advantages:
│   │   ├─ Faster acquisition (3-5 minutes)
│   │   ├─ Better visualization of slow flow (venous system, neck vessels)
│   │   └─ No overestimation of stenosis (vs TOF)
│   ├─ Disadvantages:
│   │   ├─ Requires contrast (NSF risk in renal failure)
│   │   └─ Timing-dependent (must catch arterial phase)
│   └─ Clinical uses: Extracranial carotid/vertebral arteries, venous imaging
│
└─ Aneurysm detection ⭐⭐:
    ├─ Sensitivity: 85-95% for aneurysms >3 mm (TOF MRA)
    ├─ MRA vs DSA:
    │   ├─ MRA: Non-invasive screening, detects 90% of aneurysms >5 mm
    │   └─ DSA: Gold standard, higher resolution, required for treatment planning
    └─ Limitations: Small aneurysms (<3 mm), aneurysms near bone (skull base)

---

### MAGNETIC RESONANCE VENOGRAPHY (MRV) ⭐⭐:

├─ Techniques:
│   ├─ 2D TOF MRV: Venous flow appears bright
│   └─ Contrast-enhanced MRV: Gadolinium + delayed venous phase imaging
│
├─ Venous sinus anatomy ⭐⭐:
│   ├─ Superior sagittal sinus (SSS): Runs along superior falx cerebri
│   ├─ Inferior sagittal sinus: Runs along inferior free edge of falx
│   ├─ Straight sinus: Continuation of inferior sagittal sinus + vein of Galen
│   ├─ Transverse sinuses: Lateral course from torcula (confluence of sinuses)
│   ├─ Sigmoid sinuses: S-shaped, drain into internal jugular veins
│   └─ Cavernous sinuses: Lateral to sella, contain ICA and CN III, IV, V1, V2, VI
│
└─ Cerebral venous sinus thrombosis (CVST) ⭐⭐⭐:
    ├─ MRI/MRV findings:
    │   ├─ T1: Hyperintense thrombus (subacute, methemoglobin)
    │   ├─ T2/FLAIR: Hyperintense thrombus, loss of normal flow void ⭐
    │   ├─ MRV: Absence of flow in affected sinus ("empty delta sign") ⭐⭐
    │   └─ Parenchymal changes: Vasogenic edema, hemorrhagic venous infarcts
    ├─ Clinical presentation:
    │   ├─ Headache (90%), papilledema (↑ ICP from impaired venous drainage)
    │   ├─ Seizures (40%), focal neurological deficits
    │   └─ Risk factors: Hypercoagulable states, pregnancy, oral contraceptives, infection
    └─ Treatment:
        ├─ Anticoagulation (heparin, warfarin) - even with hemorrhagic infarcts ⭐
        └─ Endovascular thrombectomy if severe/progressive

## III. DIGITAL SUBTRACTION ANGIOGRAPHY (DSA) ⭐⭐⭐

### TECHNICAL PRINCIPLES AND PROCEDURAL DETAILS ⭐⭐⭐:

├─ Technique:
│   ├─ Femoral artery access (Seldinger technique): 5-6F sheath
│   ├─ Catheterization: Guide catheter in common carotid or vertebral artery
│   ├─ Contrast injection: 4-6 mL/sec, 8-12 mL total per run
│   ├─ Digital subtraction: Pre-contrast mask subtracted from contrast images → isolates vessels
│   └─ Projections: AP, lateral, obliques (multiple angles for 3D vessel visualization)
│
├─ Advantages over CTA/MRA ⭐⭐:
│   ├─ Highest spatial resolution (0.2-0.3 mm vs 0.5-1 mm for CTA/MRA)
│   ├─ Dynamic flow visualization: Temporal resolution (arterial, capillary, venous phases)
│   ├─ Treatment capability: Diagnostic + therapeutic (coiling, stenting, embolization)
│   └─ Gold standard for aneurysm characterization, AVM angioarchitecture
│
└─ Complications:
    ├─ Stroke: 0.5-1.5% (embolic from catheter manipulation, dissection)
    ├─ Groin hematoma: 2-5%
    ├─ Contrast-induced nephropathy: Dose-dependent (use minimum contrast)
    └─ Contrast allergy: Anaphylaxis (<0.1%)

---

### ANEURYSM CHARACTERIZATION FOR TREATMENT PLANNING ⭐⭐⭐:

├─ Morphological assessment ⭐⭐⭐:
│   ├─ Size (maximum diameter): Small <7mm, large 7-24mm, giant ≥25mm
│   ├─ Neck dimensions ⭐:
│   │   ├─ Narrow-necked: <4 mm or dome-to-neck ratio >2 (favorable for coiling) ⭐
│   │   └─ Wide-necked: ≥4 mm or dome-to-neck ratio ≤2 (requires adjunctive techniques)
│   ├─ Daughter sac/bleb: Irregular outpouching (rupture risk marker)
│   ├─ Aspect ratio: Dome height / neck width (>1.6 = increased rupture risk)
│   └─ Relationship to parent vessel and branches (en passage vs sidewall)
│
├─ Endovascular treatment planning ⭐⭐⭐:
│   ├─ Coil embolization:
│   │   ├─ Ideal: Narrow-necked, dome-to-neck ratio >2, no major branch incorporation
│   │   ├─ Technique: Platinum coils deployed via microcatheter, pack aneurysm
│   │   ├─ Adjuncts for wide-necked:
│   │   │   ├─ Balloon-assisted coiling: Temporary balloon in parent vessel (prevents coil herniation)
│   │   │   └─ Stent-assisted coiling: Permanent stent scaffolds neck ⭐
│   │   └─ Outcomes (ISAT trial):
│   │       ├─ Coiling vs clipping: 23.7% vs 30.6% death/dependence at 1 year (ruptured)
│   │       └─ Recanalization: 20-30% (requires follow-up imaging)
│   ├─ Flow diversion ⭐⭐:
│   │   ├─ Indications: Large/giant aneurysms, wide-necked, fusiform
│   │   ├─ Device: Pipeline embolization device (PED), flow diverter stent
│   │   ├─ Mechanism: Diverts flow away from aneurysm → thrombosis, endothelialization
│   │   └─ Requirements: Dual antiplatelet therapy (aspirin + clopidogrel)
│   └─ Parent vessel occlusion:
│       ├─ Indications: Fusiform aneurysm, dissecting aneurysm (uncoilable)
│       └─ Prerequisites: Adequate collateral flow (balloon test occlusion)
│
└─ Surgical treatment planning ⭐⭐:
    ├─ Microsurgical clipping:
    │   ├─ Indications:
    │   │   ├─ Young patients (avoid long-term recanalization risk)
    │   │   ├─ Wide-necked aneurysms (unfavorable for coiling)
    │   │   ├─ MCA aneurysms (surgical access easier than endovascular)
    │   │   └─ Hemorrhage evacuation needed (ICH + aneurysm)
    │   ├─ DSA assessment:
    │   │   ├─ Aneurysm projection (superiorly vs posteriorly → craniotomy approach)
    │   │   ├─ Surrounding vessel anatomy (perforators, branch vessels)
    │   │   └─ Optimal clip trajectory (perpendicular to neck)
    │   └─ Common approaches by location:
    │       ├─ AComA: Pterional or interhemispheric
    │       ├─ PComA: Pterional
    │       ├─ MCA: Pterional (lateral sylvian fissure split)
    │       └─ Basilar tip: Subtemporal, orbitozygomatic, or transpetrosal
    └─ Outcomes: Occlusion rate 90-95%, retreatment rare (<5%)

---

### ARTERIOVENOUS MALFORMATION (AVM) ANGIOARCHITECTURE ⭐⭐⭐:

├─ AVM components ⭐⭐⭐:
│   ├─ Nidus:
│   │   ├─ Tangle of abnormal vessels (direct arteriovenous shunting, no capillaries)
│   │   ├─ Size classification (Spetzler-Martin grade):
│   │   │   ├─ Small: <3 cm (1 point)
│   │   │   ├─ Medium: 3-6 cm (2 points)
│   │   │   └─ Large: >6 cm (3 points)
│   │   └─ Compact vs diffuse morphology
│   ├─ Feeding arteries ⭐:
│   │   ├─ Number, caliber, and flow velocity (high-flow shunting)
│   │   ├─ Arterial supply: Pial, dural, or mixed
│   │   └─ Steal phenomenon: Diversion of blood from normal brain (chronic ischemia)
│   └─ Draining veins:
│       ├─ Superficial vs deep venous drainage (Spetzler-Martin):
│       │   ├─ Superficial only: Cortical veins to superior sagittal sinus (0 points)
│       │   └─ Deep component: Basal vein of Rosenthal, internal cerebral veins (1 point) ⭐
│       └─ Venous stenosis: Increased rupture risk (outflow restriction)
│
├─ Spetzler-Martin grading ⭐⭐⭐:
│   ├─ Grade = Size (1-3) + Eloquence (0-1) + Venous drainage (0-1)
│   ├─ Eloquent cortex ⭐:
│   │   ├─ 1 point if involves: Sensorimotor, language, visual, thalamus, hypothalamus,
│   │   │   internal capsule, brainstem, cerebellar peduncles, deep cerebellar nuclei
│   │   └─ 0 points if non-eloquent (frontal pole, temporal pole, cerebellar hemisphere)
│   └─ Surgical outcomes by grade ⭐:
│       ├─ Grade I-II: Low surgical risk (morbidity <5%)
│       ├─ Grade III: Moderate risk (morbidity 10-15%, consider all modalities)
│       ├─ Grade IV-V: High risk (morbidity >20%, often conservative management) ⭐
│       └─ Grade VI (not resectable): Observation or staged embolization/radiosurgery
│
├─ DSA hemodynamic assessment:
│   ├─ Arterial phase: Identify all feeding arteries (may need selective injections)
│   ├─ Capillary phase: Nidus visualization
│   ├─ Venous phase: Drainage pattern, early draining vein (high-flow shunting) ⭐
│   └─ Steal phenomenon: Delayed filling of normal territories
│
└─ Treatment modalities ⭐⭐:
    ├─ Microsurgical resection:
    │   ├─ Goals: Complete nidus excision (residual nidus = persistent rupture risk)
    │   └─ Technique: Circumferential dissection, sequential feeder occlusion
    ├─ Embolization (adjunctive or primary):
    │   ├─ Goal: Reduce nidus size, obliterate deep feeders (preoperative)
    │   ├─ Agents: Onyx (ethylene-vinyl alcohol copolymer), n-BCA glue
    │   └─ Curative embolization: 10-20% (most require surgery or radiosurgery)
    └─ Stereotactic radiosurgery (SRS):
        ├─ Indications: Small (<3 cm), deep/eloquent AVMs (grade III-IV)
        ├─ Mechanism: Endothelial proliferation, vessel thrombosis (2-3 years)
        ├─ Obliteration rate: 70-90% for AVMs <3 cm
        └─ Latency period: Rupture risk persists until obliteration (2-3 years)

---

### DURAL ARTERIOVENOUS FISTULA (dAVF) ⭐⭐:

├─ Pathophysiology:
│   ├─ Abnormal connection: Dural arteries (ECA branches) → dural venous sinus or cortical vein
│   └─ Acquired lesion (vs AVM, which is congenital)
│
├─ Borden-Shucart classification (based on venous drainage) ⭐⭐:
│   ├─ Type I: Drain into dural sinus (antegrade flow) → benign, low hemorrhage risk
│   ├─ Type II: Drain into dural sinus with retrograde cortical venous reflux → intermediate risk
│   ├─ Type III: Drain directly into cortical veins (no sinus involvement) → high risk ⭐
│   │   └─ Hemorrhage/neurological deficit risk: 40-60% (aggressive treatment indicated)
│   └─ Cognard classification: Further stratifies based on flow direction and venous ectasia
│
└─ Treatment:
    ├─ Type I: Observation or embolization (if symptomatic tinnitus)
    ├─ Type II-III: Endovascular embolization (transarterial, transvenous) or surgery
    └─ Goal: Eliminate cortical venous reflux (disconnect fistula from cortical veins)

## IV. SPINE IMAGING ⭐⭐⭐

### SPINE MRI SEQUENCES AND ANATOMY ⭐⭐⭐:

├─ Standard sequences:
│   ├─ Sagittal T1: Anatomy, marrow signal (fat bright), disc dark
│   ├─ Sagittal T2: CSF bright, disc bright (hydration), cord anatomy, canal stenosis ⭐
│   ├─ Sagittal STIR (Short TI Inversion Recovery): Fat-suppressed, edema bright (trauma, infection)
│   └─ Axial T2: Disc herniation, neural foraminal stenosis, facet arthropathy
│
├─ Spinal cord anatomy on MRI ⭐⭐:
│   ├─ Conus medullaris:
│   │   ├─ Normal level: L1-L2 (termination of spinal cord) ⭐
│   │   └─ Tethered cord: Conus below L2-L3 (associated with tight filum, lipoma)
│   ├─ Cauda equina: Nerve roots below conus (L2 and below)
│   ├─ Cervical enlargement: C3-T1 (brachial plexus origin)
│   ├─ Lumbar enlargement: T11-L1 (lumbosacral plexus origin)
│   └─ Central canal: T2 dark line within cord (ependymal-lined, may contain CSF)
│
├─ Degenerative disc disease ⭐⭐⭐:
│   ├─ Disc desiccation: Loss of T2 signal (dark disc, dehydration) ⭐
│   ├─ Disc herniation classification ⭐⭐:
│   │   ├─ Bulge: Circumferential disc extension >50% of circumference (degenerative)
│   │   ├─ Protrusion: Focal disc extension <50% circumference, base wider than apex ⭐
│   │   ├─ Extrusion: Focal disc, base narrower than apex (may migrate) ⭐⭐
│   │   └─ Sequestration: Disc fragment loses continuity with parent disc (free fragment)
│   ├─ Location:
│   │   ├─ Central: Compresses thecal sac (bilateral symptoms)
│   │   ├─ Paracentral: Compresses traversing nerve root (radiculopathy) ⭐
│   │   │   └─ Example: L4-L5 paracentral disc → L5 nerve root compression
│   │   ├─ Foraminal: Compresses exiting nerve root
│   │   │   └─ Example: L4-L5 foraminal disc → L4 nerve root compression
│   │   └─ Far lateral: Lateral to foramen (extraforaminal)
│   └─ Modic endplate changes:
│       ├─ Modic I: Marrow edema (T1 dark, T2 bright) - acute/inflammatory
│       ├─ Modic II: Fatty marrow replacement (T1 bright, T2 bright) - chronic
│       └─ Modic III: Sclerosis (T1 dark, T2 dark) - end-stage
│
└─ Spinal stenosis ⭐⭐⭐:
    ├─ Central canal stenosis:
    │   ├─ Causes:
    │   │   ├─ Ligamentum flavum hypertrophy (posterior, T2 dark) ⭐
    │   │   ├─ Facet joint arthropathy and hypertrophy
    │   │   ├─ Disc bulge/herniation (anterior)
    │   │   └─ Spondylolisthesis (vertebral subluxation)
    │   ├─ Measurement:
    │   │   ├─ AP diameter: <10 mm = severe stenosis, 10-13 mm = moderate
    │   │   └─ Cross-sectional area: <100 mm² = severe
    │   └─ Clinical: Neurogenic claudication (bilateral leg pain with ambulation, relieved by sitting)
    └─ Foraminal stenosis:
        ├─ Causes: Disc, facet hypertrophy, spondylolisthesis
        └─ Clinical: Unilateral radiculopathy (dermatomal pain, weakness, reflex changes)

---

### SPINE PATHOLOGY ⭐⭐⭐:

├─ Spinal cord compression ⭐⭐⭐:
│   ├─ Epidural compression:
│   │   ├─ Epidural abscess:
│   │   │   ├─ MRI: T2 hyperintense epidural collection, rim enhancement ⭐
│   │   │   ├─ Associated: Discitis-osteomyelitis (disc + endplate T2 bright, enhancement)
│   │   │   ├─ Clinical: Back pain, fever, neurological deficit (surgical emergency if cord compression)
│   │   │   └─ Treatment: IV antibiotics + surgical drainage if neurological deficit
│   │   ├─ Epidural metastasis:
│   │   │   ├─ Most common cause of malignant cord compression (breast, lung, prostate) ⭐
│   │   │   ├─ MRI: Epidural soft tissue mass, vertebral body involvement
│   │   │   └─ Treatment: Dexamethasone + radiation therapy or surgery (decompression + stabilization)
│   │   └─ Epidural hematoma:
│   │       ├─ Causes: Trauma, anticoagulation, AVM, iatrogenic (post-lumbar puncture)
│   │       └─ MRI: Epidural collection, signal varies by age (acute T1 iso/dark, T2 dark)
│   ├─ Intradural extramedullary:
│   │   ├─ Meningioma:
│   │   │   ├─ Most common intradural extramedullary tumor (40%) ⭐
│   │   │   ├─ MRI: Homogeneous enhancement, "dural tail," may have flow voids
│   │   │   └─ Location: Thoracic spine most common (50-60%)
│   │   └─ Schwannoma (nerve sheath tumor):
│   │       ├─ Second most common (30%)
│   │       ├─ MRI: Heterogeneous enhancement, may be dumbbell-shaped (extend through foramen)
│   │       └─ Clinical: Radiculopathy, progressive myelopathy
│   └─ Intramedullary:
│       ├─ Astrocytoma:
│       │   ├─ Most common intramedullary tumor in adults (40-50%)
│       │   ├─ MRI: T2 hyperintense, variable enhancement, cord expansion, syrinx
│       │   └─ Location: Cervical and thoracic spine
│       ├─ Ependymoma:
│       │   ├─ Second most common (30-40%)
│       │   ├─ MRI: Well-circumscribed, homogeneous enhancement, "cap sign" (hemosiderin) ⭐
│       │   └─ Location: Filum terminale, conus medullaris (myxopapillary ependymoma)
│       └─ Hemangioblastoma:
│           ├─ Highly vascular, associated with von Hippel-Lindau (25-40%)
│           └─ MRI: Enhancing mural nodule with cyst, flow voids (serpentine vessels)
│
├─ Infection ⭐⭐:
│   ├─ Discitis-osteomyelitis:
│   │   ├─ MRI findings ⭐:
│   │   │   ├─ Disc space T2 hyperintense (normally dark) ⭐
│   │   │   ├─ Endplate erosion, marrow edema (T1 dark, T2 bright, STIR bright)
│   │   │   ├─ Enhancement of disc and endplates
│   │   │   └─ Paravertebral or epidural abscess (phlegmon vs rim-enhancing collection)
│   │   └─ Organisms: Staph aureus (most common), TB (Pott's disease - gibbus deformity)
│   └─ Spinal epidural abscess (see above)
│
└─ Trauma ⭐⭐⭐:
    ├─ Spinal cord injury without radiographic abnormality (SCIWORA):
    │   ├─ Spinal cord injury (myelopathy) without fracture/dislocation
    │   ├─ MRI: Cord edema (T2 hyperintense), hemorrhage (T2 dark - hemosiderin)
    │   └─ More common in children (ligamentous laxity)
    ├─ Cord signal abnormalities:
    │   ├─ T2 hyperintensity: Edema, contusion (poor prognosis if extensive)
    │   ├─ T2 hypointensity: Hemorrhage (worse prognosis than edema alone) ⭐
    │   └─ Cord transection: Complete discontinuity (worst prognosis)
    └─ Ligamentous injury:
        ├─ STIR/T2 fat-sat: Hyperintense signal in ligaments (edema/tear)
        └─ Ligaments assessed: ALL, PLL, ligamentum flavum, interspinous, facet capsules

## V. PEDIATRIC NEUROIMAGING ⭐⭐

### CONGENITAL MALFORMATIONS ⭐⭐⭐:

├─ Neural tube defects:
│   ├─ Spina bifida occulta:
│   │   ├─ Posterior bony defect (spinous process/lamina absent), dura intact
│   │   ├─ Often asymptomatic, incidental finding
│   │   └─ Cutaneous stigmata: Sacral dimple, hairy patch, lipoma
│   ├─ Meningocele:
│   │   ├─ Bony defect with herniation of meninges (CSF-filled sac), no neural elements
│   │   └─ MRI: CSF signal sac, no cord/nerve roots within
│   ├─ Myelomeningocele ⭐⭐:
│   │   ├─ Bony defect with herniation of meninges AND neural elements (cord, nerve roots)
│   │   ├─ MRI: Neural placode visible on posterior sac surface
│   │   ├─ Associated: Chiari II malformation (80-90%) ⭐
│   │   └─ Treatment: Surgical closure (prevent infection, preserve function)
│   └─ Chiari malformations ⭐⭐⭐:
│       ├─ Chiari I:
│       │   ├─ Cerebellar tonsillar descent >5 mm below foramen magnum ⭐
│       │   ├─ Presentation: Headache (worse with Valsalva), syringomyelia (50-70%)
│       │   └─ Treatment: Posterior fossa decompression (if symptomatic)
│       ├─ Chiari II (Arnold-Chiari):
│       │   ├─ Associated with myelomeningocele (nearly 100%) ⭐
│       │   ├─ Findings: Tonsillar AND vermian descent, beaked tectum, small posterior fossa
│       │   └─ Complications: Hydrocephalus (80%), brainstem dysfunction
│       └─ Chiari III: Tonsillar/vermian descent into high cervical meningocele (rare, severe)
│
├─ Holoprosencephaly:
│   ├─ Failure of forebrain cleavage (prosencephalon division)
│   ├─ Types (severity):
│   │   ├─ Alobar: Single ventricle, fused thalami, absent falx, absent corpus callosum
│   │   ├─ Semilobar: Partial cleavage, posterior falx/interhemispheric fissure present
│   │   └─ Lobar: Mild (fused frontal horns, absent septum pellucidum)
│   └─ Associated: Facial anomalies (cyclopia, proboscis, cleft lip/palate)
│
└─ Corpus callosum agenesis:
    ├─ Complete or partial absence of corpus callosum
    ├─ MRI findings:
    │   ├─ "Bull's horn" or "Viking helmet" frontal horns (lateral ventricles widely separated)
    │   ├─ Probst bundles: Aberrant white matter bundles run parallel to interhemispheric fissure
    │   ├─ High-riding third ventricle (between lateral ventricles)
    │   └─ Radial orientation of gyri (vs normal circumferential)
    └─ Associated syndromes: Aicardi, Dandy-Walker, holoprosencephaly

---

### PEDIATRIC BRAIN TUMORS ⭐⭐⭐:

├─ Posterior fossa tumors (most common in children) ⭐⭐⭐:
│   ├─ Medulloblastoma:
│   │   ├─ Most common malignant pediatric brain tumor (30-40% of posterior fossa) ⭐
│   │   ├─ Location: Midline cerebellar vermis (children), cerebellar hemisphere (adults)
│   │   ├─ MRI:
│   │   │   ├─ T1 iso/hypointense, T2 heterogeneous, avid enhancement
│   │   │   ├─ Restricted diffusion (high cellularity) ⭐
│   │   │   └─ Fourth ventricle compression → hydrocephalus (common)
│   │   ├─ WHO grade IV (highly malignant), CSF dissemination (30%)
│   │   └─ Treatment: Maximal resection + craniospinal radiation + chemotherapy
│   ├─ Pilocytic astrocytoma (juvenile pilocytic astrocytoma, JPA):
│   │   ├─ Most common pediatric brain tumor overall (20%) ⭐
│   │   ├─ Location: Cerebellum (most common), optic pathway, hypothalamus
│   │   ├─ MRI:
│   │   │   ├─ Cystic mass with enhancing mural nodule (classic "cyst + mural nodule") ⭐⭐
│   │   │   ├─ T1 hypointense, T2 hyperintense
│   │   │   └─ No restricted diffusion (low cellularity)
│   │   ├─ WHO grade I (benign), rarely disseminates
│   │   └─ Treatment: Surgical resection (curative if complete)
│   ├─ Ependymoma:
│   │   ├─ Location: Fourth ventricle floor (arises from ependymal lining) ⭐
│   │   ├─ MRI:
│   │   │   ├─ "Plastic" tumor: Extrudes through foramen of Luschka/Magendie ⭐
│   │   │   ├─ Heterogeneous T2, calcification (50%), avid enhancement
│   │   │   └─ Restricted diffusion (moderate cellularity)
│   │   └─ Treatment: Resection + radiation (CSF dissemination 10-15%)
│   └─ Brainstem glioma (diffuse intrinsic pontine glioma, DIPG):
│       ├─ Location: Pons (diffusely infiltrative) ⭐
│       ├─ MRI:
│       │   ├─ T2 hyperintense expansion of pons, no enhancement (low-grade appearance)
│       │   ├─ Encases basilar artery
│       │   └─ Diagnosis: Clinical + imaging (no biopsy needed)
│       ├─ Clinical: Cranial nerve palsies (CN VI, VII), ataxia, long tract signs
│       └─ Prognosis: Dismal (median survival 9 months), no effective treatment
│
├─ Supratentorial tumors:
│   ├─ Craniopharyngioma:
│   │   ├─ Location: Suprasellar (from Rathke's pouch remnants)
│   │   ├─ MRI:
│   │   │   ├─ Cystic and solid components, calcification (90% on CT) ⭐
│   │   │   ├─ T1 variable (proteinaceous cyst fluid may be bright)
│   │   │   └─ Enhancement: Solid nodule and cyst wall
│   │   ├─ Clinical: Visual field defects (chiasm compression), panhypopituitarism
│   │   └─ Treatment: Surgery (gross total resection challenging, high recurrence)
│   └─ Choroid plexus papilloma:
│       ├─ Location: Lateral ventricle (children), fourth ventricle (adults)
│       ├─ MRI: Intraventricular mass, intense enhancement, hydrocephalus (↑ CSF production)
│       └─ Treatment: Surgical resection (curative)
│
└─ Neuroblastoma (extracranial):
    ├─ Adrenal/paraspinal origin, metastasizes to skull/orbits ("raccoon eyes")
    └─ Imaging: Skull metastases, dural involvement, calvarial "hair-on-end" appearance

## VI. ADVANCED IMAGING TECHNIQUES ⭐⭐

### PERFUSION IMAGING (MRI) ⭐⭐:

├─ Dynamic susceptibility contrast (DSC):
│   ├─ Technique: Bolus gadolinium, T2* signal drop during first-pass (paramagnetic effect)
│   ├─ Parameters: rCBV (relative cerebral blood volume), rCBF
│   └─ Clinical use: Tumor grading (high rCBV = high-grade glioma, angiogenesis) ⭐
│
├─ Dynamic contrast-enhanced (DCE):
│   ├─ Technique: T1-weighted dynamic imaging during contrast, pharmacokinetic modeling
│   ├─ Parameters: Ktrans (transfer constant), Ve (extravascular extracellular volume)
│   └─ Clinical use: BBB permeability assessment (Ktrans high in high-grade glioma)
│
└─ Arterial spin labeling (ASL):
    ├─ Technique: Endogenous contrast (magnetically label arterial water)
    ├─ Advantage: No gadolinium needed, quantitative CBF (mL/100g/min)
    └─ Clinical use: Pediatric imaging, renal failure, CBF quantification

---

### MAGNETIC RESONANCE SPECTROSCOPY (MRS) ⭐⭐:

├─ Technique:
│   ├─ Measures metabolite concentrations (chemical shift in ppm)
│   └─ Single-voxel or multi-voxel (chemical shift imaging)
│
├─ Key metabolites ⭐⭐:
│   ├─ N-acetylaspartate (NAA, 2.0 ppm):
│   │   ├─ Neuronal marker (↓ in neuronal loss, infarction, tumor)
│   │   └─ Normal: Highest peak in healthy brain
│   ├─ Choline (Cho, 3.2 ppm):
│   │   ├─ Membrane turnover (↑ in tumors, demyelination, gliosis)
│   │   └─ Cho/NAA ratio: >2 suggests high-grade glioma ⭐
│   ├─ Creatine (Cr, 3.0 ppm):
│   │   ├─ Energy metabolism (relatively stable, used as internal reference)
│   │   └─ Cr concentration: ~8-10 mM (institutional units)
│   ├─ Lactate (Lac, 1.3 ppm):
│   │   ├─ Anaerobic metabolism (↑ in ischemia, high-grade tumor, abscess) ⭐
│   │   └─ Doublet peak (inverted at TE 144 ms)
│   ├─ Lipids (0.9-1.3 ppm):
│   │   └─ Necrosis (↑ in high-grade glioma, metastasis)
│   └─ Myo-inositol (3.6 ppm):
│       └─ Glial marker (↑ in Alzheimer's, low-grade glioma)
│
└─ Clinical applications:
    ├─ Tumor vs non-tumor:
    │   ├─ Tumor: ↑Cho, ↓NAA, presence of lactate/lipids
    │   └─ Radiation necrosis: ↓Cho, ↓NAA (vs tumor recurrence with ↑Cho)
    ├─ Abscess vs necrotic tumor:
    │   ├─ Abscess: ↑↑ lactate, acetate, succinate, amino acids (specific for abscess) ⭐
    │   └─ Tumor: Lactate present, but no acetate/succinate
    └─ Demyelination: ↑Cho (active demyelination), ↓NAA (axonal loss)

---

### POSITRON EMISSION TOMOGRAPHY (PET) ⭐⭐:

├─ FDG-PET (fluorodeoxyglucose):
│   ├─ Principle: Glucose analog uptake (metabolic activity)
│   ├─ Applications:
│   │   ├─ Tumor grading: High-grade glioma (high uptake), low-grade (low/normal uptake)
│   │   ├─ Recurrent tumor vs radiation necrosis:
│   │   │   ├─ Recurrent tumor: High FDG uptake (active metabolism) ⭐
│   │   │   └─ Radiation necrosis: Low/no uptake (hypometabolic)
│   │   └─ Epilepsy focus: Interictal hypometabolism, ictal hypermetabolism
│   └─ Limitation: Normal gray matter has high physiologic uptake (low contrast for brain tumors)
│
├─ Amino acid PET (methionine, FET, DOPA):
│   ├─ Principle: Amino acid transport in tumors (LAT1 transporter)
│   ├─ Advantage: Low background uptake in normal brain (better tumor contrast) ⭐
│   └─ Applications: Low-grade glioma detection, treatment planning, recurrence detection
│
└─ Amyloid PET (Pittsburgh Compound B, florbetapir):
    ├─ Alzheimer's disease: Detects amyloid plaques (cortical uptake)
    └─ Negative scan: Excludes Alzheimer's (high negative predictive value)

---

## VII. INTRAOPERATIVE IMAGING AND IMAGE-GUIDED SURGERY ⭐⭐⭐

### INTRAOPERATIVE MRI (iMRI) ⭐⭐:

├─ Technique:
│   ├─ MRI scanner in or adjacent to operating room
│   ├─ Intraoperative imaging: Real-time assessment of resection extent
│   └─ Magnet strengths: 0.5T (open, low-field), 1.5T, 3T (high-field, better resolution)
│
├─ Applications:
│   ├─ Glioma resection ⭐⭐:
│   │   ├─ Goal: Maximal safe resection (extent of resection correlates with survival)
│   │   ├─ iMRI benefit: Detects residual enhancing tumor (allows further resection)
│   │   └─ Outcomes: ↑ gross total resection rate (68% → 96% in one study)
│   ├─ Pituitary adenoma resection:
│   │   └─ Confirms complete resection, detects residual tumor in cavernous sinus
│   └─ Epilepsy surgery:
│       └─ Confirms complete lesionectomy (MRI-visible lesion)
│
└─ Limitations:
    ├─ Brain shift: Intraoperative changes (CSF loss, resection) alter anatomy
    ├─ Cost and logistics: Expensive, requires MRI-compatible instruments
    └─ Time: Adds 30-60 minutes per scan

---

### INTRAOPERATIVE CT ⭐:

├─ Technique: Mobile CT scanner (O-arm, CereTom)
├─ Applications:
│   ├─ Spinal instrumentation: Verify pedicle screw placement (real-time adjustment)
│   ├─ Hemorrhage detection: Identify post-resection hemorrhage
│   └─ Shunt placement: Confirm ventricular catheter position
└─ Advantage: Faster than iMRI, detects bone/hardware well

---

### FLUORESCENCE-GUIDED SURGERY ⭐⭐⭐:

├─ 5-Aminolevulinic acid (5-ALA) ⭐⭐⭐:
│   ├─ Mechanism:
│   │   ├─ 5-ALA: Oral administration 3-4 hours preoperatively
│   │   ├─ Metabolized to protoporphyrin IX (PpIX) in tumor cells
│   │   ├─ PpIX: Fluorescent under blue light (405 nm excitation, 635 nm emission - pink/red)
│   ├─ Indications: High-grade glioma (glioblastoma) ⭐
│   ├─ Sensitivity/specificity: 85% sensitivity, 90% specificity for high-grade tumor
│   ├─ Benefit: ↑ Gross total resection, ↑ progression-free survival (ALA Glioma Study)
│   └─ Limitation: Low-grade glioma (low/no fluorescence), meningioma (variable)
│
├─ Fluorescein sodium:
│   ├─ Mechanism: IV administration, fluorescence under blue-yellow light
│   ├─ Applications: Tumor resection (glioma, metastasis), vascular visualization
│   └─ Advantage: Lower cost than 5-ALA, visualizes vasculature
│
└─ Indocyanine green (ICG):
    ├─ Mechanism: IV bolus, near-infrared fluorescence (angiography)
    ├─ Applications:
    │   ├─ Vascular surgery: Aneurysm clipping (confirm clip placement, vessel patency) ⭐
    │   ├─ Bypass surgery: Assess graft patency
    │   └─ AVM resection: Confirm complete nidus resection, identify feeders
    └─ Real-time visualization: Flow dynamics, vessel patency

---

### NEURONAVIGATION (STEREOTACTIC NAVIGATION) ⭐⭐⭐:

├─ Principle:
│   ├─ Preoperative MRI/CT co-registered to patient anatomy (fiducial markers or surface registration)
│   ├─ Intraoperative tracking: Optical (infrared cameras) or electromagnetic
│   └─ Real-time display: Instrument position overlaid on MRI/CT (multiplanar views)
│
├─ Applications ⭐⭐:
│   ├─ Tumor resection:
│   │   ├─ Trajectory planning: Safest corridor to lesion (avoid eloquent cortex, vessels)
│   │   ├─ Lesion localization: Small/deep lesions (no visible/palpable landmark)
│   │   └─ Biopsy: Stereotactic biopsy of deep lesions
│   ├─ Functional neurosurgery:
│   │   ├─ DBS electrode placement: Target STN, GPi (Parkinson's), VIM (tremor)
│   │   └─ Accuracy: Submillimetric (<2 mm error)
│   ├─ Spine surgery:
│   │   └─ Pedicle screw placement: Reduces breach rate, radiation exposure
│   └─ Endoscopic procedures:
│       ├─ Endoscopic third ventriculostomy (ETV)
│       └─ Transsphenoidal pituitary surgery
│
└─ Limitations:
    ├─ Brain shift: Tissue deformation after craniotomy (CSF loss, resection) → navigation error
    │   └─ Solution: Intraoperative imaging (iMRI, iCT) to update navigation
    ├─ Registration error: Fiducial placement inaccuracy, patient movement
    └─ Does not replace anatomical knowledge (navigation is adjunct, not replacement)

---

### INTRAOPERATIVE ULTRASOUND ⭐⭐:

├─ Technique: Sterile probe applied directly to brain/cord surface
├─ Applications:
│   ├─ Tumor localization:
│   │   ├─ Real-time imaging: Identify tumor depth, margins
│   │   ├─ Echogenicity: Glioma (hypoechoic), metastasis (hyperechoic with hypoechoic center)
│   │   └─ Doppler: Assess vascularity (AVM, highly vascular tumors)
│   ├─ Resection control: Monitor cavity, detect residual tumor
│   ├─ Cyst aspiration: Guide needle placement (arachnoid cyst, tumor cyst)
│   └─ Spinal cord tumors: Myelotomy planning (identify syrinx, tumor location)
├─ Advantages: Real-time, no radiation, inexpensive
└─ Limitations: Operator-dependent, limited field of view, artifact from instruments

## VIII. EMERGENCY NEUROIMAGING PROTOCOLS ⭐⭐⭐

### ACUTE STROKE PROTOCOL ⭐⭐⭐:

├─ CT-based (code stroke < 4.5 hours):
│   ├─ NCCT head:
│   │   ├─ Exclude hemorrhage (contraindication to tPA)
│   │   ├─ Early ischemic changes: ASPECTS score ≥6 (favorable for intervention)
│   │   └─ Hyperdense vessel sign (acute thrombus)
│   ├─ CTA head and neck:
│   │   ├─ Identify large vessel occlusion (LVO): ICA, M1, M2, basilar
│   │   └─ Assess collaterals (leptomeningeal collaterals)
│   └─ CT perfusion (optional):
│       ├─ Core: CBV <2.0 mL/100g, rCBF <30%
│       └─ Penumbra: Tmax >6 sec with preserved CBV (mismatch ratio >1.8)
│
├─ MRI-based (wake-up stroke, unclear timing):
│   ├─ DWI: Acute infarction (bright, restricted diffusion)
│   ├─ FLAIR: DWI-FLAIR mismatch (DWI+ / FLAIR-) = stroke <4.5h → tPA eligible ⭐⭐
│   ├─ MRA: Vessel occlusion
│   └─ MR perfusion: Core-penumbra mismatch (extended window thrombectomy)
│
└─ Treatment decision:
    ├─ IV tPA: <4.5 hours from onset (or DWI-FLAIR mismatch), no hemorrhage, ASPECTS ≥6
    └─ Mechanical thrombectomy: <6 hours (or up to 24h with perfusion imaging), LVO, ASPECTS ≥6

### TRAUMA IMAGING (HEAD CT) ⭐⭐⭐:

├─ Indications (Canadian CT Head Rule, New Orleans Criteria):
│   ├─ GCS <15 at 2 hours
│   ├─ Suspected skull fracture (palpable, basilar signs)
│   ├─ Vomiting (≥2 episodes), age ≥65, amnesia >30 minutes
│   └─ Dangerous mechanism (fall >3 feet, high-speed MVC)
│
├─ Systematic review ⭐⭐:
│   ├─ Scalp and skull: Soft tissue swelling, fractures (linear, depressed, basilar)
│   ├─ Epidural space: Biconvex hematoma (does not cross sutures)
│   ├─ Subdural space: Crescent hematoma (crosses sutures, not tentorium/falx)
│   ├─ Subarachnoid space: Hyperdense blood in sulci, cisterns
│   ├─ Brain parenchyma: Contusions (frontal/temporal poles), DAI (corpus callosum, brainstem)
│   ├─ Ventricles: Intraventricular hemorrhage, hydrocephalus
│   ├─ Midline shift: >5 mm = significant mass effect
│   └─ Cisternal effacement: Basal cisterns compressed (↑ ICP, herniation risk) ⭐
│
└─ Marshall CT classification (TBI severity):
    ├─ Class I (diffuse injury I): No visible pathology
    ├─ Class II (diffuse injury II): Cisterns present, shift <5 mm, lesion <25 cc
    ├─ Class III (diffuse injury III - swelling): Cisterns compressed/absent, shift <5 mm
    ├─ Class IV (diffuse injury IV - shift): Shift >5 mm
    ├─ Class V (evacuated mass): Any lesion surgically evacuated
    └─ Class VI (non-evacuated mass): Lesion >25 cc not evacuated

---

### HERNIATION SYNDROMES (IMAGING FINDINGS) ⭐⭐⭐:

├─ Subfalcine (cingulate) herniation:
│   ├─ Cingulate gyrus herniates under falx cerebri (to contralateral side)
│   ├─ Imaging: Midline shift >5 mm, deviation of septum pellucidum
│   └─ Complication: ACA compression → infarction (medial frontal/parietal)
│
├─ Uncal (transtentorial) herniation ⭐⭐⭐:
│   ├─ Uncus (medial temporal lobe) herniates over tentorial edge
│   ├─ Imaging:
│   │   ├─ Effacement of ipsilateral ambient cistern ⭐
│   │   ├─ Compression of contralateral cerebral peduncle (Kernohan's notch phenomenon)
│   │   └─ PCA compression → occipital infarction
│   ├─ Clinical: Ipsilateral CN III palsy (dilated pupil, "down and out"), contralateral hemiparesis
│   └─ Duret hemorrhages: Brainstem hemorrhages (secondary to herniation, poor prognosis)
│
├─ Tonsillar herniation:
│   ├─ Cerebellar tonsils herniate through foramen magnum
│   ├─ Imaging: Tonsils >5 mm below foramen magnum, crowding of foramen magnum
│   ├─ Clinical: Respiratory arrest (medullary compression)
│   └─ Causes: Posterior fossa mass, severe supratentorial mass effect
│
└─ Upward (ascending) transtentorial herniation:
    ├─ Posterior fossa mass pushes cerebellum/brainstem upward through tentorial incisura
    └─ Imaging: Effacement of quadrigeminal cistern, aqueductal compression (hydrocephalus)
