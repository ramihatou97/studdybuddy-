## OVERVIEW: THE VERTEBRAL COLUMN & SPINAL CORD

```
├─ Vertebral Column (Spine)
│   ├─ Regions:
│   │   ├─ Cervical: 7 vertebrae (C1-C7)
│   │   ├─ Thoracic: 12 vertebrae (T1-T12)
│   │   ├─ Lumbar: 5 vertebrae (L1-L5)
│   │   ├─ Sacral: 5 fused vertebrae (S1-S5) → sacrum
│   │   └─ Coccygeal: 3-5 fused vertebrae → coccyx
│   │
│   ├─ Total vertebrae: 33 (24 mobile + 9 fused)
│   │
│   ├─ Curvatures (sagittal plane):
│   │   ├─ Cervical lordosis: Concave posteriorly (C-shaped, opens posteriorly)
│   │   ├─ Thoracic kyphosis: Convex posteriorly (rounded back)
│   │   ├─ Lumbar lordosis: Concave posteriorly (sway back)
│   │   └─ Sacral kyphosis: Convex posteriorly (fixed)
│   │
│   └─ Functions:
│       ├─ Support: Weight-bearing, upright posture
│       ├─ Protection: Spinal cord, nerve roots
│       ├─ Movement: Flexion, extension, lateral bending, rotation
│       └─ Shock absorption: Intervertebral discs, curvatures
│
└─ Spinal Cord
    ├─ Length: ~45 cm (adult male), ~43 cm (adult female)
    ├─ Extent: Foramen magnum → L1-L2 vertebral level (conus medullaris)
    ├─ Segments: 31 spinal cord segments
    │   ├─ Cervical: 8 segments (C1-C8)
    │   ├─ Thoracic: 12 segments (T1-T12)
    │   ├─ Lumbar: 5 segments (L1-L5)
    │   ├─ Sacral: 5 segments (S1-S5)
    │   └─ Coccygeal: 1 segment (Co1)
    │
    ├─ Diameter: Cervical (13 mm) > Thoracic (10 mm) < Lumbar (enlargement)
    ├─ Enlargements:
    │   ├─ Cervical enlargement: C5-T1 (innervates upper limbs - brachial plexus)
    │   └─ Lumbar enlargement: L1-S3 (innervates lower limbs - lumbosacral plexus)
    │
    └─ Termination:
        ├─ Conus medullaris: Tapered end at L1-L2 level
        ├─ Filum terminale: Fibrous extension (pia mater) → attaches to coccyx
        └─ Cauda equina: Nerve roots below conus (L2-Co1 roots)
```

## 1. CERVICAL SPINE (C1-C7)

```
├─ Overview
│   ├─ Number: 7 cervical vertebrae
│   ├─ Most mobile: Region of spine (greatest range of motion)
│   ├─ Function: Support head, protect spinal cord, allow head movement
│   └─ Unique features: Transverse foramina (for vertebral artery), bifid spinous processes (C3-C6)
│
├─ C1 ATLAS (First Cervical Vertebra) ⭐⭐⭐
│   ├─ Unique features:
│   │   ├─ NO vertebral body: Ring-shaped (anterior + posterior arches)
│   │   ├─ NO spinous process: Posterior tubercle instead
│   │   └─ Purpose: Support skull, allow nodding ("yes" motion)
│   │
│   ├─ Anatomy:
│   │   ├─ Anterior arch:
│   │   │   ├─ Anterior tubercle: Palpable midline (attachment for longus colli)
│   │   │   ├─ Facet for dens: Posterior surface (articulates with odontoid process of C2)
│   │   │   └─ Transverse ligament groove: Posterior to facet
│   │   │
│   │   ├─ Posterior arch:
│   │   │   ├─ Posterior tubercle: Rudimentary spinous process
│   │   │   ├─ Groove for vertebral artery: Superior surface (lateral to midline)
│   │   │   │   ├─ Vertebral artery: Courses through groove (C1-C2 level)
│   │   │   │   └─ Surgical importance: At risk during posterior C1-C2 fixation ⭐
│   │   │   └─ Lamina: Connects lateral masses to posterior tubercle
│   │   │
│   │   ├─ Lateral masses (paired):
│   │   │   ├─ Superior articular facets: Concave, kidney-shaped
│   │   │   │   ├─ Articulate with: Occipital condyles (atlanto-occipital joint)
│   │   │   │   └─ Motion: Flexion-extension (nodding "yes")
│   │   │   ├─ Inferior articular facets: Flat, circular
│   │   │   │   ├─ Articulate with: C2 superior facets (atlantoaxial joint)
│   │   │   │   └─ Motion: Rotation (turning "no")
│   │   │   └─ Transverse process: Projects laterally (contains transverse foramen)
│   │   │
│   │   └─ Transverse foramen (paired):
│   │       ├─ Contents: Vertebral artery, vertebral vein, sympathetic plexus
│   │       └─ Clinical: Vertebral artery ascends from C6 → C1 through these foramina
│   │
│   ├─ Ligaments:
│   │   ├─ Transverse ligament (of atlas) ⭐⭐⭐:
│   │   │   ├─ Strongest: Most important stabilizer of atlantoaxial joint
│   │   │   ├─ Attachment: Between lateral masses (posterior to dens)
│   │   │   ├─ Function: Holds dens against anterior arch of C1
│   │   │   │   └─ Prevents: Posterior displacement of dens → spinal cord compression
│   │   │   │
│   │   │   └─ Clinical:
│   │   │       ├─ Rupture: Rheumatoid arthritis (pannus erosion), trauma
│   │   │       ├─ Result: Atlantoaxial instability (ADI >3 mm = unstable)
│   │   │       └─ Treatment: Posterior C1-C2 fusion
│   │   │
│   │   ├─ Cruciate ligament: Transverse ligament + vertical bands (superior to occiput, inferior to C2 body)
│   │   ├─ Alar ligaments: Dens → occipital condyles (limit rotation)
│   │   └─ Apical ligament: Dens tip → basion (vestigial)
│   │
│   ├─ Measurements:
│   │   ├─ Atlanto-dental interval (ADI):
│   │   │   ├─ Normal: <3 mm (adults), <5 mm (children)
│   │   │   ├─ Abnormal: >3 mm = atlantoaxial instability
│   │   │   └─ Measurement: Anterior arch of C1 to dens (on lateral X-ray)
│   │   │
│   │   └─ Space available for cord (SAC):
│   │       ├─ Calculation: Spinal canal diameter - ADI - dens diameter
│   │       ├─ Normal: >13 mm (safe)
│   │       └─ <13 mm: Risk of spinal cord compression
│   │
│   └─ Clinical Pathology:
│       ├─ Jefferson fracture (C1 burst fracture):
│       │   ├─ Mechanism: Axial load (diving, fall on head)
│       │   ├─ Pattern: Fracture of anterior + posterior arches (bilateral or 4-part)
│       │   ├─ Stable vs unstable:
│       │   │   ├─ Stable: Transverse ligament intact (combined lateral mass displacement <7 mm)
│       │   │   └─ Unstable: Transverse ligament ruptured (displacement >7 mm)
│       │   ├─ Imaging: CT (fracture), MRI (ligament integrity)
│       │   └─ Treatment:
│       │       ├─ Stable: Collar or halo immobilization (6-12 weeks)
│       │       └─ Unstable: Halo or posterior C1-C2 fusion
│       │
│       └─ Atlanto-occipital dislocation (AOD):
│           ├─ Mechanism: High-energy trauma (motor vehicle accident)
│           ├─ Usually fatal: Immediate (brainstem injury)
│           ├─ Survivors: Extremely rare, require immediate stabilization
│           └─ Treatment: Occipitocervical fusion
│
├─ C2 AXIS (Second Cervical Vertebra) ⭐⭐⭐
│   ├─ Unique feature: Odontoid process (dens) - peg-like projection
│   │
│   ├─ Anatomy:
│   │   ├─ Odontoid process (dens) ⭐:
│   │   │   ├─ Projects superiorly: From C2 body
│   │   │   ├─ Articulates: With anterior arch of C1 (anterior facet)
│   │   │   ├─ Held in place: By transverse ligament (posterior)
│   │   │   ├─ Parts:
│   │   │   │   ├─ Tip: Apical ligament attachment
│   │   │   │   ├─ Waist: Narrowest (most common fracture site) ⭐
│   │   │   │   └─ Base: Junction with C2 body
│   │   │   │
│   │   │   └─ Blood supply (CRITICAL for fracture healing):
│   │   │       ├─ Apex: Apical arcade (from vertebral artery branches)
│   │   │       ├─ Waist: Watershed area (poor vascularity - nonunion risk)
│   │   │       └─ Base: Cancellous bone from C2 body (best healing)
│   │   │
│   │   ├─ Vertebral body:
│   │   │   ├─ Superior articular facets: On lateral masses (articulate with C1)
│   │   │   ├─ Inferior articular facets: Similar to typical cervical (articulate with C3)
│   │   │   └─ Pedicles: Short, thick
│   │   │
│   │   ├─ Spinous process:
│   │   │   ├─ Large, bifid: Palpable (first spinous process below skull)
│   │   │   └─ Muscle attachments: Semispinalis, multifidus, inferior oblique
│   │   │
│   │   └─ Transverse foramen: Contains vertebral artery (enters at C6, ascends through C2)
│   │
│   ├─ Atlantoaxial joint (C1-C2):
│   │   ├─ Type: Pivot joint (synovial)
│   │   ├─ Articulations:
│   │   │   ├─ Median atlantoaxial: Dens with anterior arch of C1
│   │   │   └─ Lateral atlantoaxial (paired): C1 lateral masses with C2 superior facets
│   │   ├─ Motion: Rotation (~50% of total cervical rotation occurs here)
│   │   └─ Stability: Transverse ligament (primary), alar ligaments, joint capsules
│   │
│   └─ Clinical Pathology:
│       ├─ Odontoid fracture (C2 dens fracture) ⭐⭐⭐:
│       │   ├─ Classification (Anderson-D'Alonzo):
│       │   │   ├─ Type I: Tip of dens (avulsion - alar ligament)
│       │   │   │   ├─ Rare: 5% of odontoid fractures
│       │   │   │   ├─ Stable: Usually
│       │   │   │   └─ Treatment: Collar immobilization
│       │   │   │
│       │   │   ├─ Type II: Waist/base junction (most common ⭐):
│       │   │   │   ├─ Incidence: 60% of odontoid fractures
│       │   │   │   ├─ Mechanism: Flexion or extension
│       │   │   │   ├─ Blood supply: Poor (watershed) → high nonunion rate (30-50%)
│       │   │   │   ├─ Displacement: Measure on lateral X-ray/CT
│       │   │   │   │
│       │   │   │   ├─ Treatment (controversial):
│       │   │   │   │   ├─ Non-displaced or minimally displaced (<2 mm):
│       │   │   │   │   │   ├─ Halo immobilization: 8-12 weeks (fusion rate 60-70%)
│       │   │   │   │   │   └─ Higher risk: Age >50, displacement >4-5 mm, comminution
│       │   │   │   │   │
│       │   │   │   │   ├─ Displaced (>2 mm) or elderly (>50 years):
│       │   │   │   │   │   ├─ Anterior odontoid screw: 1 or 2 screws (preserves C1-C2 rotation)
│       │   │   │   │   │   │   ├─ Contraindications: Irreducible, >6 months old, severe osteoporosis
│       │   │   │   │   │   │   └─ Fusion rate: 80-90%
│       │   │   │   │   │   │
│       │   │   │   │   │   └─ Posterior C1-C2 fusion: Harms/Goel technique (C1 lateral mass + C2 pars screws)
│       │   │   │   │   │       ├─ Indication: Irreducible, chronic nonunion, cannot do odontoid screw
│       │   │   │   │   │       ├─ Fusion rate: 95-100%
│       │   │   │   │   │       └─ Disadvantage: Loss of 50% cervical rotation
│       │   │   │   │   │
│       │   │   │   │   └─ Nonunion: 30-50% with halo alone (especially elderly, displaced)
│       │   │   │
│       │   │   └─ Type III: Extends into C2 body (through cancellous bone):
│       │   │       ├─ Incidence: 30% of odontoid fractures
│       │   │       ├─ Blood supply: Good (cancellous bone)
│       │   │       ├─ Healing: Excellent (>90% fusion with immobilization)
│       │   │       └─ Treatment:
│       │   │           ├─ Halo immobilization: 8-12 weeks (fusion rate 90-95%)
│       │   │           └─ Surgery: If unstable, displacement >5 mm, nonunion
│       │   │
│       │   └─ Imaging:
│       │       ├─ CT: Best for fracture characterization, displacement
│       │       ├─ MRI: Transverse ligament integrity, spinal cord injury
│       │       └─ Open mouth X-ray: Can visualize dens (less used now with CT)
│       │
│       ├─ Hangman's fracture (C2 traumatic spondylolisthesis):
│       │   ├─ Definition: Bilateral C2 pars interarticularis fracture
│       │   ├─ Mechanism: Hyperextension + distraction (motor vehicle accident, diving)
│       │   ├─ Classification (Levine-Edwards):
│       │   │   ├─ Type I: Minimal displacement (<3 mm), no angulation
│       │   │   │   ├─ Stable: Usually
│       │   │   │   └─ Treatment: Collar (6-12 weeks)
│       │   │   │
│       │   │   ├─ Type II: Displacement >3 mm or angulation >11°
│       │   │   │   ├─ Unstable: C2-C3 disc disruption
│       │   │   │   └─ Treatment: Halo immobilization or C2-C3 fusion
│       │   │   │
│       │   │   ├─ Type IIA: Minimal displacement, severe angulation (flexion injury)
│       │   │   │   ├─ Highly unstable: Do NOT attempt reduction (can worsen with traction!)
│       │   │   │   └─ Treatment: Extension positioning → halo or surgery
│       │   │   │
│       │   │   └─ Type III: C2-C3 facet dislocation (severe)
│       │   │       ├─ Very unstable
│       │   │       └─ Treatment: Surgical (C2-C3 fusion)
│       │   │
│       │   └─ Neurologic injury: Uncommon (spinal canal wide at C2 level - "anatomic safety net")
│       │
│       └─ Atlantoaxial instability:
│           ├─ Causes: Rheumatoid arthritis (pannus), Down syndrome (ligamentous laxity), trauma
│           ├─ Imaging: ADI >3 mm (flexion-extension X-rays)
│           └─ Treatment: Posterior C1-C2 fusion if symptomatic or ADI >5 mm
│
├─ C3-C7 (Typical Cervical Vertebrae)
│   ├─ General anatomy (all share similar features):
│   │   ├─ Vertebral body:
│   │   │   ├─ Small, wider transversely than AP
│   │   │   ├─ Uncinate processes (uncovertebral joints):
│   │   │   │   ├─ Lateral lips: Superior surface (C3-C7)
│   │   │   │   ├─ Articulate: With inferior surface of vertebra above (luschka joints)
│   │   │   │   └─ Clinical: Osteophytes → foraminal stenosis, vertebral artery compression
│   │   │   │
│   │   │   └─ Endplates: Concave (superior and inferior) - allow disc bulging
│   │   │
│   │   ├─ Pedicles:
│   │   │   ├─ Short, thick
│   │   │   ├─ Project posterolaterally
│   │   │   └─ Intervertebral foramen: Between adjacent pedicles (nerve root exits here)
│   │   │
│   │   ├─ Transverse processes:
│   │   │   ├─ Transverse foramen: Vertebral artery passes through (C1-C6)
│   │   │   ├─ Anterior tubercle: Attachment for anterior muscles (longus colli)
│   │   │   ├─ Posterior tubercle: Attachment for posterior muscles (levator scapulae)
│   │   │   └─ Carotid tubercle (Chassaignac's tubercle):
│   │   │       ├─ C6 anterior tubercle: Prominent
│   │   │       └─ Surgical landmark: Can palpate, compress carotid artery here
│   │   │
│   │   ├─ Articular processes (facet joints):
│   │   │   ├─ Superior articular facets: Face posterosuperiorly (~45° to horizontal)
│   │   │   ├─ Inferior articular facets: Face anteroinferiorly
│   │   │   ├─ Orientation: Allows flexion-extension, rotation, lateral bending
│   │   │   └─ Facet joint arthropathy: Common source of neck pain
│   │   │
│   │   ├─ Lamina:
│   │   │   ├─ Thin, flat
│   │   │   └─ Forms posterior wall: Of spinal canal
│   │   │
│   │   ├─ Spinous process:
│   │   │   ├─ C3-C6: Bifid (split) - short
│   │   │   ├─ C7: Long, non-bifid (vertebra prominens - palpable landmark)
│   │   │   └─ Muscle attachments: Splenius, semispinalis, multifidus, interspinales
│   │   │
│   │   └─ Spinal canal:
│   │       ├─ Triangular: Wide (relative to cord - safety margin)
│   │       ├─ Diameter: ~14-15 mm (AP) at C5-C6
│   │       └─ Stenosis: <13 mm = relative, <10 mm = absolute stenosis
│   │
│   ├─ Specific variations:
│   │   ├─ C3:
│   │   │   └─ Smallest spinous process of C3-C6
│   │   │
│   │   ├─ C6:
│   │   │   ├─ Carotid tubercle: Prominent anterior tubercle (Chassaignac)
│   │   │   └─ Landmark: For anterior cervical approaches (above thyroid cartilage)
│   │   │
│   │   └─ C7 (vertebra prominens):
│   │       ├─ Long, non-bifid spinous process: First prominent spinous process (palpable)
│   │       ├─ Transverse foramen: Often absent or small (vertebral artery enters at C6)
│   │       └─ Surface landmark: Counting vertebral levels
│   │
│   └─ Intervertebral disc anatomy:
│       ├─ Structure:
│       │   ├─ Nucleus pulposus: Gelatinous center (70-90% water, type II collagen, proteoglycans)
│       │   ├─ Annulus fibrosus: Concentric lamellae (type I collagen fibers, alternating orientation)
│       │   └─ Cartilaginous endplates: Interface with vertebral bodies
│       │
│       ├─ Height: ~3 mm (C2-C3) to ~6 mm (C6-C7)
│       ├─ Function: Shock absorption, allow motion
│       │
│       └─ Clinical:
│           ├─ Degeneration: Desiccation, loss of height, osteophyte formation
│           └─ Herniation: Nucleus extrudes through annulus (see pathology below)
│
├─ Cervical Ligaments ⭐⭐⭐
│   ├─ Anterior longitudinal ligament (ALL):
│   │   ├─ Location: Anterior surface of vertebral bodies (skull to sacrum)
│   │   ├─ Attachment: Firmly to vertebral bodies, loosely to discs
│   │   ├─ Function: Limits extension, maintains disc position
│   │   └─ Strength: Strong, thick
│   │
│   ├─ Posterior longitudinal ligament (PLL):
│   │   ├─ Location: Posterior surface of vertebral bodies (inside spinal canal)
│   │   ├─ Attachment: Loosely to bodies, firmly to discs
│   │   ├─ Function: Limits flexion, contains disc herniations (weakest laterally)
│   │   └─ Clinical: Disc herniations tend to be posterolateral (PLL weakest here)
│   │
│   ├─ Ligamentum flavum:
│   │   ├─ Location: Between laminae (posterior spinal canal)
│   │   ├─ Composition: Elastic fibers (80% elastin - highest in body)
│   │   ├─ Function: Recoils to prevent folding into canal during extension
│   │   ├─ Thickness: Thinnest in cervical spine (~2-3 mm)
│   │   └─ Clinical: Hypertrophy → stenosis (more common in lumbar)
│   │
│   ├─ Interspinous ligaments:
│   │   ├─ Location: Between spinous processes
│   │   └─ Function: Limits flexion
│   │
│   ├─ Supraspinous ligament:
│   │   ├─ Location: Tips of spinous processes (C7 to sacrum)
│   │   └─ Cervical: Becomes nuchal ligament
│   │
│   ├─ Nuchal ligament:
│   │   ├─ Location: External occipital protuberance → C7 spinous process (midline)
│   │   ├─ Function: Muscle attachment (trapezius, splenius), limits flexion
│   │   └─ Strong: Fibrous raphe
│   │
│   └─ Facet joint capsules:
│       ├─ Surround: Facet joints (synovial joints)
│       ├─ Function: Stabilize, proprioception (richly innervated)
│       └─ Clinical: Whiplash injuries (capsular tear → chronic pain)
│
├─ Cervical Spine Stability (3-column concept - Denis)
│   ├─ Anterior column: ALL, anterior 2/3 of vertebral body + disc
│   ├─ Middle column: PLL, posterior 1/3 of vertebral body + disc
│   ├─ Posterior column: Pedicles, facets, laminae, spinous processes, ligamentum flavum, interspinous ligaments
│   │
│   ├─ Stability:
│   │   ├─ Stable: 1 column injured
│   │   ├─ Unstable: 2+ columns injured
│   │   └─ Applies: To thoracolumbar > cervical (cervical has different stability considerations)
│   │
│   └─ Cervical-specific stability:
│       ├─ C1-C2: Depends on transverse ligament integrity (ADI <3 mm = stable)
│       ├─ Subaxial (C3-C7):
│       │   ├─ Facet dislocation: Unilateral (often stable) vs bilateral (unstable)
│       │   └─ Disc-ligamentous injury: Can be unstable despite normal X-ray (MRI needed)
│       │
│       └─ White-Panjabi checklist (subaxial instability):
│           ├─ Translation: >3.5 mm (horizontal displacement)
│           ├─ Angulation: >11° (compared to adjacent levels)
│           └─ Flexion-extension X-rays: Assess for instability
│
└─ Clinical Pathology (Cervical Spine) ⭐⭐⭐
    ├─ Cervical disc herniation:
    │   ├─ Most common levels: C5-C6 (45%), C6-C7 (40%), C4-C5 (10%)
    │   ├─ Direction:
    │   │   ├─ Posterolateral (foraminal): Most common (85%) → radiculopathy
    │   │   └─ Central: Spinal cord compression → myelopathy
    │   │
    │   ├─ Clinical presentation:
    │   │   ├─ Radiculopathy:
    │   │   │   ├─ Dermatomal pain: Radiating to arm (sharp, electric, burning)
    │   │   │   ├─ Sensory: Numbness, paresthesias (dermatomal distribution)
    │   │   │   ├─ Motor: Weakness (myotomal pattern)
    │   │   │   └─ Reflexes: Diminished or absent (specific reflex for level)
    │   │   │
    │   │   └─ Myelopathy (central herniation):
    │   │       ├─ Gait: Clumsy, wide-based, spastic
    │   │       ├─ Upper extremity: Fine motor loss (buttons, zippers), hand weakness
    │   │       ├─ Lower extremity: Spasticity, hyperreflexia
    │   │       ├─ Pathologic reflexes: Hoffman's sign, Babinski, clonus
    │   │       └─ Bowel/bladder: Late finding (severe myelopathy)
    │   │
    │   ├─ Level-specific radiculopathy:
    │   │   ├─ C5 nerve root (C4-C5 disc):
    │   │   │   ├─ Motor: Deltoid weakness (shoulder abduction), biceps
    │   │   │   ├─ Sensory: Lateral shoulder, lateral arm
    │   │   │   └─ Reflex: Biceps
    │   │   │
    │   │   ├─ C6 nerve root (C5-C6 disc) - MOST COMMON:
    │   │   │   ├─ Motor: Biceps, wrist extensors (radial deviation)
    │   │   │   ├─ Sensory: Thumb, radial forearm, lateral forearm
    │   │   │   └─ Reflex: Brachioradialis
    │   │   │
    │   │   ├─ C7 nerve root (C6-C7 disc) - SECOND MOST COMMON:
    │   │   │   ├─ Motor: Triceps, wrist flexors, finger extensors
    │   │   │   ├─ Sensory: Middle finger, middle hand
    │   │   │   └─ Reflex: Triceps
    │   │   │
    │   │   └─ C8 nerve root (C7-T1 disc):
    │   │       ├─ Motor: Hand intrinsics (finger flexors, interossei)
    │   │       ├─ Sensory: Ring and small fingers, ulnar forearm
    │   │       └─ Reflex: None specific
    │   │
    │   ├─ Imaging:
    │   │   ├─ MRI: Gold standard (visualizes soft tissues - disc, cord, nerve roots)
    │   │   ├─ CT myelogram: If MRI contraindicated
    │   │   └─ EMG/NCS: Confirm radiculopathy, assess chronicity, exclude other causes
    │   │
    │   └─ Treatment:
    │       ├─ Conservative (first-line):
    │       │   ├─ NSAIDs, muscle relaxants
    │       │   ├─ Physical therapy (after acute phase)
    │       │   ├─ Epidural steroid injection (cervical selective nerve root block)
    │       │   └─ Success rate: 70-80% improve within 6-12 weeks
    │       │
    │       └─ Surgical:
    │           ├─ Indications:
    │           │   ├─ Absolute: Myelopathy (progressive spinal cord compression)
    │           │   ├─ Relative: Severe refractory radiculopathy (>6-12 weeks conservative treatment)
    │           │   └─ Progressive motor deficit: Significant weakness
    │           │
    │           ├─ Approaches:
    │           │   ├─ Anterior cervical discectomy and fusion (ACDF) ⭐:
    │           │   │   ├─ Indication: Single or multilevel disc herniation, spondylosis
    │           │   │   ├─ Technique:
    │           │   │   │   ├─ Smith-Robinson approach: Anterior neck (medial to SCM)
    │           │   │   │   ├─ Discectomy: Remove disc, decompress nerve root/cord
    │           │   │   │   ├─ Interbody graft: Allograft, PEEK cage, autograft
    │           │   │   │   └─ Plate fixation: Anterior cervical plate + screws
    │           │   │   ├─ Fusion rate: 95-98% (single level)
    │           │   │   ├─ Success rate: 90-95% symptom relief
    │           │   │   └─ Complications: Dysphagia (10-50% transient), RLN injury (1-2%), nonunion (5%), adjacent segment degeneration (3-5%/year)
    │           │   │
    │           │   ├─ Cervical disc replacement (arthroplasty):
    │           │   │   ├─ Indication: Single level, mobile disc, no significant spondylosis
    │           │   │   ├─ Advantage: Preserves motion (may reduce adjacent segment disease)
    │           │   │   ├─ Contraindication: Instability, ossification, multilevel
    │           │   │   └─ Success rate: Similar to ACDF (90-95%)
    │           │   │
    │           │   └─ Posterior cervical foraminotomy:
    │           │       ├─ Indication: Lateral disc herniation (foraminal), unilateral radiculopathy
    │           │       ├─ Advantage: Preserves disc, motion (no fusion)
    │           │       ├─ Technique: Laminoforaminotomy (remove superior/inferior facets)
    │           │       └─ Success rate: 90-95% for radiculopathy
    │           │
    │           └─ Myelopathy outcomes:
    │               ├─ Improvement: 60-80% (halts progression, partial recovery)
    │               ├─ Mild myelopathy: Better recovery than severe
    │               └─ Severe/long-standing: Often permanent deficits (spinal cord injury)
    │
    ├─ Cervical spondylotic myelopathy (CSM):
    │   ├─ Definition: Spinal cord dysfunction from chronic degenerative compression
    │   ├─ Pathophysiology: Disc degeneration → osteophytes, ligamentum flavum hypertrophy → canal stenosis → chronic cord compression
    │   │
    │   ├─ Presentation:
    │   │   ├─ Insidious onset: Gradual progression over months-years
    │   │   ├─ Gait difficulty: Most common presenting symptom
    │   │   ├─ Hand clumsiness: Fine motor impairment (buttoning, writing)
    │   │   ├─ Upper extremity: Weakness, atrophy (C5-T1 myotomes)
    │   │   ├─ Lower extremity: Spasticity, hyperreflexia, weakness
    │   │   ├─ Pathologic reflexes: Hoffman, Babinski, clonus
    │   │   └─ Bowel/bladder: Urinary urgency/frequency (late)
    │   │
    │   ├─ Imaging:
    │   │   ├─ MRI: Spinal cord compression, T2 hyperintensity (myelomalacia = poor prognosis)
    │   │   ├─ Canal stenosis: Sagittal diameter <13 mm
    │   │   └─ Multilevel: Often C3-C6
    │   │
    │   ├─ Natural history:
    │   │   ├─ Progressive: 70-80% worsen over time
    │   │   ├─ Stepwise decline: Periods of stability, then worsening
    │   │   └─ Severe cases: Can lead to quadriplegia
    │   │
    │   └─ Treatment:
    │       ├─ Surgery (definitive):
    │       │   ├─ Indications: Moderate-severe myelopathy, progressive
    │       │   ├─ Approaches:
    │       │   │   ├─ Anterior (ACDF or corpectomy): 1-3 levels, ventral compression
    │       │   │   ├─ Posterior (laminectomy + fusion or laminoplasty): Multilevel (>3), dorsal compression
    │       │   │   └─ Combined: Severe, circumferential compression
    │       │   ├─ Goal: Halt progression, partial neurologic recovery
    │       │   └─ Outcomes: 60-80% stabilize or improve (better if mild myelopathy, no T2 signal change)
    │       │
    │       └─ Conservative: NOT recommended for moderate-severe myelopathy (surgical disease)
    │
    └─ Cervical spine trauma (see above for C1, C2 fractures):
        └─ Subaxial (C3-C7) classification (SLIC score) assesses stability, guides treatment
```

## 2. THORACIC SPINE (T1-T12)

```
├─ Overview
│   ├─ Number: 12 thoracic vertebrae
│   ├─ Least mobile: Region of spine (rib cage provides stability)
│   ├─ Function: Protect thoracic organs, attach ribs, weight-bearing
│   └─ Unique feature: Costal facets (for rib articulation)
│
├─ General Anatomy
│   ├─ Vertebral body:
│   │   ├─ Heart-shaped: Wider transversely than AP (transition from cervical to lumbar)
│   │   ├─ Size: Progressively larger from T1 → T12 (increasing weight-bearing)
│   │   ├─ Costal facets (demifacets):
│   │   │   ├─ Superior demifacet: Articulates with head of corresponding rib
│   │   │   ├─ Inferior demifacet: Articulates with head of rib below
│   │   │   ├─ Exception: T1, T10-T12 have full facets (not demifacets)
│   │   │   └─ Rib head: Spans two vertebrae (articulates with both facets)
│   │   │
│   │   └─ Endplates: Relatively flat (compared to cervical/lumbar)
│   │
│   ├─ Pedicles:
│   │   ├─ Short, thick
│   │   ├─ Project posteriorly (vs posterolaterally in cervical)
│   │   └─ Pedicle screw trajectory: Requires steep angle (higher risk of neurologic injury)
│   │
│   ├─ Transverse processes:
│   │   ├─ Long, project posterolaterally
│   │   ├─ Costal facet: On anterior surface (articulates with rib tubercle)
│   │   │   ├─ Costotransverse joint: Rib tubercle to transverse process
│   │   │   └─ Absent: T11-T12 (ribs 11-12 are "floating ribs")
│   │   │
│   │   └─ Muscle attachments: Erector spinae, multifidus, rotators
│   │
│   ├─ Articular processes (facet joints):
│   │   ├─ Superior facets: Face posteriorly (~60-80° to horizontal - more vertical)
│   │   ├─ Inferior facets: Face anteriorly
│   │   ├─ Orientation: Primarily allows rotation, limits flexion-extension
│   │   └─ "Corona" orientation: Lies in coronal plane
│   │
│   ├─ Lamina:
│   │   ├─ Thick, broad
│   │   └─ Overlap: "Shingle pattern" (overlap inferiorly) - limits laminectomy access
│   │
│   ├─ Spinous process:
│   │   ├─ Long, slender
│   │   ├─ Inferiorly directed: Sharp downward angle (overlap)
│   │   ├─ T5-T8: Most severe downward angulation
│   │   └─ Clinical: Difficult to palpate individual levels
│   │
│   └─ Spinal canal:
│       ├─ Small: Narrowest canal diameter (11-12 mm) - least space for cord ⭐
│       ├─ Circular to oval
│       └─ Clinical: Minimal reserve - any stenosis or disc herniation is significant
│
├─ Regional Variations
│   ├─ T1:
│   │   ├─ Transitional: Between cervical and thoracic
│   │   ├─ Costal facet: Full facet for rib 1 (superior only)
│   │   └─ T1 nerve root: Exits above T1 (like cervical - C8 exits above T1)
│   │
│   ├─ T10-T12:
│   │   ├─ Transitional: Between thoracic and lumbar
│   │   ├─ Costal facets: Full facets (not demifacets)
│   │   ├─ T11-T12: No costotransverse joints
│   │   └─ More lumbar-like: Larger bodies, less vertical facets
│   │
│   └─ T12:
│       ├─ "Thoracolumbar" vertebra
│       ├─ Superior facets: Thoracic orientation (coronal)
│       ├─ Inferior facets: Lumbar orientation (sagittal)
│       └─ Susceptible: To fractures (transition point - different mechanics)
│
├─ Rib Cage Articulations
│   ├─ Costovertebral joints (rib head):
│   │   ├─ Articulation: Rib head with vertebral bodies (usually two vertebrae)
│   │   ├─ Ligaments: Radiate ligament, intra-articular ligament
│   │   └─ Motion: Minimal gliding
│   │
│   ├─ Costotransverse joints (rib tubercle):
│   │   ├─ Articulation: Rib tubercle with transverse process
│   │   ├─ Ligaments: Costotransverse ligament, lateral costotransverse, superior costotransverse
│   │   └─ Absent: T11-T12
│   │
│   └─ Function:
│       ├─ Stability: Rib cage provides significant stability (limits motion)
│       └─ Protection: Thoracic organs (heart, lungs, great vessels)
│
├─ Thoracic Spine Stability
│   ├─ Inherently stable: Rib cage (major stabilizer)
│   ├─ 3-column model: Still applies (but less mobile)
│   └─ Fractures: Can be unstable despite rib cage (see pathology)
│
└─ Clinical Pathology (Thoracic Spine) ⭐⭐⭐
    ├─ Thoracic disc herniation:
    │   ├─ Rare: 0.5-4% of all disc herniations (most are asymptomatic)
    │   ├─ Most common levels: T11-T12 (40%), T10-T11 (20%)
    │   ├─ Direction: Central or centrolateral (calcified discs common)
    │   │
    │   ├─ Presentation:
    │   │   ├─ Myelopathy: Most common (narrow canal - minimal reserve)
    │   │   │   ├─ Lower extremity weakness, spasticity
    │   │   │   ├─ Sensory level: At or below level of compression
    │   │   │   └─ Bowel/bladder: If severe cord compression
    │   │   │
    │   │   ├─ Radiculopathy: Band-like chest/abdominal pain (dermatomal)
    │   │   └─ Axial pain: Midline back pain
    │   │
    │   ├─ Imaging:
    │   │   ├─ MRI: Gold standard (disc, cord signal changes)
    │   │   ├─ CT: Visualize calcification (common in thoracic discs)
    │   │   └─ Myelography: If MRI contraindicated
    │   │
    │   └─ Treatment:
    │       ├─ Conservative: If mild symptoms, no myelopathy
    │       └─ Surgical (if myelopathy or severe radiculopathy):
    │           ├─ Approach:
    │           │   ├─ Costotransversectomy: Lateral (decompress without cord retraction)
    │           │   ├─ Transthoracic: Anterolateral (best for ventral disc, calcified)
    │           │   ├─ Thoracoscopic: Minimally invasive
    │           │   └─ Avoid: Posterior laminectomy alone (requires cord retraction - high risk)
    │           │
    │           └─ Outcomes: 70-85% good outcomes (depends on pre-op myelopathy severity)
    │
    ├─ Thoracic fractures (see below - combined with thoracolumbar)
    │
    ├─ Scheuermann's disease (kyphosis):
    │   ├─ Definition: Structural kyphosis from anterior wedging of vertebrae (≥3 adjacent levels, ≥5° wedging each)
    │   ├─ Age: Adolescence (10-16 years)
    │   ├─ Location: T7-T9 (most common)
    │   │
    │   ├─ Imaging:
    │   │   ├─ Anterior wedging: ≥5° (3+ consecutive vertebrae)
    │   │   ├─ Endplate irregularities: Schmorl's nodes (disc herniation into vertebral body)
    │   │   ├─ Kyphosis: >40-45° (normal thoracic 20-40°)
    │   │   └─ Decreased disc height
    │   │
    │   ├─ Clinical: Back pain, cosmetic deformity (round back)
    │   │
    │   └─ Treatment:
    │       ├─ Bracing: If immature skeleton, <55° kyphosis
    │       ├─ Surgery: Severe kyphosis (>70-75°), pain refractory to conservative treatment
    │       └─ Approach: Posterior spinal fusion (instrumentation + fusion)
    │
    └─ Spinal cord at risk: Narrow canal - any pathology can cause myelopathy

## 3. LUMBAR SPINE (L1-L5)

```
├─ Overview
│   ├─ Number: 5 lumbar vertebrae
│   ├─ Most mobile: In flexion-extension (after cervical)
│   ├─ Weight-bearing: Maximum load through lumbar spine
│   └─ Function: Support upper body, allow bending/lifting, protect cauda equina
│
├─ General Anatomy
│   ├─ Vertebral body:
│   │   ├─ Large: Largest vertebral bodies (increasing weight-bearing)
│   │   ├─ Kidney-shaped: Wider transversely than AP
│   │   ├─ Size progression: L1 < L2 < L3 < L4 < L5 (L5 largest)
│   │   ├─ Endplates: Concave (allow disc bulging)
│   │   └─ Trabecular bone: Dense (load-bearing)
│   │
│   ├─ Pedicles:
│   │   ├─ Short, thick, strong
│   │   ├─ Project posteriorly
│   │   ├─ Pedicle screw fixation: Most common site for instrumentation ⭐
│   │   │   ├─ Trajectory: Straight back-to-front (easiest to instrument)
│   │   │   ├─ Size: Diameter 8-12 mm (accommodates large screws)
│   │   │   └─ Safety: Medial wall breach → nerve root injury, lateral → vascular
│   │   │
│   │   └─ Pars interarticularis: Between superior and inferior facets (site of spondylolysis)
│   │
│   ├─ Transverse processes:
│   │   ├─ Long, thin
│   │   ├─ L3: Longest (lumbar plexus landmark)
│   │   ├─ Psoas muscle: Attaches here
│   │   └─ No costal facets: (unlike thoracic)
│   │
│   ├─ Articular processes (facet joints):
│   │   ├─ Superior facets: Face posteromedially (~45° to sagittal plane)
│   │   ├─ Inferior facets: Face anterolaterally
│   │   ├─ "Sagittal" orientation: Allows flexion-extension, limits rotation
│   │   ├─ Facet tropism: Asymmetric facet orientation (can predispose to instability)
│   │   └─ Clinical: Facet arthropathy common cause of low back pain
│   │
│   ├─ Lamina:
│   │   ├─ Thick, broad
│   │   └─ Laminectomy: Easier than thoracic (less overlap)
│   │
│   ├─ Spinous process:
│   │   ├─ Short, broad, rectangular
│   │   ├─ Horizontal orientation: (vs downward in thoracic)
│   │   ├─ Palpable: Easier to palpate individual levels
│   │   └─ Interspinous space: Wider (easier epidural access)
│   │
│   └─ Spinal canal:
│       ├─ Triangular: Wider than thoracic
│       ├─ Diameter: ~15-25 mm (more room than thoracic)
│       ├─ Contents: Cauda equina (nerve roots L2-S5, coccygeal)
│       │   └─ No spinal cord: Cord ends at L1-L2 (conus medullaris)
│       │
│       └─ Stenosis:
│           ├─ Central stenosis: <12 mm (AP diameter), <100 mm² (area)
│           └─ Foraminal stenosis: <4 mm (height of foramen)
│
├─ Regional Variations
│   ├─ L5:
│   │   ├─ Largest: Vertebral body
│   │   ├─ Transverse process: Often fused to sacrum (transitional anatomy)
│   │   ├─ L5-S1 disc: Largest disc, most weight-bearing
│   │   └─ Most common: Site of spondylolisthesis (isthmic)
│   │
│   └─ Transitional vertebrae:
│       ├─ Sacralization of L5: L5 partially or fully fused to sacrum
│       ├─ Lumbarization of S1: S1 not fused, appears as 6th lumbar vertebra
│       └─ Clinical: Can alter biomechanics, confuse level counting
│
├─ Lumbar Intervertebral Discs
│   ├─ Largest discs: L4-L5 and L5-S1 (greatest load)
│   ├─ Height: ~10-12 mm (thickest in spine)
│   ├─ Structure:
│   │   ├─ Nucleus pulposus: Gelatinous, high water content (80% young → 70% adult)
│   │   ├─ Annulus fibrosus: 15-25 concentric layers
│   │   │   ├─ Outer annulus: Type I collagen (strong)
│   │   │   ├─ Inner annulus: Type II collagen (more elastic)
│   │   │   └─ Posterior annulus: Thinner, weaker (herniation site)
│   │   │
│   │   └─ Endplate: Hyaline cartilage (nutrient diffusion)
│   │
│   ├─ Function:
│   │   ├─ Load distribution: Nucleus acts as hydraulic cushion
│   │   ├─ Motion: Allows flexion, extension, lateral bending
│   │   └─ Shock absorption: Viscoelastic properties
│   │
│   └─ Degeneration:
│       ├─ Desiccation: Loss of water content, proteoglycans
│       ├─ Loss of height: Reduced disc space
│       ├─ Annular tears: Radial, circumferential (painful)
│       └─ Progression: Degenerative disc disease → spondylosis → stenosis
│
├─ Lumbar Ligaments
│   ├─ Anterior longitudinal ligament (ALL): Stronger in lumbar than cervical
│   ├─ Posterior longitudinal ligament (PLL): Narrows as descends (weakest lateral)
│   ├─ Ligamentum flavum:
│   │   ├─ Thickest: In lumbar spine (~5-6 mm, can be >10 mm if hypertrophy)
│   │   ├─ Hypertrophy: Common with age → central stenosis
│   │   └─ Surgical: Must be removed during decompression (laminectomy)
│   │
│   ├─ Interspinous ligaments: Strong in lumbar
│   ├─ Supraspinous ligament: From C7 to sacrum
│   └─ Iliolumbar ligaments: L5 transverse process → iliac crest (stabilize L5-S1)
│
├─ Lumbar Spine Stability
│   ├─ 3-column model (Denis) ⭐:
│   │   ├─ Anterior column: ALL, anterior 2/3 vertebral body + disc
│   │   ├─ Middle column: PLL, posterior 1/3 vertebral body + disc
│   │   ├─ Posterior column: Pedicles, facets, lamina, spinous process, posterior ligaments
│   │   │
│   │   ├─ Stable: 1 column disrupted
│   │   ├─ Unstable: 2+ columns disrupted
│   │   └─ Most useful: For thoracolumbar fractures
│   │
│   └─ Instability criteria (White-Panjabi):
│       ├─ Translation: >4.5 mm (on flexion-extension)
│       ├─ Angulation: >15° at L1-L4, >20° at L4-L5, >25° at L5-S1
│       └─ Clinical correlation needed: Pain, neurologic deficit
│
└─ Clinical Pathology (Lumbar Spine) ⭐⭐⭐
    ├─ Lumbar disc herniation:
    │   ├─ Most common: Spine pathology requiring surgery
    │   ├─ Most common levels:
    │   │   ├─ L4-L5: 45-50% (most common)
    │   │   ├─ L5-S1: 40-45% (second most common)
    │   │   └─ L3-L4: 5-10%
    │   │
    │   ├─ Classification:
    │   │   ├─ Bulge: Annulus intact, disc extends beyond endplate (>50% circumference)
    │   │   ├─ Protrusion: Annular tear, nucleus extends through (base wider than neck)
    │   │   ├─ Extrusion: Fragment extends through annulus (base narrower than neck)
    │   │   ├─ Sequestration: Fragment completely separated from disc
    │   │   │
    │   │   └─ Location:
    │   │       ├─ Central: Midline (can compress multiple roots or cauda equina)
    │   │       ├─ Paracentral: Lateral to midline (most common - compresses traversing root)
    │   │       ├─ Foraminal: In foramen (compresses exiting root)
    │   │       └─ Far lateral (extraforaminal): Outside foramen (rare, compresses exiting root)
    │   │
    │   ├─ Nerve root anatomy (KEY FOR LOCALIZATION):
    │   │   ├─ Nerve root numbering: Exits BELOW corresponding vertebra
    │   │   │   └─ Example: L4 nerve root exits L4-L5 foramen (below L4 pedicle)
    │   │   │
    │   │   ├─ Root compression patterns:
    │   │   │   ├─ Paracentral herniation: Compresses TRAVERSING root (root going to next level)
    │   │   │   │   └─ Example: L4-L5 paracentral herniation → compresses L5 root
    │   │   │   │
    │   │   │   └─ Foraminal herniation: Compresses EXITING root (root exiting at that level)
    │   │   │       └─ Example: L4-L5 foraminal herniation → compresses L4 root
    │   │   │
    │   │   └─ Must distinguish: To predict symptoms correctly ⭐
    │   │
    │   ├─ Clinical presentation:
    │   │   ├─ Radiculopathy:
    │   │   │   ├─ Pain: Radiating to leg (dermatomal - sharp, burning, electric)
    │   │   │   ├─ Worse with: Sitting, bending forward, Valsalva
    │   │   │   ├─ Better with: Standing, lying, walking
    │   │   │   ├─ Sensory: Numbness, paresthesias (dermatomal)
    │   │   │   ├─ Motor: Weakness (myotomal)
    │   │   │   └─ Reflexes: Diminished or absent
    │   │   │
    │   │   └─ Axial back pain: Discogenic pain (annular tear)
    │   │
    │   ├─ Level-specific radiculopathy ⭐⭐:
    │   │   ├─ L3 nerve root (L2-L3 disc):
    │   │   │   ├─ Motor: Hip flexion (iliopsoas), knee extension (quadriceps)
    │   │   │   ├─ Sensory: Anterior thigh
    │   │   │   └─ Reflex: None specific
    │   │   │
    │   │   ├─ L4 nerve root (L3-L4 disc):
    │   │   │   ├─ Motor: Knee extension (quadriceps), ankle dorsiflexion (tibialis anterior)
    │   │   │   ├─ Sensory: Medial leg, medial malleolus
    │   │   │   └─ Reflex: Patellar (knee jerk)
    │   │   │
    │   │   ├─ L5 nerve root (L4-L5 disc) - MOST COMMON:
    │   │   │   ├─ Motor: Ankle dorsiflexion (tibialis anterior), great toe extension (EHL), foot eversion
    │   │   │   ├─ Sensory: Dorsum of foot, first web space
    │   │   │   ├─ Reflex: None specific (medial hamstring)
    │   │   │   └─ Gait: Footdrop (cannot heel walk)
    │   │   │
    │   │   └─ S1 nerve root (L5-S1 disc) - SECOND MOST COMMON:
    │   │       ├─ Motor: Ankle plantar flexion (gastrocnemius-soleus), foot eversion (peroneals)
    │   │       ├─ Sensory: Lateral foot, small toe
    │   │       ├─ Reflex: Achilles (ankle jerk)
    │   │       └─ Gait: Cannot toe walk
    │   │
    │   ├─ Special syndromes:
    │   │   ├─ Cauda equina syndrome (CES) ⭐⭐⭐:
    │   │   │   ├─ Definition: Compression of multiple lumbosacral nerve roots (L2-S5)
    │   │   │   ├─ Cause: Large central disc herniation, tumor, abscess, hematoma
    │   │   │   │
    │   │   │   ├─ Clinical presentation (EMERGENCY):
    │   │   │   │   ├─ Bilateral leg pain: Radicular (often asymmetric initially)
    │   │   │   │   ├─ Saddle anesthesia: Perineum, buttocks (S2-S5 dermatomes) ⭐
    │   │   │   │   ├─ Bowel dysfunction: Fecal incontinence, loss of rectal tone
    │   │   │   │   ├─ Bladder dysfunction:
    │   │   │   │   │   ├─ Urinary retention (most common - painless, overflow incontinence)
    │   │   │   │   │   ├─ Incontinence: True incontinence (late)
    │   │   │   │   │   └─ Post-void residual: >200 mL (detrusor areflexia)
    │   │   │   │   ├─ Sexual dysfunction: Impotence, loss of genital sensation
    │   │   │   │   └─ Lower extremity: Weakness (bilateral), reduced reflexes
    │   │   │   │
    │   │   │   ├─ Workup:
    │   │   │   │   ├─ MRI emergent: Large central disc herniation
    │   │   │   │   ├─ Post-void residual: Bladder scan (>200 mL = retention)
    │   │   │   │   └─ Rectal exam: Tone, perianal sensation
    │   │   │   │
    │   │   │   └─ Treatment:
    │   │   │       ├─ Surgical decompression: URGENT (<48 hours for best outcomes)
    │   │   │       │   ├─ Timing: Sooner better (within 24 hours optimal)
    │   │   │       │   ├─ Approach: Laminectomy + discectomy (wide decompression)
    │   │   │       │   └─ Outcomes:
    │   │   │       │       ├─ Pain: 80-90% improve
    │   │   │       │       ├─ Motor: 60-80% improve
    │   │   │       │       ├─ Bladder: 50-70% recover (worse if delayed >48 hours)
    │   │   │       │       └─ Sexual: Poorest recovery (30-50%)
    │   │   │       │
    │   │   │       └─ Poor prognostic factors: Delayed surgery (>48 hours), complete paralysis, complete anesthesia
    │   │   │
    │   │   └─ Conus medullaris syndrome:
    │   │       ├─ Definition: Injury to conus medullaris (terminal spinal cord at L1-L2)
    │   │       ├─ Differs from CES: More sudden onset, more symmetric, more severe
    │   │       ├─ Presentation: Saddle anesthesia, areflexic bladder/bowel, UMN + LMN signs
    │   │       └─ Treatment: Same as CES (urgent decompression)
    │   │
    │   ├─ Imaging:
    │   │   ├─ MRI: Gold standard (disc, nerve root compression, Modic changes in endplates)
    │   │   ├─ CT: If MRI contraindicated
    │   │   └─ X-rays: Limited utility (assess alignment, instability)
    │   │
    │   ├─ Natural history:
    │   │   ├─ Spontaneous resorption: 50-70% improve within 6 weeks (body reabsorbs disc fragment)
    │   │   ├─ Sequestered fragments: Better resorption than contained herniations
    │   │   └─ Pain resolution: 80-90% at 1 year (conservative treatment)
    │   │
    │   └─ Treatment:
    │       ├─ Conservative (first-line):
    │       │   ├─ NSAIDs: Anti-inflammatory, analgesia
    │       │   ├─ Activity modification: Avoid heavy lifting, prolonged sitting
    │       │   ├─ Physical therapy: After acute phase (2-4 weeks)
    │       │   ├─ Epidural steroid injection: If refractory to conservative treatment (50-70% improve)
    │       │   └─ Duration: 6-12 weeks trial
    │       │
    │       └─ Surgical (microdiscectomy):
    │       │   ├─ Indications:
    │       │   │   ├─ Absolute: Cauda equina syndrome, progressive motor deficit
    │       │   │   ├─ Relative: Severe refractory radiculopathy (>6-12 weeks conservative treatment)
    │       │   │   └─ Elective: Persistent pain affecting quality of life
    │       │   │
    │       │   ├─ Technique:
    │       │   │   ├─ Approach: Posterior midline or paramedian
    │       │   │   ├─ Laminotomy: Small bone removal (preserve facet joint)
    │       │   │   ├─ Flavectomy: Remove ligamentum flavum
    │       │   │   ├─ Root identification: Identify and protect nerve root
    │       │   │   ├─ Discectomy: Remove herniated fragment + loose disc material
    │       │   │   └─ Minimally invasive: Tubular retractors (faster recovery)
    │       │   │
    │       │   ├─ Success rate:
    │       │   │   ├─ Leg pain relief: 85-95% at 1 year
    │       │   │   ├─ Back pain: 60-70% (less predictable)
    │       │   │   └─ Patient satisfaction: 80-90%
    │       │   │
    │       │   └─ Complications:
    │       │       ├─ Dural tear: 3-10% (usually repairable)
    │       │       ├─ Nerve root injury: <1%
    │       │       ├─ Infection: 1-3%
    │       │       ├─ Recurrent herniation: 5-15% (same level)
    │       │       └─ Failed back surgery syndrome: 10-20% (chronic pain despite surgery)
    │       │
    │       └─ Long-term outcomes:
    │           ├─ Surgery vs conservative at 1 year: Similar outcomes
    │           ├─ Surgery: Faster pain relief (weeks vs months)
    │           └─ Patient selection: Key to success (correct diagnosis, appropriate indications)
    │
    ├─ Lumbar spinal stenosis ⭐⭐⭐:
    │   ├─ Definition: Narrowing of spinal canal/neural foramina → nerve root/cauda equina compression
    │   │
    │   ├─ Classification by location:
    │   │   ├─ Central stenosis:
    │   │   │   ├─ Narrowing of central spinal canal
    │   │   │   ├─ Affects cauda equina/multiple roots
    │   │   │   └─ Imaging: AP diameter <10 mm (normal 15-25 mm)
    │   │   │
    │   │   ├─ Lateral recess stenosis:
    │   │   │   ├─ Narrowing of lateral recess (between pedicle and superior articular process)
    │   │   │   ├─ Affects exiting root (before foramen)
    │   │   │   └─ Imaging: Lateral recess height <3 mm (normal 5 mm)
    │   │   │
    │   │   └─ Foraminal stenosis:
    │   │       ├─ Narrowing of neural foramen
    │   │       ├─ Affects exiting root (in foramen)
    │   │       └─ Imaging: Foraminal height <15 mm (normal 20-25 mm)
    │   │
    │   ├─ Etiology (degenerative cascade):
    │   │   ├─ Disc degeneration: Loss of height → abnormal loading
    │   │   ├─ Facet joint arthropathy: Hypertrophy → posterior compression
    │   │   ├─ Ligamentum flavum hypertrophy: Thickening (>4 mm) → posterior compression
    │   │   ├─ Disc bulging: Anterior compression
    │   │   └─ Spondylolisthesis: Subluxation → combined stenosis
    │   │
    │   ├─ Clinical presentation:
    │   │   ├─ Neurogenic claudication ⭐:
    │   │   │   ├─ Leg pain/weakness with walking/standing (extension → narrows canal)
    │   │   │   ├─ Improved with sitting/flexion (flexion → opens canal)
    │   │   │   ├─ Bilateral symptoms (70%) > unilateral
    │   │   │   ├─ "Shopping cart sign": Leaning forward on cart relieves symptoms
    │   │   │   └─ Symptom duration: Minutes to hours (vs vascular claudication: seconds to minutes)
    │   │   │
    │   │   ├─ Differentiation from vascular claudication:
    │   │   │   ├─ Neurogenic: Positional (worse with extension), improved with sitting, normal pulses
    │   │   │   └─ Vascular: Activity-related (not positional), improved with standing still, diminished pulses
    │   │   │
    │   │   ├─ Radiculopathy: Single/multiple root symptoms (pain, numbness, weakness)
    │   │   └─ Cauda equina syndrome: Rare (severe central stenosis)
    │   │
    │   ├─ Physical examination:
    │   │   ├─ Wide-based gait: Compensatory for instability
    │   │   ├─ Limited lumbar extension: Reproduces symptoms
    │   │   ├─ Variable motor/sensory deficits: Depending on roots involved
    │   │   ├─ Reflexes: Often normal (vs disc herniation with reflex loss)
    │   │   └─ Straight leg raise: Usually negative (vs disc herniation usually positive)
    │   │
    │   ├─ Imaging:
    │   │   ├─ MRI (gold standard):
    │   │   │   ├─ T2 axial: Best for canal/foraminal stenosis
    │   │   │   ├─ T2 sagittal: Best for multilevel stenosis
    │   │   │   └─ Grading (Schizas classification):
    │   │   │       ├─ Grade A: No stenosis (CSF visible around roots)
    │   │   │       ├─ Grade B: Moderate stenosis (some CSF obliterated)
    │   │   │       ├─ Grade C: Severe stenosis (no CSF visible, roots distinguishable)
    │   │   │       └─ Grade D: Extreme stenosis (no CSF, roots not distinguishable)
    │   │   │
    │   │   └─ CT myelogram: If MRI contraindicated (pacemaker)
    │   │       └─ Better for bony stenosis than MRI
    │   │
    │   ├─ Natural history:
    │   │   ├─ Nonprogressive in 70%: Stable or improvement
    │   │   ├─ Progressive symptoms: 30% worsen over 1-2 years
    │   │   └─ Spontaneous improvement: 15-20% (rare)
    │   │
    │   └─ Treatment:
    │       ├─ Conservative (initial management 3-6 months):
    │       │   ├─ NSAIDs: Anti-inflammatory (modest benefit)
    │       │   ├─ Physical therapy: Flexion-based exercises (open canal)
    │       │   ├─ Epidural steroid injection: Transforaminal or interlaminar
    │       │   │   ├─ Success rate: 50-70% short-term relief (<3 months)
    │       │   │   └─ Long-term relief: 25-40%
    │       │   ├─ Activity modification: Avoid extension activities
    │       │   └─ Walking aids: Cane/walker with forward lean
    │       │
    │       └─ Surgical decompression ⭐:
    │           ├─ Indications:
    │           │   ├─ Failed conservative treatment (3-6 months)
    │           │   ├─ Progressive neurological deficit
    │           │   ├─ Severe disability affecting quality of life
    │           │   └─ Cauda equina syndrome (emergency)
    │           │
    │           ├─ Technique: Decompressive laminectomy ± fusion
    │           │   ├─ Laminectomy alone (no instability):
    │           │   │   ├─ Remove lamina, hypertrophied ligamentum flavum, facet hypertrophy
    │           │   │   ├─ Preserve facet joints if possible (<50% resection → maintains stability)
    │           │   │   └─ Bilateral decompression via unilateral approach (preserve midline structures)
    │           │   │
    │           │   └─ Laminectomy + fusion (with instability):
    │           │       ├─ Indications for fusion:
    │           │       │   ├─ Spondylolisthesis (grade II or higher)
    │           │       │   ├─ Iatrogenic instability (>50% facet resection)
    │           │       │   ├─ Degenerative scoliosis (>20° curve)
    │           │       │   └─ Recurrent stenosis
    │           │       │
    │           │       └─ Fusion technique:
    │           │           ├─ Posterolateral fusion: Bone graft on transverse processes
    │           │           ├─ Posterior lumbar interbody fusion (PLIF): Cage in disc space
    │           │           ├─ Transforaminal lumbar interbody fusion (TLIF): Unilateral approach
    │           │           └─ Instrumentation: Pedicle screws + rods (provides immediate stability)
    │           │
    │           ├─ Outcomes (SPORT trial - multicenter RCT):
    │           │   ├─ Pain relief: 70-80% significant improvement at 2 years
    │           │   ├─ Function: 60-70% improvement in walking distance/tolerance
    │           │   ├─ Patient satisfaction: 70-80%
    │           │   └─ Surgery vs conservative: Surgery better at all time points (4 years)
    │           │
    │           ├─ Complications:
    │           │   ├─ Dural tear: 10-20% (higher than discectomy due to adhesions)
    │           │   ├─ Nerve root injury: 1-3%
    │           │   ├─ Infection: 2-5%
    │           │   ├─ Instability: 5-15% (if >50% facet resected → requires fusion)
    │           │   ├─ Adjacent segment disease: 10-15% at 5 years (if fused)
    │           │   └─ Hematoma: 1-3% (epidural hematoma → reoperation)
    │           │
    │           └─ Predictors of poor outcome:
    │               ├─ Obesity: BMI >30 (worse outcomes)
    │               ├─ Depression: Worse pain outcomes
    │               ├─ Workers' compensation: Lower satisfaction
    │               ├─ Smoking: Higher complication rates
    │               └─ Multilevel disease: More complex recovery
    │
    ├─ Spondylolisthesis ⭐⭐⭐:
    │   ├─ Definition: Forward subluxation of vertebra on vertebra below
    │   │
    │   ├─ Grading (Meyerding classification - % of vertebral body translation):
    │   │   ├─ Grade I: 0-25% (most common, usually asymptomatic)
    │   │   ├─ Grade II: 25-50% (symptomatic in 50%)
    │   │   ├─ Grade III: 50-75% (symptomatic in 80%)
    │   │   ├─ Grade IV: 75-100% (symptomatic in 95%)
    │   │   └─ Grade V (spondyloptosis): >100% (complete subluxation, rare)
    │   │
    │   ├─ Classification by etiology (Wiltse classification):
    │   │   ├─ Type I: Dysplastic (congenital):
    │   │       ├─ Dysplastic posterior arch (inadequate facets)
    │   │       ├─ Usually L5-S1 (90%)
    │   │       ├─ Presents in childhood/adolescence
    │   │       └─ High risk of progression
    │   │   │
    │   │   ├─ Type II: Isthmic (most common - 50%) ⭐:
    │   │   │   ├─ Defect in pars interarticularis ("spondylolysis")
    │   │   │   ├─ Subtypes:
    │   │   │   │   ├─ IIA: Lytic (fatigue fracture) - most common
    │   │   │   │   ├─ IIB: Elongated pars (healed stress fracture)
    │   │   │   │   └─ IIC: Acute fracture
    │   │   │   │
    │   │   │   ├─ Etiology: Repetitive hyperextension (gymnastics, football linemen, weightlifting)
    │   │   │   ├─ Location: L5-S1 (85%), L4-L5 (10%)
    │   │   │   ├─ Incidence: 6% general population, 30% athletes (hyperextension sports)
    │   │   │   ├─ "Scottie dog" sign on oblique X-ray: "Collar" on dog's neck = pars defect
    │   │   │   └─ Bilateral pars defect → allows spondylolisthesis
    │   │   │
    │   │   ├─ Type III: Degenerative (40%) ⭐:
    │   │   │   ├─ Facet joint arthropathy + disc degeneration → instability
    │   │   │   ├─ Location: L4-L5 (70%) > L3-L4 (25%) (NOT L5-S1 - facets too vertical)
    │   │   │   ├─ Pars interarticularis intact
    │   │   │   ├─ Age: >50 years (elderly)
    │   │   │   ├─ Gender: Female:Male = 6:1 (ligamentous laxity)
    │   │   │   └─ Grade: Usually I or II (rarely progresses beyond 30%)
    │   │   │
    │   │   ├─ Type IV: Traumatic:
    │   │   │   ├─ Acute fracture of posterior elements (NOT pars)
    │   │   │   └─ High-energy trauma
    │   │   │
    │   │   ├─ Type V: Pathological:
    │   │   │   ├─ Bone disease (tumor, infection, Paget's disease)
    │   │   │   └─ Weakened posterior elements
    │   │   │
    │   │   └─ Type VI: Iatrogenic:
    │   │       ├─ Post-surgical instability
    │   │       └─ Excessive posterior element resection
    │   │
    │   ├─ Clinical presentation:
    │   │   ├─ Asymptomatic: 50-70% (incidental finding)
    │   │   ├─ Low back pain: Mechanical (worse with activity, improved with rest)
    │   │   ├─ Radiculopathy: Root compression (from subluxation/stenosis)
    │   │   │   └─ Isthmic L5-S1: L5 root compressed (at pars defect site)
    │   │   ├─ Hamstring tightness: Compensatory for lumbosacral kyphosis
    │   │   ├─ Palpable "step-off": Prominent spinous process above (L4 on L5)
    │   │   └─ Neurogenic claudication: If associated stenosis (degenerative type)
    │   │
    │   ├─ Imaging:
    │   │   ├─ X-ray (initial):
    │   │   │   ├─ Lateral: Shows % slip (Meyerding grade)
    │   │   │   ├─ Oblique: "Scottie dog" with collar (pars defect - isthmic type)
    │   │   │   └─ Flexion-extension: Assess instability (>3 mm motion or >10° angulation)
    │   │   │
    │   │   ├─ CT: Best for pars defect visualization
    │   │   │   └─ "Napoleon hat" sign: Sagittal view of pars defect
    │   │   │
    │   │   ├─ MRI: Best for nerve root compression, disc degeneration
    │   │   │   └─ Shows stenosis, disc pathology, nerve root compression
    │   │   │
    │   │   └─ SPECT bone scan: If pars defect seen on CT/X-ray
    │   │       ├─ "Hot" (increased uptake): Acute/healing fracture (may heal with conservative)
    │   │       └─ "Cold" (no uptake): Chronic/established defect (will not heal)
    │   │
    │   ├─ Natural history:
    │   │   ├─ Progression risk (isthmic):
    │   │   │   ├─ Grade I: 25% progress (mostly during growth)
    │   │   │   ├─ Grade II-IV: 50% progress
    │   │   │   ├─ Age <15 years: Higher risk
    │   │   │   └─ After skeletal maturity: Rarely progresses
    │   │   │
    │   │   └─ Progression risk (degenerative):
    │   │       └─ Rarely progresses beyond grade II (facets prevent further slip)
    │   │
    │   └─ Treatment:
    │       ├─ Conservative (initial management):
    │       │   ├─ Activity modification: Avoid hyperextension
    │       │   ├─ Physical therapy: Core strengthening, hamstring stretching
    │       │   ├─ Bracing: For acute isthmic (if SPECT "hot")
    │       │   │   ├─ Rigid TLSO: Immobilize for 3-6 months
    │       │   │   └─ Success rate: 50-80% healing in acute pars fracture
    │       │   └─ NSAIDs: Symptomatic relief
    │       │
    │       └─ Surgical fusion ⭐:
    │           ├─ Indications:
    │           │   ├─ Failed conservative treatment (6 months)
    │           │   ├─ Progressive neurological deficit
    │           │   ├─ Grade III or higher (high-grade slip)
    │           │   ├─ Progressive slip (>50% increase on serial X-rays)
    │           │   └─ Severe disability/pain affecting quality of life
    │           │
    │           ├─ Technique options:
    │           │   ├─ Pars repair (isthmic, low-grade, young patients):
    │           │   │   ├─ Direct pars repair: Screws/wiring across pars defect
    │           │   │   ├─ Preserves motion segment (no fusion)
    │           │   │   ├─ Indications: Grade I, minimal disc degeneration, age <25 years
    │           │   │   └─ Success rate: 80-90% (if appropriate selection)
    │           │   │
    │           │   ├─ Posterolateral fusion (PLF):
    │           │   │   ├─ Bone graft on transverse processes + pedicle screw fixation
    │           │   │   ├─ Fusion rate: 80-85% (without instrumentation: 50-60%)
    │           │   │   └─ Requires intact anterior column (disc) for support
    │           │   │
    │           │   ├─ Posterior lumbar interbody fusion (PLIF):
    │           │   │   ├─ Cage insertion via posterior approach (bilateral laminectomy)
    │           │   │   ├─ Advantages: Direct decompression, anterior column support
    │           │   │   ├─ Fusion rate: 90-95%
    │           │   │   └─ Disadvantage: Nerve root retraction required
    │           │   │
    │           │   └─ Transforaminal lumbar interbody fusion (TLIF):
    │           │       ├─ Cage insertion via unilateral transforaminal approach
    │           │       ├─ Advantages: Less nerve retraction, unilateral approach
    │           │       ├─ Fusion rate: 90-95%
    │           │       └─ Most popular technique currently ⭐
    │           │
    │           ├─ Reduction vs in situ fusion:
    │           │   ├─ In situ fusion: Fuse without reducing slip (most common)
    │           │   │   ├─ Safer (less nerve root tension)
    │           │   │   └─ Acceptable outcomes for grade I-II
    │           │   │
    │           │   └─ Reduction: Reduce slip before fusion
    │           │       ├─ Indications: High-grade slip (III-V), sagittal imbalance
    │           │       ├─ Risk: L5 nerve root injury (10-30% neurological complications)
    │           │       └─ Technique: Gradual reduction with pedicle screws
    │           │
    │           ├─ Outcomes:
    │           │   ├─ Pain relief: 70-85% significant improvement
    │           │   ├─ Function: 70-80% improvement
    │           │   ├─ Patient satisfaction: 75-85%
    │           │   └─ Fusion rate: 85-95% (with instrumentation)
    │           │
    │           └─ Complications:
    │               ├─ Pseudarthrosis (nonunion): 5-15% (higher if smoker, no instrumentation)
    │               ├─ Adjacent segment disease: 10-20% at 5 years
    │               ├─ Nerve root injury: 1-5% (higher with reduction: 10-30%)
    │               ├─ Dural tear: 5-10%
    │               ├─ Infection: 2-5%
    │               └─ Hardware failure: 1-5% (screw breakage/loosening)
    │
    └─ SACRUM & COCCYX:
        ├─ Incidence: 9 vertebrae fused (5 sacral + 4 coccygeal typically)
        │
        ├─ Sacrum (S1-S5 fused) ⭐⭐:
        │   ├─ Shape: Triangular bone (base superior, apex inferior)
        │   │
        │   ├─ Anterior (pelvic) surface:
        │   │   ├─ Smooth, concave surface
        │   │   ├─ 4 pairs of anterior sacral foramina:
        │   │   │   ├─ Transmit: Anterior rami of S1-S4 spinal nerves
        │   │   │   └─ Note: S1 largest, decreasing size S2-S4
        │   │   │
        │   │   └─ Transverse ridges: Fusion lines between sacral vertebrae
        │   │
        │   ├─ Posterior (dorsal) surface:
        │   │   ├─ Rough, convex surface
        │   │   ├─ Median sacral crest: Fused spinous processes (4 tubercles)
        │   │   ├─ Intermediate sacral crest: Fused articular processes (paired)
        │   │   ├─ Lateral sacral crest: Fused transverse processes (paired)
        │   │   ├─ 4 pairs of posterior sacral foramina:
        │   │   │   ├─ Transmit: Posterior rami of S1-S4 spinal nerves
        │   │   │   └─ Smaller than anterior foramina
        │   │   │
        │   │   └─ Sacral hiatus:
        │   │       ├─ Deficiency in posterior wall (lamina of S5 not fused)
        │   │       ├─ Covered by sacrococcygeal ligament
        │   │       ├─ Bordered by sacral cornua (inferior articular processes of S5)
        │   │       └─ Clinical: Access point for caudal epidural anesthesia ⭐
        │   │
        │   ├─ Sacral canal:
        │   │   ├─ Continuation of spinal canal
        │   │   ├─ Triangular in cross-section
        │   │   ├─ Contains:
        │   │   │   ├─ Cauda equina (proximal): L2-S1 nerve roots
        │   │   │   ├─ Sacral nerve roots: S2-S5 (exiting via foramina)
        │   │   │   ├─ Filum terminale: From conus medullaris (L1-L2) to coccyx
        │   │   │   └─ Dural sac ends at S2 level (or lower) ⭐
        │   │   │
        │   │   └─ Variation: Dural sac can end S1-S3 (important for caudal epidural)
        │   │
        │   ├─ Sacral promontory:
        │   │   ├─ Anterior projecting ridge at L5-S1 junction
        │   │   ├─ Clinical landmark: Pelvic brim (divides true/false pelvis)
        │   │   └─ Surgical: Anterior lumbar interbody fusion (ALIF) landmark
        │   │
        │   ├─ Sacroiliac joint (SI joint) ⭐⭐:
        │   │   ├─ Type: Synovial joint (anterior 1/3) + syndesmosis (posterior 2/3)
        │   │   ├─ Articulation: Auricular surface of sacrum with ilium
        │   │   ├─ Motion: Minimal (2-4° rotation, 1-2 mm translation)
        │   │   │
        │   │   ├─ Ligaments (strongest in body):
        │   │   │   ├─ Anterior sacroiliac ligament: Weak, thin
        │   │   │   ├─ Posterior sacroiliac ligament: Strong (dorsal/ventral components)
        │   │   │   ├─ Interosseous sacroiliac ligament: Strongest ligament in body ⭐
        │   │   │   ├─ Sacrotuberous ligament: Sacrum → ischial tuberosity
        │   │   │   └─ Sacrospinous ligament: Sacrum → ischial spine
        │   │   │
        │   │   ├─ Innervation: L4-S3 (posterior rami) - complex, variable
        │   │   │
        │   │   └─ Clinical: SI joint dysfunction (low back pain source in 15-30%)
        │   │
        │   ├─ Sacral fractures:
        │   │   ├─ Classification (Denis zones):
        │   │   │   ├─ Zone I: Sacral ala (lateral to foramina)
        │   │   │   │   ├─ Most common zone (50%)
        │   │   │   │   ├─ Mechanism: Lateral compression, avulsion (ligaments)
        │   │   │   │   └─ Neurological injury: Rare (<5%)
        │   │   │   │
        │   │   │   ├─ Zone II: Foraminal region (through foramina)
        │   │   │   │   ├─ Incidence: 35%
        │   │   │   │   ├─ Neurological injury: 28% (L5 or sacral roots)
        │   │   │   │   └─ Bowel/bladder dysfunction: 10-15%
        │   │   │   │
        │   │   │   └─ Zone III: Central canal (medial to foramina)
        │   │   │       ├─ Incidence: 15% (least common but most severe)
        │   │   │       ├─ Mechanism: Axial load (fall from height)
        │   │   │       ├─ Neurological injury: 57% (highest rate) ⭐
        │   │   │       ├─ Cauda equina syndrome: Common (40%)
        │   │   │       └─ Bowel/bladder dysfunction: 30-50%
        │   │   │
        │   │   ├─ Imaging:
        │   │   │   ├─ X-ray: Often missed (complex anatomy, overlapping structures)
        │   │   │   ├─ CT: Best for fracture detection and classification ⭐
        │   │   │   └─ MRI: Best for neural injury (nerve roots, dural tears)
        │   │   │
        │   │   └─ Treatment:
        │   │       ├─ Conservative (stable, no neural injury):
        │   │       │   ├─ Weight bearing as tolerated
        │   │       │   ├─ Pain management: NSAIDs, opioids
        │   │       │   └─ Duration: 6-12 weeks for healing
        │   │       │
        │   │       └─ Surgical (unstable, neural injury):
        │   │           ├─ Indications:
        │   │           │   ├─ Unstable fracture (vertical shear pattern)
        │   │           │   ├─ Progressive neurological deficit
        │   │           │   ├─ Cauda equina syndrome
        │   │           │   └─ Displacement >1 cm
        │   │           │
        │   │           ├─ Technique:
        │   │           │   ├─ Posterior approach: Lumbopelvic fixation (L4/5 to ilium)
        │   │           │   ├─ Iliosacral screws: Percutaneous or open
        │   │           │   └─ Decompression: If canal compromise
        │   │           │
        │   │           └─ Outcomes: Neurological recovery in 50-70% (if treated early)
        │   │
        │   └─ Sacral nerve root anatomy (S1-S5):
        │       ├─ S1 nerve root:
        │       │   ├─ Motor: Gastrocnemius, soleus (ankle plantarflexion)
        │       │   ├─ Sensory: Lateral foot, posterior leg
        │       │   └─ Reflex: Achilles (ankle jerk)
        │       │
        │       ├─ S2 nerve root:
        │       │   ├─ Motor: Intrinsic foot muscles, hamstrings (contribution)
        │       │   └─ Sensory: Posterior thigh, posterior leg
        │       │
        │       ├─ S3 nerve root:
        │       │   ├─ Motor: Bladder detrusor, anal sphincter (external)
        │       │   └─ Sensory: Perineum (medial)
        │       │
        │       ├─ S4 nerve root:
        │       │   ├─ Motor: Bladder, bowel (parasympathetic)
        │       │   └─ Sensory: Perineum, perianal
        │       │
        │       └─ S5 nerve root:
        │           ├─ Motor: Anal sphincter (external)
        │           └─ Sensory: Perianal skin
        │
        └─ Coccyx (Co1-Co4 fused) ⭐:
            ├─ Shape: Small triangular bone (vestigial tail)
            ├─ Composition: 3-5 rudimentary vertebrae (usually 4)
            ├─ Articulation: Sacrococcygeal joint (fibrocartilaginous)
            │
            ├─ Attachments:
            │   ├─ Gluteus maximus: Superficial fibers
            │   ├─ Levator ani: Pelvic floor muscle
            │   ├─ Sacrococcygeal ligaments: Anterior/posterior/lateral
            │   ├─ Sacrospinous ligament: Lateral support
            │   └─ Filum terminale: Terminal attachment of spinal cord
            │
            ├─ Clinical conditions:
            │   ├─ Coccydynia (coccyx pain):
            │   │   ├─ Etiology: Trauma (fall), childbirth, repetitive strain
            │   │   ├─ Presentation: Pain with sitting, standing from sitting
            │   │   ├─ Examination: Tenderness on rectal exam (external palpation)
            │   │   └─ Treatment:
            │   │       ├─ Conservative: Cushions, NSAIDs, PT (90% improve)
            │   │       ├─ Injections: Steroid/local anesthetic (70% improve)
            │   │       └─ Surgical: Coccygectomy (last resort, 60-80% success)
            │   │
            │   └─ Coccygeal fracture:
            │       ├─ Etiology: Direct trauma (fall onto buttocks)
            │       ├─ Imaging: Lateral X-ray (CT if unclear)
            │       └─ Treatment: Conservative (cushions, NSAIDs, time)


## SPINAL CORD ANATOMY ⭐⭐⭐

### GROSS ANATOMY:

├─ Length & position:
│   ├─ Adult spinal cord length: 42-45 cm (male), 40-43 cm (female)
│   ├─ Diameter: 1-1.5 cm (varies by region)
│   │
│   ├─ Extent:
│   │   ├─ Superior: Continuous with medulla oblongata at foramen magnum
│   │   ├─ Inferior: Conus medullaris ends at L1-L2 level (adult) ⭐
│   │   │   └─ Newborn: Conus at L3 level (ascends during development)
│   │   │
│   │   └─ Filum terminale:
│   │       ├─ Pia mater extension from conus → coccyx
│   │       ├─ Internal filum terminale: Conus → dural sac (S2)
│   │       └─ External filum terminale (coccygeal ligament): Dural sac → coccyx
│   │
│   ├─ Segments: 31 spinal cord segments
│   │   ├─ Cervical: 8 segments (C1-C8)
│   │   ├─ Thoracic: 12 segments (T1-T12)
│   │   ├─ Lumbar: 5 segments (L1-L5)
│   │   ├─ Sacral: 5 segments (S1-S5)
│   │   └─ Coccygeal: 1 segment (Co1)
│   │
│   └─ Enlargements (increased neurons for limb innervation):
│       ├─ Cervical enlargement (C5-T1):
│       │   ├─ Largest diameter: 13-14 mm
│       │   ├─ Innervates: Upper limb (brachial plexus C5-T1)
│       │   └─ Vertebral level: C4-T1 vertebrae
│       │
│       └─ Lumbosacral enlargement (L2-S3):
│           ├─ Diameter: 12-13 mm
│           ├─ Innervates: Lower limb (lumbosacral plexus L2-S3)
│           └─ Vertebral level: T9-T12 vertebrae (cord segments higher than vertebrae)

├─ Meninges (from external to internal):
│   ├─ Dura mater (outermost):
│   │   ├─ Tough fibrous layer
│   │   ├─ Extends from foramen magnum → S2 vertebral level (dural sac) ⭐
│   │   ├─ Epidural space: Between dura and vertebral canal (fat, veins)
│   │   └─ Clinical: Epidural anesthesia, epidural abscess, epidural hematoma
│   │
│   ├─ Arachnoid mater (middle):
│   │   ├─ Thin, avascular membrane
│   │   ├─ Subarachnoid space: Between arachnoid and pia (contains CSF)
│   │   ├─ Arachnoid trabeculae: Web-like connections to pia mater
│   │   └─ Clinical: Spinal anesthesia (intrathecal), CSF sampling (lumbar puncture)
│   │
│   └─ Pia mater (innermost):
│       ├─ Delicate membrane adherent to spinal cord surface
│       ├─ Denticulate ligaments: 21 pairs
│       │   ├─ Pia extensions from cord → dura
│       │   ├─ Located between dorsal and ventral roots
│       │   └─ Function: Suspend and stabilize cord within dural sac
│       │
│       └─ Filum terminale: Pia extension from conus → coccyx

├─ External surface landmarks:
│   ├─ Anterior median fissure:
│   │   ├─ Deep groove (3 mm depth)
│   │   └─ Contains: Anterior spinal artery
│   │
│   ├─ Posterior median sulcus:
│   │   └─ Shallow groove (posterior midline)
│   │
│   ├─ Anterolateral sulcus:
│   │   ├─ Location: Lateral to anterior median fissure
│   │   └─ Emergence: Ventral (motor) nerve roots
│   │
│   └─ Posterolateral sulcus:
│       ├─ Location: Lateral to posterior median sulcus
│       └─ Entrance: Dorsal (sensory) nerve roots

├─ Spinal nerve roots:
│   ├─ Dorsal (posterior) roots:
│   │   ├─ Function: Sensory (afferent)
│   │   ├─ Dorsal root ganglion: Contains sensory neuron cell bodies
│   │   └─ Location: In intervertebral foramen (or just proximal)
│   │
│   ├─ Ventral (anterior) roots:
│   │   ├─ Function: Motor (efferent)
│   │   └─ Cell bodies: In anterior horn of spinal cord gray matter
│   │
│   └─ Spinal nerve formation:
│       ├─ Dorsal + ventral roots merge → spinal nerve
│       ├─ Location: At/just distal to dorsal root ganglion
│       └─ Mixed nerve: Sensory + motor

└─ Cauda equina ⭐:
    ├─ Definition: "Horse's tail" - collection of nerve roots below conus medullaris
    ├─ Composition: Lumbar, sacral, coccygeal nerve roots (L2-Co1)
    ├─ Location: Within dural sac from L1-L2 (conus) to S2 (dural termination)
    ├─ Function: Innervation of lower limbs, bladder, bowel, perineum
    └─ Clinical: Cauda equina syndrome (see lumbar disc herniation section)


### SPINAL CORD INTERNAL ANATOMY:

├─ Cross-sectional organization:
│   ├─ Gray matter (central):
│   │   ├─ Appearance: Butterfly or "H" shaped
│   │   ├─ Composition: Neuronal cell bodies, dendrites, unmyelinated axons, glia
│   │   └─ Location: Central canal surrounded by gray matter
│   │
│   └─ White matter (peripheral):
│       ├─ Appearance: Surrounds gray matter
│       ├─ Composition: Myelinated axons (ascending/descending tracts)
│       └─ Organization: Three funiculi (columns) on each side

├─ GRAY MATTER ORGANIZATION ⭐⭐:
│   ├─ Rexed laminae: 10 layers (I-X) for histological classification
│   │
│   ├─ Dorsal (posterior) horn:
│   │   ├─ Location: Dorsal aspect of gray matter
│   │   ├─ Function: Sensory processing (pain, temperature, touch, proprioception)
│   │   │
│   │   ├─ Rexed laminae I-VI:
│   │   │   ├─ Lamina I (posteromarginal nucleus):
│   │   │   │   ├─ Input: Small diameter fibers (Aδ, C fibers) - pain, temperature
│   │   │   │   └─ Output: Spinothalamic tract (contralateral)
│   │   │   │
│   │   │   ├─ Lamina II-III (substantia gelatinosa) ⭐:
│   │   │   │   ├─ Input: Pain, temperature (Aδ, C fibers)
│   │   │   │   ├─ Function: Modulation of pain signals (gate control theory)
│   │   │   │   └─ Clinical: Target for spinal cord stimulation (chronic pain)
│   │   │   │
│   │   │   ├─ Lamina IV: Receives touch, pressure (Aβ fibers)
│   │   │   ├─ Lamina V: Receives visceral pain, wide dynamic range neurons
│   │   │   └─ Lamina VI: Proprioception (mechanoreceptors)
│   │   │
│   │   └─ Nucleus proprius: Laminae III-VI (sensory relay)
│   │
│   ├─ Ventral (anterior) horn:
│   │   ├─ Location: Ventral aspect of gray matter
│   │   ├─ Function: Motor control (alpha and gamma motor neurons)
│   │   │
│   │   ├─ Rexed lamina IX:
│   │   │   ├─ Alpha motor neurons ⭐:
│   │   │   │   ├─ Cell bodies: Large multipolar neurons
│   │   │   │   ├─ Axons: Exit via ventral roots → skeletal muscle
│   │   │   │   ├─ Organization: Somatotopic (medial→proximal, lateral→distal)
│   │   │   │   └─ Lower motor neuron (LMN): Direct innervation of muscle
│   │   │   │
│   │   │   └─ Gamma motor neurons:
│   │   │       ├─ Function: Innervate muscle spindles (maintain sensitivity)
│   │   │       └─ Regulates: Muscle tone
│   │   │
│   │   └─ Motor organization (somatotopic):
│   │       ├─ Medial ventral horn: Axial/proximal muscles (trunk, shoulder, hip)
│   │       └─ Lateral ventral horn: Distal muscles (hand, foot)
│   │           └─ Most prominent in cervical/lumbar enlargements
│   │
│   ├─ Lateral horn (intermediolateral cell column):
│   │   ├─ Location: Lateral aspect (only T1-L2 and S2-S4)
│   │   ├─ Rexed lamina VII
│   │   │
│   │   ├─ T1-L2 (thoracolumbar outflow) ⭐:
│   │   │   ├─ Sympathetic preganglionic neurons
│   │   │   ├─ Function: Sympathetic nervous system
│   │   │   └─ Target: Sympathetic chain ganglia
│   │   │
│   │   └─ S2-S4 (sacral outflow):
│   │       ├─ Parasympathetic preganglionic neurons
│   │       ├─ Function: Bladder, bowel, sexual function
│   │       └─ Target: Pelvic ganglia (near target organs)
│   │
│   ├─ Central canal:
│   │   ├─ Location: Center of gray matter (Rexed lamina X)
│   │   ├─ Contains: CSF (continuous with 4th ventricle)
│   │   ├─ Lined by: Ependymal cells
│   │   └─ Note: Often obliterated in adults (not patent)
│   │
│   └─ Intermediate zone (lamina VII):
│       ├─ Location: Between dorsal and ventral horns
│       ├─ Function: Interneurons (local circuits, reflexes)
│       └─ Clarke's nucleus (nucleus dorsalis):
│           ├─ Location: C8-L2 (medial base of dorsal horn)
│           ├─ Input: Proprioception from muscle spindles/Golgi tendon organs
│           └─ Output: Dorsal spinocerebellar tract → cerebellum

└─ WHITE MATTER ORGANIZATION ⭐⭐⭐:
    ├─ Three funiculi (columns) on each side:
    │   ├─ Dorsal (posterior) funiculus: Between posterior median sulcus and dorsal horn
    │   ├─ Lateral funiculus: Between dorsal and ventral horns
    │   └─ Ventral (anterior) funiculus: Between anterior median fissure and ventral horn
    │
    ├─ Organization principle:
    │   ├─ Ascending tracts: Sensory information to brain
    │   ├─ Descending tracts: Motor commands from brain
    │   └─ Somatotopic organization: Body map (sacral→lateral, cervical→medial)
    │
    └─ Clinical correlation:
        ├─ Central cord syndrome: Preferential damage to cervical (medial) fibers
        └─ Peripheral spinal cord syndromes: Preserve medial sacral fibers


### SPINAL TRACTS ⭐⭐⭐

## ASCENDING (SENSORY) TRACTS:

├─ Dorsal column-medial lemniscal pathway ⭐⭐⭐:
│   ├─ Modality: Fine touch, vibration, proprioception (conscious)
│   ├─ Location: Dorsal (posterior) funiculus
│   │
│   ├─ Two fasciculi:
│   │   ├─ Fasciculus gracilis (medial):
│   │   │   ├─ Fibers from: Lower limb and lower trunk (below T6)
│   │   │   ├─ Somatotopy: Sacral → lateral, lumbar → medial (within fasciculus)
│   │   │   └─ Terminates: Nucleus gracilis (medulla)
│   │   │
│   │   └─ Fasciculus cuneatus (lateral):
│   │       ├─ Fibers from: Upper limb and upper trunk (above T6)
│   │       ├─ Somatotopy: Thoracic → medial, cervical → lateral (within fasciculus)
│   │       └─ Terminates: Nucleus cuneatus (medulla)
│   │
│   ├─ Pathway (3-neuron pathway):
│   │   ├─ First-order neuron:
│   │   │   ├─ Cell body: Dorsal root ganglion
│   │   │   ├─ Enters: Dorsal root → ascends ipsilaterally in dorsal column
│   │   │   └─ Terminates: Nucleus gracilis/cuneatus (medulla)
│   │   │
│   │   ├─ Second-order neuron:
│   │   │   ├─ Cell body: Nucleus gracilis/cuneatus (medulla)
│   │   │   ├─ Decussation: Internal arcuate fibers cross midline → medial lemniscus
│   │   │   └─ Terminates: VPL nucleus of thalamus (contralateral)
│   │   │
│   │   └─ Third-order neuron:
│   │       ├─ Cell body: VPL thalamus
│   │       └─ Terminates: Primary sensory cortex (postcentral gyrus)
│   │
│   └─ Clinical testing:
│       ├─ Vibration: 128 Hz tuning fork on bony prominences
│       ├─ Proprioception: Joint position sense (move toe up/down with eyes closed)
│       └─ Fine touch: Two-point discrimination, light touch localization

├─ Spinothalamic tract (anterolateral system) ⭐⭐⭐:
│   ├─ Modality: Pain, temperature, crude touch
│   ├─ Location: Lateral funiculus (lateral spinothalamic) + ventral funiculus (anterior spinothalamic)
│   │
│   ├─ Lateral spinothalamic tract ⭐:
│   │   ├─ Modality: Pain, temperature
│   │   ├─ Location: Lateral funiculus
│   │   ├─ Somatotopy: Sacral → lateral, cervical → medial ("SCLT": Sacral Cervical Lateral/medial)
│   │   │
│   │   ├─ Pathway (3-neuron pathway):
│   │   │   ├─ First-order neuron:
│   │   │   │   ├─ Cell body: Dorsal root ganglion
│   │   │   │   ├─ Fibers: Small diameter (Aδ, C fibers)
│   │   │   │   ├─ Enters: Dorsal root → dorsal horn (laminae I, II, V)
│   │   │   │   └─ Terminates: Second-order neuron (dorsal horn)
│   │   │   │
│   │   │   ├─ Second-order neuron:
│   │   │   │   ├─ Cell body: Dorsal horn (laminae I, V)
│   │   │   │   ├─ Decussation: Anterior white commissure → crosses to contralateral side
│   │   │   │   ├─ Ascends: 1-2 segments before crossing (delayed crossing) ⭐
│   │   │   │   ├─ Location: Lateral funiculus (contralateral)
│   │   │   │   └─ Terminates: VPL nucleus of thalamus
│   │   │   │
│   │   │   └─ Third-order neuron:
│   │   │       ├─ Cell body: VPL thalamus
│   │   │       └─ Terminates: Primary sensory cortex (postcentral gyrus)
│   │   │
│   │   └─ Clinical significance: Pain/temperature loss CONTRALATERAL to lesion (crosses in cord)
│   │
│   └─ Anterior spinothalamic tract:
│       ├─ Modality: Crude touch, pressure
│       ├─ Location: Ventral funiculus
│       ├─ Pathway: Similar to lateral (crosses in anterior white commissure)
│       └─ Clinical: Less important (redundant with dorsal column)

├─ Spinocerebellar tracts ⭐⭐:
│   ├─ Modality: Unconscious proprioception (for motor coordination)
│   ├─ Location: Lateral funiculus (peripheral)
│   │
│   ├─ Dorsal spinocerebellar tract (DSCT):
│   │   ├─ Input: Lower limb and lower trunk (below L2)
│   │   ├─ Location: Lateral funiculus (most peripheral, posterolateral)
│   │   │
│   │   ├─ Pathway (2-neuron pathway):
│   │   │   ├─ First-order neuron:
│   │   │   │   ├─ Cell body: Dorsal root ganglion
│   │   │   │   ├─ Input: Muscle spindles, Golgi tendon organs
│   │   │   │   └─ Terminates: Clarke's nucleus (C8-L2, lamina VII)
│   │   │   │
│   │   │   └─ Second-order neuron:
│   │   │       ├─ Cell body: Clarke's nucleus
│   │   │       ├─ Ascends: IPSILATERAL (does NOT cross) ⭐
│   │   │       ├─ Enters cerebellum: Inferior cerebellar peduncle
│   │   │       └─ Terminates: Cerebellar cortex (vermis, intermediate zone)
│   │   │
│   │   └─ Clinical: IPSILATERAL ataxia if lesioned
│   │
│   ├─ Ventral spinocerebellar tract (VSCT):
│   │   ├─ Input: Lower limb (information about ongoing motor activity)
│   │   ├─ Location: Lateral funiculus (anterolateral)
│   │   │
│   │   ├─ Pathway (2-neuron pathway):
│   │   │   ├─ First-order neuron: Same as DSCT
│   │   │   │
│   │   │   └─ Second-order neuron:
│   │   │       ├─ Cell body: Spinal gray matter (lamina VII, L1-S2)
│   │   │       ├─ Ascends: CONTRALATERAL (crosses once in cord)
│   │   │       ├─ Enters cerebellum: Superior cerebellar peduncle
│   │   │       └─ Then crosses AGAIN in cerebellum (double crossing = ipsilateral representation) ⭐
│   │   │
│   │   └─ Clinical: Less clinically significant than DSCT
│   │
│   ├─ Cuneocerebellar tract (upper limb equivalent of DSCT):
│   │   ├─ Input: Upper limb and upper trunk (C1-T6)
│   │   ├─ Pathway: Similar to DSCT but from accessory cuneate nucleus (medulla)
│   │   └─ Enters: Inferior cerebellar peduncle (ipsilateral)
│   │
│   └─ Rostral spinocerebellar tract (upper limb equivalent of VSCT):
│       ├─ Input: Upper limb
│       └─ Pathway: Similar to VSCT (double crossing)

└─ Other ascending tracts:
    ├─ Spinoreticular tract:
    │   ├─ Modality: Arousal, consciousness (pain-related)
    │   ├─ Location: Lateral funiculus (mixed with spinothalamic)
    │   ├─ Pathway: Spinal cord → reticular formation → thalamus
    │   └─ Function: Alerting/arousal response to pain
    │
    ├─ Spinotectal tract:
    │   ├─ Modality: Spinovisual reflexes
    │   ├─ Pathway: Spinal cord → superior colliculus
    │   └─ Function: Turns head/eyes toward painful stimulus
    │
    └─ Spinohypothalamic tract:
        ├─ Modality: Pain-related autonomic responses
        ├─ Pathway: Spinal cord → hypothalamus
        └─ Function: Autonomic/endocrine response to pain


## DESCENDING (MOTOR) TRACTS:

├─ Corticospinal (pyramidal) tract ⭐⭐⭐:
│   ├─ Function: Voluntary motor control (especially fine motor - hands, fingers)
│   ├─ Origin: Primary motor cortex (30%), premotor cortex (30%), sensory cortex (40%)
│   ├─ Location: Lateral funiculus (lateral CST) + ventral funiculus (anterior CST)
│   │
│   ├─ Pathway (upper motor neuron pathway):
│   │   ├─ Origin: Motor cortex (precentral gyrus, Brodmann area 4)
│   │   │   ├─ Betz cells: Large pyramidal neurons (3-5% of CST)
│   │   │   └─ Somatotopy: "Motor homunculus" (leg medial, arm lateral, face most lateral)
│   │   │
│   │   ├─ Descends through:
│   │   │   ├─ Corona radiata
│   │   │   ├─ Posterior limb of internal capsule (somatotopic)
│   │   │   ├─ Cerebral peduncle (middle 3/5 of crus cerebri)
│   │   │   ├─ Basilar pons (scattered among pontine nuclei)
│   │   │   └─ Medullary pyramids (forms pyramids on ventral medulla)
│   │   │
│   │   ├─ Decussation at cervicomedullary junction:
│   │   │   ├─ Lateral corticospinal tract (90%): Crosses at pyramidal decussation ⭐
│   │   │   │   ├─ Descends: Contralateral lateral funiculus
│   │   │   │   ├─ Somatotopy: Cervical → medial, sacral → lateral
│   │   │   │   └─ Terminates: Anterior horn (alpha motor neurons, lamina IX)
│   │   │   │
│   │   │   └─ Anterior corticospinal tract (10%): Remains ipsilateral initially
│   │   │       ├─ Descends: Ipsilateral ventral funiculus
│   │   │       ├─ Crosses: At segmental level (via anterior white commissure)
│   │   │       └─ Function: Axial/proximal muscles (bilateral control)
│   │   │
│   │   └─ Synapse: On alpha motor neurons (direct) or interneurons (indirect)
│   │
│   ├─ Clinical significance:
│   │   ├─ Upper motor neuron (UMN) signs (lesion ABOVE spinal cord):
│   │   │   ├─ Weakness: Contralateral to lesion (after decussation)
│   │   │   ├─ Spasticity: Increased muscle tone (velocity-dependent)
│   │   │   ├─ Hyperreflexia: Increased deep tendon reflexes
│   │   │   ├─ Clonus: Rhythmic muscle contractions (ankle, knee)
│   │   │   ├─ Babinski sign: Upgoing toe (pathological)
│   │   │   └─ No atrophy: Muscle bulk preserved (disuse atrophy only)
│   │   │
│   │   └─ Lower motor neuron (LMN) signs (lesion IN/BELOW spinal cord):
│   │       ├─ Weakness: Ipsilateral to lesion (segmental)
│   │       ├─ Flaccidity: Decreased muscle tone
│   │       ├─ Hyporeflexia: Decreased/absent reflexes
│   │       ├─ Fasciculations: Visible muscle twitching
│   │       ├─ Atrophy: Muscle wasting (denervation)
│   │       └─ Downgoing toe: Normal Babinski response
│   │
│   └─ Somatotopic organization:
│       ├─ Within lateral CST: Cervical → medial, sacral → lateral
│       └─ Clinical: Central cord syndrome affects cervical (medial) > sacral (lateral)

├─ Rubrospinal tract:
│   ├─ Function: Facilitates flexor motor neurons (upper limb)
│   ├─ Origin: Red nucleus (midbrain)
│   ├─ Location: Lateral funiculus (anterior to lateral CST)
│   │
│   ├─ Pathway:
│   │   ├─ Origin: Red nucleus (magnocellular part)
│   │   ├─ Decussation: Immediately in midbrain (ventral tegmental decussation)
│   │   ├─ Descends: Contralateral lateral funiculus
│   │   └─ Terminates: Flexor motor neurons (anterior horn)
│   │
│   └─ Clinical: Limited role in humans (well-developed CST compensates)

├─ Vestibulospinal tracts:
│   ├─ Function: Postural control, balance, extensor tone
│   │
│   ├─ Lateral vestibulospinal tract ⭐:
│   │   ├─ Origin: Lateral vestibular nucleus (Deiters' nucleus)
│   │   ├─ Location: Ventral funiculus
│   │   ├─ Descends: Ipsilateral (entire spinal cord)
│   │   ├─ Function: Facilitates extensor motor neurons (antigravity muscles)
│   │   └─ Clinical: Decerebrate posturing (extension) if brainstem lesion below red nucleus
│   │
│   └─ Medial vestibulospinal tract:
│       ├─ Origin: Medial/inferior vestibular nuclei
│       ├─ Location: Ventral funiculus (medial longitudinal fasciculus)
│       ├─ Descends: Bilateral (cervical and upper thoracic only)
│       └─ Function: Head/neck position (stabilize head during movement)

├─ Reticulospinal tracts:
│   ├─ Function: Posture, locomotion, muscle tone modulation
│   │
│   ├─ Pontine (medial) reticulospinal tract:
│   │   ├─ Origin: Pontine reticular formation
│   │   ├─ Location: Ventral funiculus
│   │   ├─ Descends: Ipsilateral
│   │   └─ Function: Facilitates extensors (antigravity)
│   │
│   └─ Medullary (lateral) reticulospinal tract:
│       ├─ Origin: Medullary reticular formation
│       ├─ Location: Lateral funiculus
│       ├─ Descends: Bilateral
│       └─ Function: Facilitates flexors, inhibits extensors

└─ Tectospinal tract:
    ├─ Function: Reflex turning of head toward visual/auditory stimuli
    ├─ Origin: Superior colliculus (midbrain)
    ├─ Location: Ventral funiculus
    ├─ Descends: Contralateral (crosses in dorsal tegmental decussation)
    ├─ Distribution: Cervical cord only (C1-C4)
    └─ Terminates: Neck muscles (via anterior horn)


### SPINAL CORD VASCULAR SUPPLY ⭐⭐⭐

## ARTERIAL SUPPLY:

├─ Overview:
│   ├─ Supplied by: Anterior spinal artery (ASA) + paired posterior spinal arteries (PSA)
│   ├─ Reinforced by: Radicular/segmental arteries at multiple levels
│   └─ Watershed zones: Areas vulnerable to ischemia (poor collateral flow)

├─ Anterior spinal artery (ASA) ⭐⭐⭐:
│   ├─ Origin: Two branches from vertebral arteries (at foramen magnum)
│   │   └─ Merge to form single midline artery in anterior median fissure
│   │
│   ├─ Course: Descends in anterior median fissure (entire spinal cord length)
│   │
│   ├─ Territory supplied (anterior 2/3 of spinal cord):
│   │   ├─ Anterior horns (bilateral) - motor neurons
│   │   ├─ Central gray matter
│   │   ├─ Lateral corticospinal tracts (bilateral)
│   │   ├─ Spinothalamic tracts (bilateral)
│   │   └─ Anterior/lateral white matter
│   │
│   ├─ Reinforcement: Radicular arteries join at multiple levels
│   │   ├─ Most critical: Artery of Adamkiewicz (see below)
│   │   └─ Without reinforcement: ASA alone insufficient below cervical cord
│   │
│   └─ Caliber: Smaller in thoracic region → vulnerable to ischemia

├─ Posterior spinal arteries (PSA) - paired ⭐:
│   ├─ Origin: Vertebral arteries (or PICA) at foramen magnum
│   ├─ Course: Descend posterolaterally (one on each side, along dorsal roots)
│   │
│   ├─ Territory supplied (posterior 1/3 of spinal cord):
│   │   ├─ Dorsal horns (sensory processing)
│   │   ├─ Dorsal columns (fasciculus gracilis, fasciculus cuneatus)
│   │   └─ Dorsal/lateral white matter
│   │
│   └─ Reinforcement: Numerous posterior radicular arteries → better collaterals than ASA

├─ Radicular/segmental arteries ⭐⭐:
│   ├─ Origin: Segmental arteries from aorta (thoracic/lumbar) or vertebral/subclavian (cervical)
│   ├─ Course: Enter via intervertebral foramina, travel along nerve roots
│   │
│   ├─ Anterior radicular arteries (feed ASA):
│   │   ├─ Incidence: Only 6-8 total along entire cord (sparse)
│   │   ├─ Distribution:
│   │   │   ├─ Cervical: 2-3 radicular arteries (adequate)
│   │   │   ├─ Thoracic: 1-2 radicular arteries (sparse - watershed zone)
│   │   │   └─ Lumbar: 1-2 radicular arteries (including Adamkiewicz)
│   │   │
│   │   └─ Artery of Adamkiewicz (arteria radicularis magna) ⭐⭐⭐:
│   │       ├─ Largest anterior radicular artery (diameter 0.5-1.0 mm)
│   │       ├─ Critical: Supplies most of lower thoracic and lumbosacral cord
│   │       ├─ Location (variable):
│   │       │   ├─ Most common: T9-L2 (80%)
│   │       │   ├─ Peak incidence: T10-T11 (40%)
│   │       │   └─ Side: Left (75%) > right (25%)
│   │       │
│   │       ├─ "Hairpin turn" on angiography: Characteristic appearance
│   │       │
│   │       └─ Clinical importance ⭐:
│   │           ├─ Thoracoabdominal aortic aneurysm repair: Risk of cord ischemia
│   │           ├─ Spinal epidural/subdural hematoma: Can compress artery
│   │           ├─ Protective measures during surgery:
│   │           │   ├─ Preoperative angiography: Identify Adamkiewicz location
│   │           │   ├─ CSF drainage: Lower intraspinal pressure (↑ perfusion)
│   │           │   ├─ Proximal MAP >80-90 mmHg: Maintain cord perfusion
│   │           │   └─ Avoid prolonged aortic cross-clamp (>30 min without protection)
│   │           │
│   │           └─ Injury consequences: Anterior spinal artery syndrome (paraplegia)
│   │
│   └─ Posterior radicular arteries (feed PSA):
│       ├─ Incidence: 15-20 along entire cord (more abundant than anterior)
│       └─ Better collateral circulation than ASA territory

├─ Spinal cord vascular territories (cross-sectional):
│   ├─ Central arteries (from ASA):
│   │   ├─ Sulcocommissural arteries: Branch from ASA alternating left/right
│   │   ├─ Supply: Anterior horns, central gray matter, anterior/lateral white matter
│   │   └─ Vulnerable: Small end arteries (little collateral)
│   │
│   ├─ Peripheral pial plexus (arterial corona):
│   │   ├─ Anastomotic network: ASA + PSA connections around cord surface
│   │   ├─ Supply: Outer white matter (superficial)
│   │   └─ Better collaterals than central territory
│   │
│   └─ Watershed zones (vulnerable areas):
│       ├─ Thoracic cord (T4-T9): "Watershed" - furthest from major feeders ⭐
│       │   ├─ ASA smallest caliber in thoracic region
│       │   ├─ Fewest radicular arteries
│       │   └─ Most vulnerable to systemic hypotension
│       │
│       └─ Border zones: Between ASA and PSA territories (lateral columns)

└─ Clinical syndromes - arterial occlusion:
    ├─ Anterior spinal artery syndrome ⭐⭐⭐:
    │   ├─ Etiology:
    │   │   ├─ Aortic surgery: Thoracoabdominal aneurysm repair (most common iatrogenic)
    │   │   ├─ Aortic dissection: Stanford type A or B (extends to radicular arteries)
    │   │   ├─ Systemic hypotension: Prolonged (cardiac arrest, shock)
    │   │   ├─ Atherosclerotic occlusion: Radicular artery
    │   │   ├─ Spinal cord compression: Disc, tumor, hematoma (venous congestion)
    │   │   └─ Hypercoagulable states: Thrombosis of ASA
    │   │
    │   ├─ Territory affected: Anterior 2/3 of cord (ASA distribution)
    │   │
    │   ├─ Clinical presentation (below lesion level):
    │   │   ├─ Motor loss: Bilateral (corticospinal tracts) ⭐
    │   │   │   ├─ Acute: Flaccid paralysis (spinal shock)
    │   │   │   └─ Chronic: Spastic paraplegia/quadriplegia (UMN signs)
    │   │   │
    │   │   ├─ Pain/temperature loss: Bilateral (spinothalamic tracts) ⭐
    │   │   │   └─ Dissociated sensory loss (see below)
    │   │   │
    │   │   ├─ Vibration/proprioception: PRESERVED (dorsal columns spared) ⭐
    │   │   │   └─ Key distinguishing feature from complete cord transection
    │   │   │
    │   │   ├─ Autonomic dysfunction:
    │   │   │   ├─ Bladder/bowel: Urinary retention → neurogenic bladder
    │   │   │   ├─ Sexual dysfunction: Impotence
    │   │   │   └─ Vasomotor: Hypotension (loss of sympathetic tone)
    │   │   │
    │   │   └─ Level: Depends on site of vascular compromise
    │   │       ├─ Adamkiewicz occlusion: T10-L2 level (paraplegia)
    │   │       └─ Upper thoracic/cervical: Quadriplegia
    │   │
    │   ├─ Diagnosis:
    │   │   ├─ MRI (T2): Hyperintense signal in anterior cord ("owl's eyes" on axial)
    │   │   ├─ Timing: Changes visible 24-48 hours after ischemia
    │   │   └─ Angiography: Rarely indicated (diagnosis usually clinical + MRI)
    │   │
    │   ├─ Treatment:
    │   │   ├─ Supportive: No specific treatment for established infarction
    │   │   ├─ Blood pressure support: Maintain MAP >85 mmHg (optimize perfusion)
    │   │   ├─ CSF drainage: If elevated pressure (improves cord perfusion)
    │   │   └─ Rehabilitation: Physical/occupational therapy
    │   │
    │   └─ Prognosis:
    │       ├─ Poor: Most patients remain paraplegic
    │       ├─ Some recovery: If incomplete infarction or small territory
    │       └─ Timing: Recovery typically minimal after 6 months
    │
    ├─ Posterior spinal artery syndrome (rare):
    │   ├─ Etiology: Uncommon (excellent collaterals from multiple posterior radicular arteries)
    │   ├─ Clinical: Loss of vibration/proprioception, preserved motor/pain/temperature
    │   └─ Presentation: Sensory ataxia (loss of proprioception)
    │
    └─ Central cord syndrome (see separate section):
        └─ Often vascular in elderly (ASA sulcocommissural artery occlusion)


## VENOUS DRAINAGE:

├─ Overview:
│   ├─ Valveless venous system: Bidirectional flow possible
│   ├─ Communicates freely: Pelvic/abdominal veins ↔ spinal veins ↔ intracranial veins
│   └─ Clinical: Route for metastatic spread (especially prostate, breast, lung to spine)

├─ Intrinsic venous drainage:
│   ├─ Anterior/posterior median spinal veins: On cord surface (midline)
│   ├─ Radial veins: Drain from center of cord → surface
│   └─ Coronal venous plexus: Pial plexus on cord surface

├─ Extrinsic venous drainage:
│   ├─ Anterior/posterior radicular veins: Accompany nerve roots
│   └─ Drain into: Epidural venous plexus

└─ Epidural venous plexus (Batson's plexus) ⭐⭐:
    ├─ Location: Epidural space (between dura and vertebral canal)
    ├─ Composition: Valveless venous channels
    │
    ├─ Connections:
    │   ├─ Superiorly: Intracranial venous sinuses (via foramen magnum)
    │   ├─ Anteriorly: Basivertebral veins (from vertebral bodies)
    │   ├─ Laterally: Segmental veins → azygos/hemiazygos (thoracic), IVC (lumbar)
    │   └─ Inferiorly: Pelvic veins (sacral, iliac)
    │
    ├─ Function:
    │   ├─ Venous drainage: Low-pressure system (no valves)
    │   └─ Alternative pathway: When IVC/SVC obstructed (collateral flow)
    │
    └─ Clinical significance ⭐:
        ├─ Metastatic spread (Batson's hypothesis):
        │   ├─ Mechanism: Retrograde venous spread (when intra-abdominal pressure ↑)
        │   ├─ Primary tumors: Prostate, breast, lung, kidney, thyroid
        │   ├─ Destination: Vertebral bodies, epidural space, spinal cord
        │   └─ Pattern: Explains spinal metastases without pulmonary involvement
        │
        ├─ Epidural hematoma:
        │   ├─ Causes: Trauma, anticoagulation, vascular malformation, epidural procedure
        │   ├─ Presentation: Acute back pain → progressive myelopathy
        │   └─ Treatment: Emergency surgical decompression
        │
        └─ Spinal dural arteriovenous fistula (dAVF):
            ├─ Pathophysiology: Arteriovenous shunt → venous hypertension → cord ischemia
            ├─ Presentation: Progressive myelopathy (stepwise or gradual)
            └─ Treatment: Surgical/endovascular obliteration of fistula


### SPINAL CORD SYNDROMES ⭐⭐⭐

## COMPLETE SPINAL CORD TRANSECTION:

├─ Etiology: Trauma (fracture-dislocation), complete cord compression (tumor, abscess)
│
├─ Clinical presentation (below lesion level):
│   ├─ Motor: Complete bilateral paralysis (all motor lost below level)
│   ├─ Sensory: Complete bilateral sensory loss (all modalities)
│   ├─ Autonomic: Bladder/bowel dysfunction, vasomotor instability
│   └─ Reflexes: Initially absent (spinal shock) → hyperreflexia (chronic)
│
├─ Spinal shock:
│   ├─ Definition: Temporary loss of ALL reflexes below lesion (acute phase)
│   ├─ Duration: Days to weeks (typically 1-6 weeks)
│   ├─ Clinical:
│   │   ├─ Flaccid paralysis (not spastic)
│   │   ├─ Areflexia (absent reflexes)
│   │   ├─ Urinary retention (bladder atonic)
│   │   └─ Priapism (in males, acute phase)
│   │
│   ├─ Resolution:
│   │   ├─ First reflex to return: Bulbocavernosus reflex (anal wink)
│   │   └─ Timing: Marks end of spinal shock (days to weeks)
│   │
│   └─ Chronic phase (after spinal shock):
│       ├─ Spastic paralysis: UMN signs (hyperreflexia, spasticity, clonus)
│       ├─ Neurogenic bladder: Reflex bladder (spontaneous emptying)
│       └─ Autonomic dysreflexia: If lesion T6 or above (see below)
│
└─ Autonomic dysreflexia ⭐⭐:
    ├─ Definition: Uncontrolled sympathetic response to stimulus below lesion
    ├─ Requirement: Lesion at T6 or above (splanchnic outflow intact below)
    │
    ├─ Pathophysiology:
    │   ├─ Trigger: Noxious stimulus below lesion (bladder distention, bowel impaction)
    │   ├─ Sympathetic surge: Vasoconstriction below lesion → hypertension
    │   ├─ Baroreceptor response: Vasodilation above lesion (cannot descend past lesion)
    │   └─ Result: Severe hypertension + bradycardia
    │
    ├─ Clinical presentation:
    │   ├─ Hypertension: Severe (SBP >200 mmHg) ⭐
    │   ├─ Headache: Pounding (from hypertension)
    │   ├─ Diaphoresis: Above lesion (face, neck)
    │   ├─ Flushing: Above lesion
    │   ├─ Piloerection: "Goosebumps" below lesion
    │   ├─ Bradycardia: Baroreceptor reflex (not always present)
    │   └─ Complications: Stroke, seizure, myocardial infarction if untreated
    │
    ├─ Common triggers:
    │   ├─ Bladder distention: Most common (blocked catheter)
    │   ├─ Bowel impaction: Fecal impaction
    │   ├─ Urinary tract infection: UTI
    │   └─ Skin irritation: Pressure ulcer, ingrown toenail
    │
    └─ Treatment (EMERGENCY):
        ├─ Sit patient upright: Lower blood pressure (orthostatic effect)
        ├─ Remove trigger: Empty bladder (catheter), remove impaction
        ├─ Antihypertensives: If BP remains elevated (nifedipine, nitrates)
        └─ Prevention: Regular bladder/bowel care, avoid triggers


## INCOMPLETE SPINAL CORD SYNDROMES:

├─ Central cord syndrome ⭐⭐⭐:
│   ├─ Most common incomplete cord syndrome (50-60%)
│   │
│   ├─ Etiology:
│   │   ├─ Hyperextension injury: Elderly with cervical spondylosis/stenosis (most common)
│   │   ├─ Mechanism: Cord "pinched" between anterior osteophyte and posterior ligamentum flavum
│   │   ├─ Falls: Minor trauma in elderly (no fracture often)
│   │   └─ Syringomyelia: Central cavitation (chronic)
│   │
│   ├─ Pathophysiology:
│   │   ├─ Central cord damage: Gray matter + medial white matter
│   │   ├─ Somatotopy: Cervical fibers medial, sacral fibers lateral in corticospinal tract
│   │   └─ Result: Arms > legs (cervical fibers preferentially damaged) ⭐
│   │
│   ├─ Clinical presentation:
│   │   ├─ Motor loss (characteristic pattern):
│   │   │   ├─ Upper extremities > lower extremities ⭐
│   │   │   ├─ Hands most affected (fine motor loss)
│   │   │   ├─ "Upside-down" pattern (proximal > distal or distal > proximal)
│   │   │   └─ May walk but cannot use hands
│   │   │
│   │   ├─ Sensory loss:
│   │   │   ├─ "Cape-like" distribution: Suspended sensory level
│   │   │   ├─ Pain/temperature loss: Spinothalamic crossing fibers (central)
│   │   │   └─ Variable: Often patchy, incomplete
│   │   │
│   │   ├─ Bladder dysfunction: Variable (often recovers early)
│   │   └─ Sacral sparing: Preserved (lateral fibers spared) ⭐
│   │
│   ├─ Diagnosis:
│   │   ├─ MRI: Hyperintense T2 signal in central cord
│   │   └─ Cervical stenosis/spondylosis: Often present (predisposing factor)
│   │
│   ├─ Treatment:
│   │   ├─ Conservative (most cases):
│   │   │   ├─ Observation: No fracture, no cord compression
│   │   │   ├─ Cervical collar: Immobilization initially
│   │   │   └─ Rehabilitation: Early mobilization (after acute phase)
│   │   │
│   │   └─ Surgical decompression:
│   │       ├─ Indications: Severe stenosis, progressive deficit, persistent compression
│   │       ├─ Timing: Controversial (early vs late)
│   │       └─ Options: Anterior discectomy/corpectomy or posterior laminectomy
│   │
│   └─ Prognosis:
│       ├─ Best prognosis of all incomplete cord syndromes ⭐
│       ├─ Recovery pattern: Legs → bladder → arms → hands (reverse order of loss)
│       ├─ Ambulation: 75-90% regain walking
│       ├─ Hand function: Most difficult to recover (50-60%)
│       └─ Age: Younger patients recover better

├─ Brown-Séquard syndrome (hemisection) ⭐⭐⭐:
│   ├─ Etiology: Penetrating trauma (stab/gunshot), lateral cord compression (tumor, disc)
│   │
│   ├─ Pathophysiology: Hemisection of spinal cord (one side only)
│   │
│   ├─ Clinical presentation (classic triad):
│   │   ├─ IPSILATERAL motor loss (below lesion) ⭐:
│   │   │   ├─ Corticospinal tract: Descends ipsilaterally
│   │   │   ├─ UMN signs: Spastic paralysis, hyperreflexia
│   │   │   └─ LMN signs: At level of lesion (anterior horn damage)
│   │   │
│   │   ├─ IPSILATERAL loss of vibration/proprioception (below lesion) ⭐:
│   │   │   ├─ Dorsal columns: Ascend ipsilaterally
│   │   │   └─ Sensory ataxia: Loss of proprioception
│   │   │
│   │   ├─ CONTRALATERAL loss of pain/temperature (below lesion, 1-2 levels below) ⭐:
│   │   │   ├─ Spinothalamic tract: Crosses 1-2 segments below entry
│   │   │   └─ "Dissociated sensory loss"
│   │   │
│   │   └─ At level of lesion:
│   │       ├─ IPSILATERAL: Loss of all sensation (dorsal horn damage)
│   │       └─ IPSILATERAL: LMN weakness (anterior horn damage)
│   │
│   ├─ Key distinguishing feature: "Crossed" deficit (motor/vibration ipsilateral, pain/temp contralateral)
│   │
│   └─ Prognosis:
│       ├─ Good: Second-best prognosis (after central cord)
│       ├─ Recovery: 75-90% regain functional ambulation
│       └─ Pure hemisection rare: Usually partial Brown-Séquard

├─ Anterior spinal artery syndrome ⭐⭐⭐:
│   └─ See vascular supply section above (detailed)

├─ Posterior cord syndrome (rare):
│   ├─ Etiology: Rare (dorsal column or PSA ischemia, tabes dorsalis, MS)
│   │
│   ├─ Pathophysiology: Dorsal columns damaged
│   │
│   ├─ Clinical presentation:
│   │   ├─ Loss of vibration/proprioception: Bilateral (below lesion)
│   │   ├─ Sensory ataxia: Wide-based gait (cannot sense foot position)
│   │   ├─ Romberg sign: Positive (falls with eyes closed)
│   │   └─ Motor preserved: Corticospinal tracts spared
│   │   └─ Pain/temperature preserved: Spinothalamic tracts spared
│   │
│   └─ Key feature: Motor and pain/temperature preserved, proprioception lost

├─ Conus medullaris syndrome ⭐⭐:
│   ├─ Location: Conus medullaris (L1-L2 vertebral level, S2-S5 cord segments)
│   │
│   ├─ Etiology: Fracture, tumor, vascular (ASA)
│   │
│   ├─ Clinical presentation:
│   │   ├─ Bladder/bowel dysfunction: Early and prominent (S2-S4 parasympathetic) ⭐
│   │   │   ├─ Areflexic bladder: Atonic, overflow incontinence
│   │   │   └─ Fecal incontinence: Anal sphincter dysfunction
│   │   │
│   │   ├─ Saddle anesthesia: S2-S5 dermatomes (perineum, perianal)
│   │   ├─ Sexual dysfunction: Erectile dysfunction, loss of genital sensation
│   │   ├─ Lower extremity weakness: Variable (L2-S2 roots)
│   │   │   └─ Often symmetric
│   │   │
│   │   ├─ Reflexes:
│   │   │   ├─ Bulbocavernosus reflex: Absent
│   │   │   ├─ Anal wink: Absent
│   │   │   └─ Achilles reflex: May be absent (S1-S2)
│   │   │
│   │   └─ Back pain: Sudden onset (if vascular or traumatic)
│   │
│   ├─ Differentiation from cauda equina syndrome:
│   │   ├─ Conus: Symmetric, early bladder, UMN signs above lesion, hyperreflexia possible
│   │   └─ Cauda equina: Asymmetric, later bladder, LMN signs only, areflexia
│   │
│   └─ Prognosis: Bladder/bowel function rarely recovers (sacral segments)

└─ Cauda equina syndrome ⭐⭐⭐:
    └─ See lumbar disc herniation section above (detailed)


## SPINAL LEVEL LOCALIZATION ⭐⭐:

├─ Rule of thumb: Spinal cord segment ≠ vertebral level (except C1-C7)
│
├─ Cervical spine (C1-C7 vertebrae, C1-C8 cord segments):
│   ├─ Rule: Cord level = vertebral level
│   │   └─ C5 vertebra ≈ C5 cord segment
│   │
│   └─ Note: 8 cervical cord segments, 7 cervical vertebrae
│       └─ C8 root exits between C7-T1 vertebrae
│
├─ Upper thoracic (T1-T6 vertebrae):
│   └─ Rule: Cord level = vertebral level + 1
│       └─ T5 vertebra ≈ T6 cord segment
│
├─ Lower thoracic (T7-T9 vertebrae):
│   └─ Rule: Cord level = vertebral level + 2
│       └─ T8 vertebra ≈ T10 cord segment
│
├─ T10-T12 vertebrae:
│   └─ Rule: T10 ≈ L1-L2, T11 ≈ L3-L4, T12 ≈ L5-S1
│
└─ L1-L2 vertebrae:
    ├─ Conus medullaris: Ends at L1-L2 vertebral level (S2-Co1 cord segments)
    └─ Below L2 vertebra: Only cauda equina (nerve roots, not cord)


## KEY SENSORY/MOTOR LEVELS ⭐⭐:

├─ Cervical levels:
│   ├─ C2: Posterior scalp (occiput)
│   ├─ C3: Supraclavicular fossa
│   ├─ C4: Shoulder cape (top of shoulder)
│   ├─ C5: Lateral arm (deltoid badge area)
│   │   └─ Motor: Deltoid (shoulder abduction), biceps
│   ├─ C6: Thumb and radial forearm
│   │   └─ Motor: Wrist extensors (wrist drop if lost), biceps reflex
│   ├─ C7: Middle finger
│   │   └─ Motor: Triceps (elbow extension), triceps reflex
│   └─ C8: Pinky finger and ulnar forearm
│       └─ Motor: Hand intrinsics (finger abduction/adduction)
│
├─ Thoracic levels (dermatomes):
│   ├─ T1: Medial arm/forearm
│   ├─ T4: Nipple line ⭐
│   ├─ T6: Xiphoid process
│   ├─ T10: Umbilicus ⭐
│   └─ T12: Inguinal ligament/pubic symphysis
│
├─ Lumbar levels:
│   ├─ L1: Inguinal crease
│   ├─ L2: Anterior/medial thigh
│   │   └─ Motor: Hip flexion (iliopsoas)
│   ├─ L3: Medial knee
│   │   └─ Motor: Knee extension (quadriceps)
│   ├─ L4: Medial leg/ankle
│   │   └─ Motor: Ankle dorsiflexion (tibialis anterior), knee reflex (patellar)
│   └─ L5: Dorsum of foot (first webspace)
│       └─ Motor: Great toe extension (EHL), ankle dorsiflexion
│
└─ Sacral levels:
    ├─ S1: Lateral foot, posterior leg
    │   └─ Motor: Ankle plantarflexion (gastrocnemius), ankle reflex (Achilles)
    ├─ S2: Posterior thigh
    │   └─ Motor: Knee flexion (hamstrings)
    ├─ S3-S4: Perineum (medial)
    │   └─ Motor: Bladder, bowel (parasympathetic)
    └─ S5: Perianal
        └─ Motor: External anal sphincter


## CLINICAL PEARLS ⭐:

├─ Sacral sparing: Preservation of sacral sensation (perianal) in incomplete lesions
│   ├─ Indicates: Incomplete spinal cord injury (better prognosis)
│   └─ Test: Perianal sensation, rectal tone, voluntary anal contraction
│
├─ "Cape-like" sensory loss: Central cord syndrome, syringomyelia
│   └─ Suspended sensory level (pain/temperature lost at level, preserved below)
│
├─ Dissociated sensory loss: Loss of pain/temperature with preserved vibration/proprioception
│   ├─ Causes: Anterior spinal artery syndrome, syringomyelia, Brown-Séquard
│   └─ Mechanism: Spinothalamic damage, dorsal columns spared
│
├─ "Onion skin" pattern: Trigeminal nerve sensory loss (perioral → peripheral)
│   └─ Relevant for: Brainstem lesions extending to upper cervical cord
│
└─ Lhermitte's sign: Electric shock sensation down spine with neck flexion
    ├─ Indicates: Posterior column pathology (cervical cord)
    └─ Causes: MS, cervical spondylosis, B12 deficiency, radiation myelopathy
