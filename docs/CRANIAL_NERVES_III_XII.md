# Cranial Nerves III-XII - Ultra-Granular Surgical Anatomy

**Document**: Complete CN III-XII Surgical Anatomy
**Purpose**: Detailed anatomy for neurosurgical procedures
**Scope**: Segments, foramina, function, testing, surgical considerations

---

## CN III - OCULOMOTOR NERVE

```
├─ Type: Motor (to extraocular muscles) + Parasympathetic (pupillary constriction)
│
├─ Origin & Nucleus
│   ├─ Motor nucleus: Midbrain (ventral periaqueductal gray, level of superior colliculus)
│   ├─ Edinger-Westphal nucleus: Parasympathetic (pupil + accommodation)
│   ├─ Fascicles: Traverse through red nucleus and medial cerebral peduncle
│   └─ Exit: Interpeduncular fossa (medial aspect of cerebral peduncle)
│
├─ Segments
│   ├─ 1. Cisternal (Interpeduncular/Crural)
│   │   ├─ Course: Anterolaterally through interpeduncular cistern
│   │   ├─ Length: 24-28 mm
│   │   ├─ Relations:
│   │   │   ├─ Medial: CN III contralateral
│   │   │   ├─ Lateral: Uncus of temporal lobe
│   │   │   ├─ Superior: Posterior cerebral artery (P1-P2 segments)
│   │   │   ├─ Inferior: Superior cerebellar artery (SCA)
│   │   │   └─ Posterior communicating artery (PCoA): Runs lateral/superior to CN III
│   │   │
│   │   ├─ Vascular relationships (CRITICAL):
│   │   │   ├─ PCoA aneurysm: Most common cause of CN III palsy
│   │   │   ├─ Basilar apex aneurysm: May compress CN III
│   │   │   ├─ SCA aneurysm: Rare cause of CN III palsy
│   │   │   └─ Microvascular ischemia: "Medical" CN III palsy (DM, HTN)
│   │   │
│   │   └─ Uncal herniation: Compresses CN III against tentorial edge
│   │       ├─ Pupil dilates ("blown pupil") - parasympathetic fibers compressed
│   │       ├─ Mechanism: Parasympathetic fibers run on surface of nerve
│   │       └─ Sign: Ipsilateral dilated pupil = uncal herniation until proven otherwise!
│   │
│   ├─ 2. Cavernous Sinus Segment
│   │   ├─ Entry: Through dural fold into lateral wall of cavernous sinus
│   │   ├─ Course: Superiormost nerve in lateral wall of CS
│   │   ├─ Length: 20-25 mm
│   │   ├─ Relations in CS:
│   │   │   ├─ Medial: ICA (C4-C6 segments)
│   │   │   ├─ Superior: Anterior clinoid, cavernous ICA roof
│   │   │   ├─ Lateral: Middle cranial fossa dura
│   │   │   ├─ Inferior: CN IV (below), V1 (below), V2 (inferiormost)
│   │   │   └─ CN VI: Within sinus (not in lateral wall)
│   │   │
│   │   └─ Division in CS:
│   │       ├─ Superior division: Superior rectus, levator palpebrae
│   │       └─ Inferior division: Medial rectus, inferior rectus, inferior oblique + parasympathetics
│   │
│   ├─ 3. Orbital Segment
│   │   ├─ Entry: Superior orbital fissure (within annulus of Zinn)
│   │   ├─ Divisions:
│   │   │   ├─ Superior division:
│   │   │   │   ├─ Superior rectus muscle
│   │   │   │   └─ Levator palpebrae superioris (eyelid elevation)
│   │   │   │
│   │   │   └─ Inferior division:
│   │   │       ├─ Medial rectus (eye adduction)
│   │   │       ├─ Inferior rectus (eye depression)
│   │   │       ├─ Inferior oblique (eye elevation + extorsion)
│   │   │       └─ Parasympathetic branch to ciliary ganglion →
│   │   │           ├─ Pupillary sphincter (pupil constriction)
│   │   │           └─ Ciliary muscle (lens accommodation)
│   │   │
│   │   └─ Course: Branches spread to respective muscles
│   │
│   └─ Foramen: Superior orbital fissure (within tendinous ring - annulus of Zinn)
│
├─ Function
│   ├─ Motor:
│   │   ├─ Eye elevation (superior rectus)
│   │   ├─ Eye adduction (medial rectus)
│   │   ├─ Eye depression (inferior rectus)
│   │   ├─ Eye elevation + extorsion (inferior oblique)
│   │   └─ Eyelid elevation (levator palpebrae superioris)
│   │
│   └─ Parasympathetic:
│       ├─ Pupillary constriction (sphincter pupillae)
│       └─ Lens accommodation (ciliary muscle)
│
├─ Testing
│   ├─ Eye movements:
│   │   ├─ "Look up, down, in"
│   │   ├─ H-pattern: Check all directions
│   │   └─ Diplopia (double vision) assessment
│   │
│   ├─ Pupillary light reflex:
│   │   ├─ Direct reflex (light in one eye → constriction same eye)
│   │   ├─ Consensual reflex (light in one eye → constriction other eye)
│   │   └─ Afferent (CN II) vs efferent (CN III) defect
│   │
│   ├─ Ptosis assessment:
│   │   ├─ Eyelid position (levator palpebrae function)
│   │   └─ Complete vs partial ptosis
│   │
│   └─ Accommodation:
│       ├─ Look at distant object, then near object
│       └─ Pupil should constrict with near gaze (accommodation reflex)
│
├─ Clinical Syndromes
│   ├─ Complete CN III Palsy:
│   │   ├─ Eye: Down and out position (lateral rectus + superior oblique unopposed)
│   │   ├─ Ptosis: Complete (levator palpebrae paralysis)
│   │   ├─ Pupil: Dilated and unreactive (parasympathetic damage)
│   │   └─ Diplopia: Severe (cannot adduct, elevate, or depress eye)
│   │
│   ├─ Pupil-Sparing CN III Palsy:
│   │   ├─ Motor deficit present (ptosis, ophthalmoplegia)
│   │   ├─ Pupil NORMAL (reactive, not dilated)
│   │   ├─ Cause: Microvascular ischemia (DM, HTN) - "medical CN III"
│   │   ├─ Mechanism: Core of nerve ischemic, periphery (parasympathetics) spared
│   │   └─ Management: Observation, treat risk factors (usually resolves 3-6 months)
│   │
│   ├─ Pupil-Involving CN III Palsy:
│   │   ├─ Motor deficit + dilated, unreactive pupil
│   │   ├─ Cause: Compression (aneurysm, tumor, herniation)
│   │   ├─ Mechanism: Peripheral nerve fibers (parasympathetics) compressed first
│   │   ├─ EMERGENCY: Assume PCoA aneurysm until proven otherwise!
│   │   └─ Workup: Immediate CTA/MRA, consider angiogram
│   │
│   ├─ Aberrant Regeneration:
│   │   ├─ After CN III injury, axons regrow to wrong muscles
│   │   ├─ Signs:
│   │   │   ├─ Eyelid elevation with downgaze ("pseudo-von Graefe sign")
│   │   │   ├─ Pupil constriction with adduction
│   │   │   └─ Pupil constriction with upgaze
│   │   ├─ Causes: Chronic compression (meningioma, aneurysm, cavernous sinus lesion)
│   │   └─ Never seen with: Microvascular ischemia (ischemia → no regeneration)
│   │
│   └─ Neurological localization:
│       ├─ Nuclear (brainstem): Bilateral ptosis (levator nucleus midline)
│       ├─ Fascicular (midbrain): CN III + other brainstem signs
│       │   ├─ Weber syndrome: CN III + contralateral hemiplegia (cerebral peduncle)
│       │   ├─ Benedikt syndrome: CN III + contralateral tremor (red nucleus)
│       │   └─ Nothnagel syndrome: CN III + cerebellar signs
│       ├─ Cisternal: Isolated CN III, consider aneurysm/tumor
│       ├─ Cavernous: CN III + other CNs (IV, V1, V2, VI)
│       └─ Orbital: CN III + other orbital signs (proptosis, vision changes)
│
└─ Surgical Considerations
    ├─ PCoA aneurysm clipping:
    │   ├─ Most common intraop CN III visualization
    │   ├─ Aneurysm usually lateral to CN III
    │   ├─ Dissection: Identify CN III early, protect throughout
    │   ├─ Temporary clips: Avoid CN III compression
    │   └─ Postop palsy: May be temporary (manipulation) or permanent (injury)
    │
    ├─ Cavernous sinus surgery:
    │   ├─ CN III in superior lateral wall
    │   ├─ Identify and preserve if possible
    │   ├─ Meningiomas: Often cannot preserve (subtotal resection preferred)
    │   └─ Recovery: May take 6-12 months if stretched but not transected
    │
    ├─ Uncal herniation:
    │   ├─ Dilated pupil = CN III compression = herniation
    │   ├─ Treat cause: Evacuate hematoma, osmotic therapy, decompress
    │   └─ Recovery: If nerve not infarcted, pupil may recover over weeks
    │
    └─ Microvascular decompression (rare for CN III):
        ├─ CN III compression by SCA or PCA
        ├─ Approach: Subtemporal, pterional, or orbitozygomatic
        └─ Technique: Interpose Teflon between vessel and nerve
```

## CN IV - TROCHLEAR NERVE

```
├─ Type: Motor (to superior oblique muscle only)
│
├─ Unique Features
│   ├─ Only CN to exit brainstem DORSALLY (posterior midbrain)
│   ├─ Only CN to DECUSSATE completely (crosses midline in anterior medullary velum)
│   ├─ Longest intracranial course (~75 mm)
│   ├─ Thinnest CN (smallest diameter)
│   └─ Most vulnerable CN to head trauma (due to long course + thin diameter)
│
├─ Origin & Nucleus
│   ├─ Motor nucleus: Dorsal midbrain (inferior colliculus level)
│   ├─ Fascicles: Dorsal course through midbrain tegmentum
│   ├─ Decussation: Crosses midline in anterior medullary velum (just posterior to inferior colliculus)
│   └─ Exit: Dorsal midbrain (just below inferior colliculus)
│       └─ CRITICAL: Left nucleus → right CN IV, right nucleus → left CN IV
│
├─ Segments
│   ├─ 1. Cisternal (Ambient/Quadrigeminal Cistern)
│   │   ├─ Course: Curves anterolaterally around midbrain (in ambient cistern)
│   │   ├─ Length: 20-25 mm
│   │   ├─ Relations:
│   │   │   ├─ Medial: Midbrain tectum
│   │   │   ├─ Lateral: Temporal lobe (uncus)
│   │   │   ├─ Superior: Tentorium cerebelli (free edge)
│   │   │   ├─ Inferior: Superior cerebellar artery (SCA)
│   │   │   └─ Crosses above: CN III (CN IV more superior)
│   │   │
│   │   ├─ Pierces dura: Just posterior to CN III entry into cavernous sinus
│   │   │
│   │   └─ Vascular relations:
│   │       ├─ SCA loops around midbrain (may compress CN IV)
│   │       ├─ Tentorial branches of ICA/meningohypophyseal trunk
│   │       └─ Rarely compressed by aneurysms (small diameter, high location)
│   │
│   ├─ 2. Cavernous Sinus Segment
│   │   ├─ Entry: Pierces dura lateral to posterior clinoid
│   │   ├─ Course: Lateral wall of cavernous sinus (just inferior to CN III)
│   │   ├─ Length: 20-25 mm
│   │   ├─ Relations in CS:
│   │   │   ├─ Superior: CN III (above)
│   │   │   ├─ Inferior: V1 (ophthalmic division of CN V)
│   │   │   ├─ Medial: ICA
│   │   │   └─ Lateral: Middle cranial fossa dura
│   │   │
│   │   └─ Pathology:
│   │       ├─ CS meningioma: May compress CN IV
│   │       ├─ CS thrombosis: CN IV palsy possible
│   │       └─ Tolosa-Hunt syndrome: Painful ophthalmoplegia (granulomatous inflammation of CS)
│   │
│   ├─ 3. Orbital Segment
│   │   ├─ Entry: Superior orbital fissure (OUTSIDE annulus of Zinn - unlike CN III)
│   │   ├─ Course: Superior to orbit, crosses over superior rectus
│   │   ├─ Target: Superior oblique muscle
│   │   └─ Trochlea: Pulley structure in superomedial orbit (superior oblique tendon loops through)
│   │
│   └─ Foramen: Superior orbital fissure (outside tendinous ring)
│
├─ Function
│   └─ Motor: Superior oblique muscle only
│       ├─ Primary action (eye abducted): Depression
│       ├─ Secondary action: Intorsion (top of eye rotates medially)
│       ├─ Tertiary action: Abduction (minor)
│       └─ Most important for: Downward gaze when eye is adducted (e.g., reading, walking downstairs)
│
├─ Testing
│   ├─ Eye movements:
│   │   ├─ "Look down and in" (best test of superior oblique)
│   │   ├─ H-pattern testing
│   │   └─ Diplopia assessment (vertical or oblique diplopia)
│   │
│   ├─ Head tilt test (Parks-Bielschowsky test):
│   │   ├─ Tilt head to right: Left hypertropia worsens (left CN IV lesion)
│   │   ├─ Tilt head to left: Right hypertropia worsens (right CN IV lesion)
│   │   └─ Mechanism: Unopposed inferior oblique causes hypertropia
│   │
│   └─ Subjective diplopia:
│       ├─ Vertical or oblique (tilted) diplopia
│       ├─ Worse with downgaze (reading, stairs)
│       └─ Compensatory head tilt (away from affected side)
│
├─ Clinical Syndromes
│   ├─ Isolated CN IV Palsy:
│   │   ├─ Hypertropia (vertical deviation): Affected eye higher
│   │   ├─ Excyclotorsion: Top of eye rotated laterally (absent intorsion)
│   │   ├─ Diplopia: Vertical/oblique, worse looking down and in
│   │   ├─ Compensatory: Head tilt to opposite shoulder
│   │   └─ Causes: Trauma (most common), microvascular, tumor, aneurysm (rare)
│   │
│   ├─ Bilateral CN IV Palsy:
│   │   ├─ Often from head trauma (thin nerve, long course, decussation injury)
│   │   ├─ Signs:
│   │   │   ├─ Bilateral hypertropia (alternating)
│   │   │   ├─ Large excyclotorsion (>10°) - best clue to bilateral involvement
│   │   │   ├─ V-pattern esotropia
│   │   │   └─ Difficulty reading (downward gaze impaired)
│   │   └─ Management: Prism glasses, or surgical correction (superior oblique tuck)
│   │
│   ├─ Congenital CN IV Palsy:
│   │   ├─ Often unrecognized until adulthood
│   │   ├─ Clues:
│   │   │   ├─ Longstanding head tilt in old photographs
│   │   │   ├─ Large vertical fusional amplitudes (compensation)
│   │   │   └─ Facial asymmetry (chronic head tilt)
│   │   └─ Treatment: Observation if compensated, or surgery if symptomatic
│   │
│   └─ Neurological localization:
│       ├─ Nuclear/fascicular (midbrain):
│       │   ├─ Crossed palsy: Right nucleus lesion → left CN IV palsy
│       │   └─ Associated signs: Horner's, ataxia, CN III involvement
│       │
│       ├─ Cisternal:
│       │   ├─ Trauma (most common): Shearing injury
│       │   ├─ SCA aneurysm/compression (rare)
│       │   └─ Tentorial meningioma
│       │
│       ├─ Cavernous:
│       │   ├─ CN IV + other CS nerves (III, V1, V2, VI)
│       │   └─ Painful if CS inflammation (Tolosa-Hunt)
│       │
│       └─ Orbital:
│           ├─ CN IV + orbital signs
│           └─ Trauma, tumor, inflammation
│
└─ Surgical Considerations
    ├─ Trauma:
    │   ├─ Most common CN injured in closed head injury
    │   ├─ Long, thin course + tethering at brainstem and SOF = shearing injury
    │   ├─ Often bilateral (decussation injury)
    │   └─ Recovery: May improve over 6-12 months, but often incomplete
    │
    ├─ Microvascular decompression:
    │   ├─ Indication: CN IV compression by SCA
    │   ├─ Approach: Infratentorial supracerebellar (sitting or lateral position)
    │   ├─ Technique: Identify CN IV on superior surface of cerebellum, interpose Teflon
    │   └─ Success rate: 80-90% for vascular compression syndrome
    │
    ├─ Tentorial meningiomas:
    │   ├─ May compress CN IV against tentorium
    │   ├─ Approach: Subtemporal, or supratentorial-infratentorial combined
    │   └─ Prognosis: CN IV may recover if decompressed early
    │
    └─ Surgical palsy (iatrogenic):
        ├─ Can occur during:
        │   ├─ Petrosal approach (CN IV in tentorial edge)
        │   ├─ Resection of tentorial meningioma
        │   └─ Trauma during retraction (thin nerve)
        └─ Prevention: Early identification, gentle handling, avoid excessive retraction
```

## CN VI - ABDUCENS NERVE

```
├─ Type: Motor (to lateral rectus muscle only)
│
├─ Unique Features
│   ├─ Longest intracranial course after CN IV
│   ├─ Only nerve running WITHIN cavernous sinus (in blood!)
│   ├─ Most vulnerable CN to increased ICP (false localizing sign)
│   └─ Sharp angle around petrous apex (Dorello's canal)
│
├─ Origin & Nucleus
│   ├─ Motor nucleus: Dorsal pons (pontomedullary junction)
│   │   ├─ Location: Beneath floor of 4th ventricle
│   │   ├─ Facial colliculus: CN VII loops around CN VI nucleus (surface landmark)
│   │   └─ Paramedian pontine reticular formation (PPRF) adjacent
│   │
│   ├─ Exit: Pontomedullary junction (ventral surface)
│   │   └─ Medial to CN VII-VIII complex
│   │
│   └─ Internuclear connections:
│       ├─ MLF (medial longitudinal fasciculus)
│       ├─ Coordinates: CN III (medial rectus) with CN VI (lateral rectus)
│       └─ Horizontal conjugate gaze: Both eyes move together
│
├─ Segments
│   ├─ 1. Cist

ernal (Pontine Cistern to Dorello's Canal)
│   │   ├─ Course: Anterosuperiorly from pontomedullary junction
│   │   ├─ Length: 15-20 mm
│   │   ├─ Relations:
│   │   │   ├─ Basilar artery: Runs lateral to basilar (often compressed by vessel)
│   │   │   ├─ Inferior: Anterior inferior cerebellar artery (AICA)
│   │   │   ├─ Lateral: CN VII-VIII complex, petrosal vein
│   │   │   └─ Ascends: Along clivus toward petrous apex
│   │   │
│   │   ├─ Vulnerability:
│   │   │   ├─ Increased ICP: CN VI has longest cisternal course
│   │   │   ├─ Stretching: Downward brainstem herniation stretches nerve
│   │   │   ├─ "False localizing sign": CN VI palsy doesn't localize lesion
│   │   │   └─ Compression: Mass in posterior fossa, hydrocephalus
│   │   │
│   │   └─ Dorello's Canal Entry
│       │       ├─ Location: Petrous apex (beneath petrosphenoidal ligament - Gruber's ligament)
│       │       ├─ Sharp angle: ~120° turn around petrous apex
│       │       ├─ Fixed: Nerve tethered at Dorello's canal
│       │       └─ Petrous apicitis (Gradenigo syndrome): Inflammation → CN VI palsy
│       │
│   ├─ 2. Dorello's Canal (Petrous Apex)
│   │   ├─ Location: Beneath petrosphenoidal ligament (Gruber's ligament)
│   │   ├─ Length: 5-10 mm
│   │   ├─ Contents: CN VI + inferior petrosal sinus (IPS)
│   │   ├─ Relations:
│   │   │   ├─ Inferior: Inferior petrosal sinus
│   │   │   ├─ Superior: Petrosphenoidal ligament
│   │   │   └─ Lateral: Mastoid air cells, petrous apex bone
│   │   │
│   │   └─ Clinical:
│   │       ├─ Gradenigo syndrome (petrous apicitis):
│   │       │   ├─ Triad: CN VI palsy + facial pain (CN V) + otitis media
│   │       │   ├─ Cause: Infection spreads from middle ear to petrous apex
│   │       │   └─ Treatment: Antibiotics, drainage if abscess
│   │       │
│   │       └─ Petrous apex tumors: CN V + CN VI palsies
│   │
│   ├─ 3. Cavernous Sinus Segment
│   │   ├─ Course: Lateral to ICA, WITHIN sinus (not in lateral wall!)
│   │   ├─ Length: 20-25 mm
│   │   ├─ Location: Medial and inferior (within venous blood)
│   │   │
│   │   ├─ Relations:
│   │   │   ├─ Medial: ICA (immediately lateral to artery)
│   │   │   ├─ Superior: CN III in lateral wall (above)
│   │   │   ├─ Lateral: CN IV, V1, V2 in lateral wall
│   │   │   └─ Surrounded: By venous blood (unique among CNs!)
│   │   │
│   │   ├─ Vulnerability (HIGHEST of all CNs in cavernous sinus):
│   │   │   ├─ Cavernous sinus thrombosis: CN VI affected first (within blood)
│   │   │   ├─ Mass effect: Any CS lesion compresses CN VI early
│   │   │   ├─ Carotid-cavernous fistula: Venous hypertension → CN VI palsy
│   │   │   └─ Pituitary apoplexy: Lateral extension → CN VI compression
│   │   │
│   │   └─ Pathology affecting CN VI in CS:
│   │       ├─ Cavernous sinus meningioma
│   │       ├─ Carotid aneurysm (cavernous ICA)
│   │       ├─ Nasopharyngeal carcinoma (perineural spread)
│   │       └─ Tolosa-Hunt syndrome (painful ophthalmoplegia)
│   │
│   └─ 4. Orbital Segment
│       ├─ Entry: Superior orbital fissure (within annulus of Zinn)
│       ├─ Course: Directly to lateral rectus muscle (medial surface)
│       ├─ Target: Lateral rectus muscle (sole innervation)
│       └─ No branches (simplest motor nerve - single muscle)
│
├─ Foramen: Superior orbital fissure (within tendinous ring - annulus of Zinn)
│
├─ Function
│   └─ Motor: Lateral rectus muscle only
│       ├─ Eye abduction (turn eye outward, away from nose)
│       └─ Horizontal gaze (with CN III medial rectus for conjugate gaze)
│
├─ Testing
│   ├─ Eye movements: "Look to the right, look to the left"
│   ├─ H-pattern testing
│   ├─ Diplopia assessment: Horizontal diplopia (worse looking to affected side)
│   └─ Abduction failure: Eye cannot move laterally past midline
│
├─ Clinical Syndromes
│   ├─ Complete CN VI Palsy:
│   │   ├─ Eye position: Esotropia (eye deviated medially - adducted)
│   │   ├─ Unopposed: Medial rectus (CN III) pulls eye inward
│   │   ├─ Abduction failure: Cannot move eye laterally
│   │   ├─ Diplopia: Horizontal, worse in far gaze to affected side
│   │   └─ Head turn: To affected side (compensatory - reduces diplopia)
│   │
│   ├─ Causes by Location:
│   │   ├─ Nuclear/fascicular (pons):
│   │   │   ├─ Stroke: Pontine infarct (often with hemiparesis)
│   │   │   ├─ Demyelination: MS plaque
│   │   │   ├─ Wernicke's encephalopathy: Thiamine deficiency
│   │   │   └─ Associated: PPRF involvement → horizontal gaze palsy (not just abduction)
│   │   │
│   │   ├─ Cisternal:
│   │   │   ├─ Increased ICP: Most common (false localizing sign)
│   │   │   │   └─ Mechanism: Downward pressure → stretching over petrous apex
│   │   │   ├─ Meningitis: Bacterial, carcinomatous
│   │   │   ├─ Trauma: Skull base fracture, petrous fracture
│   │   │   └─ Chiari malformation: Brainstem compression
│   │   │
│   │   ├─ Dorello's canal:
│   │   │   ├─ Gradenigo syndrome (petrous apicitis): CN VI + CN V pain + otitis
│   │   │   ├─ Petrous apex tumor: Chondrosarcoma, chordoma
│   │   │   └─ Petrous fracture: Trauma
│   │   │
│   │   ├─ Cavernous sinus:
│   │   │   ├─ Cavernous sinus thrombosis: CN VI first, then III, IV, V
│   │   │   ├─ Carotid-cavernous fistula: Pulsatile proptosis, bruit
│   │   │   ├─ Cavernous sinus meningioma
│   │   │   ├─ Pituitary apoplexy: Sudden headache, vision loss, CN VI palsy
│   │   │   └─ Nasopharyngeal carcinoma: Perineural spread
│   │   │
│   │   └─ Orbital:
│   │       ├─ Orbital tumor
│   │       ├─ Orbital trauma
│   │       └─ Thyroid eye disease (rare - usually mechanical restriction)
│   │
│   ├─ Bilateral CN VI Palsy:
│   │   ├─ Causes: Increased ICP, basilar tip aneurysm, pontine lesion
│   │   ├─ Presentation: Bilateral esotropia, cannot abduct either eye
│   │   └─ Diplopia: Horizontal, worse in far gaze (both directions)
│   │
│   ├─ Microvascular (Vasculopathic) CN VI Palsy:
│   │   ├─ Risk factors: Diabetes, hypertension, age >50
│   │   ├─ Mechanism: Ischemia to nerve (vasa nervorum occlusion)
│   │   ├─ Presentation: Sudden onset, isolated CN VI palsy, no other deficits
│   │   ├─ Recovery: Usually spontaneous over 3-6 months
│   │   └─ Workup: If typical (age, risk factors), observe; if atypical → MRI
│   │
│   └─ Duane Retraction Syndrome:
│       ├─ Congenital absence of CN VI nucleus
│       ├─ CN III aberrantly innervates lateral rectus
│       ├─ Presentation: Limited abduction + globe retraction on adduction
│       └─ Not a true CN VI palsy (developmental anomaly)
│
└─ Surgical Considerations
    ├─ Petrous apex surgery:
    │   ├─ Dorello's canal: Identify and protect CN VI
    │   ├─ Petrous apicectomy: Risk CN VI injury
    │   └─ Recovery: Variable if nerve stretched but not transected
    │
    ├─ Cavernous sinus surgery:
    │   ├─ CN VI most vulnerable (within blood, adjacent to ICA)
    │   ├─ Often cannot preserve if extensive tumor
    │   └─ Postop CN VI palsy: Common, may improve over 6-12 months
    │
    ├─ Clivus/skull base surgery:
    │   ├─ Transclival approach: CN VI at risk
    │   └─ Midline lesions: Can have bilateral CN VI injury
    │
    └─ Management of CN VI palsy:
        ├─ Acute: Patch eye or prism glasses (reduce diplopia)
        ├─ Subacute: Observe 6 months (may recover)
        ├─ Chronic (>6 mo, no recovery): Strabismus surgery
        │   ├─ Recession-resection: Weaken medial rectus, strengthen lateral rectus
        │   └─ Transposition procedures: Vertical recti to assist abduction
        └─ Botulinum toxin: Temporary weakening of medial rectus
```

## CN VII - FACIAL NERVE

```
├─ Type: Mixed - Motor (major) + Sensory + Parasympathetic
│
├─ Components (Complex!)
│   ├─ Motor (Branchial motor - special visceral efferent):
│   │   └─ Muscles of facial expression (all)
│   │
│   ├─ Sensory (Special sensory - taste):
│   │   └─ Taste from anterior 2/3 tongue (via chorda tympani)
│   │
│   ├─ Parasympathetic (General visceral efferent):
│   │   ├─ Lacrimation (lacrimal gland - via greater petrosal nerve)
│   │   ├─ Salivation (submandibular/sublingual glands - via chorda tympani)
│   │   └─ Nasal/palatine glands (via greater petrosal nerve → pterygopalatine ganglion)
│   │
│   └─ Sensory (General sensory):
│       └─ Small area external ear (concha, EAC)
│
├─ Origin & Nuclei
│   ├─ Motor nucleus:
│   │   ├─ Location: Ventrolateral pons (pontine tegmentum)
│   │   ├─ Organization:
│   │   │   ├─ Upper face (forehead): BILATERAL cortical input
│   │   │   └─ Lower face (below eyes): CONTRALATERAL cortical input only
│   │   └─ Clinical: UMN lesion spares forehead, LMN lesion affects entire face
│   │
│   ├─ Superior salivatory nucleus:
│   │   ├─ Parasympathetic preganglionic neurons
│   │   └─ Location: Pontine tegmentum (rostral to inferior salivatory nucleus)
│   │
│   ├─ Nucleus of solitary tract (gustatory portion):
│   │   ├─ Taste information
│   │   └─ Rostral portion (anterior to CN IX/X taste input)
│   │
│   └─ Spinal trigeminal nucleus:
│       └─ General sensory from ear (small contribution)
│
├─ Segments (7 Segments - Complex Course!)
│   │
│   ├─ 1. Cisternal (Pontomedullary Junction to IAC)
│   │   ├─ Origin: Pontomedullary junction (cerebellopontine angle)
│   │   ├─ Two roots emerge:
│   │   │   ├─ Motor root (larger, lateral)
│   │   │   └─ Nervus intermedius (Wrisberg) - sensory/parasympathetic (smaller, medial)
│   │   │
│   │   ├─ Course: Laterally across CPA cistern with CN VIII
│   │   ├─ Length: 15-24 mm
│   │   │
│   │   ├─ Relations:
│   │   │   ├─ CN VIII: Immediately adjacent (CN VII superior-anterior, CN VIII inferior-posterior)
│   │   │   ├─ AICA: Loops in CPA, may compress nerves (hemifacial spasm, tinnitus)
│   │   │   ├─ Flocculus (cerebellum): Posteroinferior
│   │   │   └─ Pons: Medial
│   │   │
│   │   ├─ Entry: Internal auditory canal (IAC) - porus acusticus
│   │   │
│   │   └─ Clinical:
│   │       ├─ CPA tumors: Acoustic neuroma (most common), meningioma, epidermoid
│   │       ├─ CN VII + CN VIII dysfunction: Hearing loss, facial weakness, tinnitus, vertigo
│   │       └─ Hemifacial spasm: AICA compression of CN VII at REZ → involuntary facial contractions
│   │
│   ├─ 2. Meatal (Internal Auditory Canal)
│   │   ├─ Course: Lateral through IAC with CN VIII
│   │   ├─ Length: 8-10 mm
│   │   ├─ Organization within IAC:
│   │   │   ├─ Anterior-superior: CN VII
│   │   │   ├─ Anterior-inferior: Cochlear nerve
│   │   │   ├─ Posterior-superior: Superior vestibular nerve
│   │   │   └─ Posterior-inferior: Inferior vestibular nerve
│   │   │
│   │   ├─ Labyrinthine artery: Travels with nerves in IAC
│   │   │
│   │   └─ Clinical:
│   │       ├─ Acoustic neuroma: Originates from CN VIII but compresses CN VII
│   │       ├─ Surgical exposure: Translabyrinthine or retrosigmoid approach
│   │       └─ Hearing preservation: Depends on approach and tumor size
│   │
│   ├─ 3. Labyrinthine (Narrowest Segment!)
│   │   ├─ Course: IAC fundus → geniculate ganglion
│   │   ├─ Length: 3-5 mm
│   │   ├─ Diameter: 0.61 mm (NARROWEST part of entire nerve!)
│   │   │
│   │   ├─ Relations:
│   │   │   ├─ Cochlea: Anterior and inferior
│   │   │   ├─ Vestibule: Posterior
│   │   │   └─ Bone: Surrounded by fallopian canal (bony canal)
│   │   │
│   │   └─ Clinical (CRITICAL!):
│   │       ├─ Bell's palsy: Viral inflammation (HSV-1) → swelling → compression in narrow canal
│   │       ├─ "Bottleneck effect": Swelling here causes ischemia → facial paralysis
│   │       ├─ Temporal bone fracture: Labyrinthine segment most vulnerable
│   │       └─ Decompression surgery: Opens bone around this segment
│   │
│   ├─ 4. Geniculate Ganglion (NOT a segment, but key landmark)
│   │   ├─ Location: Sharp anterior bend ("genu" = knee)
│   │   ├─ Contains: Sensory neuron cell bodies (for taste - geniculate ganglion)
│   │   ├─ Branch: Greater petrosal nerve exits here
│   │   │
│   │   └─ Greater Petrosal Nerve (Parasympathetic Branch)
│   │       ├─ Course: Anteromedially from geniculate ganglion
│   │       ├─ Hiatus: Exits via hiatus of facial canal (middle cranial fossa)
│   │       ├─ Joins: Deep petrosal nerve (sympathetics from ICA plexus) → forms Vidian nerve
│   │       ├─ Destination: Pterygopalatine ganglion (via pterygoid canal)
│   │       ├─ Synapse: Preganglionic CN VII → postganglionic at pterygopalatine ganglion
│   │       │
│   │       └─ Function:
│   │           ├─ Lacrimation: Lacrimal gland (via V2 zygomatic → V1 lacrimal)
│   │           ├─ Nasal secretion: Nasal glands
│   │           └─ Palatine secretion: Palatine glands
│   │
│   ├─ 5. Tympanic (Horizontal Segment)
│   │   ├─ Course: Posterolaterally along medial wall of middle ear
│   │   ├─ Length: 11-13 mm
│   │   │
│   │   ├─ Relations:
│   │   │   ├─ Superior: Lateral semicircular canal
│   │   │   ├─ Inferior: Oval window, stapes footplate
│   │   │   ├─ Anterior: Cochlea (cochleariform process)
│   │   │   └─ Medial: Bone (fallopian canal)
│   │   │
│   │   ├─ Branch: Nerve to stapedius
│   │   │   ├─ Exits: From tympanic segment
│   │   │   ├─ Target: Stapedius muscle
│   │   │   └─ Function: Stapedius reflex (dampens loud sounds, protects inner ear)
│   │   │
│   │   └─ Clinical:
│   │       ├─ Chronic otitis media: Can erode fallopian canal → facial weakness
│   │       ├─ Cholesteatoma: Erosion of bone → CN VII exposure/injury
│   │       ├─ Middle ear surgery: CN VII at risk (identify before drilling)
│   │       └─ Hyperacusis: If nerve to stapedius injured (sounds abnormally loud)
│   │
│   ├─ 6. Mastoid (Vertical/Descending Segment)
│   │   ├─ Course: Descends vertically in mastoid portion of temporal bone
│   │   ├─ Length: 10-14 mm
│   │   │
│   │   ├─ Relations:
│   │   │   ├─ Lateral: Mastoid air cells
│   │   │   ├─ Medial: Medial wall of tympanic cavity
│   │   │   ├─ Posterior: Tympanic sinus
│   │   │   └─ Anterior: Tympanic membrane (posterior)
│   │   │
│   │   ├─ Branch: Chorda Tympani (Parasympathetic + Taste)
│   │   │   ├─ Origin: Exits CN VII ~6 mm above stylomastoid foramen
│   │   │   ├─ Course: Anterosuperiorly → enters middle ear cavity
│   │   │   │   └─ Crosses: Medial surface of tympanic membrane (behind eardrum)
│   │   │   │   └─ Between: Handle of malleus and long process of incus
│   │   │   │
│   │   │   ├─ Exit middle ear: Petrotympanic fissure (Glaserian fissure)
│   │   │   ├─ Joins: Lingual nerve (CN V3) in infratemporal fossa
│   │   │   │
│   │   │   ├─ Components:
│   │   │   │   ├─ Taste: Anterior 2/3 of tongue (special sensory)
│   │   │   │   └─ Parasympathetic: Submandibular + sublingual glands (salivation)
│   │   │   │
│   │   │   ├─ Pathway for salivation:
│   │   │   │   ├─ Preganglionic: CN VII → chorda tympani → lingual nerve
│   │   │   │   ├─ Synapse: Submandibular ganglion
│   │   │   │   └─ Postganglionic: To submandibular + sublingual glands
│   │   │   │
│   │   │   └─ Clinical:
│   │   │       ├─ Middle ear surgery: Chorda tympani often sacrificed
│   │   │       ├─ Results: Loss of taste anterior 2/3 tongue + decreased salivation
│   │   │       ├─ Tympanoplasty: Risk chorda tympani injury
│   │   │       └─ Metallic taste: Regeneration phenomenon
│   │   │
│   │   └─ Other branches:
│   │       └─ Posterior auricular nerve (exits near stylomastoid foramen)
│   │           ├─ Motor: Occipitalis muscle, posterior auricular muscle
│   │           └─ Sensory: Small area behind ear
│   │
│   └─ 7. Extratemporal (After Stylomastoid Foramen)
│       ├─ Exit: Stylomastoid foramen (between styloid and mastoid processes)
│       │
│       ├─ Branches immediately after exit:
│       │   ├─ Posterior auricular nerve (see above)
│       │   ├─ Nerve to posterior belly of digastric
│       │   └─ Nerve to stylohyoid muscle
│       │
│       ├─ Entry into parotid gland:
│       │   ├─ Passes lateral to styloid process
│       │   ├─ Enters: Posterior surface of parotid gland
│       │   ├─ Superficial to: Retromandibular vein, external carotid artery
│       │   └─ Divides: Within parotid into terminal branches
│       │
│       └─ Terminal Branches (5-6 major branches)
│           ├─ Temporal (Frontal) Branch:
│           │   ├─ Course: Superiorly across zygomatic arch
│           │   ├─ Motor:
│           │   │   ├─ Frontalis (forehead elevation, eyebrow raise)
│           │   │   ├─ Orbicularis oculi (superior portion - eye closure)
│           │   │   └─ Corrugator supercilii (eyebrow depression - frowning)
│           │   │
│           │   └─ Clinical: Injury → inability to raise eyebrow, incomplete eye closure (lagophthalmos risk)
│           │
│           ├─ Zygomatic Branch:
│           │   ├─ Course: Across zygoma
│           │   ├─ Motor:
│           │   │   ├─ Orbicularis oculi (lateral portion - eye closure)
│           │   │   └─ Muscles of lateral face
│           │   │
│           │   └─ Anastomoses with temporal branch (protective redundancy)
│           │
│           ├─ Buccal Branch(es):
│           │   ├─ Usually 2-3 branches
│           │   ├─ Motor:
│           │   │   ├─ Buccinator (cheek compression - chewing, whistling)
│           │   │   ├─ Orbicularis oris (upper lip - lip closure)
│           │   │   ├─ Levator anguli oris (smile - elevate angle of mouth)
│           │   │   └─ Zygomaticus major/minor (smile - elevate upper lip)
│           │   │
│           │   └─ Multiple branches: Good anastomoses (protective)
│           │
│           ├─ Marginal Mandibular Branch:
│           │   ├─ Course: Along inferior border of mandible
│           │   ├─ Relationship to mandible:
│           │   │   ├─ Runs 1-2 cm BELOW inferior border (usually)
│           │   │   ├─ Can be up to 4 cm below in some people
│           │   │   └─ At risk during submandibular gland surgery, neck dissection
│           │   │
│           │   ├─ Motor:
│           │   │   ├─ Depressor anguli oris (depress corner of mouth - frown)
│           │   │   ├─ Depressor labii inferioris (depress lower lip)
│           │   │   ├─ Mentalis (chin elevation - pouting)
│           │   │   └─ Orbicularis oris (lower lip)
│           │   │
│           │   └─ Clinical:
│           │       ├─ Injury: Asymmetric smile (drooping corner on affected side)
│           │       ├─ Preservation: Key goal in parotidectomy, neck surgery
│           │       └─ Identification: Intraoperative nerve monitoring
│           │
│           └─ Cervical Branch:
│               ├─ Course: Inferiorly into neck
│               ├─ Motor: Platysma (neck skin movement)
│               └─ Clinical: Least clinically important branch
```
