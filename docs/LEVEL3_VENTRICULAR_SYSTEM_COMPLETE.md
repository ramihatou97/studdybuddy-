# LEVEL 3: VENTRICULAR SYSTEM - ULTRA-GRANULAR SPECIFICATION

**Purpose**: Complete anatomical specification of the ventricular system for StudyBuddy neurosurgical knowledge base

**Scope**:
- Four ventricles (lateral, third, aqueduct, fourth)
- Choroid plexus anatomy and CSF production
- CSF flow pathways and dynamics
- Hydrocephalus classification and pathophysiology
- Ventricular access points and surgical techniques
- Clinical pathology and syndromes
- Surgical approaches to each ventricle

**Format**: Ultra-granular hierarchical tree structure for easy extraction

---

## OVERVIEW: THE VENTRICULAR SYSTEM

```
├─ Components
│   ├─ Lateral ventricles (2): Largest, C-shaped, in cerebral hemispheres
│   ├─ Third ventricle (1): Midline, between thalami
│   ├─ Cerebral aqueduct (1): Narrow channel connecting 3rd → 4th ventricle
│   └─ Fourth ventricle (1): Posterior fossa, between pons/medulla and cerebellum
│
├─ Total CSF Volume
│   ├─ Ventricular: ~25 mL (20%)
│   ├─ Subarachnoid space: ~100 mL (80%)
│   └─ Total: ~125-150 mL
│
├─ CSF Production
│   ├─ Rate: 20 mL/hour (480 mL/day) - entire volume replaced 3-4x daily
│   ├─ Source: Choroid plexus (70%), ependyma + capillaries (30%)
│   └─ Total daily production: ~500 mL/day
│
├─ CSF Absorption
│   ├─ Primary: Arachnoid granulations (into superior sagittal sinus)
│   ├─ Secondary: Spinal nerve root sheaths, ependymal absorption
│   └─ Rate: Matches production (20 mL/hr) to maintain constant volume
│
└─ Functions
    ├─ Mechanical: Buoyancy (brain weight 1400g → effective 50g in CSF)
    ├─ Protection: Shock absorption, cushioning
    ├─ Chemical stability: Ion homeostasis (K+, Ca2+, Mg2+)
    ├─ Waste removal: Metabolic byproducts, proteins
    └─ Nutrient transport: Glucose, hormones
```

## 1. LATERAL VENTRICLES (Ventriculi Laterales)

```
├─ Overview
│   ├─ Number: Two (right and left)
│   ├─ Location: Within cerebral hemispheres
│   ├─ Shape: C-shaped (follows arc of corpus callosum)
│   ├─ Size: Largest ventricles (~10 mL each)
│   └─ Symmetry: Usually symmetric, but asymmetry common (can be normal variant)
│
├─ Divisions (5 parts)
│   ├─ 1. BODY (Corpus / Central Part):
│   │   ├─ Location: Frontal/parietal lobe (from foramen of Monro to splenium)
│   │   ├─ Roof: Corpus callosum
│   │   ├─ Floor:
│   │   │   ├─ Medial: Body of caudate nucleus (prominent bulge)
│   │   │   ├─ Lateral: Dorsal surface of thalamus
│   │   │   └─ Choroid plexus: Along choroid fissure (thalamus + fornix)
│   │   ├─ Medial wall: Septum pellucidum (separates left from right)
│   │   └─ Lateral wall: Corpus callosum (radiating fibers)
│   │
│   ├─ 2. FRONTAL (ANTERIOR) HORN:
│   │   ├─ Location: Frontal lobe (anterior to foramen of Monro)
│   │   ├─ Extends: From foramen of Monro → frontal pole
│   │   ├─ Roof + anterior wall: Corpus callosum (genu + rostrum)
│   │   ├─ Floor: Rostrum of corpus callosum
│   │   ├─ Medial wall: Septum pellucidum
│   │   ├─ Lateral wall: Head of caudate nucleus (prominent convex bulge)
│   │   │
│   │   └─ Clinical:
│   │       ├─ VP shunt placement: Anterior horn commonly targeted (Kocher's point)
│   │       ├─ Frontal horn ratio: FH/skull width - measures hydrocephalus severity
│   │       └─ Caudate indentation: Lost in hydrocephalus (ventricular dilation)
│   │
│   ├─ 3. OCCIPITAL (POSTERIOR) HORN:
│   │   ├─ Location: Occipital lobe
│   │   ├─ Extends: Posteriorly from trigone (variable length - 0-30 mm)
│   │   ├─ Roof: Corpus callosum (tapetum - thin)
│   │   ├─ Medial wall:
│   │   │   ├─ Bulb of posterior horn: Forceps major (corpus callosum fibers)
│   │   │   └─ Calcar avis: Medial eminence (from calcarine sulcus)
│   │   ├─ Lateral wall: Optic radiation fibers (tapetum)
│   │   │
│   │   ├─ Variability: Highly variable (can be asymmetric or absent)
│   │   │
│   │   └─ Clinical:
│   │       ├─ Optic radiation: Runs in lateral wall (injury → homonymous hemianopsia)
│   │       └─ Occipital horn syndrome: Rare (collapsin deficiency - X-linked)
│   │
│   ├─ 4. TEMPORAL (INFERIOR) HORN:
│   │   ├─ Location: Temporal lobe (curves inferiorly + anteriorly)
│   │   ├─ Extends: From trigone → temporal pole (35-40 mm length)
│   │   ├─ Roof:
│   │   │   ├─ Tail of caudate nucleus (follows along roof)
│   │   │   ├─ Stria terminalis (small bundle)
│   │   │   └─ Tapetum (corpus callosum fibers)
│   │   ├─ Floor:
│   │   │   ├─ Hippocampus: Prominent medial elevation ⭐
│   │   │   └─ Collateral eminence: Lateral elevation (from collateral sulcus)
│   │   ├─ Medial wall:
│   │   │   ├─ Choroid plexus: Extends into temporal horn (prominent)
│   │   │   ├─ Choroid fissure: Between fimbria (fornix) and thalamus
│   │   │   └─ Fimbria of fornix: White matter tract along hippocampus
│   │   │
│   │   ├─ Relations:
│   │   │   ├─ Lateral: Temporal lobe white matter
│   │   │   ├─ Medial: Ambient cistern, uncus, parahippocampal gyrus
│   │   │   ├─ Anterior: Amygdala (terminates near amygdala)
│   │   │   └─ Inferior: Middle cerebral artery branches (in Sylvian fissure)
│   │   │
│   │   └─ Clinical:
│   │       ├─ Mesial temporal sclerosis: Hippocampal atrophy → temporal horn dilation
│   │       ├─ Temporal lobectomy: For epilepsy (must preserve choroid plexus pedicle)
│   │       ├─ Trans-temporal approach: Access posterior 3rd ventricle, pineal region
│   │       └─ Meyer's loop: Optic radiation (anterior temporal - injury → "pie in sky" defect)
│   │
│   └─ 5. ATRIUM (TRIGONE / COLLATERAL TRIGONE):
│       ├─ Location: Junction of body, occipital horn, temporal horn
│       ├─ Shape: Triangular confluence
│       ├─ Roof: Corpus callosum
│       ├─ Floor: Collateral trigone (from collateral sulcus impression)
│       ├─ Medial:
│       │   ├─ Pulvinar of thalamus (medial prominence)
│       │   └─ Crus of fornix
│       ├─ Choroid plexus: Most prominent here (largest glomus)
│       │
│       └─ Clinical:
│           ├─ Choroid plexus papilloma: Often arises here (children)
│           ├─ Glomus: Highly vascular (ACA, PCA supply) - bleeding risk in surgery
│           ├─ Trigone meningioma: Can arise from choroid plexus stroma
│           └─ Posterior transcortical approach: Access trigone lesions (parieto-occipital)
│
├─ Boundaries Summary
│   ├─ Superior: Corpus callosum (entire roof)
│   ├─ Inferior: Caudate, thalamus, hippocampus (depends on region)
│   ├─ Medial: Septum pellucidum (anteriorly), fornix (posteriorly)
│   ├─ Lateral: White matter of hemispheres
│   └─ Anterior: Genu of corpus callosum
│
├─ Communications
│   ├─ Foramen of Monro (Interventricular Foramen) ⭐⭐⭐:
│   │   ├─ Location: Anteromedial wall of lateral ventricle body
│   │   ├─ Connects: Each lateral ventricle → third ventricle
│   │   ├─ Boundaries:
│   │   │   ├─ Anterior: Column of fornix
│   │   │   ├─ Posterior: Anterior tubercle of thalamus
│   │   │   ├─ Superior: Body of fornix
│   │   │   └─ Inferior: Hypothalamus
│   │   ├─ Size: ~3-4 mm diameter (narrow - can obstruct easily)
│   │   │
│   │   ├─ Contents:
│   │   │   ├─ Choroidal arteries: Anterior choroidal artery (AChA), lateral posterior choroidal artery
│   │   │   ├─ Choroid plexus: Continuous from lateral → 3rd ventricle
│   │   │   └─ Thalamostriate vein: Courses along ventricular floor → joins internal cerebral vein
│   │   │
│   │   └─ Clinical:
│   │       ├─ Obstruction:
│   │       │   ├─ Colloid cyst of 3rd ventricle: "Ball-valve" obstruction (classic ⭐⭐⭐)
│   │       │   ├─ Foramen of Monro glioma: Subependymal giant cell astrocytoma (tuberous sclerosis)
│   │       │   └─ Hydrocephalus: Unilateral or bilateral (depending on obstruction)
│   │       ├─ Endoscopic third ventriculostomy (ETV): Pass endoscope through Monro foramen
│   │       └─ VP shunt: Catheter must pass through Monro to drain both lateral ventricles
│   │
│   └─ No direct communication between left and right lateral ventricles:
│       └─ Separated by septum pellucidum (can be absent - cavum septum pellucidum variant)
│
├─ Choroid Plexus of Lateral Ventricle ⭐⭐
│   ├─ Location:
│   │   ├─ Extends: From foramen of Monro → along body → atrium → temporal horn
│   │   ├─ Does NOT extend: Into frontal horn or occipital horn
│   │   └─ Largest at: Atrium (glomus choroideum)
│   │
│   ├─ Attachment:
│   │   ├─ Choroid fissure: Between fornix (superiorly) and thalamus (inferiorly)
│   │   ├─ Tela choroidea: Double layer of pia mater (forms roof)
│   │   └─ Ependyma: Lines ventricular surface
│   │
│   ├─ Blood supply:
│   │   ├─ Anterior choroidal artery (AChA) - ICA branch: Anterior 2/3
│   │   ├─ Lateral posterior choroidal artery (LPChA) - PCA branch: Posterior 1/3, atrium
│   │   └─ Highly vascular: Can bleed significantly during surgery
│   │
│   ├─ Venous drainage:
│   │   ├─ Choroid veins → superior choroidal vein
│   │   └─ Drains into: Internal cerebral vein (at foramen of Monro)
│   │
│   └─ Clinical:
│       ├─ Choroid plexus papilloma: Benign (WHO grade I), children > adults
│       │   ├─ Location: Trigone (children), 4th ventricle (adults)
│       │   ├─ Hydrocephalus: Overproduction of CSF (1-5x normal)
│       │   └─ Treatment: Surgical resection (cure)
│       ├─ Choroid plexus carcinoma: Malignant (WHO grade III), rare
│       ├─ Choroid plexus cauterization: For hydrocephalus (endoscopic, reduce CSF production)
│       └─ Xanthogranuloma: Degenerative calcification (incidental, elderly)
│
├─ Surgical Access Points
│   ├─ Frontal Horn Access (Anterior Transcortical):
│   │   ├─ Entry point: Kocher's point (1-2 cm anterior to coronal suture, 2-3 cm lateral to midline)
│   │   ├─ Indications:
│   │   │   ├─ VP shunt placement (most common)
│   │   │   ├─ EVD (external ventricular drain) placement
│   │   │   ├─ Colloid cyst resection (transcallosal or transcortical)
│   │   │   └─ Intraventricular hemorrhage evacuation
│   │   ├─ Trajectory: Perpendicular to skull, aim toward ipsilateral medial canthus
│   │   ├─ Depth: 5-6 cm from skull surface
│   │   │
│   │   └─ Complications:
│   │       ├─ Caudate nucleus injury: If too lateral (usually well-tolerated)
│   │       ├─ Hemorrhage: Cortical vessels, subependymal veins
│   │       └─ Seizures: Cortical injury (1-5%)
│   │
│   ├─ Parietal Access (Posterior Transcortical):
│   │   ├─ Entry point: Frazier's point (3 cm posterior + 3 cm lateral to bregma)
│   │   ├─ Indications:
│   │   │   ├─ Atrium lesions (choroid plexus papilloma)
│   │   │   ├─ Posterior 3rd ventricle lesions
│   │   │   └─ Pineal region tumors (posterior transcallosal-transventricular)
│   │   ├─ Trajectory: Aim toward atrium
│   │   │
│   │   └─ Complications:
│   │       ├─ Optic radiation injury: If too lateral (homonymous hemianopsia)
│   │       ├─ Choroid plexus bleeding: Highly vascular at glomus
│   │       └─ Cortical injury: Visual cortex (parieto-occipital)
│   │
│   ├─ Temporal Approach (Transtemporal):
│   │   ├─ Entry: Through middle temporal gyrus (T2)
│   │   ├─ Indications:
│   │   │   ├─ Mesial temporal sclerosis (temporal lobectomy)
│   │   │   ├─ Temporal lobe tumor with ventricular involvement
│   │   │   └─ Access to ambient cistern, posterior 3rd ventricle
│   │   ├─ Anatomy: Enter through roof or lateral wall
│   │   │
│   │   └─ Complications:
│   │       ├─ Meyer's loop injury: Anterior optic radiation ("pie in sky" defect)
│   │       ├─ Hippocampal damage: Memory impairment (if dominant hemisphere)
│   │       ├─ Choroid plexus injury: Bleeding
│   │       └─ Language: If dominant (left) temporal lobe (aphasia)
│   │
│   └─ Transcallosal Approach:
│       ├─ Entry: Through corpus callosum (interhemispheric)
│       ├─ Indications: Midline lesions (colloid cyst, septum pellucidum cyst, 3rd ventricle tumor)
│       ├─ Anatomy: Split corpus callosum in midline (between pericallosal arteries)
│       │
│       └─ Complications:
│           ├─ Disconnection syndrome: If extensive callosal section (rare with small callosotomy)
│           ├─ Venous infarction: Injury to bridging veins draining to superior sagittal sinus
│           └─ Mutism: Bilateral supplementary motor area injury (temporary usually)
│
└─ Clinical Pathology
    ├─ Ventriculomegaly:
    │   ├─ Mild: Frontal horn width 10-15 mm (age-related atrophy vs early hydrocephalus)
    │   ├─ Moderate: 15-20 mm
    │   └─ Severe: >20 mm (overt hydrocephalus)
    │
    ├─ Asymmetry:
    │   ├─ Normal variant: Up to 20% population (usually benign)
    │   ├─ Pathologic:
    │   │   ├─ Unilateral obstruction (foramen of Monro)
    │   │   ├─ Hemispheric atrophy (ex vacuo dilation)
    │   │   └─ Mass effect (tumor, hemorrhage compressing one ventricle)
    │   │
    │   └─ Radiographic: Septum pellucidum shift indicates mass effect
    │
    ├─ Colpocephaly:
    │   ├─ Definition: Disproportionate dilation of occipital horns
    │   ├─ Causes: Hypoplasia/agenesis of corpus callosum
    │   └─ Associated: Developmental delay, seizures
    │
    └─ Cavum Septum Pellucidum (CSP):
        ├─ Normal variant: Present in 85% of premature infants, 15% of adults
        ├─ Structure: CSF-filled cavity between laminae of septum pellucidum
        ├─ Clinical: Usually asymptomatic, rarely causes obstruction
        └─ If symptomatic: Can fenestrate septum (endoscopic septostomy)
```

## 2. THIRD VENTRICLE (Ventriculus Tertius)

```
├─ Overview
│   ├─ Location: Midline structure between two thalami
│   ├─ Shape: Thin, slit-like vertical cavity
│   ├─ Volume: ~1-2 mL (smallest of main ventricles)
│   └─ Orientation: Vertical (aligned with plane of lamina terminalis)
│
├─ Boundaries (Walls)
│   ├─ LATERAL WALLS (Most Important) ⭐⭐⭐:
│   │   ├─ Upper portion: Thalamus (medial surface)
│   │   ├─ Lower portion: Hypothalamus
│   │   ├─ Interthalamic adhesion (massa intermedia):
│   │   │   ├─ Gray matter bridge: Connects left and right thalami
│   │   │   ├─ Crosses: Midline of 3rd ventricle
│   │   │   ├─ Present: 70-80% of population
│   │   │   └─ No functional significance (absent in 20-30%)
│   │   │
│   │   └─ Hypothalamic sulcus:
│   │       ├─ Groove: Separates thalamus (above) from hypothalamus (below)
│   │       └─ Landmark: Runs anteroposterior along lateral wall
│   │
│   ├─ ANTERIOR WALL:
│   │   ├─ Upper part: Column of fornix (left and right, entering foramen of Monro)
│   │   ├─ Middle part: Anterior commissure (white matter tract crossing midline)
│   │   ├─ Lower part: Lamina terminalis (thin membrane - rostral end of neural tube)
│   │   │
│   │   └─ Landmarks:
│   │       ├─ Anterior commissure: Key landmark (separates chiasmatic from infundibular recess)
│   │       └─ Lamina terminalis: Thinnest part (site for endoscopic third ventriculostomy)
│   │
│   ├─ POSTERIOR WALL:
│   │   ├─ Upper: Pineal recess (points posteriorly toward pineal gland)
│   │   ├─ Middle: Habenular commissure + posterior commissure
│   │   │   ├─ Habenular commissure: Superior, connects habenular nuclei
│   │   │   └─ Posterior commissure: Inferior, white matter tract (pupillary light reflex)
│   │   └─ Lower: Cerebral aqueduct opening (connection to 4th ventricle)
│   │   │
│   │   └─ Suprapineal recess:
│   │       ├─ Small diverticulum: Above pineal recess
│   │       └─ Clinical: Site of pineal cysts (usually incidental)
│   │
│   ├─ ROOF:
│   │   ├─ Tela choroidea: Double layer of pia mater (vascular membrane)
│   │   ├─ Choroid plexus: Attached to tela choroidea (produces CSF)
│   │   ├─ Internal cerebral veins: Run above tela choroidea (in velum interpositum)
│   │   └─ Fornix: Body of fornix courses above roof (left and right)
│   │   │
│   │   └─ Velum interpositum:
│   │       ├─ Space: Between tela choroidea and fornix (contains internal cerebral veins)
│   │       └─ Surgical: Must preserve internal cerebral veins during transcallosal approach
│   │
│   └─ FLOOR:
│       ├─ Anterior to posterior structures:
│       │   ├─ 1. Optic chiasm: Most anterior (bulge in floor)
│       │   ├─ 2. Infundibular recess: Funnel-shaped depression (above pituitary stalk)
│       │   ├─ 3. Tuber cinereum: Gray matter of hypothalamus
│       │   ├─ 4. Mammillary bodies: Paired round eminences (posterior floor)
│       │   └─ 5. Posterior perforated substance: Posterior limit (above interpeduncular fossa)
│       │
│       └─ Clinical:
│           ├─ Hypothalamic hamartoma: Arises from tuber cinereum (gelastic seizures)
│           ├─ Craniopharyngioma: Often involves infundibular recess, optic chiasm
│           └─ Colloid cyst: Can prolapse through foramen of Monro (mobile "ball-valve")
│
├─ Recesses (Extensions of 3rd Ventricle)
│   ├─ Optic recess (Chiasmatic recess):
│   │   ├─ Location: Above optic chiasm (anterosuperior floor)
│   │   └─ Clinical: Suprasellar cistern communication
│   │
│   ├─ Infundibular recess:
│   │   ├─ Location: Into infundibulum (pituitary stalk)
│   │   ├─ Depth: ~5 mm
│   │   └─ Clinical: Craniopharyngioma often involves this area
│   │
│   ├─ Pineal recess:
│   │   ├─ Location: Extends posteriorly above pineal gland
│   │   ├─ Depth: ~3-5 mm
│   │   └─ Clinical: Pineal region tumors (germinoma, pineoblastoma)
│   │
│   └─ Suprapineal recess:
│       ├─ Location: Above pineal recess (small)
│       └─ Clinical: Pineal cysts (common incidental finding, 20-40% population)
│
├─ Communications
│   ├─ Superior: Two foramina of Monro → left and right lateral ventricles
│   └─ Inferior: Cerebral aqueduct → fourth ventricle
│
├─ Choroid Plexus of Third Ventricle
│   ├─ Location: Roof (attached to tela choroidea)
│   ├─ Extent: Extends from foramen of Monro to rostral aqueduct
│   ├─ Continuous with: Lateral ventricle choroid plexus (at foramen of Monro)
│   │
│   ├─ Blood supply:
│   │   ├─ Medial posterior choroidal arteries (MPChA): From PCA/P1 segment
│   │   └─ Anterior choroidal artery: Minor contribution
│   │
│   └─ Clinical:
│       ├─ Choroid plexus papilloma: Rare in 3rd ventricle (more common in lateral, 4th)
│       └─ Xanthogranuloma: Calcification (incidental finding in elderly)
│
├─ Surgical Approaches
│   ├─ Transcallosal-Transventricular Approach:
│   │   ├─ Entry: Through corpus callosum (between pericallosal arteries)
│   │   ├─ Trajectory: Through lateral ventricle (foramen of Monro) → 3rd ventricle
│   │   ├─ Indications:
│   │   │   ├─ Colloid cyst (classic indication ⭐⭐)
│   │   │   ├─ 3rd ventricle tumors (craniopharyngioma, hypothalamic glioma)
│   │   │   └─ Septum pellucidum cysts
│   │   │
│   │   ├─ Technique:
│   │   │   ├─ Interhemispheric dissection: Between cerebral hemispheres
│   │   │   ├─ Callosotomy: 1-2 cm incision (between pericallosal arteries)
│   │   │   ├─ Enter: Lateral ventricle (identify foramen of Monro)
│   │   │   ├─ Column of fornix: Visualize (posterior boundary of foramen)
│   │   │   ├─ Access: 3rd ventricle via widened foramen of Monro
│   │   │   └─ Avoid: Internal cerebral veins (in roof), thalamus (lateral)
│   │   │
│   │   └─ Complications:
│   │       ├─ Venous injury: Bridging veins, internal cerebral veins (venous infarction)
│   │       ├─ Fornix injury: Memory impairment (avoid bilateral fornix injury!)
│   │       ├─ Thalamic injury: If too lateral (sensory/motor deficits)
│   │       └─ Mutism: Supplementary motor area injury (usually temporary)
│   │
│   ├─ Subfrontal (Pterional) Approach:
│   │   ├─ Entry: Lateral to optic nerve, subfrontal corridor
│   │   ├─ Indications:
│   │   │   ├─ Craniopharyngioma (suprasellar)
│   │   │   ├─ Optic chiasm glioma
│   │   │   └─ Hypothalamic lesions
│   │   ├─ Access: Through lamina terminalis (anterior 3rd ventricle wall)
│   │   │
│   │   └─ Complications:
│   │       ├─ Visual: Optic nerve, chiasm injury (visual field defects, blindness)
│   │       ├─ Endocrine: Hypothalamic/pituitary stalk injury (DI, panhypopituitarism)
│   │       └─ Vascular: ACA, AComm injury (stroke)
│   │
│   └─ Endoscopic Third Ventriculostomy (ETV) ⭐⭐⭐:
│       ├─ Technique:
│       │   ├─ Entry: Frontal burr hole (Kocher's point)
│       │   ├─ Trajectory: Through lateral ventricle → foramen of Monro → 3rd ventricle
│       │   ├─ Identify: Mammillary bodies (posterior floor), infundibular recess
│       │   ├─ Fenestration site: Floor of 3rd ventricle (between mammillary bodies and infundibulum)
│       │   ├─ Balloon dilation: Create opening (6-8 mm) in floor (lamina terminalis/tuber cinereum)
│       │   └─ Communication: CSF flows from 3rd ventricle → prepontine cistern (bypasses aqueduct)
│       │
│       ├─ Indications:
│       │   ├─ Obstructive hydrocephalus:
│       │   │   ├─ Aqueductal stenosis (classic indication)
│       │   │   ├─ 4th ventricle outlet obstruction (Dandy-Walker, tectal glioma)
│       │   │   ├─ Pineal region tumor (with obstructive hydrocephalus)
│       │   │   └─ Posterior fossa tumor (with aqueductal compression)
│       │   │
│       │   └─ NOT indicated:
│       │       ├─ Communicating hydrocephalus (CSF absorption problem - ETV won't help)
│       │       └─ Normal pressure hydrocephalus (NPH - needs VP shunt)
│       │
│       ├─ Success rate:
│       │   ├─ Children: 60-80% (depends on etiology, age >6 months better)
│       │   ├─ Adults: 70-90% (aqueductal stenosis best outcomes)
│       │   └─ ETV Success Score (ETVSS): Predicts success based on age, etiology, prior shunt
│       │
│       └─ Complications:
│           ├─ Vascular injury ⭐⭐⭐:
│           │   ├─ Basilar artery: Lies below floor (posterior) - CATASTROPHIC if injured
│           │   ├─ Posterior communicating artery: Lateral to basilar
│           │   └─ Prevention: Stay midline, between mammillary bodies and infundibulum
│           ├─ Endocrine:
│           │   ├─ Diabetes insipidus: Injury to hypothalamus/infundibulum (transient 5-10%)
│           │   ├─ Electrolyte imbalance: SIADH (rare)
│           │   └─ Usually temporary (resolves in days-weeks)
│           ├─ Hemorrhage: Choroid plexus injury (obscures visualization)
│           ├─ Infection: Meningitis (1-2%)
│           └─ Failure: Stoma closure (5-10%) - may need repeat ETV or VP shunt
│
└─ Clinical Pathology
    ├─ Colloid Cyst ⭐⭐⭐:
    │   ├─ Location: Roof of 3rd ventricle (near foramen of Monro)
    │   ├─ Origin: Embryologic remnant (endoderm)
    │   ├─ Presentation:
    │   │   ├─ Hydrocephalus: Obstructive (both foramina of Monro)
    │   │   ├─ Headache: Positional (worse with bending forward, Valsalva)
    │   │   ├─ "Ball-valve" phenomenon: Mobile cyst intermittently obstructs foramen
    │   │   ├─ Sudden death: Rare (acute obstructive hydrocephalus)
    │   │   └─ Gait ataxia: From hydrocephalus
    │   │
    │   ├─ Imaging:
    │   │   ├─ CT: Hyperdense round mass at foramen of Monro (classic)
    │   │   ├─ MRI: Variable T1 (depends on protein content), T2 hyperintense, no enhancement
    │   │   └─ Hydrocephalus: Dilated lateral ventricles, normal 3rd/4th ventricle (if complete obstruction)
    │   │
    │   └─ Treatment:
    │       ├─ Observation: Asymptomatic, small (<1 cm), incidental finding
    │       ├─ Surgery (if symptomatic):
    │       │   ├─ Endoscopic resection: First-line (minimally invasive, faster recovery)
    │       │   ├─ Transcallosal microsurgical resection: If endoscopic fails or large/adherent
    │       │   └─ Goal: Complete resection (cure), avoid recurrence
    │       └─ Emergency: EVD if acute hydrocephalus, then definitive resection
    │
    ├─ Hypothalamic Hamartoma:
    │   ├─ Location: Tuber cinereum (floor of 3rd ventricle)
    │   ├─ Presentation:
    │   │   ├─ Gelastic seizures: "Laughing" seizures (pathognomonic)
    │   │   ├─ Precocious puberty: Central (GnRH secretion)
    │   │   └─ Cognitive/behavioral: Learning difficulties, aggression
    │   │
    │   ├─ Imaging: Non-enhancing mass at tuber cinereum, isointense to gray matter
    │   │
    │   └─ Treatment:
    │       ├─ Stereotactic radiosurgery: First-line (Gamma Knife - 60-80% seizure control)
    │       ├─ Endoscopic resection: Alternative (laser ablation)
    │       └─ Observation: If asymptomatic
    │
    └─ Pineal Region Tumors (see separate section, affect posterior 3rd ventricle):
        ├─ Germinoma: Most common (40%), CSF dissemination
        ├─ Pineoblastoma: Malignant, children
        ├─ Pineocytoma: Benign, adults
        └─ Tectal glioma: Obstructs aqueduct → hydrocephalus
```

## 3. CEREBRAL AQUEDUCT (Aqueduct of Sylvius)

```
├─ Overview
│   ├─ Location: Midbrain (connects 3rd → 4th ventricle)
│   ├─ Length: ~15-18 mm
│   ├─ Diameter: ~1-2 mm (narrowest part of ventricular system ⭐)
│   ├─ Shape: Tubular channel
│   └─ Most common site: Of CSF pathway obstruction
│
├─ Boundaries
│   ├─ Anterior (Ventral): Tegmentum of midbrain
│   │   ├─ Contains: Oculomotor nucleus (CN III), red nucleus, substantia nigra
│   │   └─ Relation: Periaqueductal gray matter (surrounds aqueduct)
│   │
│   ├─ Posterior (Dorsal): Tectum of midbrain
│   │   ├─ Contains: Superior and inferior colliculi (quadrigeminal plate)
│   │   └─ Clinical: Tectal tumors (glioma) compress aqueduct → hydrocephalus
│   │
│   └─ Periaqueductal gray matter (PAG):
│       ├─ Surrounds: Aqueduct circumferentially
│       ├─ Functions: Pain modulation, autonomic control
│       └─ Clinical: Lesions → vertical gaze palsy (dorsal PAG/posterior commissure)
│
├─ Divisions
│   ├─ Upper (Rostral) part: Near posterior commissure
│   ├─ Middle part: Narrowest region
│   └─ Lower (Caudal) part: Opens into 4th ventricle (at upper pons)
│
├─ Relations
│   ├─ Superior: Posterior commissure (pupillary light reflex fibers)
│   ├─ Anterior: Cerebral peduncles, interpeduncular cistern
│   ├─ Posterior: Quadrigeminal cistern, pineal gland
│   └─ Lateral: Midbrain tegmentum, periaqueductal gray
│
├─ Clinical Significance
│   ├─ Aqueductal Stenosis ⭐⭐⭐:
│   │   ├─ Definition: Narrowing/obstruction of cerebral aqueduct
│   │   ├─ Incidence: Most common cause of congenital hydrocephalus (20-30%)
│   │   │
│   │   ├─ Types:
│   │   │   ├─ Congenital:
│   │   │   │   ├─ Primary aqueductal stenosis: Idiopathic, forking/septum
│   │   │   │   ├─ X-linked hydrocephalus: LICAM gene mutation (L1 cell adhesion molecule)
│   │   │   │   └─ Neural tube defects: Chiari II malformation
│   │   │   │
│   │   │   └─ Acquired:
│   │   │       ├─ Infection: Prior meningitis, ventriculitis (gliosis/scarring)
│   │   │       ├─ Hemorrhage: Intraventricular, subarachnoid (clot, gliosis)
│   │   │       ├─ Tumor: Tectal glioma, pineal tumor (extrinsic compression)
│   │   │       └─ Idiopathic: Late-onset (adults)
│   │   │
│   │   ├─ Presentation:
│   │   │   ├─ Infants: Macrocephaly, bulging fontanelle, "sun-setting" eyes
│   │   │   ├─ Children/Adults: Headache, nausea/vomiting, gait ataxia, Parinaud syndrome
│   │   │   └─ Parinaud syndrome: Upward gaze palsy, light-near dissociation, convergence-retraction nystagmus
│   │   │
│   │   ├─ Imaging:
│   │   │   ├─ CT/MRI: Dilated lateral + 3rd ventricles, normal-sized 4th ventricle (classic ⭐⭐)
│   │   │   ├─ Aqueduct: Not visible or very thin
│   │   │   ├─ Flow study: Phase-contrast MRI (absent CSF flow through aqueduct)
│   │   │   └─ Rule out: Tectal tumor (MRI with contrast)
│   │   │
│   │   └─ Treatment:
│   │       ├─ Endoscopic third ventriculostomy (ETV): First-line ⭐⭐
│   │       │   ├─ Success rate: 70-90% (excellent for isolated aqueductal stenosis)
│   │       │   └─ Advantage: Avoids shunt (no hardware, no shunt complications)
│   │       │
│   │       └─ VP shunt: If ETV fails or contraindicated
│   │           └─ Success: 85-90%, but shunt-dependent (lifelong)
│   │
│   ├─ Tectal Glioma:
│   │   ├─ Location: Quadrigeminal plate (tectum) - compresses aqueduct
│   │   ├─ Presentation:
│   │   │   ├─ Hydrocephalus: Aqueductal compression (most common presentation)
│   │   │   ├─ Parinaud syndrome: Upward gaze palsy (tectal compression)
│   │   │   └─ Precocious puberty: Rare (pineal region involvement)
│   │   │
│   │   ├─ Imaging: T2 hyperintense tectal thickening, variable enhancement
│   │   │
│   │   └─ Treatment:
│   │       ├─ ETV: For hydrocephalus (avoid VP shunt if possible)
│   │       ├─ Observation: If no growth (many are low-grade, stable)
│   │       ├─ Biopsy: If atypical features (consider higher-grade)
│   │       └─ Radiation: If progressive or symptomatic (chemotherapy if high-grade)
│   │
│   └─ Parinaud Syndrome (Dorsal Midbrain Syndrome):
│       ├─ Lesion: Dorsal midbrain (superior colliculus, posterior commissure, periaqueductal gray)
│       ├─ Causes: Pineal tumor, tectal glioma, stroke, hydrocephalus (upward pressure)
│       │
│       ├─ Classic triad:
│       │   ├─ 1. Upward gaze palsy: Cannot look up (downgaze preserved)
│       │   ├─ 2. Light-near dissociation: Pupils don't constrict to light, but DO constrict to accommodation
│       │   └─ 3. Convergence-retraction nystagmus: Eyes converge + retract with upward saccades
│       │
│       ├─ Additional features:
│       │   ├─ Lid retraction: "Collier's sign" (wide-eyed appearance)
│       │   ├─ Pupil abnormalities: Large, poorly reactive pupils
│       │   └─ Eyelid lag: Slow downward eye movement
│       │
│       └─ Treatment: Address underlying cause (ETV for hydrocephalus, treat tumor)
│
└─ Surgical Considerations
    ├─ Not directly accessible: Too narrow and deep (surrounded by midbrain)
    ├─ ETV bypasses: Aqueductal obstruction (fenestrate floor of 3rd ventricle)
    └─ Aqueductoplasty: Rarely done (endoscopic dilation of stenotic aqueduct - high failure rate)
```

## 4. FOURTH VENTRICLE (Ventriculus Quartus)

```
├─ Overview
│   ├─ Location: Posterior fossa (between brainstem anteriorly, cerebellum posteriorly)
│   ├─ Shape: Tent-shaped cavity (rhomboid on sagittal view)
│   ├─ Volume: ~3-5 mL
│   ├─ Extends: From cerebral aqueduct (superiorly) → obex/central canal (inferiorly)
│   └─ Most surgically accessible: Ventricle (via multiple approaches)
│
├─ Boundaries
│   ├─ ROOF (Posterior Wall - Complex ⭐⭐⭐):
│   │   ├─ Upper part (Superior roof):
│   │   │   ├─ Superior medullary velum: Thin white matter sheet
│   │   │   │   ├─ Extends: From superior cerebellar peduncles
│   │   │   │   └─ Contains: Decussation of trochlear nerve (CN IV)
│   │   │   └─ Lingula of cerebellum: Inferior vermis projection
│   │   │
│   │   ├─ Lower part (Inferior roof):
│   │   │   ├─ Inferior medullary velum: Thin membrane (ependyma + pia)
│   │   │   ├─ Tela choroidea: Vascular membrane (double layer of pia)
│   │   │   ├─ Choroid plexus: Suspended from tela choroidea (T-shaped)
│   │   │   └─ Nodulus of cerebellum: Inferior vermis (touches roof)
│   │   │
│   │   └─ Clinical:
│   │       ├─ Telovelar approach: Entry through inferior medullary velum (preserves cerebellum)
│   │       ├─ Transvermian approach: Split vermis (more direct, but cerebellar injury)
│   │       └─ Inferior vermis syndrome: If vermis injured (ataxia, truncal instability)
│   │
│   ├─ FLOOR (Anterior Wall - Rhomboid Fossa) ⭐⭐⭐:
│   │   ├─ Formed by: Dorsal surface of pons (upper) + medulla (lower)
│   │   ├─ Shape: Diamond-shaped (widest at pontomed

ullary junction)
│   │   │
│   │   ├─ Midline structures (rostral to caudal):
│   │   │   ├─ 1. Median sulcus: Midline groove (entire length)
│   │   │   ├─ 2. Median eminence: Slight elevation on either side of sulcus
│   │   │   ├─ 3. Facial colliculus: Elevation over abducens nucleus + facial nerve genu (KEY LANDMARK ⭐)
│   │   │   │   ├─ Location: Pontomedullary junction level
│   │   │   │   └─ Contents: Abducens nucleus (CN VI), internal genu of facial nerve (CN VII)
│   │   │   ├─ 4. Hypoglossal trigone: Triangle (lateral to median sulcus, lower medulla)
│   │   │   │   └─ Overlies: Hypoglossal nucleus (CN XII)
│   │   │   ├─ 5. Vagal trigone: Triangle (lateral to hypoglossal trigone)
│   │   │   │   ├─ Overlies: Dorsal motor nucleus of vagus (CN X)
│   │   │   │   └─ Clinical: "Safe entry zone" for 4th ventricle surgery (between trigones)
│   │   │   └─ 6. Area postrema: Chemoreceptor trigger zone (most caudal floor)
│   │   │       ├─ Function: Vomiting center (detects toxins in blood)
│   │   │       └─ Clinical: Outside blood-brain barrier (can sense systemic toxins)
│   │   │
│   │   ├─ Lateral structures:
│   │   │   ├─ Vestibular area: Lateral to sulcus limitans (overlies vestibular nuclei - CN VIII)
│   │   │   ├─ Sulcus limitans: Groove separating motor (medial) from sensory (lateral) nuclei
│   │   │   └─ Lateral recess: Extensions laterally toward CP angle
│   │   │
│   │   └─ Clinical:
│   │       ├─ Floor landmarks: Critical during 4th ventricle surgery (avoid CN nuclei)
│   │       ├─ Safe entry zones:
│   │       │   ├─ Midline (between hypoglossal trigones): Safest for superior medullary lesions
│   │       │   ├─ Lateral recess: For lateral 4th ventricle tumors
│   │       │   └─ Inferior medullary velum: Telovelar approach (avoids floor entirely)
│   │       └─ CN nuclei injury:
│   │           ├─ CN VI (abducens): Lateral gaze palsy (cannot abduct eye)
│   │           ├─ CN VII (facial): Facial weakness (LMN pattern - all facial muscles)
│   │           ├─ CN X (vagus): Dysphagia, hoarseness, aspiration (life-threatening)
│   │           └─ CN XII (hypoglossal): Tongue deviation (toward lesion side)
│   │
│   ├─ LATERAL WALLS:
│   │   ├─ Upper: Superior cerebellar peduncle (brachium conjunctivum)
│   │   ├─ Lower: Inferior cerebellar peduncle (restiform body), cuneate tubercle
│   │   └─ Lateral recesses: Extensions toward cerebellopontine angle (CPA)
│   │       ├─ Contain: Choroid plexus extending laterally
│   │       └─ Open at: Foramina of Luschka (paired, lateral outlets)
│   │
│   └─ INFERIOR POINT (Obex):
│       ├─ Location: Caudal end of 4th ventricle (at level of foramen magnum)
│       ├─ Transition: 4th ventricle → central canal of spinal cord
│       └─ Landmark: Surgical landmark for identifying caudal extent
│
├─ Foramina (CSF Outflow - CRITICAL ⭐⭐⭐)
│   ├─ Foramen of Magendie (Median Aperture):
│   │   ├─ Location: Midline, inferior roof (caudal 4th ventricle)
│   │   ├─ Opens into: Cisterna magna (cerebellomedullary cistern)
│   │   ├─ Size: Largest outlet (~5-8 mm)
│   │   ├─ CSF flow: Primary outlet (70-80% of CSF drainage from 4th ventricle)
│   │   │
│   │   └─ Clinical:
│   │       ├─ Obstruction: Dandy-Walker malformation (absent or stenotic Magendie)
│   │       ├─ Chiari I: Tonsillar herniation can compress Magendie → hydrocephalus
│   │       └─ Posterior fossa surgery: Must preserve Magendie opening
│   │
│   └─ Foramina of Luschka (Lateral Apertures) - PAIRED:
│       ├─ Location: Lateral recesses of 4th ventricle (bilateral)
│       ├─ Opens into: Cerebellopontine angle cistern (CPA)
│       ├─ Size: Smaller than Magendie (~2-3 mm each)
│       ├─ CSF flow: Secondary outlets (20-30% combined)
│       │
│       ├─ Contents:
│       │   ├─ Choroid plexus: Extends through Luschka (visible in CPA cistern)
│       │   └─ Clinical: Choroid plexus visible on MRI protruding into CPA
│       │
│       └─ Clinical:
│           ├─ Obstruction: Rare isolated, usually with Magendie obstruction
│           ├─ Acoustic neuroma: Can compress Luschka (hydrocephalus uncommon)
│           └─ Surgical: Identify during CPA surgery (avoid choroid plexus bleeding)
│
├─ Choroid Plexus of Fourth Ventricle ⭐⭐
│   ├─ Shape: T-shaped or anchor-shaped
│   ├─ Location:
│   │   ├─ Vertical limb: Midline (along median sulcus of roof)
│   │   └─ Horizontal limbs: Extend laterally into lateral recesses (to foramina of Luschka)
│   │
│   ├─ Attachment: Tela choroidea (inferior medullary velum)
│   │
│   ├─ Blood supply:
│   │   ├─ PICA (posterior inferior cerebellar artery): Primary ⭐
│   │   ├─ AICA (anterior inferior cerebellar artery): Lateral portions
│   │   └─ Highly vascular: Significant bleeding risk during surgery
│   │
│   ├─ Venous drainage:
│   │   └─ Drains into: Inferior vermian veins → straight sinus/torcular
│   │
│   └─ Clinical:
│       ├─ Choroid plexus papilloma:
│       │   ├─ Adults: 4th ventricle most common site (50%)
│       │   ├─ Children: Lateral ventricle atrium more common
│       │   ├─ Presentation: Hydrocephalus (CSF overproduction + outlet obstruction from tumor mass)
│       │   └─ Treatment: Surgical resection (control bleeding from PICA first)
│       │
│       ├─ Choroid plexus coagulation: For hydrocephalus (endoscopic, reduce CSF production)
│       └─ Telovelar approach: Splits tela choroidea (preserves choroid plexus pedicle = less bleeding)
│
├─ Recesses
│   ├─ Lateral recesses (paired):
│   │   ├─ Extend: Laterally toward CPA
│   │   ├─ Open at: Foramina of Luschka
│   │   └─ Clinical: Lateral 4th ventricle tumors (ependymoma)
│   │
│   └─ Fastigial recess:
│       ├─ Small superior extension: Into fastigium of cerebellum
│       └─ Usually: Not clinically significant
│
├─ Relations
│   ├─ Anterior: Pons (upper), medulla (lower) - floor of 4th ventricle
│   ├─ Posterior: Cerebellum (vermis, cerebellar hemispheres)
│   ├─ Superior: Cerebral aqueduct (connection to 3rd ventricle)
│   ├─ Inferior: Obex, central canal (connection to spinal cord)
│   ├─ Lateral: Cerebellar peduncles, CPA cistern
│   └─ Surrounding cisterns:
│       ├─ Cisterna magna: Posterior + inferior (largest posterior fossa cistern)
│       ├─ CPA cistern: Lateral (contains CN VII, VIII, AICA)
│       └─ Prepontine cistern: Anterior (contains basilar artery)
│
├─ Surgical Approaches ⭐⭐⭐
│   ├─ Telovelar Approach (Inferior Approach):
│   │   ├─ Position: Sitting or prone (head flexed)
│   │   ├─ Entry: Through inferior medullary velum (between cerebellar tonsils)
│   │   ├─ Landmarks:
│   │   │   ├─ Tonsils: Retract laterally (gentle)
│   │   │   ├─ Tela choroidea: Incise in midline
│   │   │   ├─ Inferior medullary velum: Open to access 4th ventricle
│   │   │   └─ Avoid: PICA branches (tonsillary, choroidal)
│   │   │
│   │   ├─ Advantages:
│   │   │   ├─ Avoids cerebellar incision: No vermis splitting (less ataxia)
│   │   │   ├─ Direct access: To inferior 4th ventricle, obex
│   │   │   ├─ Preserves floor: Less risk to CN nuclei
│   │   │   └─ Natural corridor: Between tonsils
│   │   │
│   │   ├─ Disadvantages:
│   │   │   ├─ Limited superior access: Cannot reach superior 4th ventricle well
│   │   │   ├─ PICA at risk: Must preserve (brainstem infarct if injured)
│   │   │   └─ Narrow corridor: If tonsils bulky or herniating (Chiari I)
│   │   │
│   │   └─ Indications:
│   │       ├─ Medulloblastoma (midline, inferior 4th ventricle)
│   │       ├─ Ependymoma (floor or inferior 4th ventricle)
│   │       └─ Subependymoma (inferior 4th ventricle)
│   │
│   ├─ Transvermian Approach (Midline Superior Approach):
│   │   ├─ Position: Sitting or prone
│   │   ├─ Entry: Split cerebellar vermis (inferior vermis) in midline
│   │   ├─ Exposure: Superior + midportion of 4th ventricle
│   │   │
│   │   ├─ Advantages:
│   │   │   ├─ Wide exposure: Excellent visualization of entire 4th ventricle
│   │   │   ├─ Access: To superior 4th ventricle, aqueduct
│   │   │   └─ Familiar: Traditional approach
│   │   │
│   │   ├─ Disadvantages:
│   │   │   ├─ Cerebellar injury: Vermis splitting → ataxia (truncal > appendicular)
│   │   │   ├─ Mutism: Posterior fossa mutism syndrome (10-25% in children)
│   │   │   └─ Inferior vermis syndrome: Gait ataxia, truncal instability
│   │   │
│   │   └─ Indications:
│   │       ├─ Large 4th ventricle tumors (need wide exposure)
│   │       ├─ Superior 4th ventricle lesions
│   │       └─ Historically more common (now telovelar preferred if feasible)
│   │
│   ├─ Lateral Recess Approach (Retrosigmoid):
│   │   ├─ Position: Lateral (park bench or supine with head turned)
│   │   ├─ Entry: Retrosigmoid craniotomy → through lateral recess
│   │   ├─ Indications:
│   │   │   ├─ Lateral 4th ventricle tumor (ependymoma with CPA extension)
│   │   │   ├─ Foramen of Luschka lesion
│   │   │   └─ Combined CPA + 4th ventricle pathology
│   │   │
│   │   └─ Advantages:
│   │       ├─ Avoids: Vermis, cerebellar incision
│   │       └─ Access: To CPA + lateral 4th ventricle
│   │
│   └─ Transcerebellar Approach:
│       ├─ Entry: Through cerebellar hemisphere (not vermis)
│       ├─ Indications: Lateral cerebellar lesion with 4th ventricle involvement
│       └─ Disadvantage: Cerebellar hemisphere injury (appendicular ataxia, dysmetria)
│
├─ Complications (4th Ventricle Surgery)
│   ├─ Cranial nerve deficits ⭐⭐⭐:
│   │   ├─ CN VI (abducens): Lateral rectus palsy (cannot abduct eye) - diplopia
│   │   ├─ CN VII (facial): Facial weakness (all facial muscles - LMN)
│   │   ├─ CN IX, X (glossopharyngeal, vagus): Dysphagia, aspiration (LIFE-THREATENING)
│   │   │   ├─ Management: NGT/G-tube feeding, vocal cord medialization
│   │   │   └─ Bilateral: Respiratory compromise (may need tracheostomy)
│   │   └─ CN XII (hypoglossal): Tongue deviation (usually well-tolerated if unilateral)
│   │
│   ├─ Cerebellar mutism syndrome (Posterior fossa mutism):
│   │   ├─ Incidence: 10-25% after vermian splitting (children > adults)
│   │   ├─ Onset: Delayed (24-72 hours post-op)
│   │   ├─ Features:
│   │   │   ├─ Mutism: Complete inability to speak
│   │   │   ├─ Emotional lability: Crying, agitation
│   │   │   ├─ Hypotonia: Generalized
│   │   │   └─ Ataxia: Truncal + appendicular
│   │   ├─ Duration: Weeks to months (recovery usually gradual)
│   │   ├─ Pathophysiology: Dentate nucleus, superior cerebellar peduncle injury (cerebello-cerebral pathways)
│   │   └─ Prevention: Avoid extensive vermian splitting (use telovelar approach when possible)
│   │
│   ├─ Vascular complications:
│   │   ├─ PICA injury: Lateral medullary syndrome (Wallenberg), cerebellar infarct
│   │   │   ├─ Prevention: Identify and preserve PICA branches
│   │   │   └─ If sacrificed: May tolerate if contralateral PICA intact
│   │   ├─ Choroid plexus bleeding: Can obscure surgical field (control with bipolar, hemostatic agents)
│   │   └─ Vertebral artery: At risk during far lateral approaches
│   │
│   ├─ Hydrocephalus:
│   │   ├─ Postoperative: Obstruction of 4th ventricle outlets (blood, debris, edema)
│   │   ├─ Incidence: 10-30% after posterior fossa tumor resection
│   │   └─ Management: EVD (temporary), VP shunt (if persistent)
│   │
│   ├─ CSF leak: 5-10% (through wound, from dural opening)
│   ├─ Meningitis: 2-5%
│   ├─ Aseptic meningitis: Chemical (blood products in CSF - self-limited)
│   └─ Brainstem injury:
│       ├─ Respiratory arrest: Medullary injury (requires intubation, ICU monitoring)
│       ├─ Cardiovascular instability: Autonomic centers in medulla
│       └─ "Locked-in syndrome": Rare (bilateral ventral pontine injury)
│
└─ Clinical Pathology
    ├─ Medulloblastoma ⭐⭐⭐:
    │   ├─ Most common: Malignant posterior fossa tumor in children
    │   ├─ Location: Midline inferior vermis, extends into 4th ventricle
    │   ├─ Age: Peak 5-10 years (also young adults 20-30)
    │   │
    │   ├─ Presentation:
    │   │   ├─ Hydrocephalus: 4th ventricle obstruction (increased ICP)
    │   │   ├─ Ataxia: Truncal (midline vermis involvement)
    │   │   ├─ Headache, nausea/vomiting: Increased ICP
    │   │   └─ Head tilt: Tonsillar herniation (may be presenting sign)
    │   │
    │   ├─ Imaging:
    │   │   ├─ CT: Hyperdense, midline, 4th ventricle mass, hydrocephalus
    │   │   ├─ MRI: T1 iso/hypointense, T2 variable, avid enhancement
    │   │   ├─ Restricted diffusion: High cellularity (DWI hyperintense)
    │   │   └─ Spine MRI: CSF dissemination (25-40% at presentation)
    │   │
    │   ├─ Molecular subtypes (WHO 2021):
    │   │   ├─ WNT-activated: Best prognosis (>95% 5-year survival)
    │   │   ├─ SHH-activated: Intermediate prognosis (variable)
    │   │   ├─ Group 3: Poor prognosis (MYC amplification, metastases)
    │   │   └─ Group 4: Intermediate prognosis
    │   │
    │   └─ Treatment:
    │       ├─ Surgery: Maximal safe resection (goal: gross total resection)
    │       ├─ Radiation: Craniospinal irradiation (CSI 23.4-36 Gy) + tumor boost
    │       ├─ Chemotherapy: Adjuvant (cisplatin, vincristine, cyclophosphamide)
    │       └─ Prognosis: 70-80% 5-year survival (depends on subtype, extent of resection, age)
    │
    ├─ Ependymoma:
    │   ├─ Second most common: Posterior fossa tumor in children (after medulloblastoma)
    │   ├─ Location:
    │   │   ├─ Floor of 4th ventricle (most common) - arises from ependymal lining
    │   │   ├─ Can extend: Through foramen of Luschka into CPA
    │   │   └─ "Plastic tumor": Molds to ventricle shape, grows through foramina
    │   │
    │   ├─ Presentation: Similar to medulloblastoma (hydrocephalus, ataxia)
    │   │
    │   ├─ Imaging:
    │   │   ├─ MRI: Heterogeneous enhancement, calcification (40-50%), cysts
    │   │   ├─ "Toothpaste tumor": Extrudes through foramina of Luschka
    │   │   └─ Less diffusion restriction: Compared to medulloblastoma
    │   │
    │   └─ Treatment:
    │       ├─ Surgery: Gross total resection (critical - most important prognostic factor)
    │       ├─ Radiation: Focal (not craniospinal unless dissemination)
    │       ├─ Chemotherapy: Limited role (not as chemo-sensitive as medulloblastoma)
    │       └─ Prognosis: 60-70% 5-year survival (GTR > STR)
    │
    ├─ Pilocytic Astrocytoma (Juvenile Pilocytic Astrocytoma - JPA):
    │   ├─ Location: Cerebellar hemisphere > midline vermis (can involve 4th ventricle)
    │   ├─ Age: Children, young adults
    │   ├─ WHO grade 1: Benign, slow-growing
    │   │
    │   ├─ Imaging:
    │   │   ├─ Classic: Cystic mass with enhancing mural nodule
    │   │   ├─ T2 hyperintense, avid enhancement
    │   │   └─ No diffusion restriction (low cellularity)
    │   │
    │   └─ Treatment:
    │       ├─ Surgery: Gross total resection → CURE (no need for further treatment)
    │       ├─ If incomplete resection: Observation vs adjuvant therapy
    │       └─ Prognosis: Excellent (>95% 10-year survival if GTR)
    │
    ├─ Hemangioblastoma:
    │   ├─ Location: Cerebellar hemisphere, can involve 4th ventricle
    │   ├─ Age: Adults (30-40 years)
    │   ├─ Association: Von Hippel-Lindau (VHL) disease (25-30% of cases)
    │   │
    │   ├─ Imaging:
    │   │   ├─ Classic: Cystic + enhancing mural nodule (like JPA, but in adults)
    │   │   ├─ Flow voids: Hypervascular (prominent feeding vessels on angiography)
    │   │   ├─ T2 hyperintense cyst, intensely enhancing nodule
    │   │   └─ Angiography: Hypervascular blush
    │   │
    │   ├─ Clinical:
    │   │   ├─ Polycythemia: EPO secretion (10-20%)
    │   │   └─ Spine MRI: Screen for spinal hemangioblastomas (VHL)
    │   │
    │   └─ Treatment:
    │       ├─ Surgery: Resect nodule (drain cyst, resect enhancing nodule only)
    │       ├─ Preoperative embolization: If very vascular
    │       └─ Prognosis: Excellent if complete nodule resection (cure)
    │
    ├─ Subependymoma:
    │   ├─ Location: 4th ventricle floor (60%), lateral ventricle (40%)
    │   ├─ Age: Middle-aged adults, elderly
    │   ├─ Benign: WHO grade 1, slow-growing
    │   │
    │   ├─ Imaging:
    │   │   ├─ T1 hypointense, T2 hyperintense
    │   │   ├─ Minimal/no enhancement (differentiates from ependymoma)
    │   │   └─ Calcification possible
    │   │
    │   └─ Treatment:
    │       ├─ Observation: If asymptomatic, incidental
    │       ├─ Surgery: If symptomatic (hydrocephalus), gross total resection → cure
    │       └─ Prognosis: Excellent
    │
    └─ Dandy-Walker Malformation:
        ├─ Definition: Congenital malformation with 4th ventricle cystic dilation
        ├─ Components (classic triad):
        │   ├─ 1. Cystic dilation: 4th ventricle (large posterior fossa cyst)
        │   ├─ 2. Hypoplasia/agenesis: Cerebellar vermis (absent or incomplete)
        │   ├─ 3. Enlarged posterior fossa: Elevated torcular, lambdoid sutures
        │
        ├─ Pathophysiology:
        │   ├─ Atresia/stenosis: Foramen of Magendie (CSF outflow obstruction)
        │   ├─ Result: 4th ventricle dilates, balloons into posterior fossa
        │   └─ Vermis: Fails to develop properly (embryologic defect)
        │
        ├─ Clinical presentation:
        │   ├─ Hydrocephalus: 80-90% (communicating or obstructive)
        │   ├─ Macrocephaly: Enlarged head (infants)
        │   ├─ Developmental delay: Cognitive impairment (variable severity)
        │   ├─ Ataxia: Truncal instability (vermis hypoplasia)
        │   └─ Associated anomalies: Agenesis of corpus callosum, cardiac defects (50%)
        │
        ├─ Imaging:
        │   ├─ CT/MRI: Large posterior fossa cyst, absent/hypoplastic vermis
        │   ├─ 4th ventricle: Continuous with cyst (differentiates from arachnoid cyst)
        │   ├─ Torcular: Elevated ("torcular-lambdoid inversion")
        │   └─ Hydrocephalus: Dilated lateral + 3rd ventricles
        │
        └─ Treatment:
            ├─ VP shunt: For hydrocephalus (primary treatment)
            ├─ Cyst fenestration: Rarely needed (shunt usually sufficient)
            ├─ ETV: Not effective (communicating hydrocephalus usually)
            └─ Prognosis: Variable (depends on associated anomalies, cognitive function)
```
