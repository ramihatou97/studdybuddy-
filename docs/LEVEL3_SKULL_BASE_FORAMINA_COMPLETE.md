# Level 3: Skull Base Foramina & Contents - Ultra-Granular Surgical Anatomy

**Document**: Complete Skull Base Foramina Surgical Reference
**Purpose**: Ultra-granular anatomy for neurosurgical procedures
**Scope**: All skull base foramina with contents, surgical approaches, clinical syndromes
**Connection**: Links directly to CN III-XII anatomy (nerves pass through these foramina)

---

## Organization by Cranial Fossa

The skull base is divided into three fossae, each containing specific foramina:

```
├─ Anterior Cranial Fossa
│   ├─ Cribriform plate foramina (multiple)
│   └─ Foramen cecum
│
├─ Middle Cranial Fossa (Most Complex)
│   ├─ Optic canal
│   ├─ Superior orbital fissure
│   ├─ Foramen rotundum
│   ├─ Foramen ovale
│   ├─ Foramen spinosum
│   ├─ Foramen lacerum
│   └─ Carotid canal
│
└─ Posterior Cranial Fossa
    ├─ Internal auditory canal
    ├─ Jugular foramen
    ├─ Hypoglossal canal
    ├─ Foramen magnum
    ├─ Stylomastoid foramen
    └─ Mastoid foramen
```

## ANTERIOR CRANIAL FOSSA

### 1. CRIBRIFORM PLATE FORAMINA

```
├─ Location
│   ├─ Bone: Ethmoid bone (horizontal lamina)
│   ├─ Fossa: Anterior cranial fossa (floor, midline)
│   ├─ Structure: Thin, perforated bone (sieve-like - "cribrum")
│   └─ Vulnerability: Weakest part of skull base (CSF leak risk)
│
├─ Contents (Multiple Small Foramina - ~20 on each side)
│   ├─ Olfactory nerve filaments (CN I):
│   │   ├─ ~20 bundles of unmyelinated axons
│   │   ├─ From: Olfactory mucosa (superior nasal cavity)
│   │   ├─ To: Olfactory bulb (on cribriform plate, superior surface)
│   │   └─ Function: Smell
│   │
│   └─ Anterior ethmoidal artery:
│       ├─ Branch: Ophthalmic artery
│       ├─ Course: Through anterior ethmoidal foramen → cribriform plate
│       └─ Supplies: Anterior ethmoid, anterior nasal cavity
│
├─ Anatomical Relations
│   ├─ Superior: Olfactory bulbs and tracts (on superior surface)
│   ├─ Inferior: Nasal cavity roof (mucosa directly below bone)
│   ├─ Lateral: Ethmoid air cells
│   └─ Thickness: 0.05-0.5 mm (EXTREMELY THIN - paper thin)
│
├─ Clinical Significance
│   │
│   ├─ Anosmia (Loss of Smell):
│   │   ├─ Causes:
│   │   │   ├─ Head trauma: Shearing of olfactory filaments
│   │   │   ├─ Anterior skull base fracture: Cribriform plate disruption
│   │   │   ├─ Meningioma: Olfactory groove meningioma compression
│   │   │   └─ Viral URI: Temporary (COVID-19, influenza)
│   │   │
│   │   └─ Prognosis: Head trauma anosmia often permanent (axons cannot regenerate through scar)
│   │
│   ├─ CSF Rhinorrhea:
│   │   ├─ Mechanism: Cribriform plate fracture → dural tear → CSF leak into nose
│   │   ├─ Presentation:
│   │   │   ├─ Clear fluid dripping from nose (increased with Valsalva, bending forward)
│   │   │   ├─ Salty taste (CSF vs rhinorrhea)
│   │   │   ├─ "Reservoir sign": Fluid pools when supine, drains when upright
│   │   │   └─ Anosmia: Often coexists (trauma to CN I)
│   │   │
│   │   ├─ Diagnosis:
│   │   │   ├─ Beta-2 transferrin: Gold standard (specific to CSF)
│   │   │   ├─ Halo sign: Blood with clear ring on gauze (not specific)
│   │   │   ├─ CT cisternography: Contrast in CSF, identify leak site
│   │   │   └─ High-resolution CT: Thin cuts (0.6 mm), coronal views
│   │   │
│   │   ├─ Complications:
│   │   │   ├─ Meningitis: Direct pathway for bacteria (pneumococcal most common)
│   │   │   ├─ Recurrent meningitis: Always suspect CSF leak
│   │   │   └─ Pneumocephalus: Air entering intracranial space
│   │   │
│   │   └─ Management:
│   │       ├─ Conservative (small leaks): Bed rest, head elevation, acetazolamide (↓CSF production)
│   │       ├─ Lumbar drain: 5-7 days (allows dural healing)
│   │       ├─ Surgical repair: If conservative fails (2 weeks)
│   │       │   ├─ Endoscopic endonasal: Direct repair with graft (fascia lata, fat)
│   │       │   └─ Craniotomy: If large defect or failed endoscopic
│   │       │
│   │       └─ Success: 90-95% with endoscopic repair
│   │
│   └─ Iatrogenic Injury:
│       ├─ Endoscopic sinus surgery: Risk cribriform plate violation
│       ├─ Prevention: CT navigation, recognize anatomical landmarks
│       └─ Recognition: Immediate CSF leak, olfactory bulb/frontal lobe injury
│
├─ Surgical Approaches
│   ├─ Endoscopic endonasal:
│   │   ├─ Access: Through nasal cavity, superior to middle turbinate
│   │   ├─ Advantages: Minimally invasive, direct visualization of leak
│   │   ├─ Disadvantages: Limited to midline lesions
│   │   └─ Uses: CSF leak repair, olfactory groove meningioma (small)
│   │
│   ├─ Transcranial (bifrontal craniotomy):
│   │   ├─ Access: Frontal bone removal, elevate frontal lobes
│   │   ├─ Advantages: Wide exposure, bilateral access
│   │   ├─ Disadvantages: Brain retraction, anosmia (sacrifice CN I)
│   │   └─ Uses: Olfactory groove meningioma (large), anterior skull base tumors
│   │
│   └─ Anosmia: Expected after transcranial approach (olfactory tract transection)
│
└─ Tumors Involving Cribriform Plate
    ├─ Olfactory groove meningioma:
    │   ├─ Origin: Dura over cribriform plate
    │   ├─ Presentation: Anosmia, frontal lobe symptoms (personality change, seizures)
    │   └─ Surgery: Transcranial vs endoscopic (depends on size)
    │
    ├─ Esthesioneuroblastoma (Olfactory neuroblastoma):
    │   ├─ Origin: Olfactory epithelium (nasal cavity)
    │   ├─ Grows: Through cribriform plate into anterior cranial fossa
    │   ├─ Kadish staging: A (nasal), B (paranasal sinuses), C (beyond sinuses/brain)
    │   └─ Treatment: Surgery + radiation (Kadish C: chemotherapy)
    │
    └─ Meningocele/Encephalocele:
        ├─ Herniation: Brain/meninges through cribriform defect
        ├─ Congenital or acquired (trauma)
        └─ Treatment: Surgical repair (endoscopic or transcranial)
```

### 2. FORAMEN CECUM

```
├─ Location
│   ├─ Bone: Frontal bone (anterior to crista galli)
│   ├─ Fossa: Anterior cranial fossa (anterior midline)
│   └─ Appearance: Small blind pit (usually closed in adults)
│
├─ Contents
│   ├─ Emissary vein (variable - often obliterated):
│   │   ├─ Connects: Nasal cavity veins → superior sagittal sinus
│   │   └─ Obliterated: In 90% of adults (patent in children)
│   │
│   └─ Clinical: Usually no contents (vestigial)
│
├─ Embryology
│   ├─ Developmental: Site of anterior neuropore closure
│   └─ Persistent: If patent, can be pathway for infection or tumors
│
└─ Clinical Significance
    ├─ Nasal dermoid sinus:
    │   ├─ Congenital: Epithelial tract from nasal skin → intracranial (via foramen cecum)
    │   ├─ Presentation: Midline nasal pit/mass, recurrent meningitis
    │   └─ Treatment: Complete excision (prevent meningitis)
    │
    └─ Meningitis pathway: If patent (rare in adults)
```

## MIDDLE CRANIAL FOSSA

### 3. OPTIC CANAL

```
├─ Location
│   ├─ Bone: Lesser wing of sphenoid bone
│   ├─ Fossa: Medial end of superior orbital fissure, junction of middle fossa and orbit
│   ├─ Shape: Oval tunnel (length ~10 mm)
│   └─ Direction: Anteromedial (from middle fossa to orbit)
│
├─ Contents
│   ├─ Optic nerve (CN II):
│   │   ├─ Segment: Intracanalicular portion (10 mm)
│   │   ├─ Meninges: Surrounded by dura, arachnoid, pia (optic nerve sheath)
│   │   ├─ CSF: In subarachnoid space around nerve
│   │   └─ Relationship: Inferolateral to ophthalmic artery
│   │
│   └─ Ophthalmic artery:
│       ├─ Branch: Internal carotid artery (just distal to cavernous segment)
│       ├─ Relationship: Usually inferolateral to CN II (90%)
│       │   └─ Can be superomedial (10% - surgical variation)
│       │
│       └─ Branches: Central retinal artery (critical for vision)
│
├─ Anatomical Relations
│   ├─ Superior: Anterior clinoid process (can pneumatize - optic strut)
│   ├─ Inferior: Sphenoid sinus (thin bone - can erode)
│   ├─ Medial: Sphenoid sinus, tuberculum sellae
│   ├─ Lateral: Superior orbital fissure, cavernous sinus
│   └─ Anterior: Orbital apex
│
├─ Clinical Significance
│   │
│   ├─ Optic Nerve Compression:
│   │   ├─ Causes:
│   │   │   ├─ Trauma: Optic canal fracture → nerve edema/hematoma
│   │   │   ├─ Meningioma: Tuberculum sellae or sphenoid wing (medial)
│   │   │   ├─ Aneurysm: Ophthalmic artery or superior hypophyseal artery
│   │   │   ├─ Paget's disease: Bone thickening → canal narrowing
│   │   │   └─ Fibrous dysplasia: Bone expansion
│   │   │
│   │   ├─ Presentation:
│   │   │   ├─ Progressive vision loss (gradual vs acute)
│   │   │   ├─ Optic atrophy: Pale disc on ophthalmoscopy
│   │   │   ├─ RAPD: Relative afferent pupillary defect (Marcus Gunn pupil)
│   │   │   └─ Visual field defect: Central scotoma or sector defect
│   │   │
│   │   └─ Management:
│   │       ├─ Trauma: Controversial (steroids ± optic nerve decompression)
│   │       ├─ Tumor: Surgical decompression + tumor resection
│   │       └─ Time-sensitive: Irreversible damage after prolonged compression
│   │
│   ├─ Traumatic Optic Neuropathy (TON):
│   │   ├─ Mechanism: Blunt trauma → optic nerve shear/edema/ischemia
│   │   ├─ Canal fracture: 10-30% of TON cases
│   │   ├─ Treatment (controversial):
│   │   │   ├─ Observation: Many recover spontaneously
│   │   │   ├─ High-dose steroids: Methylprednisolone (limited evidence)
│   │   │   └─ Optic canal decompression: If bony fragments, no recovery (controversial)
│   │   │
│   │   └─ Prognosis: Variable (some recover, many permanent vision loss)
│   │
│   └─ Sphenoid Sinus Disease:
│       ├─ Mucocele: Can expand, erode into optic canal
│       ├─ Fungal sinusitis: Aspergillus invasion → vision loss
│       └─ Optic canal decompression: Endoscopic (remove medial wall of canal)
│
├─ Surgical Approaches
│   ├─ Transcranial (pterional or frontal):
│   │   ├─ Exposure: Unroof optic canal from superior
│   │   ├─ Technique: Drill lesser wing of sphenoid, open optic canal roof
│   │   ├─ Optic nerve sheath: Can fenestrate (decompress if swollen)
│   │   └─ Uses: Meningioma, aneurysm, trauma decompression
│   │
│   ├─ Endoscopic endonasal:
│   │   ├─ Exposure: Through sphenoid sinus (medial and inferior walls)
│   │   ├─ Advantages: Direct access, less brain retraction
│   │   ├─ Disadvantages: Limited to medial canal wall
│   │   └─ Uses: Sphenoid sinus mucocele, medial optic canal decompression
│   │
│   └─ Lateral orbitotomy:
│       ├─ Exposure: Lateral orbital wall removal
│       ├─ Uses: Orbital apex lesions, optic nerve tumors
│       └─ Preserves: Vision (nerve not manipulated directly)
│
└─ Tumors Involving Optic Canal
    ├─ Optic nerve sheath meningioma:
    │   ├─ Origin: Dura around CN II (can extend into canal)
    │   ├─ Presentation: Progressive vision loss, "tram-track" sign on MRI
    │   ├─ Treatment: Observation (slow) vs radiation (preserve vision)
    │   └─ Surgery: High risk of blindness (nerve manipulation)
    │
    ├─ Tuberculum sellae meningioma:
    │   ├─ Compresses: Optic nerves bilaterally (can be asymmetric)
    │   ├─ Presentation: Bitemporal hemianopia (chiasm) or vision loss
    │   └─ Surgery: Transcranial (unroof canals, decompress chiasm)
    │
    └─ Optic nerve glioma:
        ├─ Usually: Pilocytic astrocytoma (children, NF1)
        ├─ Expands: Within and around nerve, can involve canal
        └─ Treatment: Observation (slow) vs chemotherapy (if vision threatened)
```

### 4. SUPERIOR ORBITAL FISSURE

```
├─ Location
│   ├─ Bone: Between greater and lesser wings of sphenoid
│   ├─ Fossa: Between middle cranial fossa and orbit
│   ├─ Shape: Comma-shaped, widest medially
│   └─ Divisions: Medial (within annulus of Zinn) and lateral compartments
│
├─ Contents (Complex - 7 Structures Pass Through)
│   │
│   ├─ WITHIN Annulus of Zinn (Tendinous Ring):
│   │   ├─ CN III (Oculomotor): Superior and inferior divisions
│   │   ├─ CN VI (Abducens): Lateral rectus motor
│   │   ├─ Nasociliary nerve (CN V1): Branch of ophthalmic division
│   │   └─ Sympathetics: To pupil dilator (from ICA plexus)
│   │
│   ├─ OUTSIDE Annulus of Zinn (Lateral to Ring):
│   │   ├─ CN IV (Trochlear): Superior oblique motor
│   │   ├─ Lacrimal nerve (CN V1): To lacrimal gland
│   │   ├─ Frontal nerve (CN V1): Supraorbital and supratrochlear sensation
│   │   └─ Superior ophthalmic vein: Orbit drainage → cavernous sinus
│   │
│   └─ Mnemonic: "Live Frankly To See Absolutely No Insult"
│       └─ Lacrimal, Frontal, Trochlear, Superior ophthalmic vein (outside annulus)
│           Abducens, Nasociliary, Inferior & superior divisions of CN III (inside annulus)
│
├─ Anatomical Relations
│   ├─ Medial: Optic canal (separated by optic strut)
│   ├─ Lateral: Greater wing of sphenoid
│   ├─ Superior: Lesser wing of sphenoid, frontal lobe dura
│   ├─ Inferior: Maxillary nerve foramen (foramen rotundum)
│   └─ Posterior: Cavernous sinus (CN III-VI before entering fissure)
│
├─ Clinical Significance
│   │
│   ├─ Superior Orbital Fissure Syndrome:
│   │   ├─ Definition: Dysfunction of all structures through SOF (CN III, IV, V1, VI + vein)
│   │   ├─ Causes:
│   │   │   ├─ Trauma: Sphenoid fracture, orbital fracture
│   │   │   ├─ Tumor: Cavernous sinus meningioma, metastasis
│   │   │   ├─ Infection: Cavernous sinus thrombosis (septic)
│   │   │   ├─ Inflammation: Tolosa-Hunt syndrome (painful ophthalmoplegia)
│   │   │   └─ Aneurysm: ICA aneurysm (cavernous segment)
│   │   │
│   │   ├─ Presentation (Complete Ophthalmoplegia + V1 Loss):
│   │   │   ├─ CN III palsy: Ptosis, eye down and out, dilated pupil
│   │   │   ├─ CN IV palsy: Difficulty looking down when eye adducted
│   │   │   ├─ CN VI palsy: Cannot abduct eye (esotropia)
│   │   │   ├─ CN V1 loss: Forehead, upper eyelid, corneal sensation loss
│   │   │   └─ Proptosis: If venous congestion (superior ophthalmic vein thrombosis)
│   │   │
│   │   └─ vs Orbital Apex Syndrome (adds CN II):
│   │       └─ If vision loss: Orbital apex syndrome (optic nerve involved)
│   │
│   ├─ Cavernous Sinus Thrombosis:
│   │   ├─ Route: Facial infection → angular vein → superior ophthalmic vein → cavernous sinus
│   │   ├─ Presentation:
│   │   │   ├─ Proptosis: Venous engorgement
│   │   │   ├─ Chemosis: Conjunctival edema
│   │   │   ├─ Ophthalmoplegia: CN III, IV, VI (SOF involvement)
│   │   │   ├─ Fever, headache, altered mental status
│   │   │   └─ Bilateral: Can spread via intercavernous sinuses (50%)
│   │   │
│   │   ├─ Diagnosis: MRI with contrast (filling defect in sinus), blood cultures
│   │   ├─ Treatment: IV antibiotics (vancomycin + 3rd gen cephalosporin), anticoagulation (controversial)
│   │   └─ Mortality: 20-30% even with treatment
│   │
│   └─ Tolosa-Hunt Syndrome:
│       ├─ Definition: Painful ophthalmoplegia from granulomatous inflammation of cavernous sinus
│       ├─ Presentation: Periorbital pain + CN III, IV, V1, VI palsies (same as SOF syndrome but painful)
│       ├─ Diagnosis: MRI (cavernous sinus enhancement), diagnosis of exclusion
│       ├─ Treatment: Corticosteroids (dramatic response - diagnostic)
│       └─ Prognosis: Good response to steroids, can recur
│
├─ Surgical Approaches
│   ├─ Transcranial (pterional, orbitozygomatic):
│   │   ├─ Exposure: Lateral and superior to SOF
│   │   ├─ Can drill: Anterior clinoid, roof of SOF (widen exposure)
│   │   └─ Uses: Cavernous sinus tumor, aneurysm, SOF decompression
│   │
│   ├─ Endoscopic endonasal (extended approach):
│   │   ├─ Exposure: Medial to SOF (via sphenoid sinus)
│   │   ├─ Limited: Cannot access entire SOF
│   │   └─ Uses: Medial cavernous sinus lesions
│   │
│   └─ Lateral orbitotomy:
│       ├─ Exposure: Orbital apex (includes SOF structures in orbit)
│       └─ Uses: Orbital apex tumors, decompression
│
└─ Tumors and Pathology
    ├─ Cavernous sinus meningioma:
    │   ├─ Involves: Cavernous sinus, often extends into SOF
    │   ├─ Presentation: Progressive cranial neuropathies (CN III-VI)
    │   └─ Treatment: Observation vs radiosurgery vs surgery (high morbidity)
    │
    └─ Pituitary apoplexy:
        ├─ Acute: Hemorrhage/infarction of pituitary tumor
        ├─ Lateral extension: Compresses cavernous sinus → SOF syndrome
        └─ Emergency: Urgent decompression if vision threatened
```

### 5. FORAMEN ROTUNDUM

```
├─ Location
│   ├─ Bone: Greater wing of sphenoid
│   ├─ Fossa: Middle cranial fossa floor → pterygopalatine fossa
│   ├─ Shape: Round opening (Latin "rotundum" = round)
│   └─ Direction: Anterolateral
│
├─ Contents
│   └─ Maxillary nerve (CN V2):
│       ├─ Branch: Trigeminal nerve (second division)
│       ├─ Pure sensory: Sensation from midface
│       ├─ Distribution:
│       │   ├─ Lower eyelid, cheek
│       │   ├─ Upper lip, nasal cavity (lateral wall)
│       │   ├─ Maxillary teeth and gingiva
│       │   ├─ Hard palate, soft palate (via palatine nerves)
│       │   └─ Maxillary sinus
│       │
│       └─ Branches (in pterygopalatine fossa):
│           ├─ Zygomatic nerve: To cheek, lateral orbit
│           ├─ Infraorbital nerve: Through infraorbital foramen → midface
│           ├─ Posterior superior alveolar: Maxillary molars
│           ├─ Palatine nerves: Greater and lesser palatine
│           └─ Nasal branches: Via pterygopalatine ganglion
│
├─ Anatomical Relations
│   ├─ Medial: Sphenoid sinus (lateral wall)
│   ├─ Lateral: Infratemporal fossa
│   ├─ Superior: Middle cranial fossa (Meckel's cave - trigeminal ganglion)
│   ├─ Anterior: Pterygopalatine fossa
│   └─ Posterior: Foramen ovale (V3 exit point, inferolateral)
│
├─ Clinical Significance
│   ├─ V2 neuralgia (rare):
│   │   ├─ Similar to trigeminal neuralgia but V2 distribution
│   │   ├─ Trigger: Touch to cheek, upper lip
│   │   └─ Treatment: Carbamazepine, MVD or radiofrequency rhizotomy
│   │
│   ├─ Perineural tumor spread:
│   │   ├─ Tumors: Adenoid cystic carcinoma, squamous cell carcinoma
│   │   ├─ Route: Maxillary sinus/nasal cavity tumor → V2 → foramen rotundum → Meckel's cave → brainstem
│   │   ├─ Imaging: MRI with contrast (nerve enhancement, thickening)
│   │   └─ Prognosis: Poor (indicates extensive disease)
│   │
│   └─ Sphenoid sinus pathology:
│       ├─ Mucocele, fungal infection: Can erode into foramen rotundum
│       └─ Presentation: V2 pain, numbness
│
├─ Surgical Approaches
│   ├─ Endoscopic endonasal:
│   │   ├─ Via sphenoid sinus: Lateral recess approach
│   │   ├─ Exposure: Foramen rotundum, pterygopalatine fossa
│   │   └─ Uses: V2 schwannoma, perineural tumor spread
│   │
│   ├─ Transcranial:
│   │   ├─ Middle fossa approach: Expose Meckel's cave
│   │   └─ Uses: Trigeminal schwannoma, V2 section (neuralgia)
│   │
│   └─ V2 block (interventional):
│       ├─ Pterygopalatine fossa approach: Via infraorbital or greater palatine foramen
│       └─ Uses: V2 neuralgia, cluster headache, chronic facial pain
│
└─ Tumors
    └─ V2 schwannoma:
        ├─ Dumbbell shape: Meckel's cave → foramen rotundum → pterygopalatine fossa
        ├─ Presentation: V2 numbness, pain
        └─ Treatment: Observation vs radiosurgery vs surgery
```

### 6. FORAMEN OVALE

```
├─ Location
│   ├─ Bone: Greater wing of sphenoid (posterolateral to foramen rotundum)
│   ├─ Fossa: Middle cranial fossa floor → infratemporal fossa
│   ├─ Shape: Oval opening (Latin "ovale" = oval)
│   └─ Size: ~7 x 5 mm (largest of middle fossa foramina)
│
├─ Contents
│   ├─ Mandibular nerve (CN V3):
│   │   ├─ Branch: Trigeminal nerve (third division - LARGEST)
│   │   ├─ MIXED: Motor + sensory (only division with motor)
│   │   │
│   │   ├─ Sensory distribution:
│   │   │   ├─ Lower face: Chin, lower lip
│   │   │   ├─ Mandibular teeth and gingiva
│   │   │   ├─ Tongue: Anterior 2/3 (general sensation, NOT taste)
│   │   │   ├─ TMJ, external ear (auriculotemporal nerve)
│   │   │   └─ Buccal mucosa, floor of mouth
│   │   │
│   │   └─ Motor distribution:
│   │       ├─ Muscles of mastication:
│   │       │   ├─ Masseter, temporalis
│   │       │   ├─ Medial pterygoid, lateral pterygoid
│   │       │   └─ Function: Jaw closure, jaw movement
│   │       │
│   │       └─ Other muscles:
│   │           ├─ Tensor veli palatini: Soft palate tension
│   │           ├─ Tensor tympani: Dampens sound (middle ear)
│   │           ├─ Mylohyoid: Floor of mouth
│   │           └─ Anterior belly of digastric
│   │
│   ├─ Accessory meningeal artery:
│   │   ├─ Branch: Maxillary artery (or middle meningeal)
│   │   ├─ Supplies: Gasserian ganglion, Meckel's cave
│   │   └─ Smaller than middle meningeal (foramen spinosum)
│   │
│   ├─ Lesser petrosal nerve (exits):
│   │   ├─ Origin: CN IX (glossopharyngeal) via tympanic plexus
│   │   ├─ Exits: Via foramen ovale (or nearby small foramen)
│   │   ├─ Destination: Otic ganglion (infratemporal fossa)
│   │   └─ Function: Parasympathetic to parotid gland
│   │
│   └─ Emissary veins:
│       └─ Connect: Cavernous sinus → pterygoid venous plexus
│
├─ Anatomical Relations
│   ├─ Medial: Foramen lacerum, carotid canal
│   ├─ Lateral: Infratemporal fossa
│   ├─ Superior: Meckel's cave (trigeminal ganglion)
│   ├─ Anterior: Foramen rotundum (V2)
│   └─ Posterior: Foramen spinosum (middle meningeal artery)
│
├─ Clinical Significance
│   │
│   ├─ Trigeminal Neuralgia (V3 division):
│   │   ├─ Less common than V2 (maxillary most common overall)
│   │   ├─ Pain distribution: Lower jaw, teeth, chin, tongue
│   │   ├─ Triggers: Chewing, talking, tooth brushing
│   │   │
│   │   └─ Treatment:
│   │       ├─ Medical: Carbamazepine 200-1200 mg/day
│   │       ├─ Percutaneous procedures (via foramen ovale):
│   │       │   ├─ Radiofrequency rhizotomy: Heat lesion to V3
│   │       │   ├─ Glycerol injection: Chemical neurolysis
│   │       │   ├─ Balloon compression: Mechanical compression of ganglion
│   │       │   └─ Access: Fluoroscopy-guided needle through cheek → foramen ovale
│   │       │
│   │       └─ Microvascular decompression: See CN V document (gold standard)
│   │
│   ├─ Percutaneous Trigeminal Rhizotomy (via Foramen Ovale):
│   │   ├─ Indications: V2 or V3 neuralgia, medically refractory
│   │   │
│   │   ├─ Technique (Radiofrequency):
│   │   │   ├─ Entry point: 2.5-3 cm lateral to oral commissure
│   │   │   ├─ Trajectory: Aim for pupil (anterior-posterior), aim for tragus (lateral)
│   │   │   ├─ Fluoroscopy: Lateral and submental views
│   │   │   ├─ Pass needle: Through cheek → foramen ovale → Meckel's cave
│   │   │   ├─ CSF return: Confirms Meckel's cave entry (trigeminal cistern)
│   │   │   ├─ Stimulation: Identify V1, V2, V3 (patient awake)
│   │   │   ├─ Position: Electrode in affected division
│   │   │   ├─ Lesion: 60-80°C for 60-90 seconds (under sedation)
│   │   │   └─ Result: Selective numbness in affected division
│   │   │
│   │   ├─ Success rate: 90-98% initial pain relief
│   │   ├─ Complications:
│   │   │   ├─ Numbness: Expected (goal), can be bothersome
│   │   │   ├─ Dysesthesia: Unpleasant abnormal sensation (5-10%)
│   │   │   ├─ Anesthesia dolorosa: Painful numbness (rare, 1-2%)
│   │   │   ├─ Masseter weakness: If motor root affected (V3 rhizotomy)
│   │   │   ├─ Corneal anesthesia: If V1 involved (keratitis risk)
│   │   │   └─ Intracranial hemorrhage: Rare (<0.1%)
│   │   │
│   │   └─ Recurrence: 15-30% at 5 years (can repeat procedure)
│   │
│   ├─ Perineural Tumor Spread:
│   │   ├─ Tumors: Oral cavity SCC, skin cancer (chin/lower lip)
│   │   ├─ Route: Tumor → inferior alveolar nerve (or mental nerve) → V3 → foramen ovale → Meckel's cave
│   │   ├─ Imaging: MRI (nerve enhancement, foramen widening)
│   │   └─ Prognosis: Advanced disease
│   │
│   └─ Mandibular Nerve Block:
│       ├─ Indication: Dental anesthesia, chronic facial pain
│       ├─ Approach: Intraoral (inferior alveolar nerve) vs percutaneous (foramen ovale)
│       └─ Uses: Surgery on lower jaw, teeth, floor of mouth
│
├─ Surgical Approaches
│   ├─ Percutaneous (for rhizotomy):
│   │   └─ Via cheek, through foramen ovale to Meckel's cave
│   │
│   ├─ Middle fossa approach:
│   │   ├─ Craniotomy: Temporal craniotomy (elevate temporal lobe)
│   │   ├─ Exposure: Floor of middle fossa, Meckel's cave
│   │   └─ Uses: V3 schwannoma, trigeminal ganglion lesions
│   │
│   └─ Infratemporal fossa approach:
│       ├─ Access: From below (temporal bone, mandible)
│       └─ Uses: Skull base tumors, access to V3 in infratemporal fossa
│
└─ Tumors and Pathology
    ├─ V3 schwannoma:
    │   ├─ Dumbbell: Meckel's cave → foramen ovale → infratemporal fossa → masticator space
    │   ├─ Presentation: V3 numbness, masseter atrophy (motor loss), pain
    │   └─ Treatment: Observation vs radiosurgery vs surgery
    │
    └─ Trigeminal ganglion pathology:
        ├─ Tumors: Trigeminal schwannoma (involves ganglion, all divisions)
        └─ Infections: Herpes zoster (V3 - rare compared to V1)
```

### 7. FORAMEN SPINOSUM

```
├─ Location
│   ├─ Bone: Greater wing of sphenoid (posterolateral to foramen ovale)
│   ├─ Fossa: Middle cranial fossa floor → infratemporal fossa
│   ├─ Shape: Small, round (smallest of V-related foramina)
│   └─ Name: "Spinosum" refers to spine of sphenoid (nearby landmark)
│
├─ Contents
│   ├─ Middle meningeal artery (PRIMARY):
│   │   ├─ Branch: Maxillary artery (1st or 2nd part)
│   │   ├─ Course: Through foramen spinosum → middle cranial fossa
│   │   ├─ Divides: Into frontal (anterior) and parietal (posterior) branches
│   │   │   ├─ Frontal branch: Anterior, superior (grooves pterion area)
│   │   │   └─ Parietal branch: Posterior, superior
│   │   │
│   │   ├─ Supplies: Dura mater (majority of intracranial dura)
│   │   └─ Grooves: Inner table of skull (visible on skull - arterial grooves)
│   │
│   ├─ Middle meningeal vein:
│   │   └─ Accompanies artery
│   │
│   └─ Meningeal branch of mandibular nerve (nervus spinosus):
│       ├─ Recurrent branch: V3 → foramen spinosum → back to middle fossa
│       └─ Supplies: Dura of middle cranial fossa (sensory)
│
├─ Anatomical Relations
│   ├─ Anterior: Foramen ovale (V3)
│   ├─ Medial: Foramen lacerum, carotid canal
│   ├─ Lateral: Spine of sphenoid, infratemporal fossa
│   ├─ Superior: Middle cranial fossa, temporal lobe
│   └─ Pterion: Middle meningeal artery frontal branch grooves pterion (2-3 cm above) - CRITICAL TRAUMA LANDMARK
│
├─ Clinical Significance (TRAUMA - Most Important)
│   │
│   ├─ Epidural Hematoma (Extradural Hematoma):
│   │   ├─ Mechanism: Temporal bone fracture → middle meningeal artery tear
│   │   ├─ Location: Typically at pterion (frontal branch most commonly injured)
│   │   │   └─ Pterion: Weakest part of skull (frontal, parietal, temporal, sphenoid meet)
│   │   │
│   │   ├─ Pathophysiology:
│   │   │   ├─ Arterial bleeding: Strips dura from inner table → blood accumulates
│   │   │   ├─ Biconvex (lenticular): Dura tethered at sutures, cannot cross
│   │   │   └─ Rapid expansion: Arterial pressure (hours, not days)
│   │   │
│   │   ├─ Presentation (Classic):
│   │   │   ├─ Initial injury: Brief loss of consciousness
│   │   │   ├─ "Lucid interval": Patient awakes, seems fine (30 min - 6 hours)
│   │   │   ├─ Deterioration: Headache → vomiting → decreased GCS
│   │   │   ├─ Herniation signs:
│   │   │   │   ├─ Ipsilateral pupil dilation (CN III compression)
│   │   │   │   ├─ Contralateral hemiparesis (cerebral peduncle compression)
│   │   │   │   └─ Cushing triad (late): HTN, bradycardia, irregular breathing
│   │   │   │
│   │   │   └─ Atypical: ~50% do NOT have lucid interval (if severe initial injury)
│   │   │
│   │   ├─ CT Imaging:
│   │   │   ├─ Biconvex (lens-shaped) hyperdense collection
│   │   │   ├─ Does NOT cross suture lines (dura tethered)
│   │   │   ├─ May cross midline: If large, can have falcine EDH
│   │   │   ├─ Mass effect: Midline shift, effacement of sulci
│   │   │   └─ "Swirl sign": Active bleeding (different densities)
│   │   │
│   │   ├─ Treatment:
│   │   │   ├─ Small (<30 mL, <15 mm thick, <5 mm shift, GCS 13-15): Conservative
│   │   │   │   └─ ICU monitoring, serial CTs q4-6h, ICP monitoring
│   │   │   │
│   │   │   ├─ Surgical evacuation (URGENT):
│   │   │   │   ├─ Indications: Large (>30 mL), symptoms, GCS decline, midline shift >5 mm
│   │   │   │   ├─ Craniotomy: Centered over hematoma (pterional/temporal)
│   │   │   │   ├─ Technique:
│   │   │   │   │   ├─ Expose bone over clot
│   │   │   │   │   ├─ Evacuate clot (suction, irrigation)
│   │   │   │   │   ├─ Identify bleeding source (middle meningeal artery)
│   │   │   │   │   ├─ Hemostasis: Coagulate artery (bipolar), bone wax in foramen spinosum
│   │   │   │   │   ├─ Dural tacking sutures: Reapproximate dura to bone (prevent reaccumulation)
│   │   │   │   │   └─ Bone flap: Replace (or leave off if severe swelling)
│   │   │   │   │
│   │   │   │   └─ Prognosis: Excellent if treated early (mortality <5%), poor if delayed (>50%)
│   │   │   │
│   │   │   └─ Burr hole evacuation: Rarely sufficient (arterial bleeding usually reaccumulates)
│   │   │
│   │   └─ Outcome:
│   │       ├─ Early surgery (<4 hours): 90-95% good recovery
│   │       ├─ Delayed (>4 hours, herniation): 50-70% mortality
│   │       └─ Young patients: Better outcomes (less brain atrophy, more compensation)
│   │
│   ├─ Middle Meningeal Artery Aneurysm (Rare):
│   │   ├─ Traumatic pseudoaneurysm: After skull fracture
│   │   ├─ Presentation: Delayed EDH (days-weeks after trauma)
│   │   └─ Treatment: Endovascular embolization or surgical clipping
│   │
│   └─ Intraoperative Landmark:
│       ├─ Pterional craniotomy: MMA frontal branch marks position
│       └─ Must coagulate: When drilling bone (prevent epidural hemorrhage)
│
├─ Surgical Approaches
│   └─ Pterional craniotomy:
│       ├─ Exposure: Frontal branch of MMA encountered on inner table
│       ├─ Coagulate: Bipolar coagulation before drilling
│       └─ Key landmark: MMA grooves in bone guide surgical plane
│
└─ Variations
    ├─ Absent foramen: 1-2% (MMA enters via foramen ovale instead)
    └─ Duplication: Rare (accessory MMA branch)
```

### 8. FORAMEN LACERUM

```
├─ Location
│   ├─ Bones: Sphenoid, temporal (petrous apex), occipital (junction of all 3)
│   ├─ Fossa: Base of skull (between middle and posterior fossae)
│   ├─ Shape: Irregular, jagged opening
│   └─ Name: "Lacerum" = torn/lacerated (irregular appearance)
│
├─ Contents (NOTHING PASSES THROUGH IN LIVING - IMPORTANT!)
│   ├─ In living person:
│   │   ├─ Filled: By fibrocartilage (cartilage plug)
│   │   ├─ Closed: No neurovascular structures pass THROUGH foramen
│   │   └─ Artifact: In dry skull, appears open (cartilage removed)
│   │
│   ├─ Structures CROSSING OVER (not through):
│   │   ├─ Internal carotid artery:
│   │   │   ├─ Enters skull: Via carotid canal (inferior)
│   │   │   ├─ Turns: Horizontally at foramen lacerum (C2 segment - genu)
│   │   │   ├─ Courses: ABOVE foramen lacerum (on superior surface of cartilage)
│   │   │   └─ Enters: Cavernous sinus (anteromedial)
│   │   │
│   │   └─ Sympathetic plexus: Around ICA (follows artery)
│   │
│   └─ Structures exiting nearby:
│       ├─ Greater petrosal nerve: Emerges at anterior edge (exits to pterygoid canal)
│       └─ Deep petrosal nerve: Sympathetics join greater petrosal (form Vidian nerve)
│
├─ Anatomical Relations
│   ├─ Anterior: Pterygoid canal (Vidian canal) opening
│   ├─ Posterior: Petrous apex, foramen ovale, carotid canal
│   ├─ Medial: Clivus, sphenoid body
│   ├─ Lateral: Foramen ovale, cavernous sinus
│   └─ Superior: Meckel's cave (trigeminal ganglion), cavernous sinus
│
├─ Clinical Significance
│   ├─ ICA aneurysm (C2 segment - genu):
│   │   ├─ Location: ICA as it turns above foramen lacerum
│   │   ├─ Rare location: <1% of intracranial aneurysms
│   │   └─ Treatment: Endovascular (difficult surgical access)
│   │
│   ├─ Petrous apex lesions:
│   │   ├─ Cholesterol granuloma: Expands from petrous apex, can erode foramen lacerum
│   │   ├─ Chondrosarcoma: Arises from cartilage remnants (foramen lacerum origin possible)
│   │   └─ Imaging: MRI to differentiate
│   │
│   └─ Skull base fractures:
│       └─ Petrous temporal bone fractures: Can involve foramen lacerum (ICA injury risk)
│
├─ Surgical Approaches
│   └─ Exposure (rarely targeted directly):
│       ├─ Lateral approaches: Infratemporal fossa type A, B, C
│       ├─ Petrous apex approaches: Transsphenoidal extended, middle fossa
│       └─ Endoscopic endonasal: Can visualize superior surface
│
└─ Common Teaching Point
    └─ "Nothing goes through foramen lacerum in living person - only cartilage" ⭐
```

### 9. CAROTID CANAL

```
├─ Location
│   ├─ Bone: Petrous temporal bone (inferior surface)
│   ├─ Entry: Inferior surface of skull (between jugular foramen and foramen lacerum)
│   ├─ Course: Vertical then horizontal (S-shaped)
│   └─ Exit: Foramen lacerum region (ICA emerges superior to lacerum)
│
├─ Contents
│   ├─ Internal carotid artery (ICA):
│   │   ├─ Segments within canal:
│   │   │   ├─ C2 (petrous/vertical): Ascends vertically
│   │   │   └─ C3 (petrous/horizontal - genu): Turns horizontally at foramen lacerum
│   │   │
│   │   ├─ Branches (within canal):
│   │   │   ├─ Caroticotympanic artery: To middle ear
│   │   │   └─ Pterygoid (vidian) artery: Through pterygoid canal
│   │   │
│   │   └─ No major branches: Main trunk to cavernous sinus
│   │
│   ├─ Sympathetic plexus (carotid sympathetic plexus):
│   │   ├─ From: Superior cervical ganglion
│   │   ├─ Forms: Pericarotid plexus (mesh around ICA)
│   │   ├─ Function:
│   │   │   ├─ Pupil dilation (via long ciliary nerves)
│   │   │   ├─ Lid elevation (Müller's muscle)
│   │   │   └─ Facial sweating
│   │   │
│   │   └─ Branch: Deep petrosal nerve (exits to join greater petrosal → Vidian nerve)
│   │
│   └─ Venous plexus:
│       └─ Small veins surrounding ICA
│
├─ ICA Segments Review (Within and Near Canal)
│   ├─ C1 (cervical): Neck
│   ├─ C2 (petrous - vertical): Ascending in carotid canal
│   ├─ C3 (petrous - horizontal/genu): Horizontal turn at foramen lacerum
│   ├─ C4 (cavernous - lacerum): Enters cavernous sinus
│   ├─ C5 (cavernous - horizontal)
│   ├─ C6 (cavernous - clinoid/ophthalmic): Exits cavernous, gives off ophthalmic artery
│   └─ C7 (supraclinoid/communicating): Posterior communicating, anterior choroidal branches
│
├─ Anatomical Relations
│   ├─ Anterior: Eustachian tube (tensor tympani canal)
│   ├─ Posterior: Jugular foramen, inferior petrosal sinus
│   ├─ Medial: Petrous apex
│   ├─ Lateral: Middle ear cavity (separated by thin bone)
│   └─ Superior: Foramen lacerum, Meckel's cave, cavernous sinus
│
├─ Clinical Significance
│   │
│   ├─ Carotid Artery Dissection (Petrous ICA):
│   │   ├─ Causes: Trauma (including chiropractic manipulation), spontaneous
│   │   ├─ Presentation:
│   │   │   ├─ Horner's syndrome: Sympathetic plexus disruption (ptosis, miosis, anhidrosis)
│   │   │   ├─ Headache, neck pain (ipsilateral)
│   │   │   ├─ Stroke: If severe stenosis or thromboembolic
│   │   │   └─ Pulsatile tinnitus: Turbulent flow
│   │   │
│   │   ├─ Imaging: CTA or MRA (dissection flap, pseudoaneurysm)
│   │   └─ Treatment: Antiplatelet or anticoagulation (prevent stroke), endovascular stenting if severe
│   │
│   ├─ Petrous ICA Aneurysm:
│   │   ├─ Rare: <1% of intracranial aneurysms
│   │   ├─ Presentation:
│   │   │   ├─ Pulsatile tinnitus
│   │   │   ├─ Hearing loss (mass effect on middle ear)
│   │   │   ├─ CN VI palsy (Dorello's canal nearby)
│   │   │   └─ Subarachnoid hemorrhage: Rare (extradural location)
│   │   │
│   │   ├─ Imaging: CTA, angiography (differentiate from glomus tumor)
│   │   └─ Treatment: Endovascular (coiling, flow diversion), surgical ligation + bypass (rare)
│   │
│   ├─ Temporal Bone Fracture:
│   │   ├─ Longitudinal fracture (80%): Usually spares ICA
│   │   ├─ Transverse fracture (20%): Higher risk ICA injury
│   │   │
│   │   ├─ ICA injury (rare but devastating):
│   │   │   ├─ Occlusion: Stroke
│   │   │   ├─ Pseudoaneurysm: Delayed massive epistaxis
│   │   │   ├─ Carotid-cavernous fistula: If extends to cavernous portion
│   │   │   └─ Exsanguinating epistaxis: If connection to Eustachian tube/middle ear
│   │   │
│   │   ├─ Diagnosis: CTA (all temporal bone fractures with carotid canal involvement)
│   │   └─ Management: Endovascular (sacrifice with balloon occlusion ± bypass, or stenting)
│   │
│   ├─ Persistent Stapedial Artery:
│   │   ├─ Congenital variant: Aberrant vessel through middle ear (from petrous ICA)
│   │   ├─ Presentation: Pulsatile tinnitus, mass behind tympanic membrane
│   │   └─ Importance: Recognize before middle ear surgery (catastrophic bleeding if cut)
│   │
│   └─ Horner's Syndrome:
│       ├─ Lesion: Any pathology affecting carotid sympathetic plexus in canal
│       ├─ Causes: ICA dissection, tumor, cavernous sinus lesion
│       └─ Presentation: Ptosis, miosis, anhidrosis (face)
│
├─ Surgical Approaches
│   ├─ Infratemporal fossa approach (type A):
│   │   ├─ Access: Lateral (temporal bone resection)
│   │   ├─ Exposure: Entire petrous ICA
│   │   └─ Uses: Petrous apex tumor, ICA aneurysm, glomus jugulare
│   │
│   ├─ Middle fossa approach:
│   │   ├─ Access: Superior (elevate temporal lobe)
│   │   ├─ Exposure: Vertical petrous ICA
│   │   └─ Uses: Superior petrous apex lesions
│   │
│   └─ Endovascular:
│       ├─ Access: Femoral artery → ICA
│       └─ Uses: Aneurysm, pseudoaneurysm, dissection treatment (stenting, coiling)
│
└─ Tumors
    ├─ Glomus tympanicum:
    │   ├─ Location: Middle ear (tympanic plexus), can extend to carotid canal
    │   ├─ Presentation: Pulsatile tinnitus, conductive hearing loss, "rising sun sign" (retrotympanic mass)
    │   └─ Treatment: Embolization + surgical excision
    │
    └─ Carotid body paraganglioma extension:
        ├─ Rarely: Extends intracranially via carotid canal
        └─ Treatment: Surgical excision (may sacrifice ICA with bypass)
```

## POSTERIOR CRANIAL FOSSA FORAMINA

### 10. INTERNAL AUDITORY CANAL (IAC) (Porus Acusticus Internus)

```
├─ Location
│   ├─ Bone: Petrous temporal bone (posterior surface)
│   ├─ Length: ~8-10 mm (from porus to fundus)
│   ├─ Orientation: Lateral to medial (opens into CPA)
│   └─ Shape: Cylindrical canal
│
├─ Contents (4 structures - "7 UP, Coke DOWN" mnemonic for order at fundus)
│   ├─ Superior compartment (separated by horizontal crest):
│   │   ├─ Facial nerve (CN VII) - ANTERIOR SUPERIOR ⭐
│   │   │   ├─ Position: "7" - most anterosuperior
│   │   │   ├─ Course: IAC → labyrinthine segment → geniculate ganglion
│   │   │   └─ CRITICAL: Most at risk during IAC surgery
│   │   │
│   │   └─ Superior vestibular nerve - POSTERIOR SUPERIOR
│   │       ├─ Innervation: Horizontal + anterior semicircular canals, utricle
│   │
│   ├─ Inferior compartment:
│   │   ├─ Cochlear nerve - ANTERIOR INFERIOR
│   │   │   ├─ From: Spiral ganglion (modiolus) → cochlear nuclei
│   │   │   └─ Function: Hearing
│   │   │
│   │   └─ Inferior vestibular nerve - POSTERIOR INFERIOR
│   │       ├─ Innervation: Posterior semicircular canal, saccule
│   │
│   ├─ Labyrinthine artery (AICA branch):
│   │   ├─ Primary blood supply: To inner ear structures
│   │   ├─ Anatomical importance: Can be damaged during CPA surgery
│   │   └─ If injured: Complete hearing loss (cochlea ischemia)
│   │
│   └─ Meatal segment boundaries (Bill's Bar):
│       ├─ Horizontal crest (Bill's Bar): Separates superior from inferior compartment
│       ├─ Vertical crest (Bill's Bar vertical): Separates anterior from posterior
│       └─ Surgical landmark: Identifies facial nerve position (anterosuperior)
│
├─ Anatomical Relations
│   ├─ Medial (opens into): Cerebellopontine angle (CPA)
│   ├─ Lateral (fundus): Labyrinth (vestibule, cochlea)
│   ├─ Anterior: Cochlea (basal turn)
│   ├─ Posterior: Vestibule
│   ├─ Superior: Middle cranial fossa dura (tegmen)
│   └─ Inferior: Jugular bulb (variable - can be high-riding)
│
├─ Clinical Significance
│   ├─ Acoustic Neuroma (Vestibular Schwannoma) ⭐⭐⭐:
│   │   ├─ Origin: 95% from superior vestibular nerve (Scarpa's ganglion region)
│   │   ├─ Location: IAC → CPA (dumbbell shape)
│   │   │
│   │   ├─ Clinical presentation:
│   │   │   ├─ Hearing loss: Asymmetric sensorineural (90-95%)
│   │   │   ├─ Tinnitus: Unilateral (70%)
│   │   │   ├─ Vestibular: Imbalance (60%), rarely acute vertigo
│   │   │   ├─ Facial weakness: Late (large tumors compressing CN VII)
│   │   │   ├─ Trigeminal symptoms: Facial numbness (CN V - lateral CPA compression)
│   │   │   └─ Hydrocephalus: Very large tumors (4th ventricle compression)
│   │   │
│   │   ├─ Diagnostic tests:
│   │   │   ├─ MRI with gadolinium (gold standard): Bright enhancement, "ice cream cone" appearance
│   │   │   ├─ Audiometry: Asymmetric SNHL, poor speech discrimination
│   │   │   ├─ BAEP: Prolonged I-V latency (retrocochlear pattern)
│   │   │   └─ Tumor extends: Into IAC (diagnostic feature - differentiates from meningioma)
│   │   │
│   │   ├─ Classification (size):
│   │   │   ├─ Intracanalicular: Confined to IAC (<1 cm)
│   │   │   ├─ Small: <1.5 cm CPA extension
│   │   │   ├─ Medium: 1.5-2.5 cm
│   │   │   ├─ Large: 2.5-4 cm
│   │   │   └─ Giant: >4 cm
│   │   │
│   │   └─ Treatment (see Surgical Approaches below)
│   │
│   ├─ Facial Nerve Schwannoma:
│   │   ├─ Rare: <5% of temporal bone tumors
│   │   ├─ Location: Can involve IAC segment, geniculate, or tympanic segment
│   │   ├─ Presentation: Progressive facial weakness (vs. sudden in Bell's palsy)
│   │   └─ Treatment: Observation vs. resection with grafting
│   │
│   └─ Meningitis → Labyrinthitis Ossificans:
│       ├─ Mechanism: Infection spreads from CSF → IAC → inner ear
│       ├─ Result: Cochlear ossification (2-3 weeks post-infection)
│       └─ Implication: Cochlear implant difficult/impossible if delayed
│
├─ Surgical Approaches (for Acoustic Neuroma)
│   ├─ Retrosigmoid (Suboccipital) Approach:
│   │   ├─ Position: Lateral park bench or supine with head turned
│   │   ├─ Incision: Postauricular vertical or hockey stick
│   │   ├─ Craniotomy: Retrosigmoid (behind sigmoid sinus)
│   │   ├─ Exposure: CPA + posterior IAC
│   │   │
│   │   ├─ Advantages:
│   │   │   ├─ Hearing preservation possible (if tumor small, hearing good)
│   │   │   ├─ Familiar approach (neurosurgeons comfortable)
│   │   │   ├─ Can address brainstem compression
│   │   │   └─ Can extend superiorly (tentorium) or inferiorly (foramen magnum)
│   │   │
│   │   ├─ Disadvantages:
│   │   │   ├─ Limited IAC visualization (need to drill posterior IAC to see fundus)
│   │   │   ├─ CSF leak risk: 5-10% (dural opening, air cells)
│   │   │   └─ Headache: Posterior fossa headache common
│   │   │
│   │   ├─ Technique (key steps):
│   │   │   ├─ Identify: Sigmoid sinus, transverse sinus, emissary veins
│   │   │   ├─ Craniotomy: Behind sigmoid sinus (3-4 cm diameter)
│   │   │   ├─ Dural opening: C-shaped toward sinus
│   │   │   ├─ CSF release: Open cerebellomedullary cistern (brain relaxation)
│   │   │   ├─ Identify: CN V (superior), CN IX (inferior), flocculus (landmarks)
│   │   │   ├─ IAC identification: Lateral to CN VII-VIII complex
│   │   │   ├─ Tumor dissection:
│   │   │   │   ├─ Debulk: Internal decompression (CUSA, ultrasonic aspirator)
│   │   │   │   ├─ Capsule: Dissect from CN VII, CN VIII, brainstem
│   │   │   │   ├─ CN VII identification: Medial to tumor (use stimulator 0.05-0.1 mA)
│   │   │   │   └─ Dissection plane: Between tumor capsule and nerve (gliotic)
│   │   │   ├─ IAC drilling: Drill posterior IAC wall (high-speed drill, diamond burr)
│   │   │   ├─ Fundus exposure: Identify Bill's Bar (CN VII anterosuperior to CN VIII)
│   │   │   └─ Hemostasis: Oxidized cellulose, fibrin glue
│   │   │
│   │   └─ Facial nerve outcomes:
│   │       ├─ Small tumors (<2 cm): 90-95% normal function (HB I-II)
│   │       ├─ Large tumors (>3 cm): 50-70% normal function
│   │       └─ Monitoring: EMG, direct stimulation (essential)
│   │
│   ├─ Translabyrinthine Approach:
│   │   ├─ Position: Supine, head turned (ENT position)
│   │   ├─ Incision: Postauricular C-shaped
│   │   ├─ Mastoidectomy: Complete (remove all air cells)
│   │   ├─ Labyrinthectomy: Drill out semicircular canals, vestibule
│   │   │
│   │   ├─ Advantages:
│   │   │   ├─ Excellent IAC exposure: Complete fundus visualization
│   │   │   ├─ Early CN VII identification: At fundus (Bill's Bar)
│   │   │   ├─ No brain retraction: Direct lateral approach
│   │   │   ├─ Lower CSF leak: Fat graft closure (3-5% leak rate)
│   │   │   └─ Best for large tumors: Good working angle
│   │   │
│   │   ├─ Disadvantages:
│   │   │   ├─ Hearing sacrifice: Complete (labyrinthectomy)
│   │   │   ├─ Limited brainstem exposure: Can't address medial extension well
│   │   │   └─ Long operative time: Extensive drilling
│   │   │
│   │   ├─ Indications:
│   │   │   ├─ No serviceable hearing (SDS <50%, PTA >50 dB)
│   │   │   ├─ Large tumors (>3 cm)
│   │   │   ├─ Elderly patients (avoid brain retraction)
│   │   │   └─ Only hearing ear on opposite side
│   │   │
│   │   └─ Technique:
│   │       ├─ Mastoidectomy: Expose sigmoid sinus, tegmen, sinodural angle
│   │       ├─ Identify: Semicircular canals (lateral → superior → posterior)
│   │       ├─ Labyrinthectomy: Remove all inner ear structures
│   │       ├─ IAC exposure: Drill 270° around IAC (blue-line inferior = jugular bulb)
│   │       ├─ Dural opening: IAC dura + posterior fossa dura
│   │       ├─ Bill's Bar identification: CN VII anterosuperior to tumor
│   │       └─ Closure: Abdominal fat graft (obliterate mastoid, seal IAC)
│   │
│   └─ Middle Cranial Fossa Approach:
│       ├─ Position: Supine, head turned away
│       ├─ Incision: Preauricular vertical (temporal)
│       ├─ Craniotomy: Temporal squamosa (above EAC, anterior to sigmoid)
│       ├─ Exposure: Extradural elevation of temporal lobe → superior IAC
│       │
│       ├─ Advantages:
│       │   ├─ Best hearing preservation: Cochlea untouched (lateral approach to IAC)
│       │   ├─ Excellent fundus exposure: Superior-to-inferior view
│       │   └─ Ideal for small intracanalicular tumors
│       │
│       ├─ Disadvantages:
│       │   ├─ Limited to small tumors: <1.5 cm (cannot address CPA extension well)
│       │   ├─ Temporal lobe retraction: Risk of seizures, venous infarct
│       │   ├─ Greater superficial petrosal nerve (GSPN): Must preserve (lacrimal function)
│       │   └─ Technically demanding: Steep learning curve
│       │
│       ├─ Indications:
│       │   ├─ Small intracanalicular tumor (<1.5 cm, <1 cm CPA extension)
│       │   ├─ Good hearing (SDS >70%, PTA <30 dB)
│       │   ├─ Young patient (preserve hearing)
│       │   └─ Facial nerve schwannoma (IAC segment)
│       │
│       └─ Technique:
│           ├─ Landmarks: "House-Hitselberger triangle" boundaries:
│           │   ├─ Anterior: Greater superficial petrosal nerve (GSPN)
│           │   ├─ Posterior: Superior semicircular canal (blue-lined)
│           │   └─ Lateral: IAC (centered in triangle)
│           ├─ Drilling: Superior IAC roof (180° exposure, "eggshell thin")
│           ├─ Dural opening: IAC dura (preserve labyrinthine artery!)
│           ├─ Bill's Bar: Identifies CN VII (anterosuperior to tumor)
│           └─ Tumor removal: Microsurgical dissection from CN VII, cochlear nerve
│
├─ Complications (Acoustic Neuroma Surgery)
│   ├─ Facial nerve dysfunction:
│   │   ├─ Anatomic preservation: 95-99% (modern series)
│   │   ├─ Functional preservation (HB I-II):
│   │   │   ├─ Small tumor (<2 cm): 90-95%
│   │   │   ├─ Large tumor (>3 cm): 50-70%
│   │   │   └─ Giant tumor (>4 cm): 30-50%
│   │   └─ Monitoring: Continuous EMG + direct stimulation (mandatory)
│   │
│   ├─ Hearing loss:
│   │   ├─ Retrosigmoid: Preservation possible (30-50% if hearing good, tumor small)
│   │   ├─ Translabyrinthine: 100% loss (intentional - labyrinthectomy)
│   │   └─ Middle fossa: Best preservation (60-70% if tumor <1 cm)
│   │
│   ├─ CSF leak:
│   │   ├─ Retrosigmoid: 5-10%
│   │   ├─ Translabyrinthine: 3-5% (fat graft reduces risk)
│   │   └─ Management: Lumbar drain, reoperation if persistent
│   │
│   ├─ Meningitis:
│   │   ├─ Risk: 1-2% (all approaches)
│   │   └─ Prophylaxis: Perioperative antibiotics
│   │
│   ├─ Vascular injury:
│   │   ├─ AICA injury: Cerebellar infarct, brainstem ischemia
│   │   │   └─ Prevention: Identify early, preserve if possible
│   │   └─ Venous injury: Sigmoid sinus, superior petrosal sinus (usually tolerated)
│   │
│   └─ Brainstem injury:
│       ├─ Risk: Higher with large tumors (>3 cm), ventral extension
│       └─ Manifestation: CN deficits, ataxia, respiratory compromise
│
└─ Special Considerations
    ├─ Neurofibromatosis Type 2 (NF2):
    │   ├─ Bilateral acoustic neuromas: Pathognomonic (diagnostic)
    │   ├─ Genetics: NF2 gene mutation (chromosome 22q12)
    │   ├─ Management challenges:
    │   │   ├─ Bilateral surgery: Risk total deafness
    │   │   ├─ Hearing preservation critical: Consider auditory brainstem implant (ABI)
    │   │   └─ Timing: Operate on worse hearing side first
    │   │
    │   └─ Other features: Meningiomas, ependymomas, spinal schwannomas
    │
    └─ Hearing preservation factors:
        ├─ Favorable:
        │   ├─ Small tumor (<1.5 cm)
        │   ├─ Good preoperative hearing (SDS >70%, PTA <30 dB)
        │   ├─ Lateral tumor (no CPA extension)
        │   └─ Young age (<50 years)
        │
        └─ Unfavorable:
            ├─ Large tumor (>2.5 cm)
            ├─ Poor preoperative hearing
            └─ Tumor at brainstem
```

### 11. JUGULAR FORAMEN (Foramen Jugulare)

```
├─ Location
│   ├─ Bones: Between petrous temporal bone (anterior) and occipital bone (posterior)
│   ├─ Position: Posterior cranial fossa floor
│   ├─ Shape: Irregular, divided into compartments
│   └─ Size: Largest foramen in posterior fossa (after foramen magnum)
│
├─ Compartments (3 distinct divisions - CRITICAL ANATOMY ⭐⭐⭐)
│   ├─ ANTERIOR COMPARTMENT (Pars Nervosa):
│   │   ├─ Boundaries: Separated by jugular spine (pyramidal process)
│   │   ├─ Contents:
│   │   │   ├─ Inferior petrosal sinus (IPS): Drains cavernous sinus → jugular bulb
│   │   │   └─ Glossopharyngeal nerve (CN IX): ONLY nerve in anterior compartment
│   │   │
│   │   └─ Clinical importance: CN IX isolated in this compartment
│   │
│   ├─ INTERMEDIATE COMPARTMENT (Pars Vascularis - Posterior Part):
│   │   ├─ Contents:
│   │   │   ├─ Vagus nerve (CN X): LARGEST nerve in jugular foramen ⭐
│   │   │   │   ├─ Position: Posteromedial (closest to sigmoid sinus)
│   │   │   │   └─ Most commonly injured: During jugular foramen surgery
│   │   │   │
│   │   │   └─ Accessory nerve (CN XI - cranial root):
│   │   │       ├─ Course: Joins spinal root → exits together as CN XI
│   │   │       └─ Position: Posterior to CN X
│   │   │
│   │   └─ KEY: CN X and XI travel together (close proximity)
│   │
│   └─ POSTERIOR COMPARTMENT (Pars Vascularis - Main):
│       ├─ Contents:
│       │   ├─ Internal jugular vein (IJV) - DOMINANT structure ⭐:
│       │   │   ├─ Origin: Begins at jugular bulb (sigmoid sinus continuation)
│       │   │   ├─ Size: Largest structure in foramen
│       │   │   └─ Dominance: Usually right > left (60-70% right dominant)
│       │   │
│       │   └─ Posterior meningeal artery (small):
│       │       ├─ Origin: Ascending pharyngeal artery branch
│       │       └─ Supply: Posterior fossa dura
│       │
│       └─ Clinical: Jugular bulb position highly variable (high-riding bulb common)
│
├─ Anatomical Relations (CRITICAL for surgical approaches)
│   ├─ Medial: Medulla oblongata (CN IX, X, XI nuclei)
│   ├─ Lateral: Mastoid process, facial nerve (CN VII) runs above
│   ├─ Anterior: Carotid canal (ICA), petrous apex
│   ├─ Posterior: Occipital condyle, rectus capitis lateralis muscle
│   ├─ Superior: Petrous temporal bone, inferior petrosal sinus
│   └─ Inferior: Neck (IJV descends in carotid sheath)
│
├─ Clinical Significance
│   ├─ Jugular Foramen Syndrome (Vernet Syndrome) ⭐⭐:
│   │   ├─ Definition: CN IX, X, XI palsy (all three nerves affected)
│   │   ├─ Causes:
│   │   │   ├─ Tumor: Glomus jugulare (most common), schwannoma, meningioma
│   │   │   ├─ Trauma: Skull base fracture (temporal bone)
│   │   │   ├─ Infection: Osteomyelitis, abscess
│   │   │   └─ Vascular: IJV thrombosis, dissection
│   │   │
│   │   ├─ Clinical presentation:
│   │   │   ├─ CN IX (glossopharyngeal):
│   │   │   │   ├─ Loss of gag reflex (afferent arm)
│   │   │   │   ├─ Loss of taste: Posterior 1/3 tongue
│   │   │   │   └─ Loss of sensation: Pharynx, tonsillar fossa
│   │   │   │
│   │   │   ├─ CN X (vagus) - MOST PROMINENT SYMPTOMS:
│   │   │   │   ├─ Hoarseness: Recurrent laryngeal nerve palsy (vocal cord paralysis)
│   │   │   │   ├─ Dysphagia: Pharyngeal muscle weakness
│   │   │   │   ├─ Aspiration: Loss of laryngeal sensation + palatal weakness
│   │   │   │   ├─ Palatal weakness: Uvula deviates AWAY from lesion
│   │   │   │   └─ Cardiovascular: Rare (bradycardia, hypotension if bilateral)
│   │   │   │
│   │   │   └─ CN XI (accessory):
│   │   │       ├─ Trapezius weakness: Shoulder droop, cannot shrug
│   │   │       └─ SCM weakness: Difficulty turning head to opposite side
│   │   │
│   │   └─ Differential diagnosis:
│   │       ├─ Collet-Sicard syndrome: CN IX, X, XI, XII (adds hypoglossal)
│   │       ├─ Villaret syndrome: CN IX, X, XI, XII + Horner's syndrome
│   │       └─ Isolated CN palsies: Less common than combined
│   │
│   ├─ Glomus Jugulare Tumor (Paraganglioma) ⭐⭐⭐:
│   │   ├─ Origin: Glomus bodies (chemoreceptor cells in jugular bulb adventitia)
│   │   ├─ Location: Jugular foramen (jugular bulb), extends to middle ear
│   │   ├─ Most common: Jugular foramen tumor
│   │   │
│   │   ├─ Classification (Fisch):
│   │   │   ├─ Type A: Tumor limited to middle ear (glomus tympanicum)
│   │   │   ├─ Type B: Tumor in middle ear + mastoid, no infralabyrinthine extension
│   │   │   ├─ Type C: Tumor involves jugular bulb, petrous bone:
│   │   │   │   ├─ C1: Small involvement (jugular bulb only)
│   │   │   │   ├─ C2: Under IAC (infralabyrinthine involvement)
│   │   │   │   ├─ C3: To petrous apex (supralabyrinthine involvement)
│   │   │   │   └─ C4: Extends to clivus, cavernous sinus (medial extension)
│   │   │   └─ Type D: Intracranial extension (posterior fossa)
│   │   │       ├─ D1: <2 cm intradural
│   │   │       └─ D2: >2 cm intradural
│   │   │
│   │   ├─ Clinical presentation:
│   │   │   ├─ Pulsatile tinnitus: Synchronous with heartbeat (70-80%)
│   │   │   ├─ Hearing loss: Conductive (middle ear involvement)
│   │   │   ├─ Cranial nerve palsies: CN VII, IX, X, XI (jugular foramen syndrome)
│   │   │   ├─ Ear findings: Red "rising sun" mass behind tympanic membrane
│   │   │   ├─ Brown's sign: Blanching of mass with pneumatic otoscopy (specific)
│   │   │   └─ Catecholamine secretion: Rare (1-4%), can cause hypertension
│   │   │
│   │   ├─ Diagnosis:
│   │   │   ├─ CT temporal bone: Bone erosion, moth-eaten appearance, jugular foramen enlargement
│   │   │   ├─ MRI with contrast: Intense enhancement, "salt and pepper" appearance (flow voids)
│   │   │   ├─ Angiography: Hypervascular tumor, feeding vessels (ascending pharyngeal artery)
│   │   │   ├─ Octreotide scan: Positive (somatostatin receptor uptake)
│   │   │   └─ 24-hour urine: Catecholamines, metanephrines (rule out secreting tumor)
│   │   │
│   │   ├─ Treatment:
│   │   │   ├─ Observation: Elderly, asymptomatic, slow-growing
│   │   │   ├─ Surgery (definitive):
│   │   │   │   ├─ Infratemporal fossa approach (Fisch type A or B)
│   │   │   │   ├─ Preoperative embolization: 24-48 hours before (reduces blood loss 40-60%)
│   │   │   │   ├─ Risk: CN deficits (30-50% new or worsening CN X palsy)
│   │   │   │   └─ Cure rate: 90-95% for Fisch C or lower
│   │   │   ├─ Radiation therapy:
│   │   │   │   ├─ Stereotactic radiosurgery (SRS): Gamma Knife, CyberKnife
│   │   │   │   ├─ Indications: Elderly, poor surgical candidate, residual tumor
│   │   │   │   ├─ Control rate: 90-95% at 5 years (growth arrest, not cure)
│   │   │   │   └─ Advantage: Lower CN deficit risk (10-15%)
│   │   │   └─ Combined: Surgery + adjuvant radiation (large tumors, incomplete resection)
│   │   │
│   │   └─ Genetics:
│   │       ├─ Familial: 10% (autosomal dominant - succinate dehydrogenase mutations)
│   │       ├─ Multiple tumors: 10% (bilateral, carotid body, etc.)
│   │       └─ Screening: Genetic testing if young (<40), family history, multiple tumors
│   │
│   ├─ Lower Cranial Nerve Schwannomas:
│   │   ├─ CN IX schwannoma (rare): Isolated glossopharyngeal symptoms
│   │   ├─ CN X schwannoma: Most common lower CN schwannoma, vagal symptoms
│   │   ├─ CN XI schwannoma (very rare): SCM/trapezius weakness
│   │   │
│   │   ├─ Imaging:
│   │   │   ├─ MRI: Enhancing jugular foramen mass, dumbbell shape
│   │   │   └─ Differentiation: Less vascular than glomus (no flow voids)
│   │   │
│   │   └─ Treatment: Surgical resection (better CN outcomes than glomus jugulare)
│   │
│   ├─ Jugular Foramen Meningioma:
│   │   ├─ Location: Posterior fossa, jugular foramen region
│   │   ├─ Imaging: Dural tail, calcification (differentiates from schwannoma)
│   │   └─ Treatment: Surgery ± radiation
│   │
│   └─ Jugular Vein Thrombosis:
│       ├─ Causes: Infection (Lemierre syndrome), hypercoagulability, trauma
│       ├─ Presentation: Neck swelling, CN palsies, increased ICP
│       └─ Treatment: Anticoagulation, treat underlying cause
│
├─ Surgical Approaches (Complex - Multiple Options)
│   ├─ Infratemporal Fossa Approach (Fisch Type A):
│   │   ├─ Position: Supine, head turned away
│   │   ├─ Incision: Postauricular C-shaped extending to neck
│   │   ├─ Steps:
│   │   │   ├─ Mastoidectomy: Complete, expose sigmoid sinus
│   │   │   ├─ Facial nerve rerouting: Anterior transposition (protects nerve)
│   │   │   ├─ Sigmoid sinus ligation: Proximal + distal (for jugular bulb access)
│   │   │   ├─ Jugular bulb exposure: Open posterior fossa floor
│   │   │   ├─ Tumor removal: Piecemeal (hypervascular - blood loss can be massive)
│   │   │   └─ Closure: Fat graft, fibrin glue
│   │   │
│   │   ├─ Advantages:
│   │   │   ├─ Excellent lateral access: Direct to jugular bulb
│   │   │   ├─ Facial nerve control: Rerouted, protected
│   │   │   └─ Can extend: To petrous apex, clivus
│   │   │
│   │   └─ Disadvantages:
│   │       ├─ Facial nerve manipulation: Temporary palsy common (60-80%)
│   │       ├─ Hearing loss: Intentional (labyrinthectomy often needed)
│   │       └─ Long operative time: 8-12 hours (complex reconstruction)
│   │
│   ├─ Retrosigmoid Approach:
│   │   ├─ Access: Posterior (standard posterior fossa approach)
│   │   ├─ Advantages:
│   │   │   ├─ Familiar: Neurosurgeons comfortable
│   │   │   ├─ Intradural tumors: Good for schwannomas
│   │   │   └─ Preserves hearing: If tumor medial
│   │   ├─ Disadvantages:
│   │   │   ├─ Limited lateral exposure: Cannot access jugular bulb well
│   │   │   └─ Cerebellar retraction: Risk of edema, ischemia
│   │   │
│   │   └─ Best for: Small intradural schwannomas (not glomus jugulare)
│   │
│   ├─ Far Lateral Approach (Extreme Lateral):
│   │   ├─ Position: Lateral park bench
│   │   ├─ Craniotomy: Extends to foramen magnum (C1 laminectomy often added)
│   │   ├─ Advantages:
│   │   │   ├─ Excellent ventral access: To lower clivus, jugular foramen
│   │   │   ├─ Minimal cerebellar retraction: Gravity assists
│   │   │   └─ Can access: CN IX, X, XI rootlets (brainstem)
│   │   │
│   │   ├─ Disadvantages:
│   │   │   ├─ Vertebral artery at risk: Courses through foramen magnum
│   │   │   └─ Occipital condyle drilling: Risk C1-C2 instability (>50% condyle → fusion)
│   │   │
│   │   └─ Best for: Medially located tumors (schwannomas with ventral extension)
│   │
│   └─ Combined Approaches:
│       ├─ Infratemporal + retrosigmoid: Large tumors with intra/extracranial extension
│       └─ Staged procedures: Reduce operative time, safer (embolization between stages)
│
├─ Complications (Jugular Foramen Surgery)
│   ├─ Cranial nerve deficits ⭐⭐⭐:
│   │   ├─ CN IX palsy: Loss of gag (manageable)
│   │   ├─ CN X palsy (MOST SIGNIFICANT):
│   │   │   ├─ Hoarseness: Vocal cord paralysis (permanent in 30-50%)
│   │   │   ├─ Aspiration: Life-threatening (especially if bilateral)
│   │   │   ├─ Dysphagia: G-tube often needed (temporary or permanent)
│   │   │   └─ Management:
│   │   │       ├─ Vocal cord medialization: Injection (temporary), thyroplasty (permanent)
│   │   │       ├─ Cricopharyngeal myotomy: If upper esophageal spasm
│   │   │       └─ G-tube: If aspiration severe
│   │   ├─ CN XI palsy: Shoulder dysfunction (permanent in 20-40%)
│   │   └─ CN VII palsy: If facial nerve manipulated (temporary 60-80%, permanent 5-10%)
│   │
│   ├─ Vascular complications:
│   │   ├─ Massive hemorrhage: Glomus jugulare is hypervascular (blood loss 1-3 liters)
│   │   ├─ Sigmoid sinus injury: Air embolism, hemorrhage
│   │   ├─ ICA injury: Anterior to jugular foramen (stroke risk)
│   │   └─ Vertebral artery injury: Far lateral approach (brainstem infarct)
│   │
│   ├─ CSF leak: 10-15% (posterior fossa dural opening, air cells)
│   ├─ Meningitis: 2-5%
│   ├─ Cerebellar edema: Brain retraction (retrosigmoid)
│   └─ C1-C2 instability: If >50% occipital condyle drilled (requires fusion)
│
└─ Special Considerations
    ├─ Jugular Bulb Variants:
    │   ├─ High-riding jugular bulb: Extends above IAC floor (10-15% population)
    │   │   └─ Risk: Injury during middle ear surgery (catastrophic bleeding)
    │   ├─ Dehiscent jugular bulb: Bone deficiency (bulb protrudes into middle ear)
    │   └─ Dominant jugular vein: Usually right (60-70%), consider if bilateral pathology
    │
    ├─ Lower CN Palsy Management:
    │   ├─ Swallow evaluation: Video fluoroscopy (essential pre-discharge)
    │   ├─ Diet modification: Thickened liquids, pureed (reduce aspiration)
    │   ├─ Speech therapy: Compensatory techniques
    │   ├─ G-tube: Low threshold (aspiration pneumonia mortality high)
    │   └─ Vocal cord medialization: Early (reduces aspiration)
    │
    └─ Preoperative Embolization (for glomus jugulare):
        ├─ Timing: 24-48 hours before surgery (optimal)
        ├─ Vessels: Ascending pharyngeal artery (primary), occipital, posterior auricular
        ├─ Benefit: Reduces intraoperative blood loss by 40-60%
        └─ Risk: CN deficits (rare <5%), stroke (particles to posterior circulation)
```

### 12. HYPOGLOSSAL CANAL (Canalis Nervi Hypoglossi)

```
├─ Location
│   ├─ Bone: Occipital bone (above occipital condyle)
│   ├─ Position: Anterolateral to foramen magnum
│   ├─ Course: Oblique (posterolateral to anteromedial)
│   └─ Length: ~8-10 mm
│
├─ Contents
│   ├─ Hypoglossal nerve (CN XII): ONLY nerve ⭐
│   │   ├─ Function: Motor to all intrinsic + extrinsic tongue muscles (except palatoglossus)
│   │   └─ Course: Hypoglossal canal → exits at skull base → descends in neck
│   │
│   └─ Venous plexus (minor):
│       ├─ Connects: Basilar plexus to internal jugular vein
│       └─ Clinical: Usually minimal bleeding during surgery
│
├─ Anatomical Relations
│   ├─ Medial: Foramen magnum, medulla (hypoglossal nucleus)
│   ├─ Lateral: Jugular foramen (very close - CN IX, X, XI)
│   ├─ Inferior: Occipital condyle
│   ├─ Superior: Posterior cranial fossa floor
│   ├─ Anterior: Clivus (lower third)
│   └─ Posterior: Foramen magnum rim
│
├─ Clinical Significance
│   ├─ Isolated CN XII Palsy:
│   │   ├─ Causes:
│   │   │   ├─ Tumor: Schwannoma (most common), meningioma, metastasis
│   │   │   ├─ Trauma: Skull base fracture (occipital condyle fracture)
│   │   │   ├─ Infection: Osteomyelitis, abscess
│   │   │   └─ Vascular: Medullary infarct (medial medullary syndrome)
│   │   │
│   │   ├─ Clinical presentation (see CN XII anatomy in CRANIAL_NERVES_III_XII.md):
│   │   │   ├─ Tongue deviation: TOWARD lesion side (ipsilateral)
│   │   │   ├─ Tongue atrophy: Ipsilateral hemitongue (chronic)
│   │   │   ├─ Fasciculations: Visible twitching (LMN lesion)
│   │   │   └─ Dysarthria: Difficulty with lingual sounds (t, d, l, n)
│   │   │
│   │   └─ Functional impact:
│   │       ├─ Mild: Speech difficulty, minor swallowing issues
│   │       └─ Usually well-tolerated: Unilateral CN XII palsy less severe than CN X
│   │
│   ├─ Collet-Sicard Syndrome:
│   │   ├─ Definition: CN IX, X, XI, XII palsy (all 4 lower CNs)
│   │   ├─ Location: Skull base pathology affecting jugular foramen + hypoglossal canal
│   │   ├─ Causes: Tumor (most common), trauma, infection
│   │   │
│   │   └─ Clinical presentation:
│   │       ├─ CN IX: Loss of gag, taste (posterior 1/3 tongue)
│   │       ├─ CN X: Hoarseness, aspiration, dysphagia
│   │       ├─ CN XI: Shoulder weakness (trapezius, SCM)
│   │       └─ CN XII: Tongue deviation, atrophy
│   │
│   ├─ Hypoglossal Nerve Schwannoma:
│   │   ├─ Rare: <1% of intracranial schwannomas
│   │   ├─ Location: Hypoglossal canal → CPA (dumbbell shape)
│   │   ├─ Presentation: Isolated CN XII palsy (progressive tongue weakness)
│   │   │
│   │   ├─ Imaging:
│   │   │   ├─ MRI: Enhancing mass in hypoglossal canal, enlarges canal
│   │   │   └─ CT: Smooth bone erosion, canal enlargement
│   │   │
│   │   └─ Treatment:
│   │       ├─ Small tumors: Far lateral approach (hearing preserved)
│   │       ├─ Resection: Usually complete (benign, encapsulated)
│   │       └─ Outcome: CN XII palsy permanent (nerve sacrificed in most cases)
│   │
│   └─ Occipital Condyle Fracture:
│       ├─ Mechanism: High-energy trauma (axial loading, lateral bending)
│       ├─ Classification (Anderson-Montesano):
│       │   ├─ Type I: Impaction fracture (stable)
│       │   ├─ Type II: Extension of basilar skull fracture (variable stability)
│       │   └─ Type III: Avulsion fracture (alar ligament - UNSTABLE)
│       │
│       ├─ CN XII injury:
│       │   ├─ Incidence: 10-15% of occipital condyle fractures
│       │   ├─ Mechanism: Fracture extends through hypoglossal canal
│       │   └─ Prognosis: Often recovers (neurapraxia) if no laceration
│       │
│       └─ Treatment:
│           ├─ Type I-II stable: Collar immobilization (6-12 weeks)
│           ├─ Type III unstable: Occipitocervical fusion (C1-C2 instability)
│           └─ CN XII palsy: Usually supportive (speech therapy)
│
├─ Surgical Approaches
│   ├─ Far Lateral (Extreme Lateral) Approach:
│   │   ├─ Position: Lateral park bench
│   │   ├─ Incision: Hockey stick (postauricular to midline)
│   │   ├─ Craniotomy: Retrosigmoid + extends to foramen magnum
│   │   ├─ C1 laminectomy: Often added (increases exposure)
│   │   │
│   │   ├─ Key steps:
│   │   │   ├─ Identify: Vertebral artery (V3 segment - critical landmark)
│   │   │   ├─ Occipital condyle drilling: Partial resection if needed (<50% to avoid instability)
│   │   │   ├─ Hypoglossal canal identification: Superior to condyle
│   │   │   ├─ CN XII identification: At canal exit (use stimulator)
│   │   │   └─ Tumor dissection: From CN XII, brainstem
│   │   │
│   │   ├─ Advantages:
│   │   │   ├─ Excellent access: To hypoglossal canal, lower clivus
│   │   │   ├─ Direct visualization: CN XII, vertebral artery
│   │   │   └─ Minimal cerebellar retraction: Gravity-assisted
│   │   │
│   │   └─ Disadvantages:
│   │       ├─ Vertebral artery at risk: Must identify and protect
│   │       ├─ Occipital condyle drilling: Risk instability if >50% removed
│   │       └─ Lower CN injury: CN IX, X, XI nearby
│   │
│   └─ Transcondylar Approach:
│       ├─ Extension: Far lateral + complete condyle removal
│       ├─ Indications: Tumors with extensive anterior/lateral extension
│       ├─ Disadvantage: C1-C2 instability (requires fusion)
│       └─ Rarely needed: Most tumors accessible without full condyle removal
│
├─ Complications
│   ├─ CN XII palsy: Common (30-50% if tumor involves nerve)
│   ├─ Lower CN palsies: CN IX, X, XI if dissection extends
│   ├─ Vertebral artery injury: Brainstem infarct (devastating)
│   ├─ C1-C2 instability: If >50% condyle removed (requires fusion)
│   └─ CSF leak: 10-15%
│
└─ Special Considerations
    └─ CN XII Palsy Rehabilitation:
        ├─ Speech therapy: Compensatory articulation techniques
        ├─ Swallowing therapy: Tongue strengthening exercises
        └─ Prognosis: Usually well-tolerated (unilateral palsy less severe than bilateral)
```

### 13. FORAMEN MAGNUM (Largest Foramen)

```
├─ Location
│   ├─ Bone: Occipital bone
│   ├─ Position: Base of posterior cranial fossa
│   ├─ Shape: Oval (anteroposterior diameter > transverse)
│   └─ Size: ~35 mm (AP) × 30 mm (transverse) - LARGEST foramen
│
├─ Boundaries
│   ├─ Anterior: Clivus (basion - anterior midpoint)
│   ├─ Posterior: Occipital squama (opisthion - posterior midpoint)
│   ├─ Lateral: Occipital condyles (paired)
│   └─ Clinical: Basion-opisthion line defines foramen magnum level
│
├─ Contents (Complex - Multiple Structures) ⭐⭐⭐
│   ├─ Neural structures:
│   │   ├─ Medulla oblongata → spinal cord transition:
│   │   │   ├─ Level: Decussation of pyramids (motor tracts cross)
│   │   │   ├─ Cervicomedullary junction: C1 level anatomically
│   │   │   └─ CRITICAL: Any compression here life-threatening (respiratory centers)
│   │   │
│   │   ├─ Spinal accessory nerve (CN XI - spinal root):
│   │   │   ├─ Origin: C1-C5 spinal cord (lateral horn)
│   │   │   ├─ Course: ASCENDS through foramen magnum → joins cranial root → exits jugular foramen
│   │   │   └─ Unique: Only CN that enters skull before exiting
│   │   │
│   │   ├─ Meninges (dura, arachnoid, pia):
│   │   │   ├─ Dura: Continuous with spinal dura
│   │   │   └─ Arachnoid: Subarachnoid space contains CSF
│   │   │
│   │   └─ Dentate ligaments: Anchor spinal cord (lateral to dorsal roots)
│   │
│   ├─ Vascular structures:
│   │   ├─ Vertebral arteries (paired) - CRITICAL ⭐⭐⭐:
│   │   │   ├─ Entry: V3 segment (extraspinal) → V4 segment (intradural)
│   │   │   ├─ Course: Anterolateral to medulla
│   │   │   ├─ Join: At pontomedullary junction → basilar artery
│   │   │   └─ Branches:
│   │   │       ├─ Posterior inferior cerebellar artery (PICA): Largest branch
│   │   │       ├─ Anterior spinal artery: Ventral medulla, anterior spinal cord
│   │   │       └─ Posterior spinal arteries: Dorsal medulla, posterior spinal cord
│   │   │
│   │   ├─ Anterior spinal artery (single, midline):
│   │   │   ├─ Formation: Union of vertebral artery branches
│   │   │   └─ Supply: Anterior 2/3 of spinal cord
│   │   │
│   │   ├─ Posterior spinal arteries (paired):
│   │   │   └─ Supply: Posterior 1/3 of spinal cord (dorsal columns)
│   │   │
│   │   └─ Venous drainage:
│   │       ├─ Vertebral venous plexus (Batson's plexus)
│   │       └─ Connection: To dural sinuses, spinal veins
│   │
│   └─ Ligaments (stabilize craniocervical junction):
│       ├─ Tectorial membrane: Continuation of posterior longitudinal ligament
│       ├─ Alar ligaments: Odontoid → occipital condyles (limit rotation)
│       ├─ Apical ligament: Odontoid tip → basion
│       └─ Cruciate ligament: Transverse + vertical bands (hold odontoid)
│
├─ Anatomical Relations
│   ├─ Anterior: Clivus, odontoid process (dens of C2)
│   ├─ Posterior: Cerebellar tonsils (very close - herniation risk)
│   ├─ Lateral: Occipital condyles, hypoglossal canal, jugular foramen
│   └─ Superior: Posterior cranial fossa (cerebellum, brainstem)
│   └─ Inferior: Upper cervical spine (C1 atlas, C2 axis)
│
├─ Clinical Significance
│   ├─ Foramen Magnum Meningioma ⭐⭐⭐:
│   │   ├─ Location: Arise from dura at foramen magnum rim
│   │   ├─ Incidence: 2-3% of all intracranial meningiomas
│   │   ├─ Classification (location):
│   │   │   ├─ Anterior (30%): Ventral to brainstem (most challenging)
│   │   │   ├─ Anterolateral (40%): Most common
│   │   │   ├─ Lateral (15%): Between VA and CN XI
│   │   │   └─ Posterior (15%): Behind brainstem (easiest)
│   │   │
│   │   ├─ Clinical presentation:
│   │   │   ├─ Suboccipital pain: Occipital headache, neck pain (common)
│   │   │   ├─ Motor symptoms:
│   │   │   │   ├─ Weakness: Arms > legs (central cord pattern if anterior tumor)
│   │   │   │   ├─ Atrophy: Hand intrinsic muscles (C8-T1)
│   │   │   │   └─ Spasticity: Lower extremities
│   │   │   ├─ Sensory: Dysesthesias, numbness (variable levels)
│   │   │   ├─ Cranial nerve: CN XI, XII (if lateral tumor), CN IX, X (if high)
│   │   │   ├─ Cerebellar: Ataxia, dysdiadochokinesia
│   │   │   ├─ Respiratory: Dyspnea on exertion (medullary compression)
│   │   │   └─ Sudden death: Rare (acute medullary compression)
│   │   │
│   │   ├─ Diagnosis:
│   │   │   ├─ MRI with contrast: Enhancing dural-based mass, dural tail
│   │   │   └─ CT: Calcification (20-30%), hyperostosis, condyle erosion
│   │   │
│   │   └─ Treatment:
│   │       ├─ Surgery (gold standard):
│   │       │   ├─ Approach: Based on tumor location
│   │       │   │   ├─ Posterior/posterolateral: Midline suboccipital
│   │       │   │   ├─ Anterolateral: Far lateral (condyle drilling)
│   │       │   │   └─ Anterior: Far lateral + transcondylar (most challenging)
│   │       │   ├─ Goal: Gross total resection (Simpson grade I-II)
│   │       │   ├─ Key steps:
│   │       │   │   ├─ C1 laminectomy: Often needed for exposure
│   │       │   │   ├─ Vertebral artery identification: CRITICAL (injury = brainstem infarct)
│   │       │   │   ├─ Internal decompression: Debulk tumor first
│   │       │   │   ├─ Capsule dissection: From brainstem, spinal cord, VA
│   │       │   │   └─ Dural resection: Include involved dura (reduce recurrence)
│   │       │   │
│   │       │   └─ Outcomes:
│   │       │       ├─ GTR: 80-90% (posterior/lateral), 50-70% (anterior)
│   │       │       ├─ Neurological improvement: 60-80%
│   │       │       └─ Recurrence: 5-10% (GTR), 20-40% (STR)
│   │       │
│   │       └─ Radiation:
│   │           ├─ Stereotactic radiosurgery: Residual tumor, recurrence
│   │           └─ Fractionated RT: Large tumors, poor surgical candidate
│   │
│   ├─ Arnold-Chiari Malformation (Chiari I) ⭐⭐:
│   │   ├─ Definition: Cerebellar tonsillar herniation through foramen magnum (>5 mm below foramen magnum)
│   │   ├─ Prevalence: 0.5-1% of population (many asymptomatic)
│   │   │
│   │   ├─ Pathophysiology:
│   │   │   ├─ Small posterior fossa: Underdeveloped occipital bone
│   │   │   ├─ Tonsillar herniation: Tonsils herniate inferiorly
│   │   │   ├─ CSF flow obstruction: At foramen magnum (craniocervical junction)
│   │   │   └─ Syringomyelia: 50-70% (CSF dissects into spinal cord)
│   │   │
│   │   ├─ Clinical presentation:
│   │   │   ├─ Headache (most common):
│   │   │   │   ├─ Location: Suboccipital (base of skull)
│   │   │   │   ├─ Trigger: Cough, Valsalva, straining (increased ICP)
│   │   │   │   └─ Duration: Seconds to minutes
│   │   │   │
│   │   │   ├─ Syringomyelia symptoms (if present):
│   │   │   │   ├─ Dissociated sensory loss: Loss of pain/temperature (preserved touch) - cape-like distribution
│   │   │   │   ├─ Motor: Weakness, atrophy (hands first - C8-T1)
│   │   │   │   └─ Autonomic: Horner's syndrome, neurogenic bladder
│   │   │   │
│   │   │   ├─ Cerebellar: Ataxia, dysmetria (if tonsillar compression severe)
│   │   │   ├─ Brainstem: Dysphagia, dysarthria, sleep apnea
│   │   │   └─ Cranial nerve: Rare (CN V, XII)
│   │   │
│   │   ├─ Diagnosis:
│   │   │   ├─ MRI brain + C-spine: Tonsillar descent >5 mm, syrinx if present
│   │   │   ├─ CSF flow study: Phase-contrast MRI (assess CSF pulsation at foramen magnum)
│   │   │   └─ Criteria: Tonsils >5 mm below McRae line (basion-opisthion)
│   │   │
│   │   └─ Treatment:
│   │       ├─ Observation: Asymptomatic or minimal symptoms
│   │       ├─ Surgery (posterior fossa decompression):
│   │       │   ├─ Indications: Symptomatic, progressive symptoms, syringomyelia
│   │       │   ├─ Technique:
│   │       │   │   ├─ Suboccipital craniectomy: Remove posterior foramen magnum rim
│   │       │   │   ├─ C1 laminectomy: Remove posterior arch of C1
│   │       │   │   ├─ Duraplasty: Open dura, expand with graft (increase space)
│   │       │   │   ├─ Arachnoid dissection: Some surgeons open arachnoid (more aggressive)
│   │       │   │   └─ Goal: Restore CSF flow at craniocervical junction
│   │       │   │
│   │       │   └─ Outcomes:
│   │       │       ├─ Headache improvement: 70-90%
│   │       │       ├─ Syrinx resolution: 60-80% (gradual over months-years)
│   │       │       ├─ Neurological stabilization: 80-90%
│   │       │       └─ Complications: CSF leak (10-15%), meningitis (2-5%), instability (rare)
│   │       │
│   │       └─ Syrinx shunting: If syrinx persists after decompression (syringosubarachnoid shunt)
│   │
│   ├─ Basilar Invagination (Platybasia):
│   │   ├─ Definition: Odontoid process projects upward into foramen magnum
│   │   ├─ Causes:
│   │   │   ├─ Congenital: Underdeveloped occipital bone, atlas assimilation
│   │   │   └─ Acquired: Rheumatoid arthritis, Paget's disease, trauma
│   │   │
│   │   ├─ Diagnosis:
│   │   │   ├─ Chamberlain line: Hard palate to opisthion (odontoid >3 mm above = invagination)
│   │   │   ├─ McGregor line: Hard palate to lowest occipital point (odontoid >4.5 mm above)
│   │   │   └─ MRI: Direct visualization of odontoid, brainstem compression
│   │   │
│   │   └─ Treatment:
│   │       ├─ Posterior decompression: If reducible (traction)
│   │       ├─ Transoral odontoidectomy: If irreducible, severe compression (remove odontoid)
│   │       └─ Occipitocervical fusion: Stabilization (after decompression)
│   │
│   └─ Foramen Magnum Stenosis:
│       ├─ Causes: Achondroplasia (most common), mucopolysaccharidoses
│       ├─ Presentation: Myelopathy, sleep apnea (central), sudden death
│       └─ Treatment: Posterior decompression (suboccipital craniectomy + C1 laminectomy)
│
├─ Surgical Approaches
│   ├─ Midline Suboccipital Approach:
│   │   ├─ Position: Prone or sitting (rare now - air embolism risk)
│   │   ├─ Incision: Midline vertical (inion to C3)
│   │   ├─ Exposure: Posterior foramen magnum, posterior arch C1
│   │   │
│   │   ├─ Steps:
│   │   │   ├─ Muscle dissection: Subperiosteal (preserve midline raphe if possible)
│   │   │   ├─ Bone removal:
│   │   │   │   ├─ Suboccipital craniectomy: 3-4 cm diameter (to transverse sinus)
│   │   │   │   └─ C1 laminectomy: Posterior arch (preserve VA groove laterally)
│   │   │   ├─ Dural opening: Y-shaped or linear
│   │   │   ├─ Arachnoid: Open to release CSF (brain relaxation)
│   │   │   └─ Exposure: Cerebellar tonsils, posterior medulla, foramen of Magendie
│   │   │
│   │   ├─ Indications:
│   │   │   ├─ Chiari I decompression
│   │   │   ├─ Posterior/posterolateral foramen magnum meningioma
│   │   │   └─ Fourth ventricle tumors (inferior approach)
│   │   │
│   │   └─ Advantages:
│   │       ├─ Familiar approach: Most neurosurgeons comfortable
│   │       ├─ Excellent midline access: Posterior and inferior structures
│   │       └─ Low morbidity: If vertebral arteries avoided
│   │
│   └─ Far Lateral Approach:
│       ├─ See Hypoglossal Canal section above
│       ├─ Indications: Anterolateral/anterior foramen magnum meningioma, vertebral artery aneurysm
│       └─ Advantage: Access to ventral structures without brain retraction
│
├─ Complications (Foramen Magnum Surgery)
│   ├─ Vertebral artery injury ⭐⭐⭐:
│   │   ├─ Risk: During C1 laminectomy (VA in groove of C1 posterior arch)
│   │   ├─ Consequence: Brainstem infarct (PICA territory), death
│   │   └─ Prevention: Identify VA landmarks, careful dissection lateral to midline
│   │
│   ├─ Lower cranial nerve injury:
│   │   ├─ Risk: Lateral dissection (CN IX, X, XI, XII)
│   │   └─ Consequence: Dysphagia, aspiration, hoarseness
│   │
│   ├─ CSF leak: 10-20% (Chiari decompression with duraplasty)
│   ├─ Meningitis: 2-5%
│   ├─ Respiratory arrest: Medullary injury (respiratory centers)
│   ├─ Cerebellar injury: Retraction injury, hematoma
│   └─ C1-C2 instability: If excessive bone removal or ligament injury
│
└─ Special Considerations
    ├─ Craniocervical Junction Instability:
    │   ├─ Assessment:
    │   │   ├─ Flexion-extension X-rays: Atlantodens interval (ADI <3 mm normal)
    │   │   ├─ MRI: Odontoid-brainstem distance, cord compression
    │   │   └─ CT: Bony anatomy, fractures
    │   │
    │   └─ Fusion indications:
    │       ├─ ADI >3-5 mm (instability)
    │       ├─ >50% occipital condyle removal
    │       └─ Rheumatoid arthritis with pannus

        ├─ Foramen magnum surgery: Risk of respiratory depression
```

### 14. STYLOMASTOID FORAMEN

```
├─ Location
│   ├─ Bone: Temporal bone (between styloid and mastoid processes)
│   ├─ Position: Inferior surface of temporal bone
│   ├─ Relationship: Posterolateral to jugular foramen
│   └─ Opening: Exits between styloid process (anterior) and mastoid process (posterior)
│
├─ Contents
│   ├─ Facial nerve (CN VII): EXIT point from skull ⭐
│   │   ├─ Course: Mastoid segment (vertical/descending) → exits stylomastoid foramen → enters parotid gland
│   │   ├─ First branch after exit: Posterior auricular nerve (to occipitalis, auricular muscles)
│   │   └─ Then: Nerve to posterior belly of digastric + stylohyoid
│   │
│   └─ Stylomastoid artery (branch of posterior auricular artery):
│       ├─ Accompanies: Facial nerve
│       └─ Supply: Facial nerve, mastoid air cells
│
├─ Anatomical Relations
│   ├─ Anterior: Styloid process, parotid gland (CN VII enters immediately after exit)
│   ├─ Posterior: Mastoid process
│   ├─ Medial: Jugular foramen (CN IX, X, XI), internal jugular vein
│   ├─ Superior: Mastoid segment of CN VII (within facial canal)
│   └─ Lateral: Sternocleidomastoid muscle origin
│
├─ Clinical Significance
│   ├─ Facial Nerve Block (Anesthesia):
│   │   ├─ Technique: Local anesthetic injection at stylomastoid foramen
│   │   ├─ Landmark: Between mastoid tip and angle of mandible
│   │   ├─ Uses: Facial procedures, trigeminal neuralgia temporary relief
│   │   └─ Complications: Hematoma, transient facial paralysis (prolonged block)
│   │
│   ├─ Facial Nerve Monitoring Electrode Placement:
│   │   ├─ Intraoperative: Stimulation at stylomastoid foramen
│   │   ├─ Uses: Parotidectomy, temporal bone surgery, posterior fossa surgery
│   │   └─ Goal: Confirm nerve continuity, guide dissection
│   │
│   ├─ Mastoiditis Extension:
│   │   ├─ Route: Infection can track from mastoid air cells → stylomastoid foramen
│   │   ├─ Result: Facial nerve palsy (via mastoid segment compression or inflammation)
│   │   └─ Treatment: IV antibiotics, mastoidectomy if abscess
│   │
│   └─ Surgical Landmark:
│       ├─ Parotidectomy: CN VII identification starts at stylomastoid foramen
│       ├─ Landmarks: Tragal pointer (superior), tympanomastoid suture (medial)
│       └─ Depth: ~2-3 cm deep to mastoid tip
│
├─ Surgical Relevance
│   ├─ Parotidectomy (Facial Nerve Identification):
│   │   ├─ Landmark approach:
│   │   │   ├─ Tragal pointer: Cartilaginous pointer (6-10 mm anterior to stylomastoid foramen)
│   │   │   ├─ Tympanomastoid suture: Groove between tympanic and mastoid bone
│   │   │   ├─ Posterior belly of digastric: Muscle landmark (nerve is just superior)
│   │   │   └─ Stylomastoid foramen: Direct identification (nerve exits here)
│   │   │
│   │   ├─ Technique:
│   │   │   ├─ Expose: Mastoid tip, posterior belly of digastric
│   │   │   ├─ Identify: Stylomastoid foramen (between styloid and mastoid)
│   │   │   ├─ Dissect: Carefully free facial nerve trunk
│   │   │   ├─ Stimulate: Confirm nerve (0.5-1.0 mA)
│   │   │   └─ Follow: Nerve into parotid (divides into 5 main branches)
│   │   │
│   │   └─ Complications:
│   │       ├─ Facial nerve injury: 10-30% temporary, 1-5% permanent
│   │       └─ Frey syndrome: Auriculotemporal nerve injury (gustatory sweating)
│   │
│   └─ Facial Nerve Decompression:
│       ├─ Indication: Temporal bone fracture with CN VII palsy
│       ├─ Approach: Transmastoid (expose facial canal → stylomastoid foramen)
│       └─ Technique: Decompress nerve from geniculate → stylomastoid foramen
│
└─ Clinical Teaching Point
    └─ "The facial nerve exits the skull at the stylomastoid foramen, immediately enters the parotid gland, and divides into 5 terminal branches (temporal, zygomatic, buccal, marginal mandibular, cervical)" ⭐
```

```
    ├─ Location: Between petrous and squamous temporal bone

    ├─ Clinical: CSF leak route (middle fossa fracture)
    └─ Usually: Fuses in adults, but can remain patent
```

## CLINICAL SYNDROMES SUMMARY

### Cavernous Sinus Syndrome
- **Contents**: CN III, IV, V1, V2, VI + ICA
- **Causes**: Thrombosis, tumor, infection, CCF
- **Features**: Ophthalmoplegia, ptosis, proptosis, chemosis, trigeminal sensory loss

### Superior Orbital Fissure Syndrome
- **Affected**: CN III, IV, V1, VI (NO ICA involvement - differentiates from cavernous)
- **Causes**: Trauma, tumor, infection
- **Features**: Ophthalmoplegia, ptosis, V1 sensory loss (NO proptosis unless also involves orbit)

### Orbital Apex Syndrome
- **Affected**: SOF contents + optic nerve (CN II)
- **Features**: Vision loss + ophthalmoplegia + V1 sensory loss

### Vernet Syndrome (Jugular Foramen Syndrome)
- **Affected**: CN IX, X, XI
- **Features**: Loss of gag, hoarseness, dysphagia, shoulder weakness

### Collet-Sicard Syndrome
- **Affected**: CN IX, X, XI, XII
- **Features**: Vernet + tongue deviation

### Villaret Syndrome
- **Affected**: CN IX, X, XI, XII + Horner's (sympathetic)
- **Features**: Collet-Sicard + ptosis, miosis, anhidrosis
