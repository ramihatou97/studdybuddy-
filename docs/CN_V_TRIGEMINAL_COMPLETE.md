# CN V - TRIGEMINAL NERVE - Complete Surgical Anatomy

**The Largest Cranial Nerve | 3 Divisions | Mixed Sensory-Motor**

---

## Overview & Unique Features

```
├─ Largest cranial nerve (thickest)
├─ Mixed nerve: Sensory (major) + Motor (minor - jaw muscles)
├─ Three major divisions:
│   ├─ V1: Ophthalmic (sensory only)
│   ├─ V2: Maxillary (sensory only)
│   └─ V3: Mandibular (sensory + motor)
│
├─ Sensory innervation: Entire face, anterior scalp, oral/nasal cavities, dura
├─ Motor innervation: Muscles of mastication
└─ Clinical importance: Trigeminal neuralgia, skull base surgery landmark
```

---

## TRIGEMINAL NERVE PROPER (Main Trunk)

### Origin & Nuclei

```
├─ Sensory Nuclei (Brainstem - extensive!)
│   ├─ Mesencephalic Nucleus (Midbrain)
│   │   ├─ Location: Lateral periaqueductal gray
│   │   ├─ Function: Proprioception from jaw muscles/teeth
│   │   └─ Unique: Only primary sensory neurons with cell bodies in CNS!
│   │
│   ├─ Principal (Main/Chief) Sensory Nucleus (Pons)
│   │   ├─ Location: Lateral pontine tegmentum
│   │   ├─ Function: Discriminative touch, vibration (face)
│   │   └─ Fibers project: Contralateral thalamus (VPM nucleus)
│   │
│   └─ Spinal Trigeminal Nucleus (Pons → Medulla → C2)
│       ├─ Subdivisions (rostral to caudal):
│       │   ├─ Pars oralis (rostral pons)
│       │   ├─ Pars interpolaris (medulla)
│       │   └─ Pars caudalis (caudal medulla to C2) - most important
│       │
│       ├─ Function: Pain and temperature (face)
│       ├─ Somatotopic organization:
│       │   ├─ V1 (ophthalmic): Ventral/caudal in nucleus
│       │   ├─ V2 (maxillary): Middle
│       │   └─ V3 (mandibular): Dorsal/rostral
│       │
│       └─ Onion-skin distribution:
│           ├─ Perioral region: Rostral nucleus
│           ├─ Mid-face: Mid nucleus
│           └─ Lateral face: Caudal nucleus (extends to C2!)
│
└─ Motor Nucleus (Pons)
    ├─ Location: Medial to main sensory nucleus
    ├─ Function: Muscles of mastication (chewing)
    ├─ Travels with: V3 (mandibular division)
    └─ Innervates: Masseter, temporalis, medial/lateral pterygoids, tensor veli palatini, tensor tympani, mylohyoid, anterior belly of digastric
```

### Segments of Main Trigeminal Nerve

```
├─ 1. Cisternal Segment (Root Entry Zone to Meckel's Cave)
│   ├─ Origin: Lateral mid-pons (root entry zone)
│   │   ├─ Two roots emerge:
│   │   │   ├─ Large sensory root (portio major) - lateral
│   │   │   └─ Small motor root (portio minor) - medial/superior
│   │   └─ Separation: ~2-5 mm between roots
│   │
│   ├─ Course: Anterolaterally across cerebellopontine angle cistern
│   ├─ Length: 10-15 mm
│   │
│   ├─ Relations (CRITICAL for CPA surgery)
│   │   ├─ Superior: Tentorium cerebelli (undersurface)
│   │   ├─ Inferior: CN VII-VIII complex (facial-vestibulocochlear)
│   │   ├─ Anterior: Petrous apex, superior petrosal sinus
│   │   ├─ Posterior: Pons (anterolateral surface)
│   │   └─ Medial: Basilar artery branches
│   │
│   ├─ Vascular Relationships
│   │   ├─ Superior cerebellar artery (SCA):
│   │   │   ├─ Most important vascular relationship!
│   │   │   ├─ SCA loop may compress trigeminal root
│   │   │   ├─ Cause: Trigeminal neuralgia (tic douloureux)
│   │   │   └─ Treatment: Microvascular decompression (MVD)
│   │   │
│   │   ├─ Anterior inferior cerebellar artery (AICA):
│   │   │   └─ May also compress (less common)
│   │   │
│   │   └─ Venous compression:
│   │       ├─ Petrosal vein complex
│   │       └─ Superior petrosal sinus tributaries
│   │
│   └─ Entry into Meckel's Cave
│       ├─ Pierces dura at petrous apex
│       ├─ Enters: Meckel's cave (trigeminal cistern)
│       └─ Cave boundaries:
│           ├─ Roof: Middle cranial fossa dura (attached to petrous temporal bone)
│           ├─ Floor: Petrous apex bone
│           └─ Contents: Trigeminal ganglion in CSF-filled dural pouch
│
├─ 2. Trigeminal (Gasserian/Semilunar) Ganglion
│   ├─ Location: Meckel's cave (petrous apex, middle cranial fossa)
│   ├─ Shape: Crescent-shaped (semilunar)
│   ├─ Size: 15-20 mm long, 8-10 mm wide
│   │
│   ├─ Contains: Sensory neuron cell bodies (pseudounipolar neurons)
│   │   ├─ Peripheral processes → V1, V2, V3 divisions
│   │   └─ Central processes → brainstem sensory nuclei
│   │
│   ├─ Divisions arise from ganglion:
│   │   ├─ V1 (Ophthalmic): Anterosuperior
│   │   ├─ V2 (Maxillary): Anteromedial
│   │   └─ V3 (Mandibular): Anteroinferior
│   │
│   ├─ Motor root:
│   │   ├─ Does NOT synapse in ganglion
│   │   ├─ Passes beneath ganglion
│   │   └─ Joins V3 (mandibular division)
│   │
│   ├─ Relations
│   │   ├─ Superior: Middle cranial fossa floor (temporal lobe)
│   │   ├─ Inferior: Petrous apex bone
│   │   ├─ Medial: Cavernous sinus (posterior aspect)
│   │   ├─ Lateral: Middle fossa dura
│   │   ├─ Anterior: Foramen ovale (V3), foramen rotundum (V2), superior orbital fissure (V1)
│   │   └─ Posterior: Porus trigeminus (opening to posterior fossa)
│   │
│   └─ Surgical Access
│       ├─ Middle fossa approach (extradural):
│       │   ├─ Elevation of temporal lobe dura
│       │   ├─ Exposure of Meckel's cave roof
│       │   └─ Access to ganglion for section (V1, V2, V3 selective)
│       │
│       ├─ Retrogasserian approach (posterior fossa):
│       │   ├─ Via CPA, identify trigeminal root
│       │   ├─ Follow to Meckel's cave (porus trigeminus)
│       │   └─ Partial sensory rhizotomy for trigeminal neuralgia
│       │
│       └─ Percutaneous procedures:
│           ├─ Foramen ovale approach (Hartel's technique):
│           │   ├─ Needle via cheek → foramen ovale → Meckel's cave
│           │   ├─ Radiofrequency lesioning (thermal rhizotomy)
│           │   ├─ Glycerol injection (chemical rhizotomy)
│           │   └─ Balloon compression
│           │
│           └─ Indications: Trigeminal neuralgia (medically refractory)
│
└─ 3. Divisions Exiting Skull Base
    ├─ V1: Superior orbital fissure
    ├─ V2: Foramen rotundum
    └─ V3: Foramen ovale
```

## V1 - OPHTHALMIC DIVISION

### Anatomy & Course

```
├─ Origin: Anterosuperior aspect of trigeminal ganglion
├─ Type: Sensory only (no motor fibers)
├─ Course: Through lateral wall of cavernous sinus → superior orbital fissure
├─ Length in cavernous sinus: ~15-20 mm
│
├─ Relations in Cavernous Sinus
│   ├─ Position: Middle of lateral wall (between CN III above, V2 below)
│   ├─ Medial: ICA (C4-C6 segments)
│   ├─ Superior: CN III
│   ├─ Inferior: CN V2
│   └─ Lateral: Middle cranial fossa dura
│
├─ Divisions (occur in cavernous sinus, before SOF)
│   ├─ Lacrimal nerve
│   ├─ Frontal nerve
│   └─ Nasociliary nerve
│
└─ Foramen: Superior orbital fissure
    ├─ Passes OUTSIDE annulus of Zinn (except nasociliary)
    └─ Enters orbit
```

### Branches & Distribution

```
├─ 1. LACRIMAL NERVE
│   ├─ Course: Lateral orbit along lacrimal gland
│   ├─ Sensory innervation:
│   │   ├─ Lacrimal gland (sensation, not secretion)
│   │   ├─ Lateral upper eyelid
│   │   └─ Lateral conjunctiva
│   │
│   └─ Communicating branch:
│       ├─ Receives: Zygomatic branch of V2 (from pterygopalatine ganglion)
│       ├─ Contains: Parasympathetic fibers (CN VII → PPG → V2 → V1 lacrimal)
│       └─ Function: Parasympathetic secretomotor to lacrimal gland (tearing)
│
├─ 2. FRONTAL NERVE (Largest V1 branch)
│   ├─ Course: Superior orbit, beneath periorbita
│   │
│   ├─ Branches:
│   │   ├─ Supratrochlear nerve (medial)
│   │   │   ├─ Exit: Superomedial orbit (trochlea level)
│   │   │   ├─ Sensory: Medial forehead, medial upper eyelid, glabella
│   │   │   └─ Surface landmark: ~1 cm lateral to midline
│   │   │
│   │   └─ Supraorbital nerve (lateral)
│   │       ├─ Exit: Supraorbital notch/foramen (mid-pupillary line)
│   │       ├─ Divisions:
│   │       │   ├─ Medial branch: Forehead to vertex
│   │       │   └─ Lateral branch: Lateral forehead, frontal sinus
│   │       ├─ Sensory: Forehead, anterior scalp (to vertex), upper eyelid
│   │       └─ Surface landmark: Palpable notch in supraorbital rim
│   │
│   └─ Clinical significance:
│       ├─ Supraorbital nerve block: Anesthesia for forehead laceration repair
│       ├─ Supraorbital neuralgia: Post-traumatic, post-herpetic
│       └─ Surgical injury: Forehead/scalp numbness (cosmetic incisions)
│
└─ 3. NASOCILIARY NERVE (Most complex V1 branch)
    ├─ Course: Enters orbit within annulus of Zinn (medial orbit)
    │
    ├─ Branches (complex, multiple):
    │   ├─ Long ciliary nerves
    │   │   ├─ Course: Alongside optic nerve to posterior globe
    │   │   ├─ Pierce sclera near optic nerve
    │   │   ├─ Sensory: Cornea, iris, ciliary body (afferent for corneal reflex)
    │   │   └─ Sympathetic: Carry sympathetics to dilator pupillae (from carotid plexus)
    │   │
    │   ├─ Communicating branch to ciliary ganglion
    │   │   ├─ Sensory root of ciliary ganglion
    │   │   ├─ Fibers pass through ganglion (no synapse)
    │   │   └─ Join short ciliary nerves → globe
    │   │
    │   ├─ Posterior ethmoidal nerve
    │   │   ├─ Course: Through posterior ethmoidal foramen
    │   │   ├─ Sensory: Posterior ethmoidal air cells, sphenoid sinus
    │   │   └─ Clinical: Sphenoid sinus pathology (pain distribution)
    │   │
    │   ├─ Anterior ethmoidal nerve (terminal branch)
    │   │   ├─ Course: Through anterior ethmoidal foramen → anterior cranial fossa → cribriform plate → nasal cavity
    │   │   ├─ Divisions:
    │   │   │   ├─ Internal nasal branches: Nasal septum, lateral nasal wall
    │   │   │   └─ External nasal branch: Emerges between nasal bone and lateral cartilage
    │   │   │
    │   │   ├─ Sensory:
    │   │   │   ├─ Anterior ethmoidal air cells
    │   │   │   ├─ Frontal sinus
    │   │   │   ├─ Anterior nasal cavity (septum, lateral wall)
    │   │   │   └─ External nose (dorsum, tip)
    │   │   │
    │   │   └─ Clinical:
    │   │       ├─ Anterior ethmoidal nerve block: Nasal surgery
    │   │       ├─ Surgical injury: Nasal tip/dorsum numbness
    │   │       └─ "Charlin's syndrome": Anterior ethmoidal neuralgia (periorbital + nasal pain)
    │   │
    │   └─ Infratrochlear nerve
    │       ├─ Course: Inferior to trochlea (superior oblique pulley)
    │       ├─ Exit: Medial orbit (below medial canthus)
    │       └─ Sensory: Medial eyelids, lacrimal sac, side of nose, medial conjunctiva
    │
    └─ Corneal Reflex Pathway (V1 afferent)
        ├─ Afferent: Cornea → long ciliary nn (V1 nasociliary) → trigeminal ganglion → spinal trigeminal nucleus
        ├─ Central: Spinal trigeminal nucleus → bilateral facial nuclei
        ├─ Efferent: CN VII → orbicularis oculi
        └─ Response: Bilateral eye blink (direct + consensual)
```

### Sensory Territory

```
├─ Skin:
│   ├─ Forehead (to vertex)
│   ├─ Upper eyelid
│   ├─ Dorsum of nose (upper half)
│   ├─ Anterior scalp (to vertex of skull)
│   └─ Medial canthus region
│
├─ Mucosa:
│   ├─ Anterior nasal cavity (septum, lateral wall)
│   ├─ Frontal sinus
│   ├─ Ethmoidal air cells
│   └─ Sphenoid sinus
│
├─ Eye:
│   ├─ Cornea
│   ├─ Conjunctiva (upper)
│   ├─ Iris
│   └─ Ciliary body
│
└─ Dura mater:
    ├─ Anterior cranial fossa (via recurrent meningeal branch)
    └─ Tentorium cerebelli (anterior portion)
```

## V2 - MAXILLARY DIVISION

### Anatomy & Course

```
├─ Origin: Anteromedial aspect of trigeminal ganglion
├─ Type: Sensory only (no motor fibers)
├─ Course: Cavernous sinus → foramen rotundum → pterygopalatine fossa → infraorbital canal
│
├─ Cavernous Sinus Segment
│   ├─ Position: Inferior lateral wall (below V1, above floor)
│   ├─ Length: ~15 mm
│   ├─ Relations:
│   │   ├─ Medial: ICA
│   │   ├─ Superior: V1
│   │   ├─ Inferior: Foramen rotundum
│   │   └─ Lateral: Middle cranial fossa dura
│   │
│   └─ Branch in CS:
│       └─ Middle meningeal nerve (recurrent branch)
│           ├─ Accompanies: Middle meningeal artery (back through foramen spinosum)
│           ├─ Sensory: Middle cranial fossa dura
│           └─ Clinical: Subdural/epidural hematoma may cause V2 pain
│
├─ Foramen Rotundum
│   ├─ Location: Greater wing of sphenoid
│   ├─ Direction: Anterolateral
│   ├─ Opens into: Pterygopalatine fossa (PPF)
│   └─ Contents: V2 only (no other structures)
│
├─ Pterygopalatine Fossa Segment (CRITICAL surgical region)
│   ├─ Location: Between maxilla and pterygoid plates
│   ├─ Boundaries:
│   │   ├─ Anterior: Posterior wall of maxillary sinus
│   │   ├─ Posterior: Pterygoid plates (sphenoid)
│   │   ├─ Medial: Perpendicular plate of palatine bone
│   │   ├─ Lateral: Pterygomaxillary fissure (opens to infratemporal fossa)
│   │   ├─ Superior: Greater wing of sphenoid, inferior orbital fissure
│   │   └─ Inferior: Palatine canal
│   │
│   ├─ Contains (in addition to V2):
│   │   ├─ Pterygopalatine ganglion (suspended from V2)
│   │   ├─ Maxillary artery (terminal branches)
│   │   ├─ Vidian nerve (nerve of pterygoid canal)
│   │   └─ Nerve of pterygoid canal connections
│   │
│   ├─ Connections (7 foramina/fissures):
│   │   ├─ Foramen rotundum (to middle cranial fossa)
│   │   ├─ Inferior orbital fissure (to orbit)
│   │   ├─ Pterygomaxillary fissure (to infratemporal fossa)
│   │   ├─ Sphenopalatine foramen (to nasal cavity)
│   │   ├─ Greater & lesser palatine canals (to palate)
│   │   ├─ Pterygoid canal (to foramen lacerum)
│   │   └─ Pharyngeal canal (to nasopharynx)
│   │
│   └─ Surgical access:
│       ├─ Transnasal endoscopic approach
│       ├─ Transantral approach (through maxillary sinus)
│       ├─ Lateral approach (through infratemporal fossa)
│       └─ Indications: V2 neurectomy, PPF tumor resection, vidian neurectomy
│
└─ Infraorbital Canal/Groove
    ├─ V2 becomes infraorbital nerve
    ├─ Course: Enters orbit via inferior orbital fissure → infraorbital groove (floor of orbit) → infraorbital canal → infraorbital foramen
    ├─ Length: 15-20 mm
    └─ Emerges: Infraorbital foramen (below orbital rim, 5-8 mm)
```

### Pterygopalatine Ganglion (Complex Autonomic Connection)

```
├─ Type: Parasympathetic ganglion (suspended from V2)
├─ Location: Pterygopalatine fossa
├─ Size: 5 mm x 3 mm
│
├─ Three Roots:
│   ├─ 1. Sensory root (from V2)
│   │   ├─ Fibers: Sensory from V2
│   │   ├─ Pass through ganglion (NO SYNAPSE)
│   │   └─ Distribute: With postganglionic parasympathetics
│   │
│   ├─ 2. Parasympathetic root (Vidian nerve, nerve of pterygoid canal)
│   │   ├─ Origin: Greater petrosal nerve (CN VII, geniculate ganglion)
│   │   ├─ Course: Greater petrosal → foramen lacerum → pterygoid canal → PPF
│   │   ├─ Synapse: Preganglionic parasympathetics synapse in PPG
│   │   ├─ Postganglionic fibers:
│   │   │   ├─ Lacrimal gland secretion (via V2 zygomatic → V1 lacrimal)
│   │   │   ├─ Nasal gland secretion
│   │   │   └─ Palatine gland secretion
│   │   │
│   │   └─ Function: Secretomotor (tearing, nasal mucus, salivation)
│   │
│   └─ 3. Sympathetic root (also in Vidian nerve)
│       ├─ Origin: Deep petrosal nerve (carotid sympathetic plexus)
│       ├─ Joins: Greater petrosal → forms Vidian nerve
│       ├─ Pass through ganglion (NO SYNAPSE - already postganglionic)
│       └─ Function: Vasoconstriction of nasal, palatine mucosa
│
├─ Branches from Ganglion:
│   ├─ Orbital branches (1-2):
│   │   └─ Sensory: Periorbita, sphenoid/ethmoidal sinuses
│   │
│   ├─ Palatine nerves:
│   │   ├─ Greater palatine nerve:
│   │   │   ├─ Course: Greater palatine canal → greater palatine foramen
│   │   │   ├─ Sensory: Hard palate, palatine gingiva
│   │   │   └─ Parasympathetic: Palatine glands
│   │   │
│   │   └─ Lesser palatine nerves (2-3):
│   │       ├─ Course: Lesser palatine canals → lesser palatine foramina
│   │       ├─ Sensory: Soft palate, uvula, palatine tonsil
│   │       └─ Parasympathetic: Palatine glands
│   │
│   ├─ Posterior superior nasal branches:
│   │   ├─ Course: Through sphenopalatine foramen → nasal cavity
│   │   ├─ Medial branches: Nasal septum
│   │   │   └─ Nasopalatine nerve (longest):
│   │   │       ├─ Descends on septum → incisive canal → anterior hard palate
│   │   │       └─ Sensory: Nasal septum, anterior hard palate
│   │   │
│   │   └─ Lateral branches: Lateral nasal wall, turbinates
│   │       ├─ Sensory: Middle/inferior turbinates, lateral nasal wall
│   │       └─ Parasympathetic: Nasal glands (mucus secretion)
│   │
│   └─ Pharyngeal nerve:
│       ├─ Course: Pharyngeal canal → nasopharynx
│       └─ Sensory: Nasopharyngeal roof, Eustachian tube opening
│
└─ Clinical Procedures:
    ├─ Vidian neurectomy:
    │   ├─ Indication: Refractory vasomotor rhinitis (excessive nasal drainage)
    │   ├─ Approach: Transnasal endoscopic → pterygoid canal → cut vidian nerve
    │   ├─ Effect: Blocks parasympathetic → decreases nasal/lacrimal secretion
    │   └─ Side effect: Dry eye (lacrimal gland denervation)
    │
    ├─ Sphenopalatine ganglion block:
    │   ├─ Indication: Cluster headache, sphenopalatine neuralgia
    │   ├─ Technique: Transnasal spray (lidocaine), or needle via greater palatine canal
    │   └─ Effect: Temporary blockade of PPG
    │
    └─ PPG ablation:
        ├─ Indication: Refractory cluster headache
        ├─ Technique: Radiofrequency, cryoablation, or open surgical resection
        └─ Risks: Numbness (V2 territory), dry eye, dry mouth
```

### V2 Branches & Distribution

```
├─ Branches in Pterygopalatine Fossa:
│   ├─ Zygomatic nerve
│   │   ├─ Course: Inferior orbital fissure → lateral orbital wall
│   │   ├─ Divisions:
│   │   │   ├─ Zygomaticotemporal: Pierces zygoma → temple skin
│   │   │   └─ Zygomaticofacial: Anterior zygoma → cheek prominence
│   │   │
│   │   ├─ Sensory: Zygomatic prominence (cheekbone), anterior temple
│   │   │
│   │   └─ Communicating branch:
│   │       ├─ To lacrimal nerve (V1)
│   │       ├─ Carries: Postganglionic parasympathetics from PPG
│   │       └─ Function: Lacrimation (tearing pathway: CN VII → PPG → V2 zygomatic → V1 lacrimal → lacrimal gland)
│   │
│   ├─ Posterior superior alveolar (PSA) nerves
│   │   ├─ Origin: V2 in PPF, before infraorbital canal
│   │   ├─ Course: Descend on posterior maxilla, enter foramina
│   │   ├─ Sensory:
│   │   │   ├─ Maxillary molars (posterior) - M1, M2, M3
│   │   │   ├─ Maxillary sinus (posterolateral wall, floor)
│   │   │   └─ Buccal gingiva (molar region)
│   │   │
│   │   └─ Clinical: PSA nerve block (dental anesthesia for molars)
│   │
│   └─ Branches to/from pterygopalatine ganglion (see above)
│
├─ Infraorbital Nerve (Continuation of V2)
│   ├─ Branches in infraorbital canal:
│   │   ├─ Middle superior alveolar (MSA) nerve
│   │   │   ├─ Present: ~60-70% of people (absent: nerves from ASA and PSA overlap)
│   │   │   ├─ Course: Descends in lateral wall of maxillary sinus
│   │   │   └─ Sensory: Maxillary premolars (PM1, PM2), mesiobuccal root of M1
│   │   │
│   │   └─ Anterior superior alveolar (ASA) nerve
│   │       ├─ Course: Anteroinferior in canal, then descends in anterior maxilla
│   │       └─ Sensory:
│   │           ├─ Maxillary incisors (central, lateral)
│   │           ├─ Maxillary canine
│   │           ├─ Anterior maxillary sinus wall
│   │           └─ Floor of nasal cavity (anterior)
│   │
│   ├─ Terminal Branches (after exiting infraorbital foramen):
│   │   ├─ Inferior palpebral branches:
│   │   │   └─ Sensory: Lower eyelid, lateral canthus
│   │   │
│   │   ├─ External nasal branches:
│   │   │   └─ Sensory: Lateral nose (ala, lower side wall)
│   │   │
│   │   └─ Superior labial branches:
│   │       └─ Sensory: Upper lip, labial mucosa, upper lip skin
│   │
│   └─ Infraorbital Foramen
│       ├─ Location: Maxilla, below orbital rim
│       ├─ Surface landmark: 5-8 mm below inferior orbital rim, in line with pupil (mid-pupillary line)
│       ├─ Palpable: Can palpate notch/foramen through skin
│       └─ Clinical: Infraorbital nerve block (midface anesthesia)
```

## V3 - MANDIBULAR DIVISION

### Anatomy & Course

```
├─ Origin: Anteroinferior aspect of trigeminal ganglion
├─ Type: MIXED - Sensory (major) + Motor (muscles of mastication)
├─ Unique: Only CN V division with motor fibers
├─ Motor root joins V3: Below trigeminal ganglion (doesn't pass through ganglion)
│
├─ Course: Trigeminal ganglion → foramen ovale → infratemporal fossa
│
├─ Foramen Ovale
│   ├─ Location: Greater wing of sphenoid
│   ├─ Opens into: Infratemporal fossa
│   ├─ Contents:
│   │   ├─ V3 (sensory + motor roots combined)
│   │   ├─ Accessory meningeal artery
│   │   └─ Lesser petrosal nerve (sometimes - CN IX parasympathetics to otic ganglion)
│   │
│   ├─ Clinical access (percutaneous):
│   │   ├─ Hartel's technique: Needle from cheek → foramen ovale
│   │   ├─ Fluoroscopic guidance: Lateral and AP views
│   │   ├─ Indications:
│   │   │   ├─ Trigeminal neuralgia (V3 distribution)
│   │   │   ├─ Radiofrequency ablation
│   │   │   ├─ Glycerol rhizotomy
│   │   │   └─ Balloon compression
│   │   │
│   │   └─ Landmarks:
│   │       ├─ Entry: 2.5 cm lateral to oral commissure
│   │       ├─ Target: 3 cm anterior to EAC, 2.5 cm below zygomatic arch
│   │       └─ Confirm: CSF aspiration (in Meckel's cave), or fluoroscopy
│   │
│   └─ Surgical anatomy:
│       ├─ Skull base tumor resection (foramen ovale involvement)
│       └─ Middle fossa approach to Meckel's cave
│
└─ Infratemporal Fossa
    ├─ Boundaries:
    │   ├─ Medial: Lateral pterygoid plate
    │   ├─ Lateral: Mandibular ramus
    │   ├─ Anterior: Posterior maxilla
    │   ├─ Posterior: TMJ, styloid process, parotid gland
    │   ├─ Superior: Greater wing of sphenoid (skull base)
    │   └─ Inferior: Medial pterygoid muscle attachment
    │
    └─ Contents:
        ├─ V3 branches (see below)
        ├─ Muscles of mastication (temporalis, masseter, pterygoids)
        ├─ Maxillary artery (mandibular and pterygoid segments)
        ├─ Pterygoid venous plexus
        └─ Otic ganglion (parasympathetic - for parotid)
```

### Division Pattern (Complex!)

```
├─ Main Trunk (Short, 3-5 mm after exiting foramen ovale)
│   │
│   ├─ Divides immediately into:
│   │   ├─ Anterior division (mostly motor, smaller)
│   │   └─ Posterior division (mostly sensory, larger)
│   │
│   └─ Branch from main trunk (before division):
│       └─ Meningeal branch (nervus spinosus)
│           ├─ Reenters skull: Through foramen spinosum
│           ├─ Accompanies: Middle meningeal artery
│           └─ Sensory: Middle cranial fossa dura, mastoid air cells
│
├─ ANTERIOR DIVISION (Mostly motor + 1 sensory branch)
│   │
│   ├─ Motor branches (to muscles of mastication):
│   │   │
│   │   ├─ Deep temporal nerves (2 branches usually)
│   │   │   ├─ Anterior deep temporal nerve
│   │   │   └─ Posterior deep temporal nerve
│   │   │   ├─ Course: Ascend deep to temporalis muscle
│   │   │   └─ Motor: Temporalis muscle (jaw elevation + retraction)
│   │   │
│   │   ├─ Masseteric nerve
│   │   │   ├─ Course: Lateral, above lateral pterygoid → mandibular notch → deep surface masseter
│   │   │   ├─ Passes: Above TMJ capsule (with masseteric artery)
│   │   │   ├─ Motor: Masseter muscle (jaw elevation)
│   │   │   └─ Sensory: TMJ capsule (articular branches)
│   │   │
│   │   ├─ Lateral pterygoid nerve
│   │   │   ├─ Course: Directly into lateral pterygoid muscle (medial surface)
│   │   │   └─ Motor: Lateral pterygoid (jaw protrusion, lateral movement)
│   │   │
│   │   └─ Medial pterygoid nerve
│   │       ├─ Course: Descends medially through otic ganglion (no synapse)
│   │       ├─ Motor: Medial pterygoid (jaw elevation)
│   │       └─ Also gives branches to:
│   │           ├─ Tensor veli palatini (soft palate tension)
│   │           └─ Tensor tympani (dampens sound in middle ear)
│   │
│   └─ Sensory branch from anterior division:
│       └─ Buccal nerve (long buccal nerve)
│           ├─ Course: Between two heads of lateral pterygoid → anteroinferiorly → emerges anterior to masseter → buccal mucosa
│           ├─ Sensory ONLY (not motor to buccinator - that's CN VII!)
│           ├─ Territory:
│           │   ├─ Buccal mucosa (cheek lining)
│           │   ├─ Buccal gingiva (mandibular molars)
│           │   └─ Skin of cheek (variable, small area)
│           │
│           └─ Clinical: Buccal nerve block for dental procedures (molars)
│
└─ POSTERIOR DIVISION (Mostly sensory + 1 motor branch)
    │
    ├─ Auriculotemporal nerve (sensory)
    │   ├─ Origin: Two roots encircle middle meningeal artery → join
    │   ├─ Course: Posterior, between neck of mandible and sphenomandibular ligament →
    │   │          ascends behind TMJ → crosses zygomatic arch with superficial temporal artery → temple
    │   │
    │   ├─ Relationships:
    │   │   ├─ Parotid gland: Passes through/posterior to superficial lobe
    │   │   ├─ TMJ: Passes immediately posterior to joint capsule
    │   │   ├─ Superficial temporal artery: Nerve anterior to artery in temple
    │   │   └─ Otic ganglion: Communicating branches (carries postganglionic parasympathetics)
    │   │
    │   ├─ Sensory branches:
    │   │   ├─ Articular branches: TMJ capsule (proprioception, pain)
    │   │   ├─ Auricular branches:
    │   │   │   ├─ Anterior ear (tragus, helix, EAC anterior wall)
    │   │   │   └─ Tympanic membrane (lateral surface, anterior portion)
    │   │   │
    │   │   ├─ Parotid branches: Parotid gland sensation
    │   │   └─ Superficial temporal branches: Temple skin, anterior scalp (to vertex)
    │   │
    │   ├─ Parasympathetic communication (hitchhiking!):
    │   │   ├─ Source: Otic ganglion (CN IX parasympathetics)
    │   │   ├─ Fibers join: Auriculotemporal nerve branches
    │   │   ├─ Destination: Parotid gland
    │   │   └─ Function: Parotid salivation
    │   │       └─ Pathway: CN IX → tympanic plexus → lesser petrosal → otic ganglion →
    │   │                   postganglionic → auriculotemporal → parotid
    │   │
    │   └─ Clinical:
    │       ├─ Frey syndrome (Auriculotemporal syndrome):
    │       │   ├─ Etiology: Parotidectomy → nerve damage/aberrant regeneration
    │       │   ├─ Parasympathetics intended for parotid → aberrantly innervate facial sweat glands
    │       │   ├─ Result: Gustatory sweating/flushing (eating triggers facial sweating)
    │       │   └─ Treatment: Botulinum toxin injection, interpositional barriers
    │       │
    │       ├─ TMJ pain: Referred via auriculotemporal nerve (ear pain with TMJ pathology)
    │       └─ Auriculotemporal nerve block: For ear/temple procedures
    │
    ├─ Lingual nerve (sensory)
    │   ├─ Course: Descends medial to lateral pterygoid → between medial pterygoid and mandible →
    │   │          mandibular 3rd molar region (lingual aspect) → floor of mouth → tongue
    │   │
    │   ├─ Relationship to chorda tympani (CN VII):
    │   │   ├─ Chorda tympani (from CN VII) joins lingual nerve in infratemporal fossa
    │   │   ├─ Junction: ~2 cm below skull base, near origin of V3 from foramen ovale
    │   │   ├─ Chorda tympani fibers "hitchhike" on lingual nerve to:
    │   │   │   ├─ Submandibular ganglion (parasympathetics to submandibular/sublingual glands)
    │   │   │   └─ Anterior 2/3 tongue (taste sensation - special sensory)
    │   │   │
    │   │   └─ Clinical importance: Lingual nerve injury affects taste + salivation!
    │   │
    │   ├─ Branch: To submandibular ganglion (suspended from lingual nerve)
    │   │   ├─ Ganglion location: Above deep part of submandibular gland
    │   │   ├─ Preganglionic: Chorda tympani (CN VII) → synapse in ganglion
    │   │   ├─ Postganglionic: To submandibular and sublingual salivary glands
    │   │   └─ Function: Salivation
    │   │
    │   ├─ Sensory territory:
    │   │   ├─ Anterior 2/3 of tongue (general sensation: touch, temp, pain)
    │   │   │   └─ NOT taste (that's chorda tympani special sensory)
    │   │   ├─ Floor of mouth
    │   │   ├─ Lingual gingiva (mandibular teeth - all)
    │   │   └─ Oral mucosa (lingual surface)
    │   │
    │   └─ Clinical:
    │       ├─ Lingual nerve injury:
    │       │   ├─ Causes:
    │       │   │   ├─ Mandibular 3rd molar (wisdom tooth) extraction (most common)
    │       │   │   ├─ Floor of mouth surgery
    │       │   │   ├─ Submandibular gland excision
    │       │   │   └─ Inferior alveolar nerve block (aberrant injection)
    │       │   │
    │       │   ├─ Results:
    │       │   │   ├─ Ipsilateral tongue numbness/paresthesia (anterior 2/3)
    │       │   │   ├─ Loss of taste (anterior 2/3) - if chorda tympani involved
    │       │   │   └─ Decreased salivation - if chorda tympani involved
    │       │   │
    │       │   └─ Recovery: Variable (50-80% recover over 6-18 months)
    │       │
    │       ├─ Lingual nerve block:
    │       │   ├─ Indication: Dental procedures (lingual gingiva anesthesia)
    │       │   ├─ Technique: Inject lingual to mandible at 3rd molar level
    │       │   └─ Caution: Nerve superficial (easily injured if aggressive injection)
    │       │
    │       └─ Anatomical proximity:
    │           ├─ Mandibular 3rd molars: Nerve passes ~2-3 mm lingual to tooth roots
    │           ├─ Surgical risk: Must protect during wisdom tooth extraction
    │           └─ Retract gently (avoid stretch injury)
    │
    ├─ Inferior alveolar nerve (sensory - largest V3 branch)
    │   ├─ Course: Descends lateral to medial pterygoid → enters mandibular foramen (lingual of ramus) →
    │   │          mandibular canal (within mandible) → exits mental foramen (becomes mental nerve) +
    │   │          continues as incisive nerve
    │   │
    │   ├─ Before entering mandibular foramen:
    │   │   └─ Mylohyoid nerve (motor branch!)
    │   │       ├─ Course: Descends in mylohyoid groove (medial mandible) → mylohyoid muscle
    │   │       ├─ Motor innervation:
    │   │       │   ├─ Mylohyoid muscle (floor of mouth elevation)
    │   │       │   └─ Anterior belly of digastric muscle (jaw opening assist)
    │   │       │
    │   │       └─ Sensory: Small sensory branches to floor of mouth (variable)
    │   │
    │   ├─ Within mandibular canal:
    │   │   ├─ Inferior dental plexus branches:
    │   │   │   ├─ Sensory to all mandibular teeth:
    │   │   │   │   ├─ Molars (M1, M2, M3)
    │   │   │   │   ├─ Premolars (PM1, PM2)
    │   │   │   │   ├─ Canine
    │   │   │   │   └─ Incisors (central, lateral)
    │   │   │   │
    │   │   │   ├─ Dental pulp, periodontal ligaments
    │   │   │   └─ Buccal gingiva (partially - molars/premolars)
    │   │   │
    │   │   └─ Accompanied by: Inferior alveolar artery and vein
    │   │
    │   ├─ Mental nerve (terminal branch - exits mental foramen)
    │   │   ├─ Mental foramen location:
    │   │   │   ├─ Between PM1 and PM2 roots (usually)
    │   │   │   ├─ Or: Below PM2 apex
    │   │   │   ├─ Surface landmark: In line with pupil (mid-pupillary line)
    │   │   │   └─ Palpable through mucosa or skin
    │   │   │
    │   │   ├─ Sensory territory:
    │   │   │   ├─ Lower lip (skin and mucosa)
    │   │   │   ├─ Chin skin
    │   │   │   └─ Labial gingiva (incisors, canines)
    │   │   │
    │   │   └─ Clinical: Mental nerve block (anesthesia for lower lip, chin)
    │   │
    │   └─ Incisive nerve (terminal branch - continues anteriorly in canal)
    │       ├─ Course: Continues anterior in mandible from mental foramen
    │       ├─ Sensory: Anterior mandibular teeth (canine, incisors), anterior mandibular bone
    │       └─ Clinical: Incisive nerve block (rarely performed - usually get with IA block)
    │
    ├─ Clinical Procedures - Inferior Alveolar Nerve Block (IANB)
    │   ├─ Most common nerve block in dentistry
    │   ├─ Indications: Anesthesia for mandibular teeth, surgery
    │   │
    │   ├─ Technique:
    │   │   ├─ Landmark: Mandibular foramen (medial surface of ramus)
    │   │   │   ├─ Location: ~1 cm above occlusal plane of molars
    │   │   │   ├─ Anteroposterior: 2/3 distance from coronoid notch to posterior border of ramus
    │   │   │   └─ Depth: 20-25 mm from lateral surface of ramus
    │   │   │
    │   │   ├─ Approach:
    │   │   │   ├─ Needle insertion: Contralateral premolar region (across midline)
    │   │   │   ├─ Target: Mandibular foramen (palpate with finger lingually)
    │   │   │   ├─ Depth: Insert ~2 cm, contact bone, withdraw 1 mm, inject
    │   │   │   └─ Volume: 1.5-2 mL local anesthetic
    │   │   │
    │   │   └─ Nerves anesthetized:
    │   │       ├─ Inferior alveolar (all mandibular teeth ipsilateral)
    │   │       ├─ Mental nerve (lower lip, chin)
    │   │       ├─ Incisive nerve (anterior teeth, bone)
    │   │       └─ Lingual nerve (often blocked simultaneously - close proximity)
    │   │
    │   ├─ Complications:
    │   │   ├─ Failed block (10-20%): Anatomical variation, aberrant injection
    │   │   ├─ Hematoma: Inferior alveolar artery puncture
    │   │   ├─ Prolonged anesthesia: Direct nerve trauma (needle, intraneural injection)
    │   │   ├─ Temporary paresthesia: Nerve irritation (usually resolves)
    │   │   ├─ Permanent paresthesia: Direct nerve injury (rare, <1%)
    │   │   └─ Lingual nerve injury: If needle too medial
    │   │
    │   └─ Anatomical variations affecting block:
    │       ├─ Bifid mandibular canal (1-2%): Two canals, may need supplemental block
    │       ├─ Accessory mandibular foramina: Additional nerve entry points
    │       └─ Mylohyoid nerve: May carry sensory to molars (explains failed blocks)
    │
    └─ Motor branch from posterior division:
        └─ Nerve to mylohyoid (see inferior alveolar nerve above)
```

### Otic Ganglion (Parasympathetic)

```
├─ Type: Parasympathetic ganglion (functionally related to CN IX, anatomically to V3)
├─ Location: Infratemporal fossa, medial to V3, just below foramen ovale
├─ Size: 2-3 mm
│
├─ Four Roots:
│   ├─ 1. Parasympathetic root (lesser petrosal nerve - CN IX)
│   │   ├─ Origin: Glossopharyngeal nerve (CN IX) → tympanic plexus → lesser petrosal nerve
│   │   ├─ Course: Lesser petrosal nerve → foramen ovale (or separate foramen) → otic ganglion
│   │   ├─ Synapse: Preganglionic parasympathetics synapse in otic ganglion
│   │   └─ Postganglionic fibers: Join auriculotemporal nerve → parotid gland
│   │
│   ├─ 2. Sympathetic root
│   │   ├─ Source: Sympathetic plexus around middle meningeal artery
│   │   ├─ Pass through ganglion (NO SYNAPSE - already postganglionic)
│   │   └─ Function: Vasoconstriction (parotid vasculature)
│   │
│   ├─ 3. Motor root (nerve to medial pterygoid - V3)
│   │   ├─ Passes through ganglion (NO SYNAPSE)
│   │   ├─ Continues to: Medial pterygoid, tensor veli palatini, tensor tympani
│   │   └─ Function: Motor to these muscles
│   │
│   └─ 4. Sensory root (from V3)
│       ├─ Sensory fibers from V3
│       ├─ Pass through ganglion (NO SYNAPSE)
│       └─ Function: General sensation
│
├─ Function: Parotid gland salivation
│   └─ Pathway: CN IX → tympanic plexus → lesser petrosal → otic ganglion →
│                 postganglionic → auriculotemporal (V3) → parotid
│
└─ Clinical: Frey syndrome (see auriculotemporal nerve above)
```

### Muscles of Mastication (V3 Motor Innervation)

```
├─ Temporalis
│   ├─ Origin: Temporal fossa, temporal fascia
│   ├─ Insertion: Coronoid process, anterior ramus of mandible
│   ├─ Innervation: Deep temporal nerves (anterior and posterior)
│   ├─ Actions:
│   │   ├─ Elevation (jaw closing) - entire muscle
│   │   ├─ Retraction (jaw backward movement) - posterior fibers
│   │   └─ Ipsilateral deviation - unilateral contraction
│   │
│   └─ Testing:
│       ├─ Palpate: Over temple during jaw clenching
│       └─ Atrophy: Depression of temple with chronic weakness
│
├─ Masseter
│   ├─ Origin: Zygomatic arch (superficial and deep heads)
│   ├─ Insertion: Lateral ramus and angle of mandible
│   ├─ Innervation: Masseteric nerve
│   ├─ Actions:
│   │   ├─ Elevation (jaw closing) - powerful elevator
│   │   └─ Minor protrusion
│   │
│   └─ Testing:
│       ├─ Palpate: Angle of mandible during clenching
│       └─ Strongest elevator (most force)
│
├─ Medial Pterygoid
│   ├─ Origin: Medial pterygoid plate (sphenoid), palatine bone
│   ├─ Insertion: Medial surface of mandible (angle and ramus)
│   ├─ Innervation: Nerve to medial pterygoid (via otic ganglion)
│   ├─ Actions:
│   │   ├─ Elevation (jaw closing)
│   │   ├─ Protrusion (bilateral)
│   │   └─ Lateral deviation (contralateral side - if right contracts, jaw moves left)
│   │
│   └─ Forms sling with masseter (masseter lateral, medial pterygoid medial)
│
├─ Lateral Pterygoid
│   ├─ Two heads:
│   │   ├─ Superior head: Greater wing of sphenoid (infratemporal surface)
│   │   └─ Inferior head: Lateral pterygoid plate
│   │
│   ├─ Insertion:
│   │   ├─ TMJ capsule, articular disc
│   │   └─ Pterygoid fovea (condylar neck of mandible)
│   │
│   ├─ Innervation: Lateral pterygoid nerve
│   │
│   ├─ Actions:
│   │   ├─ Protrusion (bilateral contraction) - jaw forward
│   │   ├─ Depression (jaw opening) - pulls condyle forward
│   │   ├─ Lateral deviation (ipsilateral side - if right contracts, jaw moves right)
│   │   └─ TMJ stabilization (superior head controls disc movement)
│   │
│   └─ Unique: Only muscle that opens jaw (depresses mandible)
│
├─ Additional V3 Motor Innervation:
│   ├─ Tensor veli palatini
│   │   ├─ Origin: Scaphoid fossa (sphenoid), Eustachian tube
│   │   ├─ Insertion: Palatine aponeurosis
│   │   ├─ Action: Tense soft palate, open Eustachian tube
│   │   └─ Innervation: Nerve to medial pterygoid
│   │
│   ├─ Tensor tympani
│   │   ├─ Location: Middle ear
│   │   ├─ Insertion: Handle of malleus
│   │   ├─ Action: Dampen loud sounds (stapedius reflex with CN VII)
│   │   └─ Innervation: Nerve to medial pterygoid
│   │
│   ├─ Mylohyoid
│   │   ├─ Origin: Mylohyoid line (mandible)
│   │   ├─ Insertion: Hyoid bone
│   │   ├─ Action: Elevate floor of mouth, assist jaw opening
│   │   └─ Innervation: Mylohyoid nerve (from inferior alveolar)
│   │
│   └─ Anterior belly of digastric
│       ├─ Origin: Digastric fossa (mandible)
│       ├─ Insertion: Intermediate tendon (hyoid)
│       ├─ Action: Depress mandible (jaw opening), elevate hyoid
│       └─ Innervation: Mylohyoid nerve (from inferior alveolar)
│           └─ Note: Posterior belly innervated by CN VII (different!)
│
└─ Clinical Testing:
    ├─ Jaw elevation:
    │   ├─ "Bite down hard" - palpate masseter, temporalis
    │   └─ Test against resistance
    │
    ├─ Jaw opening:
    │   ├─ "Open your mouth" - observe deviation
    │   ├─ Unilateral weakness: Jaw deviates to WEAK side (unopposed contralateral lateral pterygoid)
    │   └─ Test against resistance (push down on chin)
    │
    ├─ Lateral deviation:
    │   ├─ "Move jaw to right, then left"
    │   └─ Weakness: Cannot move to CONTRALATERAL side (weak lateral pterygoid)
    │
    └─ Jaw jerk reflex:
        ├─ Tap on chin with mouth slightly open
        ├─ Normal: Slight jaw elevation (masseter/temporalis contraction)
        ├─ Afferent: V3 (muscle spindles in masseter) → mesencephalic nucleus
        ├─ Efferent: V3 motor → masseter, temporalis
        └─ Hyperactive: Upper motor neuron lesion (pseudobulbar palsy)
```

### Sensory Territory Summary (V3)

```
├─ Skin:
│   ├─ Lower face (below oral commissure level)
│   ├─ Chin
│   ├─ Lower lip
│   ├─ Temple (anterior scalp)
│   ├─ Anterior ear (tragus, anterior EAC)
│   └─ Cheek (buccal region)
│
├─ Mucosa:
│   ├─ Floor of mouth
│   ├─ Anterior 2/3 tongue (general sensation - NOT taste)
│   ├─ Lower gingiva (buccal and lingual)
│   └─ Lower lip (labial mucosa)
│
├─ Teeth:
│   └─ All mandibular teeth (incisors, canines, premolars, molars)
│       ├─ Dental pulp
│       └─ Periodontal ligaments
│
├─ TMJ:
│   └─ Temporomandibular joint capsule (proprioception, pain)
│
└─ Dura:
    └─ Middle cranial fossa (meningeal branch)
```

## TRIGEMINAL NEURALGIA (Tic Douloureux)

### Clinical Syndrome

```
├─ Definition: Sudden, severe, electric shock-like facial pain in CN V distribution
│
├─ Epidemiology:
│   ├─ Incidence: 4-5 per 100,000/year
│   ├─ Age: Usually >50 years (younger age → consider MS, other causes)
│   ├─ Gender: Female > Male (3:2)
│   └─ Side: Right > Left (60% right, 40% left)
│
├─ Classification:
│   ├─ Type 1 (Classical TN):
│   │   ├─ Episodic pain: Sharp, stabbing, electric shock-like
│   │   ├─ Duration: Seconds to 2 minutes per attack
│   │   ├─ Pain-free intervals between attacks
│   │   └─ Most common type (90%)
│   │
│   └─ Type 2 (Atypical TN):
│       ├─ Constant aching/burning pain
│       ├─ Sharp pain superimposed on background pain
│       ├─ Less responsive to treatment
│       └─ Less common (10%)
│
├─ Pain Characteristics:
│   ├─ Quality: Electric shock, stabbing, lancinating
│   ├─ Intensity: Severe (10/10) - "worst pain imaginable"
│   ├─ Duration: Seconds to 2 minutes per paroxysm
│   ├─ Frequency: Multiple attacks per day (can be dozens)
│   ├─ Location: Unilateral (bilateral <5%, if bilateral → think MS)
│   └─ Distribution:
│       ├─ V2 + V3 combined: 40% (most common)
│       ├─ V2 alone: 35%
│       ├─ V3 alone: 20%
│       └─ V1 alone: 5% (rare - if V1 isolated, consider other diagnoses)
│
├─ Trigger Zones:
│   ├─ Light touch triggers severe pain
│   ├─ Common triggers:
│   │   ├─ Light facial touch (washing face, shaving)
│   │   ├─ Chewing, talking
│   │   ├─ Brushing teeth
│   │   ├─ Cold air/wind on face
│   │   └─ Smiling, grimacing
│   │
│   ├─ Refractory period:
│   │   └─ After attack, cannot immediately retrigger (seconds to minutes)
│   │
│   └─ Avoidance behavior:
│       ├─ Stop eating (fear of triggering)
│       ├─ Stop talking
│       ├─ Unwashed face/unshaved
│       └─ Weight loss (avoid chewing)
│
├─ Physical Exam:
│   ├─ Sensory exam: NORMAL (key distinguishing feature!)
│   │   └─ If sensory loss present → not classical TN, consider other causes
│   │
│   ├─ Motor exam: Normal
│   ├─ Corneal reflex: Normal
│   └─ No objective findings (diagnosis is clinical)
│
└─ Diagnostic Criteria (International Headache Society):
    ├─ Paroxysmal attacks of facial pain
    ├─ Duration: Fraction of second to 2 minutes
    ├─ Severe intensity
    ├─ Electric shock-like, shooting, stabbing quality
    ├─ Trigger zones present
    └─ No neurological deficit
```

### Etiology & Pathophysiology

```
├─ Primary (Idiopathic) Trigeminal Neuralgia (90-95%)
│   ├─ Vascular Compression (most common):
│   │   ├─ Superior cerebellar artery (SCA) loop: 75-80%
│   │   ├─ AICA loop: 10-15%
│   │   ├─ Vein (petrosal vein complex): 5-10%
│   │   └─ Basilar artery: Rare (ectatic, tortuous basilar)
│   │
│   ├─ Location of compression:
│   │   ├─ Root entry zone (REZ): Most critical location
│   │   ├─ Transition: Central (CNS myelin) → peripheral (PNS myelin)
│   │   ├─ Vulnerable zone: Within 5 mm of pons
│   │   └─ Mechanism: Pulsatile compression → demyelination → ephaptic transmission
│   │
│   ├─ Pathophysiology:
│   │   ├─ Chronic compression → focal demyelination
│   │   ├─ Demyelinated axons → abnormal cross-excitation (ephaptic transmission)
│   │   ├─ Light touch fibers (Aβ) activate pain fibers (Aδ, C)
│   │   └─ Result: Allodynia (light touch perceived as severe pain)
│   │
│   └─ Age-related changes:
│       ├─ Arterial elongation/tortuosity with aging
│       ├─ Increased vascular pulsations
│       └─ Loss of protective CSF cushion around nerve
│
└─ Secondary Trigeminal Neuralgia (5-10%)
    ├─ Multiple Sclerosis (MS):
    │   ├─ Plaque in pons (brainstem demyelination)
    │   ├─ Younger patients (<40 years)
    │   ├─ Often bilateral (20% of MS-related TN is bilateral)
    │   └─ Diagnosis: MRI shows MS plaques, CSF oligoclonal bands
    │
    ├─ Posterior Fossa Tumors:
    │   ├─ Acoustic neuroma (most common tumor cause)
    │   ├─ Meningioma (CPA or petroclival)
    │   ├─ Epidermoid cyst
    │   └─ Metastases (rare)
    │
    ├─ Vascular Malformations:
    │   ├─ AVM (arteriovenous malformation)
    │   └─ Dural arteriovenous fistula
    │
    └─ Other:
        ├─ Trigeminal schwannoma
        ├─ Chiari malformation
        └─ Post-herpetic neuralgia (after shingles - different pain quality)
```

### Diagnostic Workup

```
├─ Clinical Diagnosis (primarily):
│   ├─ History: Classic paroxysmal pain, trigger zones
│   ├─ Exam: Normal sensory/motor exam (no deficits)
│   └─ Red flags for secondary causes:
│       ├─ Age <40 years (consider MS)
│       ├─ Bilateral pain (MS, other central causes)
│       ├─ Sensory loss (tumor, structural lesion)
│       ├─ Continuous pain (atypical - broader differential)
│       └─ Associated symptoms (hearing loss, ataxia, other CN deficits)
│
└─ Imaging:
    ├─ MRI Brain with Contrast + MR Angiography (MRA):
    │   ├─ Indications: ALL patients (rule out secondary causes)
    │   ├─ Sequences:
    │   │   ├─ T1 + contrast: Tumors, enhancement
    │   │   ├─ T2/FLAIR: MS plaques, other lesions
    │   │   ├─ 3D-CISS or FIESTA: High-resolution cisternography
    │   │   │   └─ Visualize neurovascular compression (SCA loop on trigeminal nerve)
    │   │   └─ MRA (Time-of-flight): Vascular anatomy, compressions
    │   │
    │   ├─ Findings in primary TN:
    │   │   ├─ Neurovascular contact (NVC): Vessel touching nerve (seen in 80-90%)
    │   │   │   └─ Note: NVC seen in 30% of asymptomatic people (not diagnostic alone)
    │   │   ├─ Nerve distortion/displacement
    │   │   ├─ Nerve atrophy (chronic compression)
    │   │   └─ Absence of tumor, MS plaque, other lesion
    │   │
    │   └─ Findings in secondary TN:
    │       ├─ Tumor compressing nerve
    │       ├─ MS plaques (pontine, brainstem)
    │       ├─ Vascular malformation
    │       └─ Structural abnormality (Chiari, etc.)
    │
    └─ Additional Tests (if secondary cause suspected):
        ├─ LP (lumbar puncture): If MS suspected (oligoclonal bands, IgG index)
        ├─ Trigeminal reflexes (blink reflex): Abnormal in secondary causes
        └─ Somatosensory evoked potentials: Rarely used
```

## TREATMENT OF TRIGEMINAL NEURALGIA

### Medical Management (First Line)

```
├─ Carbamazepine (Tegretol) - GOLD STANDARD
│   ├─ Mechanism: Sodium channel blocker (stabilizes neuronal membranes)
│   ├─ Efficacy: 70-80% initial response
│   ├─ Dosing:
│   │   ├─ Start: 100-200 mg BID
│   │   ├─ Titrate: Increase by 200 mg every 3-5 days
│   │   ├─ Target: 600-1200 mg/day (divided doses)
│   │   └─ Max: 1600 mg/day
│   │
│   ├─ Side effects:
│   │   ├─ Dizziness, drowsiness, ataxia (dose-dependent)
│   │   ├─ Hyponatremia (SIADH)
│   │   ├─ Rash (5-10%) - Stevens-Johnson syndrome (rare but serious)
│   │   ├─ Blood dyscrasias (aplastic anemia, agranulocytosis - rare)
│   │   └─ Drug interactions (CYP3A4 inducer - many interactions)
│   │
│   ├─ Monitoring:
│   │   ├─ Baseline: CBC, LFTs, electrolytes, HLA-B*1502 (if Asian descent - SJS risk)
│   │   ├─ Follow-up: CBC, LFTs at 1 month, then every 6-12 months
│   │   └─ Sodium: Monitor if symptoms of hyponatremia
│   │
│   └─ Therapeutic trial: If effective, confirms diagnosis of TN
│
├─ Oxcarbazepine (Trileptal) - Alternative to carbamazepine
│   ├─ Keto-analogue of carbamazepine
│   ├─ Advantages: Fewer drug interactions, better tolerated
│   ├─ Efficacy: Slightly less than carbamazepine (~65-70%)
│   ├─ Dosing: 300-600 mg BID, titrate to 1200-1800 mg/day
│   └─ Side effects: Similar but milder (hyponatremia still occurs)
│
├─ Other Medications (if carbamazepine/oxcarbazepine fail or not tolerated):
│   ├─ Gabapentin (Neurontin):
│   │   ├─ Dosing: 300-3600 mg/day (divided TID)
│   │   ├─ Efficacy: Moderate (50-60%)
│   │   └─ Side effects: Sedation, dizziness, weight gain
│   │
│   ├─ Pregabalin (Lyrica):
│   │   ├─ Dosing: 150-600 mg/day (divided BID)
│   │   └─ Similar efficacy to gabapentin, better bioavailability
│   │
│   ├─ Baclofen:
│   │   ├─ Dosing: 40-80 mg/day (divided TID-QID)
│   │   ├─ Mechanism: GABA-B agonist
│   │   └─ Can combine with carbamazepine (synergistic effect)
│   │
│   ├─ Lamotrigine (Lamictal):
│   │   ├─ Dosing: 200-400 mg/day (slow titration required)
│   │   ├─ Caution: Slow titration to avoid rash (SJS risk)
│   │   └─ May be effective in MS-related TN
│   │
│   └─ Phenytoin (Dilantin):
│       ├─ Older option, less commonly used
│       └─ IV phenytoin can be used for acute TN crisis
│
├─ Combination Therapy:
│   ├─ Carbamazepine + Baclofen (common combination)
│   ├─ Gabapentin + Carbamazepine
│   └─ Used when monotherapy insufficient
│
└─ Medication Failure:
    ├─ Definition: Failed 2-3 adequate medication trials
    ├─ Tolerance: Over time, efficacy decreases (years)
    ├─ Side effects: Intolerable despite dose adjustments
    └─ Indication for surgical intervention
```

### Surgical Management

#### Percutaneous Procedures (Ablative, Destructive)

```
├─ Common Features:
│   ├─ Approach: Needle through cheek → foramen ovale → Meckel's cave
│   ├─ Technique: Hartel's approach with fluoroscopic guidance
│   ├─ Anesthesia: IV sedation ± local (need patient feedback during procedure)
│   ├─ Goal: Selective destruction of pain fibers (preserve light touch if possible)
│   └─ Targeting: V2 and/or V3 (avoid V1 - risk of corneal anesthesia)
│
├─ 1. Radiofrequency Thermal Rhizotomy (RF-TR)
│   ├─ Mechanism: Heat lesioning of trigeminal ganglion/rootlets (70-90°C)
│   │
│   ├─ Technique:
│   │   ├─ Needle placement: Foramen ovale → Meckel's cave
│   │   ├─ Stimulation: Identify division (V1/V2/V3) by paresthesias
│   │   ├─ Patient feedback: "Where do you feel tingling?" (division localization)
│   │   ├─ Lesioning: Heat applied to target division (60-90 seconds, 70°C initially)
│   │   ├─ Graded approach: Start lower temp, increase if needed
│   │   └─ Endpoint: Mild facial numbness in target distribution
│   │
│   ├─ Efficacy:
│   │   ├─ Immediate pain relief: 90-95%
│   │   ├─ 1-year pain freedom: 75-85%
│   │   ├─ 5-year pain freedom: 50-60%
│   │   └─ Recurrence: Can repeat procedure
│   │
│   ├─ Complications:
│   │   ├─ Facial numbness (intended, but extent variable): 50-70%
│   │   ├─ Dysesthesias/paresthesias: 6-10% (uncomfortable but tolerable)
│   │   ├─ Anesthesia dolorosa: 1-4% (painful numbness - difficult to treat)
│   │   ├─ Corneal hypoesthesia: 1-3% (if V1 involved)
│   │   ├─ Masseter weakness: 5-10% (if target too posterior - V3 motor)
│   │   └─ Aseptic meningitis: <1%
│   │
│   └─ Best for:
│       ├─ Older patients (poor surgical candidates)
│       ├─ Medically refractory TN
│       ├─ MS-related TN (MVD less effective)
│       └─ V2/V3 distribution (avoid V1 - corneal anesthesia risk)
│
├─ 2. Glycerol Rhizotomy (Retrogasserian Glycerol Injection)
│   ├─ Mechanism: Chemical neurolysis with glycerol
│   │
│   ├─ Technique:
│   │   ├─ Needle: Foramen ovale → Meckel's cave (trigeminal cistern)
│   │   ├─ CSF aspiration: Confirms Meckel's cave location
│   │   ├─ Positioning: Sitting or semi-sitting (glycerol sinks with gravity)
│   │   ├─ Injection: 0.2-0.4 mL sterile glycerol into trigeminal cistern
│   │   ├─ Patient position: Target distribution (V2/V3 dependent on head position)
│   │   └─ Duration: Glycerol left in place 1-2 hours (patient sits upright)
│   │
│   ├─ Efficacy:
│   │   ├─ Immediate pain relief: 85-90%
│   │   ├─ 1-year pain freedom: 70-75%
│   │   ├─ 5-year pain freedom: 40-50%
│   │   └─ Lower long-term efficacy than RF, but less sensory loss
│   │
│   ├─ Complications:
│   │   ├─ Facial numbness: 30-50% (less than RF, but variable)
│   │   ├─ Dysesthesias: 5-10%
│   │   ├─ Aseptic meningitis: 5% (chemical irritation)
│   │   ├─ Herpes labialis reactivation: Common
│   │   └─ Anesthesia dolorosa: <1% (lower than RF)
│   │
│   └─ Best for:
│       ├─ Patients who want to avoid significant numbness
│       ├─ Older patients (outpatient procedure)
│       └─ Can repeat if recurs
│
└─ 3. Balloon Compression (Percutaneous Balloon Microcompression - PBC)
    ├─ Mechanism: Mechanical compression of trigeminal ganglion
    │
    ├─ Technique:
    │   ├─ Needle + balloon catheter: Foramen ovale → Meckel's cave
    │   ├─ Balloon inflation: 0.5-1.0 mL (creates pear-shaped compression)
    │   ├─ Duration: 1-4 minutes (longer = more numbness, better efficacy)
    │   ├─ Fluoroscopy: Confirm balloon shape/position (should compress ganglion)
    │   └─ Anesthesia: General anesthesia (painful during inflation)
    │
    ├─ Efficacy:
    │   ├─ Immediate pain relief: 90%
    │   ├─ 1-year pain freedom: 75-80%
    │   ├─ 5-year pain freedom: 50-55%
    │   └─ V1 distribution: Most effective of percutaneous methods for V1
    │
    ├─ Complications:
    │   ├─ Facial numbness: 50-60% (moderate, often resolves partially)
    │   ├─ Masseter weakness: 20-25% (usually temporary, resolves 3-6 months)
    │   ├─ Dysesthesias: 5-10%
    │   ├─ Anesthesia dolorosa: <1%
    │   └─ CN III, IV, VI palsies: <1% (over-inflation, petrous apex involvement)
    │
    └─ Best for:
        ├─ V1 distribution TN (better than RF or glycerol for V1)
        ├─ MS-related TN
        └─ Recurrent TN after previous procedures
```

#### Microvascular Decompression (MVD) - Definitive Treatment

```
├─ Concept: Decompress trigeminal nerve from offending vessel (non-destructive)
│
├─ Indications:
│   ├─ Medically refractory classical TN (failed 2+ medications)
│   ├─ Younger patients (<65 years preferred, but age not absolute contraindication)
│   ├─ Good surgical candidate (able to tolerate craniotomy)
│   ├─ MRI showing neurovascular compression
│   └─ Desire to avoid facial numbness (MVD preserves sensation)
│
├─ Surgical Technique:
│   ├─ Approach: Retrosigmoid (retromastoid) craniotomy
│   │   ├─ Position: Lateral (park bench) or semi-sitting
│   │   ├─ Incision: Retroauricular, vertical or hockey-stick
│   │   ├─ Craniotomy: 3-4 cm craniectomy behind sigmoid sinus
│   │   └─ Dural opening: C-shaped, based on sigmoid sinus
│   │
│   ├─ Exposure:
│   │   ├─ CSF drainage: Open cisterna magna or cerebellopontine angle
│   │   ├─ Cerebellum retraction: Gentle superior retraction
│   │   ├─ Microscope: High magnification (10-16x)
│   │   └─ Identify trigeminal nerve: At root entry zone (REZ)
│   │
│   ├─ Decompression:
│   │   ├─ Identify offending vessel:
│   │   │   ├─ SCA loop (most common, 75-80%)
│   │   │   ├─ AICA loop (10-15%)
│   │   │   ├─ Veins (5-10% - petrosal veins)
│   │   │   └─ Inspect 360° around nerve (multiple vessels possible)
│   │   │
│   │   ├─ Mobilization:
│   │   │   ├─ Dissect vessel free from nerve (sharp arachnoid dissection)
│   │   │   ├─ Gentle manipulation (avoid vasospasm)
│   │   │   └─ Preserve perforators (to brainstem)
│   │   │
│   │   ├─ Interposition:
│   │   │   ├─ Teflon felt (shredded): Place between vessel and nerve
│   │   │   ├─ OR Muscle graft, fibrin glue-soaked Gelfoam
│   │   │   ├─ Goal: Prevent re-contact of vessel with nerve
│   │   │   └─ Secure: Teflon stays in place via adhesions
│   │   │
│   │   └─ Venous compression:
│   │       ├─ Small veins: Can coagulate and divide
│   │       ├─ Large veins (petrosal): Mobilize and interpose Teflon (avoid division - venous infarct risk)
│   │       └─ Veins alone: Less common cause, but can cause TN
│   │
│   └─ Closure:
│       ├─ Dura: Watertight closure (prevent CSF leak)
│       ├─ Bone: Replace or cranioplasty
│       └─ Postop: ICU monitoring overnight, discharge POD 1-3
│
├─ Efficacy:
│   ├─ Immediate pain relief: 85-90% (some experience delayed relief over days-weeks)
│   ├─ 1-year pain freedom: 80-85%
│   ├─ 5-year pain freedom: 70-75%
│   ├─ 10-year pain freedom: 60-65%
│   └─ Recurrence: 20-30% over 10 years
│       └─ Can repeat MVD (re-exploration, 70% success on repeat MVD)
│
├─ Complications:
│   ├─ CSF leak: 5-10% (may require repair, lumbar drain)
│   ├─ Facial numbness/paresthesias: 5-10% (usually mild, temporary)
│   ├─ Hearing loss: 3-5% (usually transient, permanent <1%)
│   ├─ Facial weakness (CN VII injury): 1-2%
│   ├─ Cerebellar hematoma/infarct: <1%
│   ├─ Meningitis/wound infection: <1%
│   ├─ Stroke (brainstem perforator injury): <0.5%
│   └─ Death: <0.2-0.5%
│
├─ Advantages:
│   ├─ Only procedure that addresses cause (vascular compression)
│   ├─ Preserves facial sensation (non-destructive)
│   ├─ Highest long-term efficacy
│   └─ Can repeat if recurs
│
├─ Disadvantages:
│   ├─ Requires craniotomy (general anesthesia, hospital stay)
│   ├─ Posterior fossa surgery risks
│   ├─ Older/frail patients may not tolerate
│   └─ Longer recovery (2-4 weeks vs days for percutaneous)
│
└─ Best for:
    ├─ Younger patients (<65 years, though not absolute)
    ├─ Classical TN with vascular compression on MRI
    ├─ Patients wanting to preserve sensation
    └─ First-time surgical intervention (best long-term results)
```
