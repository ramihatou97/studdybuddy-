# LEVEL 3: FUNCTIONAL NEUROANATOMY - ULTRA-GRANULAR SPECIFICATION

**Purpose**: Complete functional neuroanatomy for neurosurgical knowledge base
**Scope**: All major functional brain networks with clinical correlations
**Format**: Hierarchical tree structure for deep extraction and RAG retrieval
**Standards**: Neurocore architecture, medical ontology accuracy

---

## INTRODUCTION TO FUNCTIONAL NEUROANATOMY ⭐⭐⭐

├─ Definition: Organization of brain regions into functional networks for specific cognitive, sensory, motor, and behavioral tasks
│
├─ Principles of functional organization:
│   ├─ Localization: Specific brain regions specialized for specific functions
│   ├─ Distributed networks: Complex functions require multiple interconnected regions
│   ├─ Hierarchical processing: Simple → complex processing (primary → association cortex)
│   ├─ Lateralization: Left/right hemisphere specialization (language, visuospatial)
│   └─ Plasticity: Networks can reorganize after injury (especially in children)
│
├─ Methods for mapping brain function:
│   ├─ Lesion studies: Correlation of brain damage with functional deficits
│   ├─ Functional MRI (fMRI): Blood oxygenation level-dependent (BOLD) signal
│   ├─ PET imaging: Metabolic activity (glucose uptake)
│   ├─ Electrophysiology: EEG, MEG (millisecond temporal resolution)
│   ├─ Direct cortical stimulation: Awake craniotomy (gold standard for surgical mapping)
│   └─ Transcranial magnetic stimulation (TMS): Non-invasive stimulation
│
└─ Clinical relevance:
    ├─ Surgical planning: Preserve eloquent cortex (motor, language, vision)
    ├─ Stroke localization: Predict deficits from vascular territory
    ├─ Neurodegenerative disease: Understanding network-specific degeneration patterns
    └─ Psychiatric disorders: Network dysfunction in depression, schizophrenia, ADHD

## I. MOTOR SYSTEMS ⭐⭐⭐

### PRIMARY MOTOR CORTEX (M1, BA 4) ⭐⭐⭐:

├─ Location: Precentral gyrus (anterior to central sulcus)
│   └─ Brodmann area 4
│
├─ Cytoarchitecture:
│   ├─ Agranular cortex: Lacks layer IV (no granule cells - motor, not sensory)
│   ├─ Giant Betz cells: Layer V (3-5% of corticospinal neurons)
│   └─ Six cortical layers: With prominent layer V (descending motor output)
│
├─ Somatotopic organization (motor homunculus) ⭐⭐:
│   ├─ Medial surface (paracentral lobule): Lower limb, foot
│   │   └─ Representation: Toes, ankle, knee, hip
│   │
│   ├─ Convexity (lateral precentral gyrus): Upper limb, trunk
│   │   ├─ Superior: Shoulder, arm
│   │   ├─ Middle: Elbow, wrist, hand
│   │   └─ Hand area: Disproportionately large (fine motor control) ⭐
│   │
│   └─ Inferior (near Sylvian fissure): Face, tongue, larynx
│       ├─ Face: Disproportionately large
│       └─ Tongue: Large representation (complex motor control)
│
├─ Motor homunculus distortions:
│   ├─ Hand: Largest representation (fine motor dexterity)
│   ├─ Face/tongue: Large (facial expression, speech)
│   └─ Trunk: Small representation (gross movements)
│
├─ Function:
│   ├─ Voluntary movement execution: Direct control of contralateral muscles
│   ├─ Force modulation: Firing rate correlates with force
│   ├─ Movement direction: Population coding (many neurons for one movement)
│   └─ Skilled movements: Fine motor control (especially hand/fingers)
│
├─ Cortical outputs:
│   ├─ Corticospinal tract (90%): Direct pathway to spinal cord α-motor neurons ⭐
│   │   ├─ Pyramidal decussation: Crosses at medulla (contralateral control)
│   │   └─ Monosynaptic connections: Especially for hand muscles (primates)
│   │
│   ├─ Corticobulbar tract: To cranial nerve motor nuclei (face, tongue, pharynx)
│   └─ Corticoreticular: To brainstem reticular formation (posture, locomotion)
│
├─ Blood supply:
│   ├─ Leg area (medial): ACA (pericallosal, callosomarginal branches)
│   ├─ Arm area (lateral): MCA (central sulcus branches)
│   └─ Face area (inferior): MCA (inferior division)
│
└─ Clinical significance ⭐⭐:
    ├─ Stroke (MCA infarct):
    │   ├─ Contralateral hemiparesis: Face + arm > leg (ACA spared)
    │   ├─ Upper motor neuron signs: Spasticity, hyperreflexia, Babinski
    │   └─ Initial flaccid paralysis: Spinal shock, then spasticity develops
    │
    ├─ Cortical stimulation (awake craniotomy):
    │   ├─ Positive motor response: Muscle contraction (twitching)
    │   ├─ Low threshold: <5 mA (vs non-eloquent cortex >10 mA)
    │   └─ Somatotopy: Confirms precentral gyrus identity
    │
    ├─ Tumor/trauma:
    │   ├─ Focal motor seizures: Jacksonian march (progressive spread along homunculus)
    │   └─ Todd's paralysis: Post-ictal weakness (hours to days)
    │
    └─ Surgical preservation:
    │   ├─ Hand knob: Omega/epsilon-shaped knob on axial MRI (hand area landmark) ⭐
    │   └─ DTI tractography: Corticospinal tract visualization (surgical planning)


### SUPPLEMENTARY MOTOR AREA (SMA, BA 6 medial) ⭐⭐:

├─ Location: Medial frontal lobe (superior frontal gyrus, medial surface)
│   ├─ Brodmann area 6 (medial)
│   └─ Anterior to M1 leg area (paracentral lobule)
│
├─ Subdivisions:
│   ├─ SMA proper (posterior):
│   │   ├─ Location: Immediately anterior to M1 leg area
│   │   ├─ Connections: Strong to M1, basal ganglia, spinal cord
│   │   └─ Function: Movement execution (especially internally generated)
│   │
│   └─ Pre-SMA (anterior):
│       ├─ Location: Anterior to SMA proper
│       ├─ Connections: Prefrontal cortex, basal ganglia (no direct spinal connections)
│       └─ Function: Movement planning, selection, inhibition
│
├─ Somatotopy: Crude (less precise than M1)
│   └─ Leg/foot medial, hand/arm lateral (similar to M1 but overlapping)
│
├─ Function ⭐:
│   ├─ Internally generated movements: Self-initiated actions (vs externally cued)
│   ├─ Bimanual coordination: Both hands working together
│   ├─ Sequential movements: Complex motor sequences (e.g., typing, playing piano)
│   ├─ Motor planning: Preparation before movement execution
│   └─ Motor inhibition: Suppress unwanted movements
│
├─ Blood supply:
│   └─ ACA (pericallosal branches)
│
└─ Clinical significance ⭐⭐:
    ├─ SMA syndrome (SMA resection/stroke):
    │   ├─ Contralateral hemiparesis: Proximal > distal (leg/shoulder)
    │   ├─ Akinesia/mutism: Reduced spontaneous movement/speech
    │   │   ├─ Duration: Days to weeks (usually recovers)
    │   │   └─ Mechanism: Loss of movement initiation
    │   ├─ Apraxia: Difficulty with bimanual tasks
    │   └─ Recovery: Excellent (contralateral SMA compensates) ⭐
    │       ├─ Time course: 4-12 weeks typically
    │       └─ Outcome: Near-complete recovery in most cases
    │
    ├─ Cortical stimulation:
    │   ├─ Negative motor response: Movement arrest (inhibition)
    │   ├─ Urge to move: Sensation of wanting to move (without actual movement)
    │   └─ Vocalization arrest: Speech inhibition
    │
    ├─ Seizures:
    │   ├─ SMA seizures: Characteristic pattern
    │   │   ├─ Tonic posturing: "Fencing posture" (contralateral arm extended, ipsilateral flexed)
    │   │   ├─ Preserved consciousness: Often (vs M1 seizures with impaired awareness)
    │   │   └─ Vocalization: Grunting, yelling
    │   └─ Surgical resection: Can resect SMA if necessary (good functional recovery)
    │
    └─ Surgical considerations:
        ├─ Can sacrifice: If tumor infiltration (unlike M1)
        ├─ Postoperative deficit: Transient hemiparesis, resolves 4-12 weeks
        └─ Awake mapping: Negative motor responses, speech arrest


### PREMOTOR CORTEX (PM, BA 6 lateral) ⭐:

├─ Location: Lateral frontal lobe (precentral gyrus, anterior to M1)
│   └─ Brodmann area 6 (lateral)
│
├─ Subdivisions:
│   ├─ Dorsal premotor cortex (PMd):
│   │   ├─ Location: Superior lateral premotor
│   │   ├─ Function: Visually guided reaching, grasping
│   │   └─ Connections: Posterior parietal cortex (dorsal stream)
│   │
│   └─ Ventral premotor cortex (PMv):
│       ├─ Location: Inferior lateral premotor (inferior frontal gyrus pars opercularis)
│       ├─ Function: Object manipulation, grasping based on object properties
│       ├─ Mirror neurons: Fire during action execution AND observation
│       └─ Connections: Inferior parietal, superior temporal (ventral stream)
│
├─ Function:
│   ├─ Externally cued movements: Movements triggered by sensory stimuli (vs SMA - internal)
│   ├─ Sensorimotor transformation: Convert sensory information → motor commands
│   ├─ Action selection: Choose appropriate movement based on context
│   └─ Motor learning: Acquisition of new motor skills
│
├─ Mirror neuron system (PMv) ⭐:
│   ├─ Discovery: Fire when observing actions performed by others
│   ├─ Function: Action understanding, imitation, social cognition
│   └─ Clinical: Implicated in autism spectrum disorder (mirror neuron dysfunction hypothesis)
│
└─ Clinical significance:
    ├─ Apraxia: Difficulty with skilled movements despite intact motor strength
    │   └─ Ideomotor apraxia: Cannot perform movement on command (but can spontaneously)
    ├─ Stroke: Less common pure premotor stroke (MCA territory)
    └─ Surgical: Can resect if necessary (less eloquent than M1)


### BASAL GANGLIA ⭐⭐⭐:

├─ Components:
│   ├─ Striatum:
│   │   ├─ Caudate nucleus: C-shaped (head, body, tail)
│   │   └─ Putamen: Lateral to internal capsule
│   │
│   ├─ Globus pallidus (pallidum):
│   │   ├─ External segment (GPe)
│   │   └─ Internal segment (GPi)
│   │
│   ├─ Substantia nigra (midbrain):
│   │   ├─ Pars compacta (SNc): Dopaminergic neurons
│   │   └─ Pars reticulata (SNr): GABAergic output neurons
│   │
│   └─ Subthalamic nucleus (STN): Only excitatory nucleus in basal ganglia
│
├─ Circuitry (direct and indirect pathways) ⭐⭐:
│   ├─ Direct pathway (facilitates movement):
│   │   ├─ Cortex → Striatum (D1 receptors) → GPi/SNr → Thalamus (VL/VA) → Cortex
│   │   ├─ Effect: Disinhibition of thalamus → increased cortical activity → movement
│   │   └─ Dopamine: Facilitates direct pathway (D1 receptors - excitatory)
│   │
│   ├─ Indirect pathway (inhibits movement):
│   │   ├─ Cortex → Striatum (D2 receptors) → GPe → STN → GPi/SNr → Thalamus → Cortex
│   │   ├─ Effect: Increased inhibition of thalamus → decreased cortical activity → movement suppression
│   │   └─ Dopamine: Inhibits indirect pathway (D2 receptors - inhibitory)
│   │
│   └─ Hyperdirect pathway:
│       ├─ Cortex → STN → GPi/SNr → Thalamus → Cortex
│       └─ Function: Rapid movement suppression (faster than indirect pathway)
│
├─ Function:
│   ├─ Movement selection: Choose appropriate motor program, suppress unwanted movements
│   ├─ Movement scaling: Adjust movement amplitude and velocity
│   ├─ Motor learning: Habit formation, procedural memory
│   └─ Non-motor functions: Cognition, emotion (limbic loop), motivation (reward processing)
│
├─ Dopaminergic modulation (SNc → striatum):
│   ├─ Facilitates direct pathway: Via D1 receptors (movement initiation)
│   ├─ Inhibits indirect pathway: Via D2 receptors (reduce movement inhibition)
│   └─ Net effect: Dopamine promotes movement
│
└─ Clinical significance ⭐⭐⭐:
    ├─ Parkinson's disease (dopamine deficiency - SNc degeneration):
    │   ├─ Pathophysiology: Loss of dopamine → underactivity of direct pathway, overactivity of indirect pathway
    │   │
    │   ├─ Cardinal features (TRAP):
    │   │   ├─ Tremor: Resting tremor (4-6 Hz, "pill-rolling")
    │   │   ├─ Rigidity: "Lead-pipe" or "cogwheel" rigidity
    │   │   ├─ Akinesia/bradykinesia: Reduced/slowed movements
    │   │   └─ Postural instability: Falls, stooped posture
    │   │
    │   ├─ Treatment:
    │   │   ├─ Levodopa: Dopamine precursor (gold standard)
    │   │   ├─ Dopamine agonists: Pramipexole, ropinirole
    │   │   ├─ MAO-B inhibitors: Selegiline, rasagiline (reduce dopamine breakdown)
    │   │   └─ Deep brain stimulation (DBS): STN or GPi stimulation ⭐
    │   │       ├─ Mechanism: Disrupts pathological oscillations in basal ganglia circuits
    │   │       ├─ Indications: Advanced PD with motor fluctuations, dyskinesias
    │   │       ├─ Targets: STN (most common), GPi (alternative)
    │   │       └─ Outcomes: 40-60% reduction in motor symptoms, reduced medication
    │   │
    │   └─ Surgical: DBS electrode placement (stereotactic, microelectrode recording guidance)
    │
    ├─ Huntington's disease (striatal degeneration - medium spiny neurons):
    │   ├─ Pathophysiology: Striatal cell loss → preferential loss of indirect pathway → excessive direct pathway
    │   ├─ Clinical: Chorea (involuntary, dance-like movements), cognitive decline, psychiatric symptoms
    │   ├─ Genetics: CAG repeat expansion (HTT gene, chromosome 4)
    │   └─ Anticipation: Earlier onset in successive generations (paternal transmission)
    │
    ├─ Hemiballismus:
    │   ├─ Etiology: Subthalamic nucleus lesion (stroke most common)
    │   ├─ Clinical: Violent, flinging movements of contralateral limbs
    │   └─ Mechanism: Loss of STN → reduced GPi activity → disinhibition of thalamus
    │
    └─ Dystonia:
        ├─ Definition: Sustained muscle contractions → twisting, repetitive movements
        ├─ Types: Focal (blepharospasm, torticollis), generalized, task-specific (writer's cramp)
        └─ Treatment: Botulinum toxin (focal), DBS (GPi - for generalized)


### CEREBELLUM ⭐⭐⭐:

├─ Gross anatomy:
│   ├─ Location: Posterior fossa (below tentorium cerebelli)
│   ├─ Subdivisions:
│   │   ├─ Vermis: Midline (axial muscle control, gait)
│   │   ├─ Paravermis (intermediate zone): Lateral to vermis (limb control)
│   │   └─ Lateral hemispheres: Most lateral (motor planning, cognitive functions)
│   │
│   └─ Three layers:
│       ├─ Molecular layer: Outermost (parallel fibers, Purkinje dendrites)
│       ├─ Purkinje cell layer: Single cell layer (output neurons) ⭐
│       └─ Granule cell layer: Innermost (most neurons in brain)
│
├─ Functional divisions (based on phylogeny):
│   ├─ Vestibulocerebellum (archicerebellum):
│   │   ├─ Anatomy: Flocculonodular lobe, part of vermis
│   │   ├─ Input: Vestibular nuclei
│   │   ├─ Output: Vestibular nuclei
│   │   └─ Function: Balance, eye movements (vestibulo-ocular reflex)
│   │
│   ├─ Spinocerebellum (paleocerebellum):
│   │   ├─ Anatomy: Vermis, paravermis
│   │   ├─ Input: Spinal cord (spinocerebellar tracts - proprioception)
│   │   ├─ Output: Fastigial nucleus (vermis), interposed nucleus (paravermis) → brainstem, spinal cord
│   │   └─ Function: Postural control, locomotion, ongoing movement correction
│   │
│   └─ Cerebrocerebellum (neocerebellum):
│       ├─ Anatomy: Lateral hemispheres (largest in humans)
│       ├─ Input: Cerebral cortex (via pontine nuclei - corticopontocerebellar pathway)
│       ├─ Output: Dentate nucleus → thalamus (VL) → motor cortex
│       └─ Function: Motor planning, motor learning, cognitive functions (working memory, language)
│
├─ Cerebellar circuitry:
│   ├─ Input pathways:
│   │   ├─ Mossy fibers: From pontine nuclei, spinal cord → granule cells
│   │   └─ Climbing fibers: From inferior olive → Purkinje cells (error signal for motor learning) ⭐
│   │
│   ├─ Purkinje cells:
│   │   ├─ Only output neurons from cerebellar cortex
│   │   ├─ Inhibitory (GABAergic) → deep cerebellar nuclei
│   │   └─ Complex spike: Triggered by climbing fiber (learning signal)
│   │
│   └─ Deep cerebellar nuclei (output):
│       ├─ Fastigial: From vermis (balance, posture)
│       ├─ Interposed (globose + emboliform): From paravermis (limb movements)
│       └─ Dentate: From lateral hemispheres (motor planning, cognitive) ⭐
│
├─ Cerebellar peduncles (white matter connections):
│   ├─ Inferior cerebellar peduncle:
│   │   ├─ Afferent: Dorsal spinocerebellar tract, cuneocerebellar tract, vestibular fibers, inferior olive
│   │   └─ Efferent: To vestibular nuclei
│   │
│   ├─ Middle cerebellar peduncle:
│   │   └─ Afferent only: Pontine nuclei → cerebellum (largest peduncle)
│   │
│   └─ Superior cerebellar peduncle:
│       ├─ Efferent primarily: Dentate, interposed nuclei → thalamus (VL)
│       └─ Afferent: Ventral spinocerebellar tract
│
├─ Function:
│   ├─ Motor coordination: Smooth, accurate movements
│   ├─ Motor learning: Adaptation, skill acquisition (riding bike, playing instrument)
│   ├─ Error correction: Compare intended vs actual movement (forward model)
│   ├─ Timing: Temporal precision of movements
│   └─ Cognitive functions: Working memory, language, executive function (lateral hemispheres)
│
├─ Lateralization: Ipsilateral control (vs cerebral cortex - contralateral)
│   ├─ Cerebellar lesion → ipsilateral deficits
│   └─ Reason: Double crossing (cortico-ponto-cerebellar → crosses at pons, then cerebello-thalamo-cortical → crosses at midbrain)
│
└─ Clinical significance ⭐⭐⭐:
    ├─ Cerebellar syndrome (DANISH):
    │   ├─ Dysdiadochokinesia: Impaired rapid alternating movements
    │   ├─ Ataxia: Incoordination, wide-based gait
    │   ├─ Nystagmus: Horizontal (fast phase toward lesion side)
    │   ├─ Intention tremor: Tremor with goal-directed movement (finger-to-nose)
    │   ├─ Scanning speech: Irregular rhythm, separation of syllables
    │   └─ Hypotonia: Reduced muscle tone
    │
    ├─ Lesion localization:
    │   ├─ Vermis: Truncal ataxia (cannot sit/stand without support), nystagmus
    │   ├─ Paravermis: Limb ataxia (ipsilateral), intention tremor
    │   ├─ Lateral hemisphere: Motor planning deficits, cognitive impairments
    │   └─ Flocculonodular lobe: Severe vertigo, nystagmus, imbalance
    │
    ├─ Posterior fossa tumors:
    │   ├─ Children: Medulloblastoma (vermis), pilocytic astrocytoma (hemispheres)
    │   ├─ Adults: Hemangioblastoma, metastases
    │   └─ Surgical approach: Midline (vermis) - telovelar, suboccipital, lateral (hemispheres) - retrosigmoid
    │
    ├─ Cerebellar stroke:
    │   ├─ PICA territory: Lateral medulla + inferior cerebellum (Wallenberg syndrome components)
    │   ├─ AICA territory: Pons + middle cerebellar peduncle
    │   ├─ SCA territory: Superior cerebellum, superior cerebellar peduncle
    │   └─ Mass effect: Risk of tonsillar herniation, hydrocephalus (fourth ventricle compression)
    │
    └─ Cerebellar mutism syndrome:
        ├─ Etiology: Vermis surgery (medulloblastoma resection)
        ├─ Clinical: Mutism (days to weeks post-op), emotional lability, hypotonia
        ├─ Mechanism: Disruption of cerebello-cerebral pathways (dentate → thalamus → frontal cortex)
        └─ Recovery: Gradual (months), may have persistent dysarthria

## II. SENSORY SYSTEMS ⭐⭐⭐

### PRIMARY SOMATOSENSORY CORTEX (S1, BA 3, 1, 2) ⭐⭐⭐:

├─ Location: Postcentral gyrus (posterior to central sulcus)
│   ├─ Brodmann areas 3a, 3b, 1, 2 (anterior to posterior)
│   └─ Primary sensory receiving area
│
├─ Cytoarchitecture:
│   ├─ Granular cortex: Prominent layer IV (granule cells - sensory input layer) ⭐
│   └─ Six cortical layers: Layer IV receives thalamic input (VPL/VPM nuclei)
│
├─ Somatotopic organization (sensory homunculus) ⭐⭐:
│   ├─ Medial surface (paracentral lobule): Lower limb, foot, genitals
│   ├─ Convexity (lateral postcentral gyrus): Upper limb, trunk
│   │   └─ Hand: Disproportionately large (fine tactile discrimination)
│   └─ Inferior (near Sylvian fissure): Face, tongue, intraoral structures
│       └─ Lips/tongue: Large representation (fine sensory discrimination)
│
├─ Subregion specialization:
│   ├─ BA 3a: Proprioception (muscle spindles)
│   ├─ BA 3b: Cutaneous touch (primary receiving area from VPL/VPM)
│   ├─ BA 1: Texture (surface properties)
│   └─ BA 2: Size, shape (object recognition by touch - stereognosis)
│
├─ Sensory modalities:
│   ├─ Fine touch: Two-point discrimination, localization
│   ├─ Vibration: Rapidly adapting receptors (Pacinian corpuscles)
│   ├─ Proprioception: Joint position, kinesthesia
│   └─ Pressure: Slow adapting receptors (Merkel discs, Ruffini endings)
│
├─ Inputs:
│   ├─ VPL thalamus: Body sensation (dorsal column-medial lemniscal pathway)
│   └─ VPM thalamus: Face sensation (trigeminal pathway)
│
├─ Blood supply:
│   ├─ Leg area: ACA (paracentral branches)
│   ├─ Arm/trunk area: MCA (central sulcus branches)
│   └─ Face area: MCA (inferior division)
│
└─ Clinical significance ⭐⭐:
    ├─ Stroke (MCA territory):
    │   ├─ Contralateral hemianesthesia: Loss of touch, proprioception, vibration
    │   ├─ Face + arm > leg: ACA territory spared
    │   └─ Astereognosis: Cannot identify objects by touch (BA 2 lesion)
    │
    ├─ Sensory seizures:
    │   ├─ Paresthesias: Tingling, numbness marching along homunculus (Jacksonian sensory march)
    │   └─ Localization: Confirms postcentral gyrus involvement
    │
    ├─ Cortical stimulation:
    │   ├─ Paresthesias: Tingling, electrical sensation (contralateral)
    │   └─ Mapping: Identify sensory cortex during awake craniotomy
    │
    └─ Tumor/surgery:
        ├─ Gliomas: Can infiltrate S1 (map with awake craniotomy)
        └─ Preservation: Avoid dense sensory deficits (impact hand function)


### POSTERIOR PARIETAL CORTEX (BA 5, 7, 39, 40) ⭐⭐:

├─ Location: Superior and inferior parietal lobules (posterior to postcentral sulcus)
│
├─ Subdivisions:
│   ├─ Superior parietal lobule (SPL, BA 5, 7):
│   │   ├─ BA 5: Somatosensory association (tactile integration)
│   │   └─ BA 7: Visuospatial integration, reaching
│   │
│   └─ Inferior parietal lobule (IPL):
│       ├─ Supramarginal gyrus (BA 40): Phonological processing, tactile object recognition
│       └─ Angular gyrus (BA 39): Reading, semantic processing, spatial cognition
│
├─ Function (sensory integration) ⭐:
│   ├─ Visuospatial processing: Where/how pathway (dorsal visual stream termination)
│   ├─ Sensorimotor integration: Coordinate vision and proprioception for reaching
│   ├─ Attention: Spatial attention (especially right hemisphere)
│   ├─ Body schema: Representation of body in space
│   └─ Higher-order somatosensation: Stereognosis, graphesthesia
│
├─ Lateralization:
│   ├─ Left hemisphere:
│   │   ├─ Language: Phonological processing (supramarginal gyrus)
│   │   ├─ Reading: Visual word processing (angular gyrus)
│   │   └─ Calculation: Number processing, arithmetic (angular gyrus)
│   │
│   └─ Right hemisphere:
│   │   ├─ Spatial attention: Directed to both hemifields ⭐
│   │   ├─ Body schema: Awareness of body parts
│   │   └─ Visuospatial construction: Drawing, block design
│
└─ Clinical significance ⭐⭐⭐:
    ├─ Gerstmann syndrome (left angular gyrus):
    │   ├─ Acalculia: Impaired calculation
    │   ├─ Agraphia: Impaired writing
    │   ├─ Finger agnosia: Cannot identify fingers
    │   └─ Left-right disorientation
    │
    ├─ Hemispatial neglect (right inferior parietal):
    │   ├─ Definition: Fail to attend to left hemispace (despite intact vision)
    │   ├─ Clinical manifestations:
    │   │   ├─ Behavioral: Ignore left side of plate, dress only right side, shave only right side
    │   │   ├─ Visual: Bisect lines to right of center (neglect left)
    │   │   ├─ Drawing: Draw only right side of clock/flower
    │   │   └─ Reading: Read only right side of words (neglect dyslexia)
    │   ├─ Mechanism: Right parietal attends to both hemifields, left parietal only to right hemifield
    │   │   └─ Right lesion → both hemifields neglected by right, only right hemifield attended by left = left neglect
    │   └─ Treatment: Visual scanning training, prism adaptation
    │
    ├─ Balint syndrome (bilateral parietal):
    │   ├─ Etiology: Bilateral parieto-occipital lesions (watershed infarcts, posterior reversible encephalopathy syndrome)
    │   ├─ Triad:
    │   │   ├─ Simultanagnosia: Cannot perceive multiple objects simultaneously (see details but not whole scene)
    │   │   ├─ Optic ataxia: Misreach for objects under visual guidance
    │   │   └─ Ocular apraxia: Difficulty voluntarily directing gaze (saccades to targets)
    │   └─ Functional impact: Cannot navigate, cannot read (see one word at a time)
    │
    ├─ Apraxia:
    │   ├─ Ideomotor apraxia: Left inferior parietal (cannot perform learned motor acts on command)
    │   └─ Constructional apraxia: Right parietal (cannot copy drawings, construct objects)
    │
    └─ Agnosias:
        ├─ Astereognosis: Cannot identify objects by touch (posterior parietal)
        └─ Autotopagnosia: Cannot localize/name body parts (left parietal)

## III. VISUAL SYSTEMS ⭐⭐⭐

### PRIMARY VISUAL CORTEX (V1, BA 17) ⭐⭐⭐:

├─ Location: Calcarine sulcus (medial occipital lobe)
│   ├─ Brodmann area 17 (striate cortex)
│   └─ Banks of calcarine sulcus (superior and inferior)
│
├─ Cytoarchitecture:
│   ├─ Striate cortex: Stripe of Gennari (layer IV - dense myelination visible to naked eye) ⭐
│   └─ Ocular dominance columns: Alternating columns for left/right eye input
│
├─ Retinotopic organization ⭐⭐:
│   ├─ Contralateral visual field: Each V1 processes opposite visual field
│   ├─ Superior/inferior organization:
│   │   ├─ Superior visual field → inferior calcarine (lingual gyrus)
│   │   └─ Inferior visual field → superior calcarine (cuneus)
│   ├─ Macular (central) vision:
│   │   ├─ Location: Posterior occipital pole (large representation - cortical magnification) ⭐
│   │   └─ Blood supply: Dual (PCA + MCA collaterals) → macular sparing in PCA stroke
│   └─ Peripheral vision: Anterior calcarine
│
├─ Function:
│   ├─ Elementary visual processing: Orientation selectivity, spatial frequency
│   ├─ Edge detection: Simple cells (oriented edges), complex cells (moving edges)
│   └─ Binocular integration: Combine inputs from both eyes (depth perception)
│
├─ Inputs:
│   ├─ Lateral geniculate nucleus (LGN): Thalamic relay (optic radiations)
│   └─ 6 layers in LGN: Segregated magnocellular (motion) and parvocellular (form/color) pathways
│
├─ Blood supply:
│   ├─ Calcarine cortex: Posterior cerebral artery (calcarine branches)
│   └─ Occipital pole: Dual supply (PCA + MCA) → macular sparing
│
└─ Clinical significance ⭐⭐:
    ├─ Homonymous hemianopia (V1 lesion):
    │   ├─ Complete loss of contralateral visual field
    │   ├─ Congruous: Identical defect in both eyes (vs optic tract - incongruous)
    │   └─ Macular sparing: Central vision preserved (dual blood supply)
    │
    ├─ Cortical blindness (bilateral V1):
    │   ├─ Etiology: Bilateral PCA infarction (cardiac arrest, vertebrobasilar occlusion)
    │   ├─ Clinical: Complete blindness despite intact pupils (no cortical vision)
    │   └─ Anton syndrome: Denial of blindness, confabulation (anosognosia)
    │
    ├─ Visual hallucinations:
    │   ├─ Simple: Flashes, geometric patterns (V1 seizures/irritation)
    │   └─ Complex: Formed objects, people (visual association cortex)
    │
    └─ Blindsight:
        ├─ Definition: Unconscious visual processing despite cortical blindness
        ├─ Mechanism: Preserved subcortical pathways (superior colliculus, pulvinar)
        └─ Clinical: Can "guess" location of objects above chance (without awareness of seeing)


### VISUAL ASSOCIATION CORTEX (V2-V5, BA 18, 19) ⭐⭐:

├─ V2 (BA 18):
│   ├─ Location: Surrounds V1 (pericalcarine cortex)
│   ├─ Function: Intermediate processing (orientation, spatial frequency, color)
│   └─ Connections: Receives from V1, projects to ventral/dorsal streams
│
├─ V3/V3A:
│   ├─ Location: Lateral occipital cortex (dorsal to V2)
│   └─ Function: Dynamic form perception, motion processing
│
├─ V4:
│   ├─ Location: Ventral occipitotemporal cortex (fusiform gyrus region)
│   ├─ Function: Color constancy, form processing
│   └─ Clinical: V4 lesion → cerebral achromatopsia (loss of color perception)
│
└─ V5/MT (middle temporal area):
    ├─ Location: Lateral occipital cortex (ascending limb of inferior temporal sulcus)
    ├─ Function: Motion perception (direction, speed)
    └─ Clinical: V5 lesion → akinetopsia (motion blindness - world appears as static snapshots)


### VENTRAL AND DORSAL VISUAL STREAMS ⭐⭐⭐:

├─ Dual visual pathway hypothesis:
│   ├─ Ventral stream: "What" pathway (object recognition)
│   └─ Dorsal stream: "Where/How" pathway (spatial location, visuomotor control)
│
├─ Ventral stream (occipitotemporal) ⭐⭐:
│   ├─ Pathway: V1 → V2 → V4 → inferior temporal cortex
│   ├─ Function: Object recognition, face perception, reading
│   │
│   ├─ Key regions:
│   │   ├─ Fusiform face area (FFA):
│   │   │   ├─ Location: Fusiform gyrus (ventral temporal)
│   │   │   ├─ Function: Face-specific processing
│   │   │   ├─ Right hemisphere dominant
│   │   │   └─ Clinical: Lesion → prosopagnosia (face blindness)
│   │   │
│   │   ├─ Parahippocampal place area (PPA):
│   │   │   ├─ Location: Parahippocampal gyrus
│   │   │   └─ Function: Scene/landmark recognition
│   │   │
│   │   └─ Visual word form area (VWFA):
│   │       ├─ Location: Left fusiform gyrus (occipitotemporal sulcus)
│   │       ├─ Function: Visual word recognition (orthographic processing)
│   │       └─ Clinical: Lesion → pure alexia (cannot read)
│   │
│   └─ Clinical: Ventral stream lesions → visual agnosias
│       ├─ Apperceptive agnosia: Cannot perceive object form
│       └─ Associative agnosia: Can perceive form but cannot recognize meaning
│
└─ Dorsal stream (occipitoparietal) ⭐⭐:
    ├─ Pathway: V1 → V2 → V3/V5 → posterior parietal cortex
    ├─ Function: Spatial vision, visuomotor control (reaching, grasping)
    │
    ├─ Key regions:
    │   ├─ V5/MT: Motion processing
    │   ├─ Lateral intraparietal area (LIP): Saccade planning, spatial attention
    │   └─ Anterior intraparietal area (AIP): Grasping, object manipulation
    │
    └─ Clinical: Dorsal stream lesions → spatial deficits
        ├─ Optic ataxia: Misreaching under visual guidance (Balint syndrome component)
        └─ Spatial neglect: Failure to attend to contralateral space (right parietal)

## IV. LANGUAGE NETWORKS ⭐⭐⭐

### BROCA'S AREA (BA 44, 45) ⭐⭐⭐:

├─ Location: Inferior frontal gyrus (left hemisphere - dominant)
│   ├─ Pars opercularis (BA 44)
│   ├─ Pars triangularis (BA 45)
│   └─ Immediately anterior to motor face area (precentral gyrus)
│
├─ Function:
│   ├─ Speech production: Articulatory planning, motor programming of speech
│   ├─ Grammar: Syntactic processing (sentence structure)
│   ├─ Phonological processing: Sound-based language
│   └─ Working memory: Verbal working memory (phonological loop)
│
├─ Blood supply:
│   └─ MCA (inferior division, frontal branches)
│
└─ Clinical significance ⭐⭐⭐:
    ├─ Broca's aphasia (expressive, non-fluent aphasia):
    │   ├─ Etiology: Left inferior frontal lesion (stroke, tumor, trauma)
    │   │   └─ Often involves subcortical white matter (arcuate fasciculus, corticospinal tract)
    │   │
    │   ├─ Classic presentation:
    │   │   ├─ Non-fluent speech ⭐: Slow, effortful, telegraphic (omit function words)
    │   │   │   ├─ Example: "Wife... hospital... bed" (vs fluent sentences)
    │   │   │   └─ Output: <50 words/minute (normal 100-150)
    │   │   │
    │   │   ├─ Agrammatism: Omit grammatical elements (articles, prepositions, verb tenses)
    │   │   │
    │   │   ├─ Preserved comprehension: Can understand spoken/written language ⭐
    │   │   │   └─ May have difficulty with complex grammatical structures (passives, embedded clauses)
    │   │   │
    │   │   ├─ Impaired repetition: Cannot repeat sentences (arcuate involvement)
    │   │   │
    │   │   └─ Awareness of deficit: Frustrated by inability to speak (vs Wernicke's - unaware)
    │   │
    │   ├─ Associated deficits:
    │   │   ├─ Right hemiparesis: Face + arm (adjacent motor cortex)
    │   │   └─ Apraxia of speech: Articulatory difficulty (motor planning deficit)
    │   │
    │   └─ Prognosis:
    │       ├─ Variable recovery: Depends on lesion size, white matter involvement
    │       ├─ Comprehension: Remains good
    │       └─ Production: May remain effortful, agrammatic
    │
    └─ Surgical considerations:
        ├─ Awake mapping: Object naming, sentence completion tasks
        ├─ Speech arrest: Direct stimulation of Broca's area
        └─ Tumor infiltration: Low-grade gliomas can infiltrate without obvious deficit (plasticity)


### WERNICKE'S AREA (BA 22 posterior) ⭐⭐⭐:

├─ Location: Posterior superior temporal gyrus (left hemisphere - dominant)
│   ├─ Brodmann area 22 (posterior portion)
│   └─ Adjacent to primary auditory cortex (Heschl's gyrus)
│
├─ Function:
│   ├─ Language comprehension: Auditory language understanding
│   ├─ Semantic processing: Word meaning
│   ├─ Phonological processing: Sound-based language recognition
│   └─ Lexical access: Retrieve words from semantic memory
│
├─ Blood supply:
│   └─ MCA (inferior division, temporal branches)
│
└─ Clinical significance ⭐⭐⭐:
    ├─ Wernicke's aphasia (receptive, fluent aphasia):
    │   ├─ Etiology: Left posterior superior temporal lesion (stroke most common)
    │   │
    │   ├─ Classic presentation:
    │   │   ├─ Fluent speech ⭐: Normal rate, rhythm, prosody (sounds fluent)
    │   │   │   └─ Output: 100-150 words/minute (normal rate)
    │   │   │
    │   │   ├─ Paraphasias (word errors):
    │   │   │   ├─ Semantic paraphasias: Related word substitution ("apple" for "orange")
    │   │   │   ├─ Phonemic paraphasias: Sound substitution ("bapple" for "apple")
    │   │   │   └─ Neologisms: Nonsense words ("flibber" - invented words)
    │   │   │
    │   │   ├─ Impaired comprehension ⭐: Cannot understand spoken or written language
    │   │   │   └─ Deficit: Severe (cannot follow simple commands)
    │   │   │
    │   │   ├─ Impaired repetition: Cannot repeat (comprehension deficit)
    │   │   │
    │   │   └─ Lack of awareness: Unaware of language deficit (anosognosia) ⭐
    │   │       └─ May produce "word salad" (jargon aphasia) without realizing it's meaningless
    │   │
    │   ├─ Associated deficits:
    │   │   ├─ No hemiparesis: Posterior temporal lobe (away from motor cortex)
    │   │   └─ Visual field defect: Possible superior quadrantanopia (optic radiations nearby)
    │   │
    │   └─ Prognosis:
    │       ├─ Poor for comprehension: Severe deficits often persistent
    │       └─ Functional impact: Greater than Broca's (cannot understand)
    │
    └─ Surgical:
        └─ Rare surgical lesions: Posterior temporal region (less common tumor location)

## V. MEMORY SYSTEMS ⭐⭐⭐

### HIPPOCAMPUS AND MEDIAL TEMPORAL LOBE ⭐⭐⭐:

├─ Anatomy:
│   ├─ Hippocampus proper (Cornu Ammonis - CA1, CA2, CA3, CA4)
│   ├─ Dentate gyrus: Input structure (receives from entorhinal cortex)
│   ├─ Subiculum: Output structure (projects to various cortical/subcortical regions)
│   └─ Entorhinal cortex: Gateway to hippocampus (layer II → dentate, layer III → CA1/subiculum)
│
├─ Circuitry (trisynaptic loop):
│   ├─ Entorhinal cortex → dentate gyrus (perforant pathway)
│   ├─ Dentate gyrus → CA3 (mossy fibers)
│   ├─ CA3 → CA1 (Schaffer collaterals)
│   └─ CA1/subiculum → back to entorhinal cortex (output)
│
├─ Function ⭐⭐:
│   ├─ Episodic memory: Encoding and retrieval of autobiographical events
│   ├─ Spatial memory: Cognitive map (place cells - Nobel Prize 2014)
│   ├─ Memory consolidation: Transfer from short-term → long-term memory
│   └─ Pattern separation (dentate gyrus): Distinguish similar experiences
│
├─ Place cells and spatial cognition:
│   ├─ Place cells: Fire when animal in specific location (Nobel Prize 2014 - O'Keefe, Moser)
│   ├─ Grid cells: Hexagonal firing pattern (entorhinal cortex)
│   └─ Head direction cells: Encode heading direction (like compass)
│
├─ Lateralization:
│   ├─ Right hippocampus: Spatial memory, navigation
│   └─ Left hippocampus: Verbal memory, episodic details
│
└─ Clinical significance ⭐⭐⭐:
    ├─ Anterograde amnesia (cannot form new memories):
    │   ├─ Bilateral hippocampal damage: Severe, permanent
    │   │   └─ Case H.M.: Bilateral medial temporal lobectomy (1953) - profound anterograde amnesia, taught us role of hippocampus
    │   ├─ Preserved: Procedural memory (motor skills), working memory, remote memories
    │   └─ Mechanism: Cannot consolidate new episodic memories
    │
    ├─ Transient global amnesia (TGA):
    │   ├─ Clinical: Sudden-onset anterograde + retrograde amnesia (hours duration)
    │   ├─ Preserved: Personal identity, procedural memory
    │   ├─ Repetitive questioning: "What happened?" "Where am I?"
    │   ├─ Imaging: DWI hyperintensity in CA1 (hippocampus)
    │   └─ Prognosis: Complete recovery within 24 hours
    │
    ├─ Alzheimer's disease:
    │   ├─ Early: Hippocampal atrophy → memory impairment (earliest symptom)
    │   ├─ Pathology: Neurofibrillary tangles, amyloid plaques (hippocampus affected early)
    │   └─ Progression: Spreads from entorhinal cortex → hippocampus → neocortex
    │
    ├─ Temporal lobe epilepsy:
    │   ├─ Hippocampal sclerosis: Most common cause (neuronal loss, gliosis in CA1/CA3)
    │   ├─ Mesial temporal seizures: Aura (déjà vu, rising epigastric sensation), staring, automatisms
    │   ├─ Surgery: Anterior temporal lobectomy with amygdalohippocampectomy
    │   └─ Memory outcome:
    │       ├─ Dominant hemisphere resection: Verbal memory decline (30-50%)
    │       ├─ Non-dominant resection: Visual/spatial memory decline (less severe)
    │       └─ Bilateral damage: Severe anterograde amnesia (avoid bilateral surgery)
    │
    └─ Surgical considerations ⭐:
        ├─ Awake mapping: Memory testing during stimulation (verbal vs visual material)
        ├─ Intracarotid amobarbital test (Wada): Assess language/memory lateralization pre-surgery
        └─ Preservation: Avoid bilateral hippocampal damage (devastating amnesia)


### PREFRONTAL CORTEX - WORKING MEMORY ⭐⭐:

├─ Working memory: Temporary storage and manipulation of information (seconds to minutes)
│
├─ Dorsolateral prefrontal cortex (DLPFC, BA 9, 46):
│   ├─ Location: Middle frontal gyrus (lateral prefrontal)
│   ├─ Function: Working memory maintenance, manipulation, executive control
│   ├─ Domain-specific: Spatial vs object working memory (right vs left)
│   └─ Clinical: DLPFC lesion → impaired working memory, distractibility
│
└─ Ventrolateral prefrontal cortex (VLPFC, BA 44, 45, 47):
    ├─ Location: Inferior frontal gyrus
    ├─ Function: Active maintenance, selection, comparison
    └─ Overlaps: Broca's area (verbal working memory component)

## VI. EXECUTIVE FUNCTION ⭐⭐⭐

### PREFRONTAL CORTEX (BA 9-12, 44-47) ⭐⭐⭐:

├─ Overview: Highest level of cognitive control, planning, decision-making
│
├─ Subdivisions (functional):
│   ├─ Dorsolateral prefrontal cortex (DLPFC, BA 9, 46) ⭐⭐:
│   │   ├─ Location: Middle frontal gyrus (lateral surface)
│   │   │
│   │   ├─ Function:
│   │   │   ├─ Working memory: Maintain and manipulate information
│   │   │   ├─ Planning: Goal-directed behavior, strategy formation
│   │   │   ├─ Cognitive flexibility: Task switching, rule learning
│   │   │   ├─ Abstract reasoning: Problem-solving, concept formation
│   │   │   └─ Inhibitory control: Suppress inappropriate responses
│   │   │
│   │   └─ Clinical (DLPFC lesion):
│   │       ├─ Executive dysfunction: Impaired planning, problem-solving
│   │       ├─ Perseveration: Difficulty switching tasks (Wisconsin Card Sorting Test failure)
│   │       ├─ Working memory deficits: Cannot hold information online
│   │       └─ Apathy: Reduced initiative, motivation
│   │
│   ├─ Ventromedial prefrontal cortex (VMPFC, BA 10, 11, 12) ⭐⭐:
│   │   ├─ Location: Medial frontal lobe, inferior to genu of corpus callosum
│   │   │
│   │   ├─ Function:
│   │   │   ├─ Emotion regulation: Integration of emotional and cognitive information
│   │   │   ├─ Decision-making: Value-based choices, risk/reward assessment
│   │   │   ├─ Social cognition: Theory of mind, empathy
│   │   │   └─ Self-referential processing: Autobiographical memory, self-reflection
│   │   │
│   │   └─ Clinical (VMPFC lesion) ⭐:
│   │       ├─ Acquired sociopathy: Impaired social judgment, disinhibition
│   │       │   └─ Case Phineas Gage: Railroad spike through VMPFC → personality change, poor decision-making
│   │       ├─ Poor decision-making: Risky choices (Iowa Gambling Task failure)
│   │       ├─ Emotional dysregulation: Flat affect or emotional lability
│   │       └─ Preserved: Intellect, memory (IQ normal)
│   │
│   └─ Orbitofrontal cortex (OFC, BA 11, 12, 47) ⭐⭐:
│       ├─ Location: Inferior frontal lobe (above orbits)
│       │
│       ├─ Function:
│       │   ├─ Reward processing: Value coding, expected reward
│       │   ├─ Inhibitory control: Response inhibition (especially emotional responses)
│       │   ├─ Decision-making: Reversal learning (adapt when contingencies change)
│       │   └─ Olfactory processing: Primary olfactory cortex (piriform) projects to OFC
│       │
│       └─ Clinical (OFC lesion):
│           ├─ Disinhibition: Impulsive behavior, socially inappropriate
│           ├─ Perseveration: Continue unrewarded behavior (reversal learning deficit)
│           ├─ Olfactory deficits: Anosmia (OFC lesion/frontal base fracture)
│           └─ Personality change: Irritability, lability, poor judgment
│
├─ Anterior cingulate cortex (ACC, BA 24, 32) ⭐⭐:
│   ├─ Subdivisions:
│   │   ├─ Dorsal ACC (dACC): Cognitive control (conflict monitoring, error detection)
│   │   └─ Ventral ACC (vACC/subgenual): Emotion regulation, autonomic control
│   │
│   ├─ Function (dACC):
│   │   ├─ Conflict monitoring: Detect response conflict (Stroop task)
│   │   ├─ Error detection: Monitor performance errors
│   │   ├─ Motivation: Effort-based decision-making
│   │   └─ Pain processing: Affective component of pain (unpleasantness)
│   │
│   └─ Clinical:
│       ├─ ACC lesion: Abulia, apathy, akinetic mutism (bilateral)
│       └─ Cingulotomy: Historically for OCD, chronic pain (disrupts affective pain processing)
│
└─ Dysexecutive syndrome (frontal lobe syndrome):
    ├─ Definition: Constellation of executive dysfunction from frontal lobe damage
    ├─ Clinical features:
    │   ├─ Cognitive: Poor planning, impaired judgment, perseveration
    │   ├─ Behavioral: Disinhibition, impulsivity, apathy
    │   ├─ Personality: Irritability, emotional lability, lack of insight
    │   └─ Preserved: Basic motor, sensory, language, memory (initially)
    ├─ Etiology: Frontal lobe tumor, trauma, stroke, frontotemporal dementia
    └─ Assessment: Wisconsin Card Sorting Test, Trail Making Test, Stroop Test

## VII. ATTENTION NETWORKS ⭐⭐⭐

### DORSAL ATTENTION NETWORK (DAN) ⭐:

├─ Components:
│   ├─ Frontal eye fields (FEF, BA 8): Posterior middle frontal gyrus
│   └─ Intraparietal sulcus (IPS): Posterior parietal cortex
│
├─ Function:
│   ├─ Top-down attention: Goal-directed, voluntary attention
│   ├─ Spatial attention: Orienting to locations in space
│   └─ Eye movements: Saccade planning and execution
│
└─ Clinical: DAN lesion → impaired voluntary attention, difficulty directing gaze


### VENTRAL ATTENTION NETWORK (VAN) ⭐:

├─ Components:
│   ├─ Temporoparietal junction (TPJ): Right hemisphere dominant
│   └─ Ventral frontal cortex (VFC): Right inferior frontal gyrus
│
├─ Function:
│   ├─ Bottom-up attention: Stimulus-driven, automatic attention
│   ├─ Reorienting: Shift attention to unexpected salient stimuli
│   └─ Circuit breaker: Interrupt ongoing activity when salient stimulus detected
│
└─ Clinical: VAN lesion (especially right) → hemispatial neglect (cannot reorient to left-sided stimuli)


### DEFAULT MODE NETWORK (DMN) ⭐⭐⭐:

├─ Components:
│   ├─ Medial prefrontal cortex (MPFC): Ventromedial prefrontal, medial frontal
│   ├─ Posterior cingulate cortex (PCC): Retrosplenial cortex, precuneus
│   ├─ Inferior parietal lobule: Angular gyrus
│   └─ Medial temporal lobe: Hippocampus, parahippocampal cortex
│
├─ Function:
│   ├─ Task-negative network: Active during rest, deactivates during goal-directed tasks ⭐
│   ├─ Self-referential thinking: Introspection, autobiographical memory
│   ├─ Mind-wandering: Spontaneous thought, daydreaming
│   ├─ Theory of mind: Inferring mental states of others
│   └─ Future planning: Episodic simulation, mental time travel
│
├─ Discovery: Raichle et al. (2001) - consistent deactivation during cognitive tasks
│
└─ Clinical significance ⭐:
    ├─ Alzheimer's disease: Early DMN disruption (PCC hypometabolism)
    ├─ Depression: Hyperactivity (rumination, negative self-referential thought)
    ├─ ADHD: Altered DMN activity (inability to suppress during tasks)
    └─ Schizophrenia: Abnormal DMN connectivity (self-referential processing deficits)

## VIII. EMOTIONAL AND LIMBIC SYSTEMS ⭐⭐⭐

### AMYGDALA ⭐⭐⭐:

├─ Anatomy:
│   ├─ Location: Medial temporal lobe (anterior to hippocampus, below uncus)
│   ├─ Nuclei: ~13 nuclei grouped into:
│   │   ├─ Basolateral complex: Receives sensory inputs, connects to cortex
│   │   └─ Central/medial nuclei: Output to hypothalamus, brainstem (autonomic/behavioral responses)
│   │
│   └─ Size: Almond-shaped (~1 cm³)
│
├─ Function:
│   ├─ Fear processing: Fear conditioning, threat detection ⭐
│   ├─ Emotional learning: Associative learning (classical conditioning)
│   ├─ Emotional memory: Enhanced memory for emotional events
│   ├─ Social cognition: Facial emotion recognition (especially fear)
│   └─ Autonomic responses: Heart rate, blood pressure, cortisol release
│
├─ Inputs:
│   ├─ Sensory cortex: Visual, auditory, olfactory (rapid threat detection)
│   ├─ Hippocampus: Contextual information
│   └─ Prefrontal cortex: Top-down regulation
│
├─ Outputs:
│   ├─ Hypothalamus: Autonomic/endocrine responses (stress response)
│   ├─ Brainstem: Behavioral responses (freezing, fight-or-flight)
│   └─ Prefrontal cortex: Modulate attention, decision-making
│
└─ Clinical significance ⭐⭐:
    ├─ Klüver-Bucy syndrome (bilateral amygdala lesion):
    │   ├─ Etiology: Bilateral temporal lobectomy, herpes encephalitis
    │   ├─ Clinical features:
    │   │   ├─ Hyperorality: Put everything in mouth (visual agnosia component)
    │   │   ├─ Hypersexuality: Inappropriate sexual behavior
    │   │   ├─ Placidity: Loss of fear, reduced aggression
    │   │   ├─ Visual agnosia: Cannot recognize objects visually
    │   │   └─ Hypermetamorphosis: Compulsive attention to visual stimuli

    │   ├─ Hyperactive amygdala: Enhanced threat detection

        ├─ Ictal fear: Overwhelming fear during seizures (amygdala involvement)
        └─ Surgery: Can resect amygdala with hippocampus (anterior temporal lobectomy)


### LIMBIC SYSTEM (PAPEZ CIRCUIT) ⭐⭐:

├─ Components (classical Papez circuit):
│   ├─ Hippocampus → fornix → mammillary bodies
│   ├─ Mammillary bodies → mammillothalamic tract → anterior thalamus
│   ├─ Anterior thalamus → cingulate cortex
│   └─ Cingulate cortex → cingulum → parahippocampal gyrus → entorhinal cortex → hippocampus
│
├─ Extended limbic system (modern view):
│   ├─ Add: Amygdala, orbitofrontal cortex, insula, ventral striatum
│   └─ Function: Emotion, motivation, memory integration
│
├─ Function:
│   ├─ Emotion: Processing and regulation
│   ├─ Memory: Emotional enhancement of memory consolidation
│   └─ Motivation: Drive, reward processing
│
└─ Clinical:
    ├─ Wernicke-Korsakoff syndrome (mammillary body damage):
    │   ├─ Etiology: Thiamine deficiency (alcoholism, malnutrition)
    │   ├─ Acute (Wernicke encephalopathy): Confusion, ophthalmoplegia, ataxia
    │   ├─ Chronic (Korsakoff syndrome): Anterograde amnesia, confabulation
    │   └─ Imaging: Mammillary body atrophy, periventricular T2 hyperintensity
    │
    └─ Limbic encephalitis:
        ├─ Etiology: Autoimmune (anti-NMDA, anti-LGI1), paraneoplastic
        ├─ Clinical: Memory impairment, seizures, psychiatric symptoms
        └─ Imaging: T2 hyperintensity in medial temporal lobes (hippocampus, amygdala)
